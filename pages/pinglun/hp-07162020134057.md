 <p>
  外界期盼已久的世界卫生组织新冠病毒溯源调查组终于成行，由一名动物卫生专家和一名流行病学家组成的先遣小组已于7月10日抵达北京。
  <br/>
  <br/>
  今年2月，世卫组织曾经到中国进行有关新冠病毒的考察，但是他们的行程被中国政府严格控制，连武汉病毒所都没去过；另外，很多重要的信息也无从接触。此后，如世卫组织驻华代表高力（Gauden Galea）所说，世卫组织多次提出到中国调查，均被中国政府拒绝。这次中国政府总算同意世卫组织来调查了，但是中国方面是否愿意积极配合而不设置障碍，世卫调查组能调查出什么结果，仍不容乐观。
  <br/>
  <br/>
  自去年12月31日中国官媒首次承认武汉地区出现不明原因肺炎以来，六个多月过去了，可是有关新冠病毒的起源问题甚至比先前还更扑朔迷离，模糊不清。早先，官媒宣布新冠肺炎病例都与武汉的华南海鲜市场有关；其后有不少专家说，新冠病毒的中间宿主应是穿山甲；可是后来官媒又报道说，在华南海鲜市场采集的数百个环境样本中，有33个样本的病毒检测结果呈阳性，但这些样本取自何处，是从门把手、砧板，还是从下水道、垃圾车采集的？却从没公布过。到目前为止，报告的阳性检测结果没有一例是来自动物的。另外，官媒早就报道过，在早期确诊病例中，有三分之一的患者，包括第一例确诊患者，都从未去过海鲜市场。中国疾控中心主任高福明确说：起先我们相信病毒来源于海鲜市场，但现在看来，海鲜市场也是个受害者。早在海鲜市场发生感染以前就有病毒了。
  <br/>
  <br/>
  按理说，以中国政府打造的堪称世界一流的大数据监控系统，中国方面很容易查出那些和海鲜市场无关的患者在那段时间的行踪，顺藤摸瓜，找出他们行踪的交集点，这就可以发现他们是怎样被感染的，从而找出病毒的源头。可是直到今天为止，中国方面都没有在这方面给出任何线索。要说他们一点线索都没有，那是不可能的。因此，更大的可能性是中国方面已经掌握了很多很重要的信息，但是却秘而不宣。我以为，这次世卫组织来调查，首先就应该从调查那些和海鲜市场无关的患者的行踪着手。
  <br/>
  <br/>
  另一个调查重点就是，病毒是不是实验室泄漏。武汉病毒所的石正丽说，她曾经把新冠病人的病毒样本，拿来和实验室原来采集保存的上千个冠状病毒样本做过比较，发现都不匹配，证明新冠病毒不是从他们的实验室泄漏的。但是这只是石正丽的一面之词。正如《华尔街日报》编辑部文章《武汉实验室理论》一文所说，石正丽自己坚称，武汉出现的冠状病毒与任何样本都不匹配，但这需要进行外部调查才能确认。我以为，这应该是世卫组织调查的重点。只不过相隔半年，如果新冠病毒真的是来自实验室，恐怕也早被藏起来了或者被销毁了。当然，还可以查实验室的工作日志，看工作日志有无缺漏，有无篡改。如此说来，世卫组织要调查出真相，不但要有科学家的专业知识，还要有福尔摩斯的破案本事，否则是查不出什么名堂的。
  <br/>
  <br/>
  我对世卫组织在中国调查会有多大成果很不乐观，不过我以为这事也不是那么重要。因为不需要世卫组织的调查，就凭已有的并得到证实的信息，我们就足以断定：在疫情初期，中国政府不准公布疫情，拒绝启动应急机制，错过了把疫情扼杀于萌芽状态、防止疫情大规模扩散的时间窗口；又由于中国政府隐瞒真相，发布不实信息和虚假数据，严重地误导了各国政府和民众。
  <br/>
  <br/>
  新冠疫情是一场全球性的大灾难，而这场大灾难本来是完全可以避免的。对于这场大灾难，中国政府难辞其咎。我们应该、也必须向中国政府追究责任。
 </p>
