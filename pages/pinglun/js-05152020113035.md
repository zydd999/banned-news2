 <p>
  5月14日开始，习近平策动的大规模海陆空实弹军演在渤海湾登场，预计历时两个半月，至7月底收场。正当大瘟疫时期，中国经济遭受空前重创，此时此刻，习当局不是集中人力、物力、财力、精力于防疫抗疫、复工复产和重振经济，而是耗费巨大人力、物力、财力、精力于大军演，意欲何为？
  <br/>
  <br/>
  从演习地域和时间选择来看，习近平策动大军演一石四鸟或一箭四雕：威胁台湾，叫板美国，防备俄罗斯和北朝鲜，震慑党内政敌。
  <br/>
  <br/>
  威胁台湾。演习地渤海湾，据说类似某处台海地形，所谓“两栖登陆，夺取岛屿”，明显有针对台湾之意。而有中共人士放话：这波军演结束后，将紧接着在南海军演，目标是夺取由台湾控制的东沙群岛。另外，5月20日，当选连任的蔡英文将再度宣誓就任台湾总统，而习近平和中共一直逼蔡英文和民进党承认所谓“九二共识”、“一个中国”，遭后者坚拒，习近平曾放话“地动天摇”，却摇不起来。故而，至少，也要在蔡英文宣誓连任的520期间制造些恐吓，为习政权挽回些颜面。
  <br/>
  <br/>
  叫板美国。正值美国和世界各国纷纷就中共隐瞒和传播大瘟疫，向中共追责索赔，中共已经用战争叫嚣和核武威胁来回应国际社会。策动大军演，是向美国和世界亮肌肉，显示中共有能力制造战争，让世界不得安宁，如果各国坚持追责索赔的话。值得注意的是，中共御用学者发言：这次大军演，“可以演练保卫首都北京的重点科目，保卫北京主要是防空反导。担负保卫首都任务的部队可以在接近实战的条件下，检验和完善作战方案。”无意间泄露，这次军演与其说是进攻性的，不如说是防御性的，防范美军对这个红色首都圈的突然打击。军演选址渤海湾乃京津门户，意图之一就在这里：拱卫京师。
  <br/>
  <br/>
  防备俄罗斯和北朝鲜。中共隐瞒和传播大瘟疫，让俄罗斯和北朝鲜深受其害。俄罗斯确诊病例，已经超过中共自己公布的中国数字的数倍；而北朝鲜虽秘而不宣，却被习近平致金正恩的口信泄密：疫情严重。普京与金正恩的恼火可想而知。以中共习性，习近平极可能用“私了”的方式，以“援助”为名，对俄罗斯和北朝鲜变相赔款，避免这两国公开加入以美国为首的国际阵营。但普京与金正恩未必满足，更绝不可能与备受国际孤立的中共站在一起，甘当冤大头。故而，习近平的大军演选定靠近俄罗斯和北朝鲜的渤海湾，也有暗中防备俄、朝两国之意。
  <br/>
  <br/>
  震慑党内政敌。对习近平个人而言，他同时面临国内和国际的反习势力。相比之下，国内或党内的反习势力更是燃眉之急。这段时间，从公安部到司法部，从北京卫戍区到军工部门，泄密、叛变、政变的风声频传，习近平频频离京外出，充分显露他的焦虑感和不安全感。搞大军演，显示习近平军权在握，纵使在党内已经人见人厌，却暗示，他可以借用手中军权压制党内政敌。选择渤海湾，扼控北京、天津、唐山、北戴河一带，无论在北京还是北戴河，若党内有变，政争或政变，习近平恫吓：他可以动用军事手段，也就是政变手段，强行解决政治问题或权力之争。

 <p>
  习近平首先要面对的，就是即将于五月下旬举行的人大、政协两会。五千多代表或委员云集，会不会出现串联、起哄、哗变，让他下不了台？习近平心中没底。但两会举行期间，操纵大军在身边军演，制造心理震慑效应，或是习近平的盘算之一。大军演尾声则接近北戴河时间，甚或与北戴河会议重叠。这个每年一度由政治老人和现任中共高层齐集的会议，在最近几年，往往成为“批习大会”或“斗习大会”，让习近平坐卧不安。他需要就近布设军队向政敌示威，为自己壮胆。而大军演，就成为一个就近布设军队的借口。
  <br/>
  <br/>
  习近平策动大军演，同时瞄准国内外反习势力，一石四鸟或一箭四雕。其实，震慑和对付党内政敌，才是习近平大军演的主要目标。正如2014年历时4个月的所谓“三军四海”大军演，虽然也以台湾、日本、美国为假想敌，但其过程与结果却与那些外部目标没有多大关联，其主要目标就是党内政敌：即，当时习近平急于排除的江派，以（选择性）“反腐”为名。（如今的主要政敌，已经变为团派、政治老人的主流派，以及权力外围，太子党和红二代的主流派。）
  <br/>
  <br/>
  同理，这波渤海湾大军演对外的目标只是烟幕弹，比如针对台湾或谋攻东沙群岛不过是虚张声势。习近平和中南海心下很清楚，无论是进攻台湾本岛、离岛、还是台湾占据的南海诸岛，都会给美军制造介入战争的最佳口实。美国等待的正是这样的机会：让中共主动挑衅、主动开战，让它玩火自焚。
