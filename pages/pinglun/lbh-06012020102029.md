 <p>
  因为武汉肺炎而推迟的北京“两会”5月下旬在北京召开。一开始人大抛出为香港制定国安法震惊全球，特朗普总统明言会有强烈反应，末代港督彭定康也呼吁西方国家不应该再向中国低头，各种谴责声浪排山倒海而来。中国官媒与外交部虽然都以战狼姿态公开对干，然而大会显得虎头蛇尾，习近平也突然隐身而放软姿态，实在值得玩味。
  <br/>
  <br/>
  人大5月27日通过港版国安法，28日闭幕，29日在人民大会堂召开《反分裂国家法》实施15周年座谈会。显然，这是事先策划的连串戏码，显示吞并港、台的决心，然而对台湾的声势显然大不如前。首先并不是纪念会而只是座谈会，其次不是习近平领军，而由党内第三号人物、全国人大委员长栗战书出马。
  <br/>
  <br/>
  栗战书虽然名为“战书”，也强打精神对台湾下战书，然而他的战书软弱无力。真正的战书简短有力，震人心扉，可是栗战书却是又臭又长的4000字裹脚布，重弹老调，令人昏昏欲睡。他说，如果“台独”势力一意孤行甚至铤而走险，将按《反分裂法》采取必要手段，粉碎“台独”分裂图谋云云。

 <p>
  哈哈，粉碎台独中共领导人讲过千万遍。最妙的是说，“如果‘台独’势力一意孤行甚至铤而走险”，那表明蔡英文并未一意孤行，也没有铤而走险；那就表示，北京默认九二共识收档与中华民国台湾存在的事实了。习近平将使台湾“地动山摇”的恐吓不但没有实现，倒是两会期间北京门头沟发生地震吓到北京人。习近平抵不过天意，还是退到幕后可以保存一些脸面。
  <br/>
  <br/>
  栗战书是习近平的亲信，但是代习近平下战书却有气无力。而在5月28日人大结束的记者会上，总理李克强的讲话却真正让习近平地动山摇。他说，中国有6亿中低收入及以下人群，平均月收入在1000元人民币左右。习近平把今年实现脱贫的小康社会作为他的政绩，是“伟大民族复兴”的重要内容。如今李克强暴露真实数据，6亿人可不是小数字，几乎人口的一半，不但表明小康社会远远没有实现，也表明中国贫富差距的悬殊。习近平8年反腐败反了什么？只是为了小集团利益的党内互相倾轧而已。
  <br/>
  <br/>
  李克强在记者会上的讲话不用政治局审批，所以他拥有短暂的“言论自由”，借这机会狠狠捅了习近平的假面具，打了他一巴掌。他在全世界面前打了习近平，也向国人宣布习近平执政的失败，可见党内斗争激烈到什么程度。李克强会有任志强的下场吗？大家等着看习近平还有多大的实力。
  <br/>
  <br/>
  习近平在两会闭幕没有强势出现，又避开反分裂国家法的纪念活动，他是不是被“禁评”了？5月底习近平再没有亮相，只是隐身发表对海南省自由贸易区的指示与祝贺六一儿童节，这头“战狼”要假装羔羊了吗？
