 <p>
  各位听众朋友大家好：
  <br/>
  <br/>
  今天的热邮信箱，我想跟大家聊一个跟”六四“纪念活动有关的话题，那就是“我们为什么要zoom举办线上纪念活动”？
  <br/>
  <br/>
  今年由于疫情关系，美国等地的六四纪念活动，一般都改在线上举行。而线上活动的平台，基本上也都是用zoom的方式。对于使用zoom，有一些朋友表示不理解，他们认为，zoom被认为有安全顾虑，使用它有可能被中共监控。我承认，这些朋友的担心，不是完全没有根据的。
  <br/>
  <br/>
  但对于这个问题，我有不同看法：
  <br/>
  <br/>
  第一，众所周知，今天中共的网上监控能力，从技术上来讲，其实是无所不在，无所不能的。真正想完全躲避开监控，几乎是不可能的。在这种情况下，我认为我们要调整心态，如果我们总是把“担心监控”放在最高考量上，那其实，什么都不能做了。而这，难道不正是中共监控要达到的恐吓人民，并因此而让人民自律，自我约束的目的吗？
  <br/>
  <br/>
  第二，我们纪念六四，是光明正大的事情，我们并没有触犯法律，更没有从事犯罪活动。我们 为什么要那么惧怕中共的监控？很早就有人讲过，如果人民不再怕政府，政府就要害怕人民了。我要说的是，如果人民永远都这么怕政府，政府就永远都不会害怕人民。1989年的时候，学生走上街头，当然也是冒着风险的，如果当时我们害怕，就没有八九民运的发生了。我们是要注意安全，但是也不能让中共的阴影影响我们的全部生命和所有行为。我必须说，纪念六四这样的公开活动，如果中共要监控，就给他监控好了，我们就是要展示我们的决心和意志给他们看，不是吗？
  <br/>
  <br/>
  第三，至于国内朋友参加，会不会有危险的问题，我觉得很简单：我们并没有强迫国内的朋友参加。国内的朋友，可以自行评估参加这个活动的风险；自认为有风险的，其实可以不参加。我相信，很多国内的朋友经过评估之后，还是会参加的，而 zoom是目前国内朋友还可以使用的不多的工具，为了让国内的朋友能够一起参与，这也是最好的选择。在这样的自愿参加的原则下，使用zoom没有太大的风险上的问题；
  <br/>
  <br/>
  最后，我认为，如果涉及到一些隐私或者安全的事情，使用zoom当然有风险，我也不建议使用；但是，六四纪念活动是公开的，集体参与的，这里没有隐私，也不涉及具体个人，发言者都是海外的人。因此，我们不必那么自我设限。
  <br/>
  <br/>
  以上纯属个人看法，不知各位是否同意？
  <br/>
  <br/>
