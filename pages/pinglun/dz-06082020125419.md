 <p>
  罢黜独裁者， 绝非一件轻而易举的事情，可能还会地动山摇，否则那就不是极权独裁政治了。注意，这里指的独裁者， 不是民主体制中的强势领导人， 也不是宪政框架下的某些长期执政者， 而是赤裸裸的独夫民贼，肆无忌惮，毫无舆论监督和体制制衡， 凌驾于宪法和民意之上。
  <br/>
  <br/>
  现代民主国家已经学会通过文明，和平，理性的选举程序， 来改变更迭国家最高权力，国家公民对最高掌权者可以选上或选下。 尽管选举政治并非完美无缺， 但确是人类文明的一大空前进步， 即权力政治开始讲究宪法程序与和平竞争。
  <br/>
  <br/>
  而独裁专制是自然 “丛林法则” 的产物，象一般动物绝对不懂什么是选举和民主，而是看谁更凶狠残暴，威权独揽， 地盘子多，谁就是山中老大， 独占鳌头， 成王败寇。这可能乍看上去不可一世， 但实际上不过是野兽的生存状态和档次罢了。
  <br/>
  <br/>
  独裁者往往视权如命，时刻为之绞尽脑汁的事情， 就是如何扩充， 强化， 巩固和确保自己的权力， 并因此必然成为人民当家作主的凶恶敌人， 因为民主与独裁是水火不相容的两重天地。对于独裁者这样的 “权力综合征”重度病人，如果试图对其天真地进行劝诫或者劝退， 那基本是对牛弹琴，与兽谋皮， 弄不好反会惹来杀身之祸。
  <br/>
  <br/>
  既然独裁极权政治的性质极端野蛮落后， 那么对其的解决和革命， 也必须是采取非常的措施和手段， 方能见效， 否则国家民族和文化就苦海无边。这里我们不妨对四个“罢黜独裁者” 的典型案例加以回顾扫描，希望历史可以告诉现实和未来，并有助找到果断解决独夫独裁， 开启现代文明新局面的勇气， 智慧和希望。
  <br/>
  <br/>
  第一个是 “推翻齐奥塞斯库模式”。1989年底， 当时身为罗马尼亚最高权力独裁者的齐奥塞斯库，忽然在全国爆发的反抗和起义浪潮下失去权力， 并遭到逮捕和处决。罗马尼亚发生巨变的大背景， 是当时苏联和东欧所谓 “社会主义阵营” 整体步入风雨飘摇，朝不保夕的状态。
  <br/>
  <br/>
  苏联集团长期失道寡助到达临界点， 终于引发惊天巨变。 权倾一时的独裁者齐奥塞斯库，在历史洪流面前立刻丢盔卸甲， 抱头鼠窜，灰飞烟灭。于是在此个模式中， 历史大势和人民起义最为关键。一旦举国上下天翻地覆， 独裁者就在劫难逃了。还譬如利比亚的独裁者卡扎菲， 也是同样遭遇和下场。
  <br/>
  <br/>
  第二个是 “罢免赫鲁晓夫模式”。 尽管赫鲁晓夫本人并非严格意义上的独裁者， 甚至当年还很有些 ”改革“ 色彩， 但他所供职效力的苏共政权， 是名副其实的独裁专制。由于苏联高层的保守顽固派密谋， 1964年10月当时正在外地休假疗养的苏共第一书记赫鲁晓夫， 突然被苏共中央召回到莫斯科， 并随即宣布被解除一切职务， 失去最高权力。
  <br/>
  <br/>
  这无疑是一场精心策划的宫廷政变，而且是趁主要领导人外出或外访的机会加以实施，做到了出其不意， 攻其不备。 反对势力得到了克格勃即秘密警察机关的密切有利配合，使得 “倒赫“ 一举成功。后来在中国，为人们所熟知的 “粉碎四人帮“ 事件， 其实也是采取的相近模式和手段。不过中国当时还算幸运，是文革极左势力遭到了粉碎。
  <br/>
  <br/>
  第三个是 “斩首萨达姆模式“。 即伊拉克独裁者萨达姆， 2003年在以美国为首的多国联军发动的军事打击下，其政权土崩瓦解， 其本人东躲西藏， 最后还是被活捉生擒， 经特别审判后被处以绞刑。曾号称 ”中东强人“ 的萨达姆， 其倒台的关键因素显然是由于外力打击和国际征讨， 也由于其空前孤立， 毫无援手，被自己的人民完全唾弃憎恨。还有如埃及的独裁者穆巴拉克， 在2010年底爆发的 “阿拉伯之春” 民主运动中，也遭遇到类似结局。
  <br/>
  <br/>
  第四个是 “突袭朴正熙模式”。1979年10月， 在韩国已经掌权18年的独裁者朴正熙， 在出席一个晚宴时， 突然遭到其一名心腹亲信的枪击刺杀， 并当场身亡， 朝鲜半岛为之震惊。此个事件再次证明独裁者没有绝对安全，只有绝对腐败和背叛。独裁者本身就是一个阴谋，恐怖和残暴的巨大载体， 其贴身周边也必然变故无常，草木皆兵，反叛难料。 独裁者常会自己打败自己。
  <br/>
  <br/>
  用上述四大模式考察中国的情况: 中国普通民众 “揭竿而起”， 一般是因为经济而非政治原因。中国知识界虽具备强烈的政治敏锐性， 但若没有普通民众响应协助， 常是孤掌难鸣， 于是中国的宏观经济形势很值得关注；搞秘密突然的宫廷政变， 对此中国的统治阶层绝不外行，能否成功要看政治实力对比演化，以及组织策划的特殊能力和周密程度。高层政变往往促成中国政治转向。
  <br/>
  <br/>
  而中国的独裁者是否会被外力击败， 那主要看其武装力量的状态和立场。如果象当年八国联军打来， 百万清军都抵挡不住，慈禧太后只好认怂投降；或出现类似苏军在1991年 “8.19事件”中临阵倒戈的情况，独裁体制也就岌岌可危。至于亲信同党谋反， 搞突袭刺杀这一套，独裁体制也是防不胜防，当年林立果的 “571工程纪要” 就是如此图谋。而且独裁者都象朴正熙一样，谁都不能真正信任，不断为自己制造死敌。
  <br/>
  <br/>
  以上历史案例有力表明独裁者是真正的纸老虎， 虽貌似强大， 但在瞬间就可以变得不堪一击， 党羽走卒们也随即树倒猢狲散。更何况21世纪的今天， 推翻消灭独裁专制，乃是时代潮流，人心所向， 人人有责。在当下中国， 最能为祖国和人民建功立业的举措，无疑就是铲除倒行逆施的独夫民贼，重启改革开放，践行民主法制， 迎来中华民族和中国文化真正复兴的重大转机！
