 <p>
  以中国当前的政治制度，傀儡式的全国人大常委会对“港区国安法”的审议和通过不过是为中共最高层的旨意盖章而已。对此举可能引发和已经引发的种种后果，中共最高层不可能毫不在意，但如果说他们胸有成竹也可能高估，最恰当的心态还是：知道有后果，甚至很严重，但心中自有一杆秤，孰轻孰重权衡过了，也有一些底气，对手的本事见识过了，天塌不下来，相信政权扛得住。
  <br/>
  <br/>
  实际上，此举的影响完全可以说撼动了中共所谓的“国际、国内两个大局”。国际上，这意味着中共与西方价值观彻底进入正面对抗状态，对全球化时代从西方得到的好处在所不惜；在国内，这意味着习在对内层面对邓时代的基石“改革开放”从阳奉阴违到玩弄于股掌之后，于对外层面与邓路线的另一基石“一国两制”公开决裂，并对引发大中华区公众中占绝对多数的邓时代认同者和获益者的强烈不满在所不惜。
  <br/>
  <br/>
  尽管众所周知，香港民众的抗争被镇压下去容易，甚至香港基本自由被剥夺，从立法、教育、言论等所有关键领域的“两制”被彻底阉割，变成澳门那样的“宦城”。但香港金融和贸易地位一落千丈，对中国整体经济的冲击绝对为中共所低估，中共手中自以为的替代手段恐怕远没有那么厉害。但是，在全局上有这样的对抗、决裂和在所不惜，香港本身的价值对中共最高层来说迅速萎缩，以及香港本身可能的抗争、动荡，自然更加在所不惜。
  <br/>
  <br/>
  <b>
   中共不惜毁灭香港暗示对台也会更粗暴
  </b>
  <br/>
  <br/>
  与台湾根本不同的是，香港除了金融和贸易，在地缘战略上毫无利用价值，更无事实上的独立地位，1997年的移交也断绝了西方更多干预的可能。但与此同时，香港的教训和局势，也必然对台湾政坛和民众在统独问题上的态度产生强烈影响。按理说，这也是中共此举最大的潜在代价之一。但中共之所以在所不惜，显然也暗示着在更大的战略视野内，习很可能也对台湾抱有更粗暴和不计代价的企图。简言之，在香港放弃“一国两制”，在台湾也没打算“一国两制”，统统“一制”了之。
  <br/>
  <br/>
  然而同样有目共睹的是，美国参议院也罕见地以全票通过了相关法案，对北京和香港涉及此事的官员实施制裁，取消美国对香港在金融和贸易上的特殊待遇也是大势所趋。对此，特别是对后一项举措的后果，诸如中国引进外资和对外贸易将受到严重冲击，北京不可能不清楚，但显然认为完全可以承受。
  <br/>
  <br/>
  这背后的底气应该来自两方面。
  <br/>
  <br/>
  <b>
   中共以为已找到民主国家弱点不惜对抗到底
  </b>
 </p>
 <p>
  <b>
  </b>
  <br/>
  <div class="image-inline captioned" style="width:1900px;">

 <p>
  其一，经过中美贸易战的交锋，美国在和平状态下能对中国造成的最大损害，北京已经有数，无非就是中美贸易全盘中断，甚至人员往来大幅萎缩。这当然后果严重，但北京显然也早就发现，特朗普实际上做不到这一点。他从用高关税将中国出口卡死到中美经济完全脱钩的狠话都说过了，内心却始终只是希望借此逼中国答应条件，而对真正重回冷战完全没有决心和准备。相反，美国经济和社会近期的巨大波动更加使他言不由衷。
  <br/>
  <br/>
  中国官方内部也有分歧，维持开放局面的压力很大，因而一度反复采取屈服姿态，但最终充分利用了美国的弱点。即使达成了协议，也完全没打算真正兑现。何况枭雄配乱世，中国全面的倒退果然引发全球疫情，一举以更惨烈的损失和混乱将中美斗法引向更复杂的局面，并激发出自己更豁得出去的对抗决心。
  <br/>
  <br/>
  <b>
   中共外交手段采用黑社会斗法模式
  </b>
  <br/>
  <br/>
  这类似于中国近代以来流行的一种黑社会斗法模式，拿出刀来不往对方身上捅，而是先一刀扎在自己大腿上，鲜血直流但岿然不动，展示的是一种轻易玩命，更何惧一切司法后果的无底线流氓本色。对方哪怕在商场、官场都谈得上心狠手辣，只要无法同样地对自己狠，只能认输。
 </p>
 <p>
  更有利的是，在前日美国发表的纲领性文件《美国对中华人民共和国战略方式》中，美国一连三处为中共的永久执政权吃下定心丸，试图以此诱使中共达成交易；同时精心选择了“竞争”这个尽量少带对抗色彩的措辞。而在对美国重要国家利益的表述上，又根本不涉及中国对国际秩序的冲击。即使再不聪明，中共仍能看穿这些混乱的目标和三心二意的决心，因而不管损失多大，北京绝不承认自己有错，不承认美国的规则有理，还对国际秩序发起更多的挑战。
  <br/>
  <br/>
  <b>
   中国人的愚昧助长了中共的无赖
  </b>
  <br/>
  <br/>
  其二，中共的底气来自在对国内局势和自身能力的评价上已能指鹿为马的境界。中国国内经济失去出口这驾马车的拉动已非一日，6亿人月收入不足千元，近千万农民工事实上失业，874万应届大学毕业生就业率爆新低……，面对这些在西方政府绝对要倒台的局面，中国领袖面不改色心不跳。近日全国26省1374万人陷入水灾，经济损失数百亿，习近平也没有露过面，官方媒体甚至能成功地使多数公众无从知晓这些灾情。
  <br/>
  <br/>
  中共应该深知民不聊生后的政治风险，但真正支撑自信的其实根本不是近年在国际上吹嘘却效果不佳的“中国模式”，而是毛的模式。当年同样民不聊生，同样四面树敌，同样你死我活，却能“风生水起”，“扬眉吐气”，这对中国相当一部分不读书的人有毒品一般的吸引力。所需要的法宝，不外乎欺骗、愚民、暴力和内斗等等，如今已是样样不缺，蔚为壮观。
  <br/>
  <b>
   <br/>
   整顿军队目标只为对付人民
  </b>
  <br/>
  <br/>
  虽然客观来说，以当今中国和世界的政治、经济、社会特点，再现这种毛的模式很难说能够如愿，甚至完全可能玩火自焚。7月1日零时起，中国全部预备役部队的指挥权也突然从原来的军地双重领导变成收归中央军委。这样的部队打美军不行，打群众还是够用，然而仍然是哪怕让地方鹰犬有所染指也放心不下。因而，也不能说他没有焦虑。但至少从目前来看，内部和外部都没有足以令习近平真正胆寒的力量和行动。相反，美国衰落而动荡，西方软弱，在中国的攻势面前，全世界都不过说说而已；国内更是哪个还敢乱说，有权有钱，有人有枪，有粮有理有话说，这或是习近平的基本认知。
  <br/>
  <br/>
  因此，香港不稳定又怎样，有抗争又怎样，北爱尔兰又怎样，新疆尚且成功了呢。对财政困难，同样不懂经济的毛就是将国民经济玩到“崩溃的边缘”，美苏也没有入侵，国内外也没有任何人要求共产党下台啊。金正恩也是在类似的极端困难下，保持着“人民选择执政党和社会制度”的独立自主地位啊。正是在这种你可以称为无知无畏的决心之下，失去香港又怎样，失去外资又怎样？有大饥荒可怕吗，有核大战可怕吗？
