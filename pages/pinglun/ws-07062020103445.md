 <p>
  <br/>
  在觉康[1]，面向曾焚身的觉沃佛[2]，
  <br/>
  众声訇然响起。那四句人人会念的
  <br/>
  祷词[3]，犹如低回的合唱祈求实现。
  <br/>
  更长的祝祷文，由一位青年男子
  <br/>
  捧经诵读，也多次提及一个尊名。
  <br/>
  <br/>
  <br/>
  各种敬供源源不绝。根敦群培[4]说过，
  <br/>
  若真心敬供，就献出自己所喜乐的。
  <br/>
  （传言中，他将放弃自我的酒奉上）
  <br/>
  我在出门前，挑选了一条上乘的哈达，
  <br/>
  又在路上买了三支黄色的色金梅朵[5]。
  <br/>
  <br/>
  <br/>
  一群边地族人风尘仆仆而至，
  <br/>
  举着上金的铜碗，扛着大袋的青稞，
  <br/>
  还捧着一个浑圆、厚实、硕大的面饼，
  <br/>
  那是有名的安多郭勒[6]，应是众位贤惠妇女的
  <br/>
  默契合作，才会烘焙得如此完美。
  <br/>
  <br/>
  <br/>
  穿制服的保安却将供上去的面饼
  <br/>
  拿了过来：“你们带走吧，加持过了。”
  <br/>
  他用力地，把饼子掰成几大块，
  <br/>
  塞给了不知所措的众人。
  <br/>
  我悄悄地看着身旁的老妇人，
  <br/>
  <br/>
  <br/>
  她怀抱被退回的可能是她亲手做的饼子，
  <br/>
  凝视着取下所有华美装饰的赤裸佛陀，
  <br/>
  反复地诵念着“嘉瓦仁波切千诺”[7]。
  <br/>
  我好像闻到了当地面粉的香味，不再翕动嘴唇，
  <br/>
  而有了清晰地、大声地，说出尊者名号的勇气。
  <br/>
  <br/>
  <br/>
  2018年7月6日写于拉萨
  <br/>
  <br/>
  注释：
  <br/>
  [1]觉康：拉萨大昭寺供奉佛祖释迦牟尼等身像的佛殿，是最重要的神圣佛殿。
  <br/>
  [2]觉沃佛：即拉萨大昭寺主供佛像——佛祖释迦牟尼等身像，藏人又尊称“觉仁波切”，意为释迦牟尼至尊之宝。
  <br/>
  [3]即尊者达赖喇嘛长寿祈请文：གངས་རི་ར་བའི་བསྐོར་བའི་ཞིང་ཁམས་སུ། །ཕན་དང་བདེ་བ་མ་ལུས་འབྱུང་བའི་གནས། །སྤྱན་རས་གཟིགས་དབང་བསྟན་འཛིན་རྒྱ་མཚོ་ཡི། །ཞབས་པད་བསྐལ་བརྒྱའི་བར་དུ་བརྟན་གྱུར་ཅིག །中译：雪山绵延环绕的净土，一切利乐事业之缘源，丹增嘉措慈悲观世音，愿其足莲恒久驻百劫。
  <br/>
  [4]根敦群培：1903年－1951年，安多热贡人，是现代图伯特史上集佛门奇僧、史学家、艺术家、地理学家于一身的杰出人物，更是一位民族主义者，一生论著达三十余部，另有绘画等。
  <br/>
  [5]色金梅朵：黄色菊花。
  <br/>
  [6]安多郭勒：指安多地区做的一种面饼。
  <br/>
  [7]嘉瓦仁波切千诺：祈祷词，意即达赖喇嘛遍知，达赖喇嘛护佑。
