 <p>
  今年上半年武汉肺炎肆虐中国，打击中国经济；6月以后的洪水淹没了长江流域的产粮区，有的地方还出现蝗灾。所以即使依照普通常识，也会判断今年中国的粮食一定减产；而民间也出现抢粮、囤粮情况。
  <br/>
  <br/>
  然而中国官媒矢口否认。7月27日《人民日报》发表号称的“重磅文章”说：今年夏粮再获丰收，产量达到2,856亿斤，增产24.2亿斤，同比增长0.9%，创历史新高；秋粮面积稳中有增，可以争取有好收成。8月19日，国家统计局公布的数据显示，今年全国早稻总产量达到2729万吨，比去年增加102.8万吨，增长3.9%，扭转了连续7年跌势。
  <br/>
  <br/>
  然而中共总书记习近平8月11日却发布“餐饮浪费现象，触目惊心、令人痛心！”老百姓的三餐总书记也开金口，显然这同粮荒有关。等于打了人民日报的耳光。
  <br/>
  <br/>
  因此《人民日报》在12日就赶紧发表评论文章解释，这是习近平要中国人民发扬勤俭美德与艰苦奋斗传统。难道北戴河会议讨论的就是这个？鬼才相信。
  <br/>
  <br/>
  一个谎言要用千百个谎言来圆谎。为解释粮食紧张情况，8月12日，国家粮储局披露主产区夏粮收购量同比减少，因为市场活跃，农民惜售。既然惜售，就是买卖减少，市场又怎么会活跃？中共的国进民退，国家可以把大型民企吃掉，还不能抢购农民惜售的粮食？而8月17日新华社的报导却说，官方统计收购量减少是因粮食收储制度改革所致。8月29日新华社在解释中国在储藏、运输和加工环节，每年损失量达350亿公斤以上粮食。可见目前即使习近平亲自领导的大内宣大外宣也陷于各显神通的一片混乱。
  <br/>
  <br/>
  事实是什么？显然是中国出现粮荒。中国每年要进口1亿吨粮食，这是无法隐瞒的，由于今年的天灾人祸，中国积极进口美国粮食，这不是中共向美国施善意，而是不得已之举。因为华为等问题，中国暗示报复澳洲、加拿大的外贸，然而中国敢停止从他们那里进口粮食吗？1960年中国大饥荒，就是用紧缺的外汇从加拿大进口大量面粉来救急。现在中共否认粮荒，就是担心一旦美加澳停止向中国进口粮食，中国人真的愿意吃草吗？
  <br/>
  <br/>
  然而习近平突然要求节约粮食，不单是粮荒问题。对毛泽东的徒子徒孙来说，他非常记得文革期间毛泽东为了同苏联争霸所说的“备战备荒为人民”，“深挖洞、广积粮、缓称霸”，但他只记住“备战备荒”与“广积粮”，担心中美开战，中国因为缺乏粮食而内乱。何况现在中国外汇被习近平花的七七八八，没有多余买粮食，只能要民众勒紧肚皮了。
  <br/>
  <br/>
  与当年大饥荒与文革备战不同的是，现在马屁精多了许多，想出节约粮食的花样真是千奇百怪，令人叹为观止。独裁者在进化，马屁精进化的更快，5千年博大精深的中华文化开花结果。
