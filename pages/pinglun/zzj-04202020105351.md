  按理说这个话题不值得严肃地探讨，因为答案显然是中共最高领导层说了算。然而中国大陆现在的状况是，最高层既不真战，又热衷于煽动或放任一股磨刀霍霍的力量不断虚张声势。虽然实际上是为配合其国内、国外政治需要，但对不了解中国情况的人，这些声浪时常成功制造出极为紧张的气氛。
  <br/>
  <br/>
  当然，在哪里言战也大有讲究。朝鲜半岛不得妄议，“中日必有一战”也不合时宜，中美决战则说了自己都不信，南海也并非玩火之地。
  <br/>
  <br/>
  唯有台海。虽然美、台从外交到防务层面正在升温，外界普遍相信中共如果武力攻台，美国很难无动于衷。但千万不要忘了，我们谈论的主题是放眼周边一大圈四处点火形成的潜在热点，哪里是最适合中共制造战争舆论，服务于国内政治的最佳“抓手”，而不是中共真地要在哪里开战。
  <br/>
  <br/>
  从这个角度说，虽然台海也事关美国重大利益，但毕竟美国对台一无邦交，二无驻军，三无盟约，也没有南海那样的“航行自由”行动。对大陆民众来说，台海问题也纯属内政，其主权断无争论和悬念。
  <br/>
  <br/>
  不过，近日出了点新的状况。
  <br/>
  <br/>
  原来，这种战争呼声往往由自称“中华鹰犬”的戴旭发出，但此人最大的风险是沉迷于对美强硬，时常脱缰，且太接近“毛左”势力。因而近年，这戏越来越多地由中共军队原南京军区副司令员、退役中将王洪光担纲。这位原来在任上只分管武器装备的前副职，现在陡然成了雄才大略的鹰派主帅。其实他最狡猾之处，还是宁愿死撑“攻台只需100小时，美军不是来不及反应，就是慑于中共防范不敢动手”等豪言，也懂得回避与美国交手的话题。

  因此，虽然王的言论除了个人上位诉求外，不排除有高层授意提供舆论配合的背景。但他的退役身份还是使他更多以“非民间，但也非官方”的面目出现，博的是半虚半实之效。
  <br/>
  <br/>
  可是2020年4月13日至15日，当实名认证为“东部战区陆军权威发布”的微信公号“人民前线”连续三天发表三篇雄文——《面对战争，时刻准备着》、《战争来临，你敢赴死一战吗》和《丢掉幻想，准备打仗》时，加上“观察者网”等中共外宣的炒作，不仅国内舆论大哗，连已经习惯了王洪光式战争烟幕的外媒也有些沉不住气了。毕竟，“东部战区陆军”可是真正的现役攻台力量。
  <br/>
  <br/>
  这就需要对当前中国战争信号的传递加以解剖。对中共建政后仅有的几次对外开战，国际学界和新闻界已分析得滚瓜烂熟，总结出一套“舆论造势、国内动员和外交配合”的模式。然而中共内政外交近几十年沧海桑田，谁能肯定一定会沿用同一模式呢？
  <br/>
  <br/>
  最起码的，以前中共开战的最重要信号之一，《人民日报》或新华社社论中的“勿谓言之不预也”一语，近年也早被中国体制内各种按捺不住的鹰派喉舌们用滥了。就是《人日》或新华社两大御用喉舌，其评论文字近年也过早透支了各种火药味。以《环球时报》为首，各种好战嗜血的官方外围媒体上窜下跳之际，到底谁代表中共高层已完全混乱；就连中共高层内部也时兴借嘴放风，话里有话地打哑谜。
  <br/>
  <br/>
  好在这次战争信号的虚空还是明显的。《人民前线报》确为1948年创刊的原南京军区机关报，但2016年就已停刊，这次的三篇雄文不过是其写手揣摩上意的曲意逢迎。它们只是名为《强军有我，使命在肩》的长篇系列评论的一部分，而且文字“假、大、空”俱全，通篇只字未提台海，连“祖国统一”也没有。而“准备打仗”这类口号，自成为习近平的最高指示以来早就洛阳纸贵，全军上下趋之若鹜。若一见这类热词就预感战争爆发，只怕早就吓成精神障碍了。
  <br/>
  <br/>
  然而的确使外界惴惴不安的是，习一心在近年一举武统台湾的消息，既有金灿荣等新晋跳梁的极力渲染，又在理论上确有可能是内外交困之下铤而走险的选项。近年两岸关系中，虽一直未再现90年代两次台海危机时期的大规模军事威胁局面，但大陆一侧的扩军备战和挑衅姿态远较江时代凌厉。
  <br/>
  <br/>
  <b>
   鹰派言论只能讨好狂热民族分子
  </b>
  <br/>
  <br/>
  结合中共因中美贸易战和新冠疫情而表现出的对美敌意和攻击性，以某种战争手段为破解国内外困局的绝招，对习这样的个性和认知来说也并非不可能。大陆已然成患的狂热民族主义和军国主义情绪也对这样的选项莫名期待，从而使这一选项即使原来不大现实，现在也俨然具有了部分真正的民意基础。近日上海沪东船厂在建的中共海军075级两栖攻击舰起火，也会被附会成“为攻台赶工而不慎起火”。
  <br/>
  <br/>
  这也是为什么，外界对这次并不合格的战争假信号也要紧张一番的原因。不过我们如果结合中国近来在疫情中对美国的外交骂街，还是不难对中共趁疫开战的前景梳理出一个可笑的逻辑困境。
  <br/>
  <br/>
  美军因疫情部分瘫痪，如果还说疫情是美军放毒只能等同于说美军白痴，放毒都不会。但是，至少从官方消息看，中共军队岿然不动，还能趁人之危摩拳擦掌准备攻台，那中国军队放毒的可能是不是更大了？它显然对全世界都未躲过的毁灭性疫情早有准备啊！
  <br/>
  <br/>
  如果中共军队承认自己也起码部分地瘫痪，它自己放毒的可能也可大大降低，否则那无异于骂自己的主子是白痴了。可是，如果中共军队自己也在疫情中瘫痪，趁疫攻台的雄才大略是不是也要算自欺欺人呢？
  <br/>
  <br/>
  或许中共可以反驳：我们历来光明正大，绝不会放毒，而且这是一把双刃剑，我们哪有那么傻。那为什么比中国强大得多的美军就只能靠暗箭伤人呢？
  <br/>
  <br/>
  美军都能被病毒瘫痪，台军只会更差，若中共以无毒之师攻入，对方必无招架之功。但是，想必共军也只能全身防护，终日不敢解袍，否则既染病又亡国（省）的台人，若鱼死网破以放毒为业，岂不也能扳回局面？
  <br/>
  <br/>
  届时若共军真的百毒不侵，或者还能对称臣者赏以安全，对抵抗者任其死于病毒，可谓奉天承运，但这样一来，共军造毒的嫌疑是不是又更大了呢？
  <br/>
  <br/>
  可见，中共趁疫开战，还不必引来美军干预，就是在当前全球最为瞩目的疫情起源和责任调查上也只能偷鸡不着蚀把米。
  <br/>
  <br/>
  显然，疫情已是一面照妖镜，不管是虚张声势还是蠢蠢欲动，个中漏洞都已被放大到有目共睹，这也算是人类不幸中的万幸之一。
