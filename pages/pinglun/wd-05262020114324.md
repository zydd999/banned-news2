 <p>
  各位听众你们好，我是王丹。北京召开“人大”会议，准备审议通过在香港实施《国安法》，引起国际舆论强烈批评。但是最关心的，还是香港人。对于港人来说，他们当初奋起反抗送中条例，迫使該恶法“寿终正寝”；但结果是换来了一个更加恶劣的法律。他们心中的愤怒，悲哀，茫然和恐惧，都是可以想象的。但是不管怎样，最终还是要面对现实。面对北京当局不顾一切也要推行的《国安法》，港人能做什么呢？
  <br/>
  <br/>
  首先我认为，双方对阵，无外乎“战斗，和谈，弃守”三种选择，现在，“和”这个选项显然是不可能的了。港人诉求的主要内容并不是“港独“，而是”双普选“，但”双普选“又是北京绝对不会答应的底线。国安法的制定，表明双方在这个问题上已经没有任何可以”和谈“的空间；换句话说，香港人已经被北京的强硬立场逼到了墙角，只剩下了两条选择：要么迎战，要么放弃。
  <br/>
  <br/>
  对于香港来说，现在面对的压力是空前的。国安法一旦通过，“跨境执法”将成为现实，每个街头抗争者都有可能会被捕，判刑，甚至送到中国审判。中国对香港有绝对的管治权力，一旦动用国家机器，港人的安全受到严重的威胁。在这种情况下，移民英国，美国或台湾等地避祸，留得青山在，等待时机变化，不失为一个理智的选择。当初九七大限来临之前，就曾经有过大规模的移民潮，后来很多移民看到香港局势并不是那么恶劣，又纷纷返回，就是历史先例。当然，不是每个人都有移民的条件，剩下的港人放弃街头抗争，保存实力，避免更大的牺牲，也是可以理解的，毕竟生命是最宝贵的。这就是“放弃”的选项。现在已经有港人着手进行这样的安排了，港星黄秋生宣布争取入籍台湾，就是一个例子。
  <br/>
  <br/>
  另一个选择就是“迎战”，也就是说，不管面对多么大的压力，港人团结一致，继续进行激烈的反抗，通过街头抗争，示威游行，争取国际支持等方式，抵制国安恶法，甚至做好坐牢的准备，不惜一切代价去表达港人争取民主自由的决心。这就是“迎战”。目前来看，香港抗争运动的主流，是已经准备好采取“迎战”这项选择的。正如年轻民主派代表人物罗冠聪所说的：“要战，便来战”。当然，这样的选择，是一定会付出相当的代价的，但港人显然已经对此有了共识，做好了付出代价的准备。以目前形势看，只要北京不是派出戒严部队在香港街头开枪，即使有国安法的威胁，恐怕也无法达到恐吓香港抗争运动的目的。对抗的结果，谁输谁赢，目前还难预料。
  <br/>
  <br/>
  如果香港人最后团结一致，采取不屈服，不投降的立场，以更大规模的抗争回应北京推出的国安法，这样的局势发展，也会令北京当局，尤其是习近平，十分难堪。习近平是希望借助在香港问题上的强硬立场，既转移国内经济形势恶化的压力，又凸显自己可以控制局势的能力，可谓“一石二鸟”。但如果他的强硬路线不仅没有恢复香港的平稳局势，反倒造成更大的动荡，他的统治权威就会受到更大的损害。因此有人说习近平在进行一场政治豪赌。当一个领导人变成了一个赌徒的时候，我们就知道，他一定有巨大的危机感才会不惜一赌。因此，这次国安法是否能够在香港产生效果，将是对习近平政治前途的一大考验。以上是自由亚洲电台特约评论员王丹为您做的评论。
