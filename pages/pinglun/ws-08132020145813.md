 <p>
  我需要一件与科技无关的隐身衣，
  <br/>
  就像传记中那枚用作法器的白海螺，
  <br/>
  在某个奥妙心咒的反复持诵后，
  <br/>
  身躯缩小，缩小，就能藏身其中，
  <br/>
  为的是不让邪魔干扰修行。
  <br/>
  <br/>
  那些如影随形的陌生人，
  <br/>
  有着侵略性很强的本事，那是野蛮的
  <br/>
  权力赋予的，在光天化日之下，
  <br/>
  投下蟒蛇般悄悄盘曲、随时攻击的阴影。
  <br/>
  蟒蛇？此地有过如此阴毒的动物吗？
  <br/>
  <br/>
  但有小青蛇，多年前，我在德仲温泉[1]的
  <br/>
  石块上见过，比竹子更细，比风更灵活，
  <br/>
  似乎没有杀伤力，与斯巴括洛[2] ……对！
  <br/>
  就是六道轮回图，与其中的蛇不同，
  <br/>
  那是嗔恚的象征，源于无明。
  <br/>
  <br/>
  偶尔我会觉得，我像困在城堡中的隐士，
  <br/>
  但不是可怜虫或丑八怪，无非持守着
  <br/>
  边缘人的立场，说出了异端分子的话语。
  <br/>
  我不会卑躬屈膝，讨要饶恕或恩惠，
  <br/>
  反而中意这貌似避难所的小小穴居，
  <br/>
  恰是属于我个人的香巴拉[3]。
  <br/>
  <br/>
  假如屈从于……屈从于不计其数的填表，
  <br/>
  有关“身份”、“民族”、“政治面貌”等等；
  <br/>
  屈从于无权看到却掌控一生的档案，
  <br/>
  里面塞满无所不用其极的恶；
  <br/>
  屈从于扔给你的碗，铁的瓷的塑料的，
  <br/>
  战战兢兢地，生怕摔碎或变形；
  <br/>
  屈从于画地为牢的圈，越长越像待宰的
  <br/>
  猪或羊或其它牲口，屠夫磨刀霍霍；
  <br/>
  屈从于到处闪烁着命名为幸福的光芒，
  <br/>
  却经不起真相的一瞥。
  <br/>
  <br/>
  妥协一次，即屈从一次，受辱一次。
  <br/>
  妥协十次，即屈从十次，受辱十次。
  <br/>
  每次经过一个个屈从之处，一个个屈辱
  <br/>
  就像他们喜欢说的铁拳，会反复地击中心脏和泪腺。
  <br/>
  似乎付出了折寿的代价，毕竟耗损了灵气和灵感，
  <br/>
  我宁可屈从于美与自由的幻象，
  <br/>
  那才是我梦寐以求的，无比漂亮的隐身衣。
  <br/>
  <br/>
  <br/>
  2018年8月9日，写于拉萨
  <br/>
  （选自于尚未出版的诗集《拉萨烈日下》）
  <br/>
  <br/>
  注释：
  <br/>
  <br/>
  [1]德仲温泉：位于今拉萨墨竹工卡县门巴乡德仲村，被认为是圣泉，附近有尼众寺院。
  <br/>
  [2]斯巴括洛：藏语，六道轮回图，藏传佛教关于六道生死轮回的描绘。
  <br/>
  [3]香巴拉：梵语音译，又称香格里拉，藏传佛教所说的净土，时轮佛法的发源地。
  <br/>
  <br/>
