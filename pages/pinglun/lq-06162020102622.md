  世卫五月十八日视频大会的一项内容，就是世界调查武汉病毒起源的建议，已经获得一百二十二国的支持，还差十几票就达到三分之二，即达到世界卫生组织必须调查的规定。而且再增加十几票达到必查规定，在全世界如此巨大的人命健康代价，和几乎全球停摆的惨重经济损害下，也是完全有可能实现的。尽管全球调查武汉病毒已经呈现强势意愿，但是真正查明武汉病毒的真相，怕依然是关山重重难以顺利开展，甚至完全可能没有真相，或数十年也难以看到真相。
  <br/>
  <br/>
  这是因为中共已经用实际行动，反对国际社会对病毒起源进行科学严格彻查。而且中共多次销毁武汉病毒原始证据，至今坚拒世界各国和世卫协助调查的合理要求，从开始承认病毒源自武汉但很快甩锅他国，为甩锅情急到完全不顾事实矛盾和情理悖论。国际上只要有一点谈及疫情可以利用甩锅的，中共便一定不顾情况涉及的前因后果，甩锅不论美国意大利或欧洲其他国家，至少也要搅浑水让武汉病毒起源更为混乱难辨。
  <br/>
  <br/>
  所以，即使国际社会支持调查者达到规定，中共也一定耍尽无赖伎俩让调查名存实亡。这从中共在国际强大的追责舆论下，不得不软化拒绝的强蛮立场，但软化言谈中却充满诡诈就不难看出。中共在世界卫生大会上首次软化了拒绝调查的蛮横，说“全球疫情得到有效控制之后”，可以接受一个由世界卫生组织领导，具有科学与专业而且不偏不倚的国际评估。中共外长王毅近来又在前提中增加设定，即病毒溯源要“坚持专业性、公正性和建设性”。
  <br/>
  <br/>
  中共所设定的这些前提条件，可以让调查难以进行、产生理解歧义和扯不清的烦难。首先，国际社会强烈呼吁要求的，是对武汉病毒的源头经行调查，而中共允许的是不偏不倚的国际评估。何为国际评估？且不谈肯定扯不清的不偏不倚，这评估与调查显然两码事。评估是对提交的事物进行评价估量，而调查则首先是理清事实真相。很显然中共在国际压力下给出的评估，是不能查找病毒源头，只能对给出的情报予以估量。
  <br/>
  <br/>
  果然在世卫组织屡屡从中共手中拿钱，尤其是习近平承诺将给予美国断供的世卫组织二十亿美元后，世卫通过了谭德赛引导的所谓独立调查。世卫已沦为中共牵线的傀儡，世卫所谓的独立调查，完全改变了世界要求调查的内容。世界气愤和强烈要求调查的是：中共是怎样刻意隐瞒了早期疫情，让世界丧失了最初防止蔓延的宝贵时机；同时世卫组织在谭德赛的控制下，遵从中共要求发出不传染、可防可控、反对旅游禁止等信息。因此要对中共的隐瞒和世卫的不当处理进行调查。但是谭德赛却将这一调查内容，改为调查所有疫情发生国的数据和应对措施。
  <br/>
  <br/>
  调查武汉病毒所遭遇的这些阻碍刁难，令人不由得想起卡廷惨案的调查。虽然两案时间相隔七十多年，但是两案的调查遇到的阻碍刁难，其手法犹如穿越时空的孪生再现。首先，苏联将逃避纳粹的数万波兰官兵全数屠杀，与武汉刻意隐瞒疫情，造成全球巨大死亡和健康危害及经济灾难，这两起案件的邪恶本质，在引起世界的恐怖关注上，性质高度一致而程度上，后者还远胜前者。

  其实任何犯下的罪错恶行难以做到水过无痕，不论是卡廷屠杀还是武汉爆发的病毒，全都留下了无数直接间接的完整证据链。例如数万波兰精锐官兵投靠苏共，这种重大军事决策必然有决策过程，有大量相关军队和政府官员参与及往还文件，不是将数万官兵尽数屠杀就可以完全灭迹的。同样，武汉病毒在全世界和大陆人的关注下，也不可能将相关人员和文件指示遮掩无痕。不过苏联斯大林和中共习近平全不亏共党铁杆信徒，不论如何铁证如山回应只有不认，抱定了自家这么大一坨料你无可奈何。

  <br/>
  制作或蓄意将制作虚假调查报告，统帅帮凶摇旗呐喊，利诱胁迫国际认可，是中共与苏共在卡廷惨案与武汉病毒上， 采取了又一不谋而合的相同卑劣伎俩。卡廷屠杀和武汉病毒祸害全球后，国际上全产生巨大强烈的调查压力。而中共与当年的苏共如出一辙，在耍尽手段仍不能逃避调查压力后， 便将重心放在如何控制和制作调查报告。苏联利用战胜国和联合国常任理事国身份，自己调查和意图迫使国际接受其嫁祸德国的所谓调查报告。中共在国际压力下虽同意调查，但调查内容已面目全非。此外，习近平为了打破国际追责，争取非洲欠债穷国不随同追责，最近同意七十七国欠款暂时不用赔还了。
