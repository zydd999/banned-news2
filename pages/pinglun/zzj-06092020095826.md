 <p>
  为“震慑”520台湾总统就职，北京派战力平庸的“辽宁”号从4月30日起在台湾周边"机动训练"近一个月，在渤海号称演练要“输送一个集团军”的能力，都其实毫无新意。国台办和军方的例行发言将“有决心、有信心、有能力、有办法”改为“有坚定的意志，充分的信心，足够的能力”，“决心”和“办法”两个关键不见了。最被看重的中国军方联合参谋部参谋长李作成在两会的发言也老生常谈。
  <br/>
  <br/>
  唯一令外界侧目的，居然是《舰船知识》杂志发布的“24小时解放台湾”视频。台媒立刻注意到该刊的“大陆官方背景”，称其为“兵推片“。实际上，这本以海军技术科普起家的杂志十年前就开始靠各种空洞的战略话题挽救发行量滑坡，其编创人员基本就是网络写手出身。所谓”兵棋推演”，用的不过是电玩圈内熟知的战略型模拟游戏，其战法也还是弹道导弹瘫痪机场和防空，随后大陆空军和两艘航母取得制空和制海权，剩下的就是上岸卸货了。
  <br/>
  <br/>
  这种纯理论、纯技术，料敌极严、料己极宽的推演在军事上毫无意义。国防大学等当真吃这碗饭的机构所做的兵推或战术技术模拟仿真有的是大相径庭的结论，只是从不公布而已。即使只从1995年算起，攻台也谋划和准备了25年，期间大陆某些装备进展确有拨动两岸军力平衡之势，但美国充分利用台湾问题牵制和对抗北京的动作有增无减，在北京官方表态中，“挫败任何形式的外部势力干涉”居然放在“分裂图谋”之前，内心焦虑一目了然。
  <br/>
  <br/>
  完全可以判断，客观上北京武统的难度是大大增加了。然而大陆形势最为吊诡的一点也在于此，主观上北京武统的调门和冲动也同步增加。王洪光、金灿荣等人的轻率预言风靡一时，四川美院军迷社的占领台北意淫图令大批青年心驰神往。
  <br/>
  <br/>
  诚然，这些嘴炮和小粉连皮肉之苦也没见识过，他们怂恿去尸山血海的也绝不是自己或家人，但越是外强中干越是鸡血好斗。如果你认为这种情况下大陆绝对只是虚张声势，根本不敢动手，显然是对军事之外的历史规律缺少了解，义和团战争正被越来越多的人用来借古讽今。如果你不相信，无非是不信中国当朝从宫廷到草根昏聩如昔，那么不妨到中国的官方文件、互联网和街头巷尾体察一番，或可得到启示。
  <br/>
  <br/>
  虽然以现代战争之迅猛威力，以中国社会与西方水乳交融的现实，的确不可能原样重现庚子之乱，但还有一个极为流行的清醒论断——以中美军事差距之巨，北京断然不敢在美国干预风险很大的情况下贸然攻台，这就又对“厉害国”国情有所不察了。
  <br/>
  <br/>
  当今中国，对历史经验教训、现实国际态势、自身军事经济实力和战略目标的无知、轻率、鸡血、虚妄之声何以成为主流，当然离不开高层的偏好和依赖。互动之下，决策荒腔走板，清醒理智之策已走投无路，已成定局。以慈禧式思维，挟洋务维新之器物成就之威，以疯狂倒车后越来越窄的选项余地和权力游戏断无退路的局面，还有什么不敢的。
  <br/>
  <br/>
  <b>
   外界或低估了中共对台的谋略
  </b>
  <br/>
  <br/>
  更关键的问题在于，外界在此往往过于低估了中共的基本智商和经验。中共若要战，怎么可能是这些弱智兵推所示的轻易砸锅卖铁、鱼死网破？最近北京又破天荒地纪念起金门炮战来，那么当年毛圣人以金门为绞索和美台同盟试金石的案例不就在眼前吗。类似的，正恩胞弟在美国绝对军事优势的碾压之下，又是何以长袖善舞，“以弱胜强”的？
  <br/>
  <br/>
  因此，当今外界几乎所有对台海可能战事的研判，都深陷从万弹齐发到万船竞渡的定势，却忘了中共手中其实也有大量花样可玩。
  <br/>
  <br/>
  其中最有新意的可谓5月12日日本共同社独家援引“消息人士”称，中国军队正计划8月在南海举行设想夺取东沙群岛的大规模登陆演习。
  <br/>
  <br/>
  众所周知，不管东海、台海、南海如何各陷僵局，中共毫不在意被多点轮番牵制，战线过长，四面树敌，近十几年将这些热点统统列为“核心利益”，处处“一寸也不能少”，已然是好斗的亢进外交，而实际上，加上朝鲜半岛危机在内，中国海洋方向多处热点背后均在美国影响之下，难以破局。
  <br/>
  <br/>
  尽管如此，习当政以来，虽然一度热闹的“冲破岛链”和建设“海洋强国”的调门虎头蛇尾，但贴近朝鲜、钓鱼岛抢入领海线巡航、南海造岛陈兵、舰机绕慑台湾，较前任果然“血性”许多。引来的美国压力也不小，局面可谓继续恶化。
  <br/>
  <br/>
  南海困局中，东沙主岛太平岛由台湾占领，而南海主权的重要依据正是中国国民政府从日占状态下收复并草草划下野心过大的“十一段线”（中共为照顾越南改为“九段线”）。无论台北当政者对中国如何定义，若要否定和放弃南海主权，东沙也就无理可据。这一内战后遗存的领土现状，也是中华民国尚在的标志之一。对此，中共本是作为两岸同属一中的重要共同利益，以及中国在南海仅有的两处稳固基点（另一处是中共稳据的西沙）看待。也就是说，至少单论南海争端，共军和台军绝对是“友军”而非“敌军”。
  <br/>
  <br/>
  然而，一旦中共要打东沙的主意，其最大影响就根本不在南海，而是意味着本已名存实亡的“一中各表”被代之以“一中代表“。南海的这两方动起手来，对南海其他声索方和外部利益相关方来说，实在是一出大跌眼镜的戏码。
  <br/>
  <br/>
  共同社认为：东沙横亘于海南岛与从巴士海峡前出太平洋的航线之间，对中国具有很大战略意义。其实不然，驻扎东沙的台强力部门根本不具有威胁中共舰队，阻止其前出巴士海峡的能力。即使临时部署远程反舰导弹，也难有生存力。即便对于武统攻台，中共也一点也不缺少针对台本岛的出发阵地。对武力解决南海争端的野心来说，中共已在南海南部增加了三个军事据点，而且东沙本身并无争端，距其他声索国距离都不近，完全谈不上有军事上夺取东沙的迫切需要。
  <br/>
  <br/>

   中共的虚张声势也是武统策略

 <p>
  哪怕中共对东沙的野心只是理论上存在，其战略含义也更倾向于对台湾事实上的独立状态得到进一步强化有所准备。这也可以联系到金门和马祖两地的取舍问题。从纯军事角度判断，任何时候，台湾不惜一切代价固守金、马，都可能是事倍功半，得不偿失。在今天大陆武统之势咄咄逼人的情况下，各界更是对金、马的前景充满悬念。
  <br/>
  <br/>
  面对武统前景，不要说本来就主张台独的民进党完全可能放弃金、马或东沙等民国内战遗存的外岛，就是国民党内的维持现状意图也将变得无枝可依，因为中共对维持现状已没有兴趣。面对武统，台湾要认同统一，就只能是是献上领土和民主的“被统一”，而断无诸多国民党人长期幻想的“民主统一”。
  <br/>
  <br/>
  当然，中共现在毕竟只有演习的风声，并不真正夺取东沙。可是，以此类方式施压，甚至在某个时机真地强行夺取，虽然或有引发两岸甚至中美冲突的风险，但也不排除一种可能：在仅以此为限，暂不再更多侵犯台湾利益的前提下，或可逼迫本来就不太关心南海主权的蔡政府主动放弃东沙，从而使中共至少在南海，也在领土主权和国家统一的耀眼舞台上有所斩获，收获掌声和欢呼。毕竟在两岸之间，台方更加无意也无力有力反击大陆。
  <br/>
  <br/>
  总之，武统绝不等于“留岛不留人“，以外围为绞索，步步收紧，试探反应，争取各种中间状态，见机行事，哪怕虚张声势耍花枪，都是更有可能的局面。共同社这一消息即使不实，也对观察台海局势是一个重要启发。前日金灿荣再放“应放开防火墙，我们完全可以逼得他们建墙”的清奇之策，虽是针对互联网，但足见当前各种投机选项对最高统治者主动出击的强烈诱惑。
