 <p>
  习近平和中共的如意算盘， 是保持香港作为一只下金蛋的鸡， 但必须是一只听话顺从的鸡。但很可惜， 这样的一厢情愿不合逻辑， 即香港要么是金鸡独立， 要么就鸡飞蛋打， 并没有中间的选择。注意， 这里的独立更多指的是香港的制度自决和人格独立。
  <br/>
  <br/>
  在这方面中共并没有充分接受历史教训。譬如1949年后的上海， 在中共治下政治运动， 整肃和迫害不断， 加上施行僵化的计划经济，号称 “东方巴黎” 的上海很快就风光不再， 并被香港所超越。甚至于改革开放后的深圳， 其实最多也就是个毗邻香港沾光的新暴发户， 缺乏现代文明的人格和灵魂。习近平还不时地鼓噪深圳或海南取代香港， 完全是天方夜谭。
  <br/>
  <br/>
  然而习近平目前气急败坏， 不惜鸡飞蛋打也要压制香港的民主， 自由和法治。而一旦香港堕落成一个中国内地那样的普通城市， 也就没有了太多存在的意义。 一如北边的所谓 “雄安新区”， 习近平在南边最起劲忽悠的是 “大湾区”， 但很可惜折腾到现在， 却并无多少港人尤其是年轻人对 “大湾区” 问津。
  <br/>
  <br/>
  习近平这样的愚顽， 恐怕一辈子也弄不懂自由， 民主和法治的崇高价值， 这三项关键指标缺位的地方， 再好也就不过是用金银垒起一个猪圈而已， 里面的人过得再奢华， 也不过是爱因斯坦所最为鄙视的 “猪栏式的生活”。中国现在竟然让习近平这样脑子完全活在可悲过去和专制迷梦中的落伍掌权， 堪称中华民族和文明的噩运不断。
  <br/>
  <br/>
  于是， 香港现实发生的这场博弈宏观和战略意义非凡。 香港虽小， 却牵一发而动天下。这就与冷战期间为什么决战前沿选择了柏林， 二战前夕选择了保卫马德里对抗纳粹， 二战后期美国和日本大战硫磺岛一样， 一个城市和小岛忽然凝缩了历史， 现实和未来， 聚集了正义， 良知和理性， 邪恶，卑鄙和愚昧， 并将一决高下和胜负。习近平试图将香港 “送中”，而香港人民注定为习近平 “送终”。
  <br/>
  <br/>
  对中国和中国人而言， 是眼睁睁目睹改革开放中途夭折， 功亏一篑， 还是挺身反击， 尽快剪除习近平及其中共顽固派妖孽， 就成了我们这个大时代所面临的最关键博弈。在这场博弈中香港人民和志士英才当仁不让， 一马当先，真心坚持，等于是率先敲响了习近平邪恶集团走向溃败的丧钟。
  <br/>
  <br/>
  对世界而言，香港目前的局面， 完全与什么主权完整，国家安全无关，只不过是中共的民族主义蛊惑宣传和欲盖弥彰而已。香港问题是“冷战”结束后， 极权专制僵尸政治文化的一次死灰复燃， 并公然向21世纪进步文明社会发起蚕食挑战。 对此， 现代世界别无选择， 必须团结起来，火力全开， 对腐朽野蛮的旧世界展开坚决反击，为邪恶送终， 为光明铺路， 为希望奠基。
