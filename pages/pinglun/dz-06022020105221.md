 <p>
  这世界上可能有许多问题， 但目前都不如习近平和他所做的事情荒诞入邪。为此，习氏只能是个大失败大悲剧，这几乎没有悬念， 无论其如何强撑门面， 自吹自擂也难逃覆辙。习近平对于中国和世界， 从看法到行为， 都全面过时， 错误和荒谬，其上台近八年来堪称昏招迭出败绩累累， 把中国重新置于险地，拖入深渊，其历史定位只能是21世纪中国和世界发展进步的一大绊脚石。
  <br/>
  <br/>
  从根源来讲，习近平分明就是 “三大历史溃败” 的沆瀣合污与回光返照：满清王朝，文革浩劫， 和苏联帝国。习的政治路线和对内政策， 充斥了文革式专制愚民，和苏联式僵化封闭； 习的对外政策与操作， 是一个满清王朝愚昧鲁莽， 及文革歇斯底里的混合怪胎； 习的价值观世界观， 严重缺乏科学理性和现代文明熏陶， 而是苏俄小农式陈腐乌托邦虚幻， 加上极左偏执变态长期扭曲塑造出来的产物。习有意无意地模仿复制这三大历史溃败， 其结局不难预见。
  <br/>
  <br/>
  习的青少年时代， 是一个以愚昧无耻为荣， 荒诞野蛮当道，把谬误吹捧成真理，把罪恶彰显成功绩的黑暗隧道，在这个隧道里被闷堵毒害时间过久， 人会发生 “集体无意识” 的异化和变态,  整整一代人为此而心残脑瘫。习近平本人， 其实也是受害者之一， 导致其早期人格和初心的重大先天缺陷。如今，当年的受害者又反过来为害人民和国家， 确是中国和中华民族的巨大厄运与悲哀。
  <br/>
  <br/>
  李光耀曾说习好像是个曼德拉（南非传奇人权和国务领袖）式的人物， 但这种 “海外观察” 有着根本的盲区和误差。曼德拉在从事政治活动之前， 已经完成了现代文明教育和正义正直人格价值观的塑造。而之后虽成为长达26年的政治囚徒，他的信念和意志坚强如钢； 习则正相反， 青少年时代不幸沦为愚昧野蛮和荒谬的长期囚徒， 待到成年后虽有遇到现代文明的环境和感召， 但已经是为时已晚， 心智难开，无可救药了。
  <br/>
  <br/>
  这令习近平根本不具备引领21世纪中国发展的基本素质，胸襟和智慧， 而却又由于制度， 文化和历史的缺陷和误会， 让他得以占据了目前中国的最高权位， 这样的后果只能是灾难和崩溃。习的所作所为， 其实才是真正与古今中外“敌对势力”里应外合，断送掉中国发展壮大的希望和前途。
  <br/>
  <br/>
  譬如习上台后，便把过去30多年中国改革开放的巨大成就， 归功于党的全面领导， 凸显其寡廉鲜耻，好大喜功， 缺乏基本的历史判断和诚实品格。 历史的真实表明，改革开放是中共在1970年代末为挽救自身和国家的生存危机，不得不做出的自我救赎。 改革开放的成功规律， 就是当中共能够谦卑自省， 向人民表现出合作和敬畏时， 中国的改革大业就阔步前进， 否则就遭遇障碍和挫折。
  <br/>
  <br/>
  习氏及其溜须拍马所鼓吹的国内外两大政绩： 内部反腐， 对外强硬， 其实都是徒有虚名， 败絮其中。 关于内部反腐， 习对腐败核心，二代直系是绝对不敢动的， 其拿下的要么是平民出身的官员， 要么是腐败核心的“白手套”而已； 再有就是愈反愈腐， 前“腐”后继， 譬如近来公开宣布的陕西省委第一书记赵正永一案， 其贪污达到创新高的七个亿。这就是因为习治标但不治本， “把权力关进笼子” 成了空头支票， 反而更加集权独断， 绝对权力导致绝对腐败。
  <br/>
  <br/>
  至于对外强硬，表面上义和团风格再现，搞所谓 “战狼外交”，实际上软弱无能， 投降自保。中美贸易战就是最生动案例。习一开始号称有三大谈判底线： 即美方必须取消已增加的关税， 中方采购量要合理适度， 协议文本要平衡照顾双方利益。而结果呢， 一个底线都没有守住， 全线被美方击溃突破。因为对习来说， 其既得权力安全才是真正的底线， 其它都是摆设而已。 于是他当然不惜出卖国家民族切身利益， 以此充当自保安全的交换筹码。
  <br/>
  <br/>
  习的个人悲剧和危机， 由于他目前所处的最高权力位置， 还正在造成中国， 以至世界的悲剧和危机。 习近平已经给中国和世界带来了文革复辟， 新疆集中营， 全球大瘟疫， 再下一步， 可以预测就是经济危机和饥荒， 全国管控新疆化， 引发对外冲突战争并可耻战败。中国和中国人民为国家富强， 人民幸福， 文化跨越， 世界和平所做出的最新一轮卓绝努力和成就， 眼睁睁又将被断送在习氏这个民族败类手里。
  <br/>
  <br/>
  以习氏对现代文明无知无解的先天缺陷， 在三大历史溃败的阴影笼罩下的价值与世界观， 由谬误和极左造就的偏执人格， 这些都规定了他必将“一条道走到黑“了。那么目前最为重大紧迫的问题就是， 中国和世界是否也要奉陪习 ”一条道走到黑“？到底还要容忍此酋折腾祸害到什么时候？ 为此还准备付出多少更加惨痛和沉重的代价？！
