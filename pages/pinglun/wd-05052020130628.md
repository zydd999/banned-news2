 <p>
  各位听众你们好，我是王丹。虽然现在新冠肺炎的疫情还在全球肆虐，中国的疫情也是扑朔迷离，但是去年香港反送中运动的大规模抗争，显然让中共痛定思痛，下决心要彻底解决香港问题。因此，不管疫情和经济如何面临挑战，北京方面出手整治香港的动作，最近一个接着一个，显然是在加紧布局。这是香港民主派和港人不能不警惕的动向。
  <br/>
  <br/>
  进入2020年以来，北京就开始动手布局。首先是抽调了骆惠宁到香港，重新整顿中联办等中共在香港的指挥系统。其次是开始制造舆论，准备重新推动“二十三条”。中联办4月中旬公开表示，他们对香港民主派立法会议员阻挠立法会审议反分裂和反叛国法程序“非常愤慨”，要求香港尽快立法，在《基本法》中通过所谓的“二十三条”。了解香港政情的人都知道，港人对于“二十三条”的担忧和反对，远胜于送中条例，这是“二十三”条立法一直搁置的主要原因。现在北京释放这样的口风，显然是在评估港人一定会强烈反抗的情况后，依然决定一意孤行。北京的目的很清楚，就是利用“二十三”条困住香港民主派的核心组织，试图从上层瓦解港人的反抗力量。果不其然，随后就发生了15名香港民主派意见领袖被分别逮捕，准备起诉的事件。
  <br/>
  <br/>
  第三步布局，就是重建香港亲中阵营。在反送中运动的冲击下，原来的建制派可以说一败涂地，没有他们的在地配合，中共的政策很难落实到香港基层。因此，北京方面双管齐下，一方面要求林郑月娥政府进行人事调整，把更加服从北京指令的人补充进港府，这就是前不久一下子调动四名港府局长的背景；而纪律部队出身的入境处处长曾国卫升任政制与内地事务局局长，也显示北京希望港府强化镇压机制；
  <br/>
  <br/>
  另一方面，北京正在试图重新凝聚社会上的亲北京势力。5月5日，由两位原来的特首董建华和梁振英领军，宣布成立“香港再出发大联盟” ，这个组织号称有1545位社会各界人士担任共同发起人，宣布要“重拾团结，不同政见、不同阶层携手同心”，为香港寻找出路。这个组织由全国人大常委会委员谭耀宗担任秘书长，全国人大常委会香港基本法委员会副主任谭惠珠等担任副秘书长，囊括了香港资本利益集团的几乎所有成员，可以说是经过反送中运动冲击之后，北京对香港政治经济各派人马的一次整合。
  <br/>
  <br/>
  而第四步，就是提升中联办在香港内部事务上的发言权。中联办前不久批评香港立法会议员，引起外界质疑，明显是故意挑起事端，借机重新确立中联办的地位。现在港府已经明确承认，中联办“是代表中央在香港的机构，获得中央授权处理香港事务的机构，有权有责监督基本法的落实”。至此，在港府之上， 中联办的太上皇地位已经公开确立。北京看来已经不耐烦通过港府，要直接处理香港内部事务了。
  <br/>
  <br/>
  这一连串动作，显示北京已经确定了对港的新的方针，而这样的方针当然是以“瓦解香港民主派，扑灭港人反抗，强化中央对香港的直接管治”为核心。我们有理由预期，未来会出现“二十三条”立法，香港民主派代表人物被判刑，甚至维园纪念“六四”的烛光晚会也会被取缔。北京正在慢慢布局，准备一举将香港彻底收复。以上是自由亚洲电台特约评论员王丹为您做的评论。
