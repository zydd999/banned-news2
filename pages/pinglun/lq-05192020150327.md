 <p>
  最近华文网络上常常见到数十年老友断交，或是子女父母争吵不休、不欢而散的文章。而导致这一现象的原因则是武汉病毒，是关于病毒的起源及是否人工合成有意害人。
  <br/>
  <br/>
  例如一位作家在网上公告，与交往数十年的老友割席断交，因老友不断发送不胜其烦的信息，如武汉病毒是美国实验室合成的，专门用来毒害华人而不伤白人，这是美国害怕并无力阻止中国崛起施展的卑鄙伎俩等。作家对雪片般发来的不可理喻言论，觉得价值和认识方法相反之人还是从此断绝往来，比较省心少厌烦。再比如一位留学澳洲的学生，与父母通电话中也出现相同问题。对世界和疫情有所了解的他，告诉父母事实绝非他们口中样子，但不论怎样解释摆事实，其父母不仅不听和思虑一下，反而气恼地甩掉电话令其难受困惑。
  <br/>
  <br/>
  不过网上虽有这些因态度立场不同，导致亲友争吵烦恼乃至断交情况，但实际情况之激烈远远超出这种温和表现。大陆社会普遍激烈的态度与立场，尽是坚信病毒是美国合成阻止中国崛起的。所以实际上不论是网上还是其他的方式，大陆社会大量民众对美国甚至是世界，宣泄更多的是极端、毫无理性的仇恨。例如，公开号召制造无效的额温枪售美，让美国检测失效造成武汉病毒失控；卖粥的餐厅挂横幅欢庆美国、日本的武汉病毒爆发；将防疫口罩擦鞋、擦脚并拍成录像，向大陆公众宣传以这类口罩出口美国；英国首相感染武汉病毒进重症病房，这条消息获得大陆四、五十万人疯狂点赞。
  <br/>
  <br/>
  一个国家有点看法、情绪偏激的人，不仅不足为奇而且可以说在所难免。但是大陆普遍如此，尤其是多达十几亿人，就不但让人深感恐惧和悲哀，而且不得不深思这可能祸害全球的意识究竟是什么样的条件、土壤，才能产生的？
  <br/>
  <br/>
  首先是中共的信息阻绝、编造谎言、扭曲真相，是能够产生举国这种意识情绪的必要条件。没有中共几代人洗脑的统治基础，绝无可能举国如此闭着眼睛颠倒黑白。
  <br/>
  其次，中共的纵容，甚至赞赏的脸色，让这种无事实、无理性的疯狂流言，成了大陆绝大多数民众口中无需证明的通理。中共对社会言论的控制力，不要说反感，就是不以为然也绝无存在空间，但是上述类似言论除偶有招致不利后果的，中共可能私下要求有所克制外，其他全是听凭越传越邪乎，说明这正是中共希望和想要的局面。
  <br/>
  <br/>
  中共以大量钱财供养着“网络五毛”，就是按照中共网络宣传的需要发帖，每发一条便可以在中共那领取五毛。据内部传出的信息，随疫情现在已加价为七毛了。网络上有为数众多的职业五毛或叫七毛，他们是中共在册的，以此讨生活的水军。而大量的各种匪夷所思的谣言，便是这些中共供养的网络水军首创的。而凭此可以从中共手中领钱，也说明这种荒诞不经的谣言与中共难脱干系。
  <br/>
  <br/>
  更直接可以说明，这些谣言和丧失理智的激愤是中共需要并刻意导致的社会结果，便是中共本身也按耐不住而频频出手。按中共的鼓点进退的中国防疫专家钟南山，首先甩锅世界，声称武汉病毒虽然在中国首先爆发，但并不一定是在中国首先产生的；而中共新上任的外交部发言人赵立坚，更是直接指控美军运动员将病毒带到武汉军运会。中共官方出面直接宣导的大量这类战狼言论，无疑令大陆社会的仇美仇外意识，更加有了依仗和肆无忌惮。
  <br/>
  <br/>
  大陆社会众多民众还有一种或明或暗的恐惧，就是怕刚刚开始的物质较充裕日子消失。这是从特朗普指责中共经济上以坑蒙拐骗偷伎俩，损害美国利益，因而开展了美中贸易战后就逐渐形成了。这种恐惧和忧虑涉及到未来生活，所以早已酝酿了对美国的强烈不满甚至仇恨，疫情爆发更促进并找到一条适合发泄的方式和渠道。
  <br/>
  <br/>
  最后，中国民众在无事实、无理性谣言的趋使下，高举爱国主义大旗疯狂宣泄，可以说有久远历史。当年的义和团暴力，就是历史上一次典型性的展现。今天义和团虽然已经120年了，但是大陆绝大多数人并没有走出义和团思维.中共夺取政权后也发生多次反美、反英、反日等，虽然规模与义和团远不是一个等级，但是思维模式和粗暴手法并无二致。所以中共的统治因素和中国的历史因素，令匪夷所思的谣言煽动起仇恨轻而易举。
