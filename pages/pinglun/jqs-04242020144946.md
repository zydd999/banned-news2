  今天上午，我在读完《方方日记接力29》之文章、并动笔写下留言时，邱毅在我的脑子里是个什么样的人物呢？一个台湾文人，在央视四台中常常露一小脸，对海峡此岸的权贵颇尽谄媚之能事。所以，我把他称为金庸笔下的“星宿派”现代传人。
  <br/>
  <br/>
  然而，当我看了“今日头条”上，“邱毅说方方日记”的视频后才知道，他的坏，深不可测。邱毅之坏，超出了金庸老爷子当年所能预想的极限，以至于没能在书中给安排一个类似的角色。
  <br/>
  <br/>
  众所周知，方方先生《我的书与国家之间没有张力》一文，一直在网上挂着。方方先生《我如果不交待》一文，虽说已被404而国人见不到，但是，在台海两岸间光鲜游走的邱毅，却根本不难读到。关于“约稿”一事，方方先生在上述二文中说得清清楚楚：是上海《收获》杂志约的稿。而这个邱毅，先是胡说方方对此“语焉不详”，接着完全无视方方的如实交代；振振有词地说：根据结果来推断，应该就是方方应美国一家出版社之约，而动笔写下所谓武汉日记的。
  <br/>
  <br/>
  邱毅摇唇鼓舌，以内行的口气阐说道：什么叫做“约稿”呢？约稿，就是按出版社定下的框架、方向，还有一些准则，在出版社给定的鸟笼子里写东西。再说下去，邱毅其人就立刻阴毒了。他说，约稿帮她出书的那家美国出版社，可是与美国中情局有密切关系的喔。于是，在邱毅嘴中，“证据链”完整了，“案情”明朗了：武汉的方方，哪是一位我手写我心的良知作家？她，乃是美国中情局在背后牵线的码字傀儡，是美国用来打击、甩锅、抹黑中国的工具、棋子、马前卒。他以痛心挽救浪子的口吻，大呼道：利益当头的方方啊，你是太想一鸟在手了，明知美国人别有居心，你，却还是写、写、写。
  <br/>
  <br/>
  邱毅文人，文人邱毅，你来到世间，好不容易做一回人，怎么可以像蔡莉对待艾芬那样，鼓着鱼眼说瞎话？造谣诬人不脸红？罗织罪名欲置方方于死地？
  <br/>
  <br/>
  50多年前，我是一名被卷入文革的高三学生。当时，我曾经说过不少错话，干过不少错事、甚至坏事。但是，纵然打死我，我也不会像邱毅这样，道貌岸然、明目张胆地造别人的谣，诛别人的心啊！
  <br/>
  <br/>
  十年文革中，丑类很多。但是，即便像姚文元这样的超级文痞，似乎也没像邱毅这样，把自个儿的底裤扯得如此果敢干脆，掉得如此彻彻底底。
  <br/>
  <br/>
  邱毅文人，不但造谣说是美国人向方方约的稿，还指控方方日记在国外的出版等于坐实了一个十分严重的问题：病毒的根源在武汉。
  <br/>
  <br/>
  邱毅好像什么都敢说，但他还是不敢说是方方日记坐实了病毒的根源在武汉。60篇方方日记，已然传向全世界；白纸黑字，一字一句，记录的是武汉的疫情。60篇方方日记中，没有一篇说了、提了、定了新冠“病毒的根源在武汉”。中国驻美大使崔天凯先生也从来没说过：病毒的根源在哪里？就在方方的日记里。大使先生说的是：病毒的根源在哪里，由科学家来解决么。
  <br/>
  <br/>
  处在翻译中的方方日记英文版和德文版，在未经方方审阅的情况下，曾将日记封面的副标题，分别定为“来自疫情最初震中的报道”和“来自新冠危机始发城市的禁忌日记”。应当说，这种描述是客观、中性的。他们都没有说：日记“来自一个病毒根源的城市”。即便如此，方方还是提出看法，将副标题改为“来自一座隔离中的城市”。邱毅文人，你怎么可以像蔡莉对待艾芬那样，愣是瞎说最初封面上的副标题是“来自一个病毒根源的城市”？文人邱毅，你又怎么可以像蔡莉对待艾芬那样，“满怀悲情”、竭力煽情地说，副标题的问世“等于坐实了对中国的指控”呢？
  <br/>
  <br/>
  邱毅的上述莫须有指控自然十分荒唐，但是还真不能说他是“精神病”。说他精神病，就全给他解套了。精神病人这么说，你能说他坏吗？精神病人这么喷，也没对错这一说。而邱毅这么说，是刻意歪曲和扭曲事实，真是坏到家了。

  <br/>

  最后，邱毅似乎比英国军情六局的人还要胸有成竹，一本正经地说：是美国总统特朗普要把方方捧成世界名人，来帮助其抹黑中国、甩锅中国，并使现代版的庚子八国联军，能够师出有名。
