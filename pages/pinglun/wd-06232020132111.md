 <p>
  各位听众你们好，我是王丹。6月6日，台湾高雄市长，国民党政治新星韩国瑜被高雄市民以接近94万的高票罢免，创下台湾民主政治发展历史上的新纪录，同时，对于刚刚输掉总统大选的国民党来说，是又一次重创。但是，对国民党最具有威胁性的隐忧，其实还不是一两次选举的失利。毕竟，台湾的选民摇摆性很强，未来民进党执政只要有令人民不满的地方，国民党重新赢得选举的机会还是会有的。真正的威胁，是国民党在两岸关系上的论述。在这个问题上，国民党内部已经展开了路线斗争。
  <br/>
  <br/>
  上次总统大选失利之后，国民党内部中生代要求修改国民党两岸政策的呼声很高，现在的党主席江启臣就是在这样的检讨声浪中上台的，而检讨的主要对象，就是国民党长期以来在两岸关系上的神主牌---九二共识。对于江启臣来说，九二共识已经被证明是票房毒药，国民党如果继续抱残守缺，未来将很难得到台湾人民的认同，因此国民党势必要丢掉九二共识的包袱，提出新的论述，才能摆脱被动挨打的局面。这样的思考当然是明智的，但是却引起了党内大老的反弹，为首的就是前总统马英九。
  <br/>
  <br/>
  众所周知，马英九能够促成与习近平的历史性见面，靠的就是坚持九二共识，对他来说，九二共识才能确保两岸之间的良性互动，是国民党能够与民进党竞争的主要优势，岂能轻易改变。针对新任主席江启臣对于九二共识的立场动摇，马英九通过身边的幕僚，明确表达反对意见，认为“九二共识并非大选失利的原因，国民党应当珍惜九二共识的价值。马英九的长期盟友，前任党主席吴敦义，也站在这个立场上，反对对九二共识进行检讨。6月23日，江启臣邀请各位国民党的前主席餐叙，马英九和吴敦义原本答应出席，但在江启臣提出”九二共识是过去扮演两岸求同存异的重要工具“的说法之后，立刻决定不出席餐叙，其不满之意完全不加掩饰。国民党内部以中生代为主的改革派与以前总统马英九，前主席吴敦义为首的保守派之间的路线斗争，已经拉开了帷幕。
  <br/>
  <br/>
  国民党地方实力仍在，目前也还掌握新北，台中等选举重镇；虽然党产被民进党穷追猛打，但瘦死的骆驼比马大，资金也还不是问题。对于国民党来说，最重要的就是台湾的定位问题，而围绕九二共识的争论的背后，实际就是定位问题。国民党是要继续扮演能够与中共沟通，能够发展两岸关系的在野党角色，还是改头换面，建立本土形象，拒绝“中共同路人”的标签，这关系到国民党未来的发展和命运。这个定位如果不搞清楚，未来仍然不是民进党的对手，甚至有可能输给柯文哲的民众党，沦为第三党，走上亡党之路。
  <br/>
  <br/>
  目前在国民党党内，现任主席江启臣，原立法院长王金平显然是主张改造国民党，放弃九二共识，他们的对立面是党内大老连战，马英九，洪秀柱和吴敦义，而党内最具有实力的原新北市长朱立伦站在比较中间的立场，实际上是想从中得利。国民党历来有党内互打的传统，这一场围绕九二共识，关于定位问题的路线之争，最后鹿死谁手，目前还看不出端倪。但可以肯定的是，国民党内部如果在这个路线问题上无法取得共识，将会继续分裂下去，东山再起必将成为一场黄粱梦。以上是自由亚洲电台特约评论员王丹为您做的评论。
