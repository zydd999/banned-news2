 <p>
  各位听众你们好，我是王丹。最近几年，在习近平强调“斗争”意识的影响下，中国的外交系统一马当先，以咄咄逼人的姿态处理国际事务，被称为“战狼式外交”。这个“战狼式外交”的代表人物，就是中国的外交部长王毅。他不仅曾经在加拿大怒斥记者不了解中国，而且最近宣称因为捷克参议长访问台湾，要让捷克“付出沉重代价”，这番言论引起欧洲国家强烈反弹，也使得他在国际上可以说是臭名昭著。
  <br/>
  <br/>
  但是，王毅不是一个外交界新人，他算是典型的职业外交官，有着长久的外交工作经历，其经历也是有案可查的。很多了解他过去外交形象的人，都认为王毅最近几年来发生了巨大的变化。众所周知，他曾经担任驻日本大使，而我认识一位日本外务省高级官员对我说，王毅在日本期间，口碑相当好，不仅日文流利，形象温文尔雅，而且私下的言谈也表达出非常开明的思想。我这位日本朋友对他这几年的变化也十分惊讶，表示“不可思议”，想不出是什么让这样一个职业外交官变成了一个红卫兵、义和团一般的“战狼”式外交斗士。
  <br/>
  <br/>
  关于王毅的转变，我们可以有两种解释。第一种解释是外界比较流行的看法，那就是王毅善于察言观色，他看到了习近平的“宁左勿右”的思想，所以从趋炎附势的角度体会上意，按照习近平的外交路线调整自己的风格，而最终的目的还是为了保住自己的官位，甚至希望借此博得习近平的欢心，争取在中共的政治体系中更上一层楼。这当然是一种合理的解释，毕竟在中共这样的政治体制中，积极表现自己与最高领导人的一致，是一套升官发财的终南捷径。不过，同样是在外交体系中，同样属于职业外交官系统出身的驻美大使崔天凯，以及王毅的顶头上司、主管外交工作的政治局委员杨洁篪，虽然也忠诚执行习近平的路线，但是也没有像王毅那样表现出非常突兀的变化，也没有那么成天气势汹汹。应当说，王毅就算是为了仕途而故意表现自己，也表现得有些“过”了。
  <br/>
  <br/>
  当然，还有另一种解释，那就是王毅的表现其实是一种“高级黑”。了解当今中共政治生态的人都知道，对于习近平在政治上各方面都大幅度倒退的表现，中共内部其他一些派系的人是非常不满的。但是一方面是因为习近平牢牢掌握权力，用几乎每个高级官员都有的腐败记录威胁反对派；另一方面也是因为担心取代习近平，中共的统治基础会受到动摇，所以那些不满习近平的领导的高级干部，只能把不满藏在心中。但是这些人其实是希望看习近平的政治路线出现问题，他们好看习近平的笑话的。从这个角度说，不仅顺从习近平的“左”的路线，而且可以把这套“左”的做法更加推向极端，既让习近平无话可说，一旦惹出麻烦，最终的责任也是习近平承担，这就是所谓的“高级黑”。这种“高级黑”的做法，是中共宫廷政治中常用的手法。王毅的形象和作为，转变得过于突兀，也过于生硬，因此是在玩弄“高级黑”政治手腕的可能性也是存在的。
  <br/>
  <br/>
  当然，我们不是王毅，不知道他内心到底是怎么想的；而且，不管是趋炎附势还是“高级黑”，其效果都是给中国的发展带来负面影响，都是应当予以批判的。但是多角度解读王毅的变化是必要的，因为中国的政治本来就不是外界能够看到的那么简单。以上是自由亚洲电台特约评论员王丹为您做的评论。
