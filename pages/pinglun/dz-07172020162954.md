 <p>
  最近《纽约时报》 爆料说美国正在考虑禁止中共党员及其家属入境旅行，而美国国家安全官员对此动议也并无否认。这下子在中国这边就炸了营， 不少人如丧考妣， 也有人幸灾乐祸， 更多人是不知所措。
  <br/>
  <br/>
  中国外交部的反应是先指责美国此举 “可悲”， 就是不知道到底是谁可悲？ 而后又说美国如果禁止共党入境， 就是 “与14亿中国人民公开作对”。不知道是我们大家的数学不好， 还是中国的数学特别， 中共迄今有九千多万党员， 即不足一亿， 而中国总人口14亿。 相比之下， 中共也还是九牛一毛的分量， 怎么能随口就胡诌代表中国人民呢？就算把直系亲属都带上， 据说也就2.7亿人， 那跟14亿相比也还是“一小撮” 和 “极少数” 嘛。 所以中共外交部就是满嘴胡诌而已。没有中共， 中国和中国人民失去的只是锁链， 而必将赢得自由和幸福。
  <br/>
  <br/>
  有气急败坏者指责美国此举过于 “疯狂”， 其实美国此举再正常理性不过。 即所谓 “共产党” 在概念上完全就是自由和民主世界的敌人，既然如此， 美国为什么要允许共产党自由出入自己的国境呢？严格地讲， 美国不禁止共党入境， 才是真昏头或者疯了。
  <br/>
  <br/>
  而且， 美国如实施禁止共党入境， 也算是法理上的 “正当防卫”。 因为中共在香港强推 “港版国安法”，实际上就是阻止一切崇尚民主自由的人们正常入境香港，正常在香港生活， 工作， 和学习， 等于是给香港人民和外来访问者戴上了专制独裁的紧箍咒。为此， 美国和自由世界国家忍无可忍， 无路可退， 坚决反击，这也叫一报还一报。
  <br/>
  <br/>
  中共必须懂得， 反对习近平和共产党， 只是政见不同，但绝不犯法。 如果楞说犯法， 那就只能是个恶法。 同理， 共产党也不要到自由世界去了， 因为那里都是"敌对反动力量"， 你共产党还跑去干什么？ 你如果一个劲还想去， 就说明自由世界挺不错， 那么你首先应把 “港版国安恶法” 取消作废了再说， 因为香港是自由民主世界的一部分， 不需要这样的恶法。
  <br/>
  <br/>
  从根本上讲， 美国此举也迫使中共必须公开澄清自己的性质， 即中共到底是否民主自由的敌人？ 如果是，很好， 美国和西方都不用来了。 如果不是， 那就必须说清楚为什么人民没有独立财产权？ 为什么没有独立的司法制度？ 为什么压制言论自由， 封网，抓捕律师和公民记者， 为什么到处党管一切？ 为什么镇压香港人民的民主自由诉求？为什么搞什么一尊独裁？
  <br/>
  <br/>
  总而言之， 美国禁共党入境， 是切中要害， 大义善哉， 顺应潮流。中共及其铁杆党徒们在21世纪何去何从， 自己要好好掂量掂量！ 受中共蒙蔽或投机入党的人们， 也好好掂量一下自己的抉择吧。
