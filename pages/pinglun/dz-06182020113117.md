 <p>
  八年前习近平上台伊始， 就总是唠叨 “不能犯颠覆性错误”。他脑子里的颠覆性错误，是指的苏联式垮台，极权专制覆灭。众所周知，这对21世纪的中国和世界，当是莫大的福音而非错误， 而习氏却要为之全力充当救命稻草。于是习近平的意识，价值和行为， 一早就是与世界潮流和中国愿景根本脱节和背道而驰的。
  <br/>
  <br/>
  目前中国不少有识之士，无论党内党外， 知识界及中产阶层， 从红二代富二代到热血市民，港台及海外民主进步力量，大都有一个基本共识：即真正颠覆性的错误，就是中共在中国发展的又一难得历史机遇和关键节点， 却把习近平这样一个落伍和反派扶持到最高权力位置。目前此个超级失误的恶果正接连涌现，把中国步步推向失败和灭顶之灾。
  <br/>
  <br/>
  必须尽快纠正这个失误， 果断解决习近平问题，把这愚暗反动的一页尽快翻过去，才能重新给中国一个机会。 希望可以和平， 理性， 程序化地罢黜习近平。但在今日中国，此路明显不通， 所以必须有非常决心和手段加以解决。不少中国民众反映， 原来对比较激烈激进的变革主张， 一般都回避或免谈， 但现在也没这些顾忌了，这应该就是常说的 “逼上梁山”吧。
  <br/>
  <br/>
  习近平对其所处深重危机，或已有所察觉， 却决意不思悔改，一条道走到黑。首先，习按照 “祖训”， 拼命抓牢掌控 “笔杆子” 和 “枪杆子”， 加紧整肃镇压异己； 同时强化其党羽 “习家军”，掌控党内官僚集团的关键位置；再就是宣传鼓吹虚假的“扶贫脱贫” 成果，以收买稳定社会底层的人心。
  <br/>
  <br/>
  在这种严酷形势下， “倒习” 的机遇最可能出自两个方面：一是外部多方重大压力和制裁，二是习本人持续的 “自我颠覆“ 。 尽管为此中国和国际社会也可能付出沉重或惨痛的代价， 但事已至此也是别无选择。 对外习可能并不在乎单打独斗， 但如果国际社会在疫情追责， 香港台湾， 一带一路等重大问题上， 形成显著的反习制共联盟，那习近平就会压力山大，顾此失彼了。
  <br/>
  <br/>
  而从习所实施的路线政策方针来看，其在三大问题上正在加速自我颠覆： 国内经济严重下滑， 新冠疫情复发， 公民力量横遭打压，结党营私导致内部分裂反叛； 与香港台湾两地区主流价值， 利益集团，和国际社会发生正面严重冲突， 侵害自由民主社会体系，遭遇普遍反抗和反制； 再就是不惜与美国开始 “冷战” 甚至引发 “热战”，招致美国的全力重手制裁和打击。
  <br/>
  <br/>
  今日中国， 已处在开启 “后习近平时代“ 的临界点， 虽然习氏仍貌似顽固嚣张，但实际大势已去， 正一步步滑向 “自我颠覆” 的万丈深渊。
