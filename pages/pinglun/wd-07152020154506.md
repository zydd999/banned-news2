  各位听众你们好，我是王丹。最近一段时间以来，美国在对华政策上发布了一系列的新政策，其覆盖范围之广，其政策发布之密集，都是前所未见的。在对台湾的支持方面，美国国务院日前宣布，正式批准售台爱国者三型飞弹延寿案，总额为6.2亿美元。这是美国总统特朗普上任后第7次批准对台军售。在国际合作上，英国最近改变过去的态度，其数位、文化、媒体暨体育大臣陶敦宣布，基于国家安全考量，英国电信商在今年12月31日后就不能採购华为5G设备，已安装的华为5G设备，也需在2027年全数移除。这个动作，背后显然有美国的推动作用。这从白宫国家安全顾问奥布莱恩发布的推文就可以看出，他说：“英国这项行动反映国际日益一致的共识”，“我们期待与英国以及其他许多伙伴和盟邦合作“。目前加拿大政府尚未明文禁用华为5G，但加国三大电信商都已宣布不採购华为5G。在“五眼联盟”中，现在仅剩新西兰尚未表态是否禁用华为5G。美国一改过去的态度，开始积极凝集西方社会的集体力量，而针对的目标，毫无疑问就是中国。
  <br/>
  <br/>
  在香港问题上，拖延了一段时间以后，美国总统特朗普在14日表示，他已签署《香港自治法》，并且颁布一项行政命令，终止了香港长久以来的优惠待遇。与此同时，出人意外地，美国也开始在南海问题上向中国发难。美国国务卿蓬佩奥在13日宣布中国的南海主权主张“不合法”，否认曾母暗沙为中国最南领土，并强调各国在南海的共同利益遭中国“空前威胁”，美国似乎并不只是嘴上说说，就在蓬佩奥对中发表声明后，美国海军伯克级驱逐舰“拉尔夫·强森号” 14日就前往南海巡航，强调自由航行为建立自由开放印太地区的基础。
  <br/>
  <br/>
  除此之外，外界早就在传言的金融战，似乎也已经蓄势待发。美国参议院已经通过一项法案，要求在美的中资企业必须改变过去的作法，公开接受会计审查，否则就得在3年后下市，这项法案之后将交由众议院与总统批准，相信通过的几率非常大。而禁止使用抖音的禁令，应当很快也就会公布。
  <br/>
  <br/>
  这一轮“组合拳”，不仅令外界看得眼花缭乱，恐怕也让中共看得不知所措。前不久从外交部长王毅到原驻英大使傅莹，都纷纷发表与以外交部发言人赵立坚，华春莹等的口径很不一致的言论，表示要谨慎处理中美关系，还几乎是放软身段地表示中国无意与美国争夺第一大国的地位。这些长期处理外交事务的资深官僚，恐怕已经从美国的动作中嗅到了不同寻常的气味。
  <br/>
  <br/>
  他们的内心不安显然是有道理的。上述这一系列对华政策的出台，从经济到政治，从军事到科技，几乎是一个“总体战“的布局。如此广泛的外交动作，不可能是美国国务院或者商务部等单一机构的自行其事，这看起来更像是从白宫到国务院，经过缜密思考，而进行的一个整体的布局。鉴于目前我们看到的各项政策还显得比较松散，可以预期，未来一段时间，美国会在对华政策上做一个总体性的宣示，而这个宣示，绝不可能是致力于中美友好，而更可能是向美国宣布全面抵制中国的方向发展。此刻的局势，正所谓是“山雨欲来风满楼”。习近平一个月就预言过的“严峻的外部环境”，已经摆在中共的面前了。
