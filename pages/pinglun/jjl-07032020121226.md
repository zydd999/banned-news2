 <p>
  中国已强大到无法阻挡了吗？这个问题像房间里的大象，围绕在欧洲外交圈已有一段时间。本周北京通过港版国安法在香港生效实施后，欧盟反应乏力，欧盟主要成员国德国和法国对香港人也未提供任何协助，欧盟和德法两国领袖都回避指出中国共产政权势力扩大正在啃食全球民主价值的事实。
  <br/>
  <br/>
  做为香港《中英联合声明》的签署国，英国约翰逊首相在中国实施香港《国安法》后重申将信守承诺，提供持有英国国民海外护照(BNO)的香港人5年英国居留签证，满6年可申请公民身份的权利。这项承诺，同时适用于持有英国国民海外护照(BNO)香港人的亲属。约翰逊政府的立场，获得英国朝野一致支持。在国际上，愿意给予香港人庇护的除了与英国情报分享“五眼联盟”的其他四国，美国、加拿大、澳大利亚、新西兰外，还有台湾。至于欧盟，27个会员国中迄今没有一个国家表达愿意提供港人庇护的意愿。
  <br/>
  <br/>
  从中国人大通过在香港推行国安法议案以来，再三表达“严重关切”的欧盟，此刻对香港人撇开头来的回应令人意外吗？面对北京越来越咄咄逼人的外交手段，欧盟在五月底举行的27国外长会议上，表达了北京决心在香港推行国安法让欧盟和中国的互信基础发生动摇的立场；六月下旬在欧盟和中国领导人的欧中视讯峰会中，欧盟委员会主席冯德莱恩告诉习近平，一旦港版国安法通过，将会对香港有"非常负面的后果" 。

 <p>
  这段期间，欧洲议会投票表决通过决议，呼吁若北京对香港实施国安法，欧盟和欧盟成员国应到海牙国际法庭上告中国，因为中国破坏了香港“一国两制”原则。但这项决议案没有法律约束力，只能提供欧盟决策机构和欧盟成员国政府做为参考。而在中国本周三在香港施行国安法后，我们看不到欧盟国家提供港人庇护，听不到制裁中国声的声音。欧盟对香港的“严重关切”，仅停留在口惠阶段。
  <br/>
  <br/>
  欧盟不是不明白中国威胁，但在政治、经贸、利益和价值观的纠结下，欧盟对北京难以采取强硬策略。政治上，欧盟目前正在中美对峙中寻找一条中间道路，但美欧关系近年来间隙丛生，美中对峙愈激化，欧洲对中欧关系就愈审慎；经贸上，欧洲和中国经济总额合计占全球的三分之一，布鲁塞尔正积极争取在今年年底前与北京签署欧中投资协议。欧盟对中国在经贸上有一定程度的依赖，当欧盟和美国的经贸谈判进行不顺畅时，这种情况就更明显。
  <br/>
  <br/>
  利益和价值观则是欧盟面对中国最大的症结。中国透过投资影响政治的战略，过去十年在欧洲取得巨大成功。欧盟许多成员国和中国建立紧密投资关系，导致欧盟内部对中国意见分裂，无法达成一致的对华策略，包括一向被欧盟认为不可妥协的人权在内。香港就是当下的例子，西藏则早已被牺牲。
  <br/>
  <br/>
  中国可以不遵守《中英联合声明》，当然也可以摧毁其他协议。尽管欧洲领袖们知道中共政权不可信任，但要欧盟或欧盟成员国以经济利益做为谴责、反制北京的代价，对中国采取强硬立场，从现状观察，无异于缘木求鱼。
  <br/>
  <br/>
  中国是否已强大到无法遏阻？答案应该是，分裂的西方民主阵营壮大了野心勃勃的中国。
