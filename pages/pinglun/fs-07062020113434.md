  <br/>
  近二十年来，大陆的“民国风”、“民国范”、“民国潮”日益流行，呼唤回归民国的声浪日益高涨，民国的概念与祖国的概念越来越契合。中国大陆光复民国、重建共和的主张，很长时间内被当作奇谈怪论，但现在已经成了人们耳熟能详的看法；已经有一些以此为目标的组织，“民国派”成为当代民主革命终结中共统治的重要力量。

  <br/>
  几十年来，海内外民主力量定位失当、目标模糊、组织涣散、路线错乱，因此实效不彰。我认为：在大变局即将到来的前夕，指望匆忙组成严密的、强大的组织似乎是不可能的。但是，形成一个能够为各派政治力量交流、议事、协商、协调、合作的主流平台，形成一个具有实效的政治运作机构却是可能的，这对于民主转型的顺利进行也是非常重要的。这个平台或机构可以是：“中华民国大陆过渡议会”（简称大陆议会）。简单来说其主要功能有：
  <br/>
  <br/>
  1、为各党各派各界各地人士提供一个平台，表达、交流并达成最大共识，形成对中国大陆产生重要影响的政治合力。
  <br/>
  <br/>
  2、包容各种政治派别和异见，深入探讨各种实际问题，促成海内外有效的合作，并为未来民主宪政的议会运作进行务实的预设铺垫。

  <br/>
  3、以中国大陆各地区分支为议会主体，尽快形成遍布各省市县的民间影子议会。

  <br/>
  4、促进大陆各地区民意的自由表达和海内外的良性交流，为克服中共恶政遗留的隐患、避免或减轻社会动荡，打下良好的基础。
