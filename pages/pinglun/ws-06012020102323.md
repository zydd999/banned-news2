 <p>
  如果你想见识洗脑术的话，
  <br/>
  可以让我做你的导游，
  <br/>
  无须走远，就在我家旁边，
  <br/>
  据说是拉萨最好的幼儿园之一，
  <br/>
  大概连锁了三、四家，
  <br/>
  在每天的正午时分，
  <br/>
  （想起《正午的黑暗》[1]）
  <br/>
  会练习“抗日神剧”[2]，
  <br/>
  分贝极高，你不听都不行，
  <br/>
  我被惊扰，索性走过去观看。
  <br/>
  <br/>
  <br/>
  说普通话的女老师手把手地教着，
  <br/>
  孩子们都是拉萨娃，
  <br/>
  分正反两种角色：
  <br/>
  正角就一个，扮大义凛然状，
  <br/>
  演的是放牛郎王二小，
  <br/>
  他是全中国人民熟悉的抗日小英雄，
  <br/>
  但历史上有没有这个人很难说，
  <br/>
  党的宣传干部虚构了太多的民间戏。
  <br/>
  反角是一群，提着各种玩具枪，
  <br/>
  歪戴帽子斜穿衣，演的是“日本鬼子”，
  <br/>
  在阴森森的音乐伴奏下，
  <br/>
  匪气十足地上场，耀武扬威地杀人，
  <br/>
  以为是好玩的游戏。
  <br/>
  <br/>
  <br/>
  并不在于演得如何，
  <br/>
  而在于我及几代人对这出戏烂熟于心，
  <br/>
  另一出神剧的台词在脑子里回响：
  <br/>
  “仇恨入心要发芽”[3]……这些两分钟仇恨[4]啊！
  <br/>
  紧接着，稚嫩的童声背诵起三字经和弟子规[5]。
  <br/>
  <br/>
  <br/>
  “请放过孩子们吧！”
  <br/>
  我很想对那个长得像党委书记的女园长说。
  <br/>
  <br/>
  <br/>
  2018-6-1，拉萨
  <br/>
  <br/>
  <br/>
  注释：
  <br/>
  [1]《正午的黑暗》，（英）阿瑟·库斯勒著，首次出版于1940年。
  <br/>
  [2] 抗日神剧：据维基中文百科，是中国网络文化对一类中国抗日战争题材电视剧的称呼，被批判为经常不符合历史，以夸张、神奇的剧情获取观众的欢心。
  <br/>
  [3] 出自中国文革戏剧《红灯记》台词。
  <br/>
  [4] 在（英）乔治·奥威尔的小说《1984》中，人人每日要接受两分钟仇恨教育。
  <br/>
  [5] 三字经、弟子规，皆是中国传统启蒙教材，为儒教基础经典。
