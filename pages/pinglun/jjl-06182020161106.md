 <p>
  中国在香港强推“港版国安法”，导致中英关系恶化。英国首相约翰逊誓言将提供300万香港英国国民海外护照(BNO)持有人英国居留权外，并积极推动针对中国影响力扩张的限制外资收购英国本土企业，同时适用于学术伙伴关系和研究计划的新法案。英国政府这些前所未见的动作，标帜着中英黄金时代告终，英国准备烹煮中国这只下金蛋的母鸡。
  <br/>
  <br/>
  过去20年，英国是中国进军欧洲的桥头堡，从投资、科技到学术教育，中英两国建立了紧密的互利关系。中国借英国重商主义本质和以商引政传统，以伦敦为通往欧洲大陆的基地；英国则瞄准兴起中国的资金，与北京为交好。
  <br/>
  <br/>
  中英两国这层相互利用的独特关系，可以从英国前首相卡梅伦2012年会晤西藏精神领袖达赖喇嘛，引发中方大动作抗议，取消中国全国人大委员长吴邦国访英，并拒绝英国外交高层访问北京，但二年后，中国国家主席习近平就接受英国伊利莎白二世女王邀请，并于2015年到英国进行国是访问，中英双方在短期内尽弃前嫌，共同打造两国黄金时代的过程窥知。
  <br/>
  <br/>
  中英关系鼎盛期，英国是中国一带一路在欧洲的领头羊，是第一个参与中国倡议的亚投行的欧洲国家；值此同时，中国在英国的投资每年以翻倍速度成长，英国国际生中超过三分之一来自中国。
  <br/>
  <br/>
  中国在英国参与的投资项目，从能源、资讯到基础建设，从港口到机场，过去20年，中国在英国从大家熟悉的“中国制造”廉货出口国，悄然变身，成为诸多关键企业的大股东。当英国人发现伦敦希斯罗国际机场和英国自来水公司都有中资为“中国拥有”时，这一惊非同小可。
  <br/>
  <br/>
  中国科技公司华为在英国深耕，从3G 、4G打开欧洲和全球市场，对英国经济资讯和科技的影响20年来则是渐近且深远的，华为提供给英国主流大学的研发经费和实习与工作机会，远超过其他外国企业。比如华为提供给英国牛津大学的研究经费就高达五百万英镑。这是为什么面对美国全球围堵华为，英国始终不愿意全然跟随的主因。但新冠疫情提高英国对习近平领导的中国共产党政权的警戒，英国政府重新评估对华为与5G网络建设有限度开放的决定，英国国内反对声浪，此刻明显骤减。
  <br/>
  <br/>
  约翰逊首相顺势，在国会中提出推动立法限制外国企业收购英国企业，以免造成国安疑虑。约翰逊明言，英国必须防范中国“收购我们的企业”。根据法案，若有外国企业试图收购英国本土企业逾25%的股权、资产或智慧财产，英国企业须据实呈报。
  <br/>
  <br/>
  这是英国政府首次制定此类法案。值得留意的是，这项法案同时涉及中国留学生。开放的教育体系是英国软实力最重要的一环，国际生的学费同时是英国高等教育的重要经费来源，英国大学中国留学生的人数目前已接近五万人，这个数目每年仍以平均增加五分之一的速度成长，中国学生已俨然成为许多英国大学最主要的经费来源，这还不包括中方出资派员前来英国学术基构进行研究的学术交流在内，类似交流通常都是百万英镑协议。
  <br/>
  <br/>
  但随着香港问题升温，伦敦街头出现中国留学生阻扰示威活动，英国大学校园内中国社团对与中国有关的敏感话题进行施压，中方甚至企图让反对北京同情台湾、西藏和新疆维吾尔族的英国学者消音。
  <br/>
  <br/>
  中国留学生和中国资金对英国仿如一只下金蛋的母鸡。但在学术自由受到威胁，担心中国利用后新冠经济衰退透过投资影响政治和国家安全的考量下，英国现在要排除华为、立法限制中国企业收购英国企业、紧缩中国留学生人数、监督来自中国的研究经费。对抗中共在香港强推港版国安法，英国外相拉布则誓言不惜以中英自由贸易协议做为代价。凡此种种，英国看来是要以烹煮金母鸡的方式，与中国脱钩。
