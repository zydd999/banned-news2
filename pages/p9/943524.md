    在官方公布卢卡申科连任成功后，在外国领导人中，中国领导人习近平最早向卢卡申科表达祝贺，并在贺电中强调他高度重视与白俄罗斯关系，“愿同卢卡申科总统携手努力，共同推进中白全面战略伙伴关系。” 8月17日，新华网也刊登一篇新闻，无视数万人走上街头抗议政府，而报道白俄罗斯人上街支持卢卡申科连任、支持政府维护国家安宁与和平。

 <p>
  旅居欧洲的中国问题专栏作家王庆民表示，中白两国皆为国际上的专制政权，十分友好，在国际议题上支持对方的立场，如果卢卡申科得以连任，中国等于在国际上多一个盟友。
  <br>
   <br>
    他说：“卢卡申科的废除任期连任与

    </span>

    <br>
     <br>
      白俄罗斯今年七月在联合国人权理事会上，即代表46个国家共同发言，支持中国的新疆政策。去年也曾与其他54个国家称赞中国在人权领域取得的成就。中白双方关系紧密，可见一斑。
      <br>
       <br>
        除此之外，白俄罗斯被视为一带一路的支点国家，为中国深入欧洲内陆的主要桥梁。中国在白俄罗斯建立“中白工业园区”，进行各种投资。王庆民指出，白俄罗斯受欧洲各国在经贸方面制裁，因此与中国、俄罗斯交往相对密切。14日，根据中国媒体报导，中国与白俄罗斯在河北曹妃甸产业园举行签约仪式，预计每年实现进出口贸易额15亿美元以上。
        <br/>
        <br/>
        卢卡申科被外界称为“欧洲最后的独裁者”，此次大规模抗议示威、警察镇压暴行也被形容为“欧洲的香港”。王庆民表示，即便中国与白俄罗斯皆为专制政权，两国仍有不同之处。
        <br/>
        <br/>
        王庆民说，中国的领导人为中共内定，完全没有人民授权，而卢卡申科于1994年透过民主选举而当选总统，随后透过修法、造假选举结果等连任至今，“但还是有全民投票，哪怕是做假，但仍有形式，但中国连这个都没有。”
        <br/>
        <br/>
        此次白俄罗斯选举结果，许多西方政府认为这是一次虚假的选举，
        <br/>
        <br/>
        美国国务卿蓬佩奥批评选举“不自由、不公正”，针对示威者持续实施暴力、拘捕反对派支持者；德国也强烈质疑选举的合法性，批评不符合民主标准，欧盟也于14日的外长会议上认为选举结果不实，决定启动制裁程序。
