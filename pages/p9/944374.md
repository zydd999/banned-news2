  <strong>

  星期四结束对阿曼访问，返回华盛顿。各国针对
  <strong>
   伊朗
  </strong>
  的武器禁运即将在今年10月18日到期，蓬佩奥此行除了敦促盟友联手对抗伊朗，也积极推动
  <strong>
   以色列
  </strong>
  与波斯湾地区阿拉伯国家的关系。
  <br>
   <br>
    美国国务卿蓬佩奥本周出访以色列、苏丹、巴林、阿联酋、以及阿曼，这是美国首席外交官在8月13日以色列和阿联酋宣布建交之后，第一次出访这个地区。
    <br>
     <br>
      蓬佩奥星期四会晤阿曼苏丹海瑟姆·本·塔里克·赛义德。这次会面的目的在于促进海湾国家的团结与区域的和平繁荣。
      <br>
       <br>
        以色列和美国正在推进更多阿拉伯国家仿效阿联酋与以色列的关系正常化。官员说，巴林与阿曼可能是接下来跟进的国家。美国强化与海湾国家的军事、安全与经济关系，对抗伊朗的威胁、推进中东和平与繁荣。
        <br>
         <br>
          目前美国的外交政策重点之一，是在各国针对伊朗实施的武器禁运到期之前，要求联合国安理会对伊朗进行“永久性武器禁运”。
          <br/>
          <br/>
          2015年签署的伊朗核协议禁止德黑兰研发核武器，只允许继续进行和平的核能发电项目。作为交换条件，安理会放松了对伊朗的经济制裁。但同时协议里附带了所谓的"快速恢复制裁"条款，以便在伊朗不遵守协议的情况下，恢复制裁。此外，在这个协议之下，对伊朗长达5年的武器禁运将在今年10月18日到期。签约国的一方，为联合国安理会5大常任理事国加德国，另外一方则为伊朗。
          <br/>
          <br/>
          8月14日，美国推动安理会表决延长对伊朗实施的武器出口禁令，不过在中国与俄罗斯的反对票、欧洲国家大多弃权的情况下，并未通过。 8月20日，蓬佩奥前往纽约联合国安理会，开启为期30天、"快速恢复制裁" 机制的评估程序。
          <br/>
          <br/>
          蓬佩奥星期四在一个推文当中表示，对伊朗的迅速恢复制裁将在9月20日半夜生效。美国官员警告，如果中国和俄罗斯违反禁令，两国也会面临美国的制裁。
