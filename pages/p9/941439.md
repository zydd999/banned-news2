 <p>
  【看中国2020年7月31日讯】尽管
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/伊朗" target="_blank">
    伊朗
   </span>
  </strong>
  表示本月早些时候泄露的伊朗与
  <strong>
   中国
  </strong>
  25年
  <strong>
   战略合作协议
  </strong>
  是“毫无根据的谣言”，但伊朗与中国正在就最后敲定这项协议进行谈判。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  这份通过网络向媒体泄露的合作协议草案要求中国未来25年向伊朗的能源、电信、基础设施、港口、铁路和其他领域投资4000亿美元，换取
  <span href="https://zh.wikipedia.org/zh/%E4%BC%8A%E6%9C%97" target="_blank">
   伊朗
  </span>
  向中国提供价格大打折扣的石油。
  <br>
   <br>
    伊朗驻巴基斯坦大使赛义德·穆罕默德·阿里·胡赛尼称，有关报道是西方媒体的宣传。
    <br>
     <br>
      胡赛尼说，“鉴于合作协议的细节形式尚未敲定，这些是毫无根据的谣言，只不过是政治和媒体揣测”。
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  巴基斯坦非政府智库伊斯兰堡政策学会（Islamabad Policy Institute）本星期主办的中伊伙伴关系网络论坛宣读了这位伊朗大使的书面发言。
  <br>
   <br>
    胡赛尼说，伊朗和中国政府依然就未来25年双边合作的“总体概况”进行磋商。他提到，双方讨论的是一项有关战略合作的全面文件，这是从中国国家主席
    <span href="https://www.secretchina.com/news/gb/tag/习近平" target="_blank">
     习近平
    </span>
    2016年访问伊朗时开始的。
    <br>
     <br/>
     胡赛尼说，“这份文件是在所有政治经济领域以平等公平的方式开展长期战略合作的全面框架”，但“这份合作文件的范围到目前为止尚未达成最后结论”。
     <br/>
     <br/>
     中国仍然是伊朗的主要贸易伙伴和伊朗石油最大的进口国。美国政府2018年退出伊朗核协议并对伊朗实施经济制裁后，中国和伊朗的合作受到影响。
     <br/>
     <br/>
     在伊朗和中国商讨战略合作的同时，美国的制裁和新冠病毒疫情正在加剧伊朗经济困境，也让伊朗总统鲁哈尼领导的政府面临国内压力。
    </br>
   </br>
  </br>
 </p>
 <p>
  伊朗前总统艾哈迈德内贾德等伊朗政界人士抨击鲁哈尼总统试图与
  <span href="https://www.secretchina.com" target="_blank">
   中国
  </span>
  达成秘密协议。他们认为这会损害伊朗的主权。
  <br/>
  <br/>
  伊朗外长贾瓦德·扎里夫不认同这些担忧。他本月早些时候对议员们说，“协议没什么要隐瞒的”，协议内容最终敲定后将会公布。
 </p>
 <p>
  北京承认习近平访问伊朗期间双方发表了有关“全面战略伙伴关系”的联合声明，但没有对泄露给媒体的战略协议内容发表评论。
  <br/>
  <br/>
  上海国际问题研究院资深研究员金良祥星期三通过网络参加伊斯兰堡这个论坛时提到，中国与伊朗之间的关系近年来趋于成熟，但伊朗国内政治问题对这项25年战略合作计划构成了挑战。
  <br/>
  <br/>
  金良祥说，“伊朗面临的最大挑战是伊朗能否就与中国的合作问题达成国内共识”。
  <br/>
  <br/>
  金良祥说，这项长期合作协议“即使存在”，也还只是草案，但相关文件对外泄露并不是好事。
 </p>
 <p>
  金良祥说，“我们在中国有个著名的说法，那就是想干就干，如果你真的想干成某件事情，就去干好了。你如果不想干，就把计划拿出来讨论讨论，辩论辩论。”
  <br/>
  <br/>
  巴基斯坦前驻伊朗大使阿西夫·杜拉尼说，伊朗官员可能是有意泄露这份草案的内容，“拿出来炒作”，试图了解国际反应。他认为伊朗驻巴基斯坦大使胡赛尼的书面发言可能是在面临中国压力的情况下所采取的损害管制措施。
  <br/>
  <br/>
  杜拉尼说，“我认为中国眼下与伊朗打交道时会十分谨慎，因为他们必须面对美国的批评”，“他们目前不会想去开启另一条战线。但中国同时又不会去招惹地区国家，尤其是不愿招惹沙特阿拉伯和阿联酋。中国与沙特和阿联酋有将近700亿美元的贸易额”。
 </p>
