  【看中国2020年6月16日讯】国际社会回应香港
  <span href="https://www.secretchina.com/news/gb/tag/反送中" target="_blank">
   反送中
  </span>
  运动与“港版国安法”的力度各有不同，拥有五十多万名香港移民及其后裔的
  <strong>
   加拿大
  </strong>
  在有关香港的政策上与其邻国美国有很大程度上的分别。加拿大的香
  <strong>
   港人
  </strong>
  组织批评现届
  <strong>
   特鲁多
  </strong>
  政府的政策，他们希望当地港人扩大融入社区，将香港各个议题连结起来，迫使加拿大政府在国际社会上为香港发声。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  在香港反送中抗极权运动刚好满一周年之际，加拿大跨省份城市的支援香港团体共同举办网上论坛，邀请香港与加拿大东西两岸各阶层代表作出回顾和前瞻。多位讲者就“港版国安法”即将实施的严峻挑战下香港运动该如何应对，以及国际社会包括加拿大怎样支援香港的抗争等议题作出探讨。
  <br>
   <br>
    <strong>
     冯玉兰：加拿大对华是“鹌鹑”
    </strong>
    <br>
     <br>
      长期推动加拿大支援香港社会运动的香港加拿大联会(港加联)会长冯玉兰直言，尽管加拿大港人的公民社会自2014年香港的“占中运动”到去年的“反送中运动”不断成熟壮大，积极支援香港人；但奈何加拿大政府为港人发声的力度与表现却强差人意。她解释，这与特鲁多政府过去的“亲中”态度有莫大关系。
      <br>
       <br>
        冯玉兰说：“看到我们自由党政府其实对中共有很多的盲点。它(加拿大政府)不仅是鹌鹑， 很多中共对我们的干预，甚至是无赖式对待，随意拘捕我们两名加拿大人- 前外交官康明凯（Michael Kovrig）和商人斯帕弗（Michael Spavor），然后作为孟晚舟被引渡事件的筹码。结果到目为止，仍然是束手无策。”
        <br>
         <br>
          冯玉兰续说，孟晚舟事件发生后，加拿大国内其实有很强烈的民意，要求政府禁止使用
          <span href="https://www.secretchina.com/news/gb/tag/华为" target="_blank">
           华为
          </span>
          的设备，但政府延至今天还未能作出决定。她认为，这一取态也印证了为何在香港反送中及港版国安法等议题上，加拿大政府的表现如此滞后。
          <br>
           <br/>
           <strong>
            加拿大港人组织伞运后蓬勃发展
           </strong>
           <br/>
           <br/>
           冯玉兰介绍说，其实自2014年香港的雨伞运动开始，很多移居加拿大的香港人已经积极参加支援香港的活动，在多伦多成立支援会，发动商讨、公投，与香港占领活动无异。她续称，去年反送中运动，加拿大多个城市更各自组织声援，互相支持，如雨后春笋，遍地开花。例如协办这次网上活动的团体中，除了港加联外，还包括了香港加拿大人之声、爱民顿香港之友、温哥华香港协进会、多伦多香港人行动组与温哥华中国民主促进会等。
           <br/>
           <br/>
           <strong>
            港加联：加国港人需积极参政迫政府为香港表态
           </strong>
           <br/>
           <br/>
           目前，在加拿大生活的香港移民约有50多万人，在香港生活的加拿大公民也有约35万。相比之下，美籍港人移民与在香港生活的美国人分别只是23万及8万5千多。
           <br/>
           <br/>
           但冯玉兰说，在大选前现届政府对香港政策并无任何作为，纵使加拿大港人组织已发展成熟，但若要有效迫使政府在香港议题上勇敢发声，必需联系其他族裔，扩大参与，融入社区，积极在各级层面参政，促使所有加拿大人明白香港议题不是单一外来无关痛痒的议题，而是与普世价值观与国家安全攸关的议题。
           <br/>
           <br/>
           冯玉兰解释说：“我们需要有渠道发出声音，也要有策略，如我们港加联最近也举办了两次有效游说国会议员的策略训练斑，教懂大家与国会议员及政客倾谈时，他们能听进多少，其实我们也有一种技考可以去学，使游说教育的工作更加有效，每一个人成为精锐部队。”
           <br/>
           <br/>
           <strong>
            香港留学生的抗议遭大陆学生反对
           </strong>
          </br>
         </br>
        </br>
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  在举行支援香港的社会运动上，香港移民除了面对打动议员、官员与其他族裔国民的困难外，他们还面对中国大陆一些移民群众的反制。如去年8月21日，在
  <span href="https://www.secretchina.com/news/gb/tag/加拿大" target="_blank">
   加拿大
  </span>
  多伦多的“撑香港警察”集会上，出现了法拉利豪华车队，这些貌似来自中国大陆的年青人一边开车，一边大骂脏话，对着香港留学生用普通话高喊“穷逼”。
  <br/>
  <br/>
  此外，去年12月2日，加拿大列治文市中学发生了中国大陆留学生群殴一名十年级的学生。据悉，这名学生在社会学课堂上解释了香港游行争取民主人权的原因，结果在吃午饭时被数名大陆留学生殴打。最后，涉嫌施暴的大陆留学生只被有关学校惩罚停课三天了事。
  <br/>
  <br/>
  在英属哥伦比亚省列治文市(Richmond)的中学学生家长12月3日更对加拿大《星岛日报》称，自己女儿告知同一班级的台湾同学因公开支持香港反送中抗议活动，结果在储物柜发现一封恐吓信。
  <br/>
  <br/>
  <strong>
   温哥华“思政学阵 ”：加拿大政府放任中国政策
  </strong>
  <br/>
  <br/>
  面对上述众多例子，这次论坛的另一位讲者 - 在温哥华的“思政学阵”成员郑隽言并不感到陌生。他解释，在反送中运动发展一年后，在加拿大留学的香港学生对来自中国大陆的同窗已有所顾忌，不会主动引发冲突。但郑隽言最关心的，倒是校方与政府相关部门能否给予香港学生应有的保护。
 </p>
 <p>
  他说：“在不同的院校，校方对于中共或大陆政府各方面的影响和渗透采取了一个放任的态度，他们不会特别关注这方面的影响，也不会明确地采取立场去制止一些不同程度的影响与渗透。”
  <br/>
  <br/>
  郑隽言分析，面对着“港版国安法”的来临，更多在加拿大的香港留学生需要面对去与留的问题。若毕业后回港参与抗争，会否承受更大的风险。如选择留下加拿大发展，在声援香港时也同样面对部份
  <span href="https://www.secretchina.com" target="_blank">
   中国
  </span>
  大陆移民的反制。
  <br/>
  <br/>
  <strong>
   占中发起人戴耀廷：国际战线同样重要
  </strong>
 </p>
 <p>
  占中发起人之一的香港大学法律系教授戴耀廷表示，尽管加拿大的例子显示香港抗争的国际战线满途荆棘，但其实它与本地战线一样重要。
  <br/>
  <br/>
  他解释，不管在香港或是在海外，抗争与支援者越受打压，越代表了国际社会有道德上的责任为香
  <span href="https://www.secretchina.com/news/gb/tag/港人" target="_blank">
   港人
  </span>
  回应发声。戴耀廷以他有份筹备即将来临九月份的香港立法会选举民主派初选协调机制为例，指出若民主派真的在夺取过半数35以上的议席后被当局以各种理由取消议席，国际社会便不能不发声。
  <br/>
  <br/>
  戴耀廷说：“但是如果在今次的立法会选举，我们赢得选举，真是取得35+(以上的议席)，他们(港府)DQ(取消)了全部人(的议席)的话，国际社会在国际法的层面，会有更强的理据去采取更有针对性的行动，去针对这个专制政权。”
  <br/>
  <br/>
  面对港版国安法即将来临之际，多位讲者都勉励港人继续打好本土线与国际线的战线，相辅相成，继续为香港抗争运动作出贡献。
 </p>
