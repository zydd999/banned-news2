 <p>
  美国总统特朗普禁止抖音和微信的行政令已经下达两周，但禁令语义模糊，引起了广泛的争议，来自不同方向的法律诉讼也已经在路上。请听本台记者王允的报道。
  <br/>
  <br/>
  人权组织“人道中国”的创办人周锋锁在忙碌中被我的采访电话打断。他感慨说，现在有太多与中国民主议题相关的工作要做。
  <br/>
  <br/>
  <b>
   不同方向的愤怒
  </b>
  <br/>
  <br/>
  谈到微信，周锋锁再次使用了“愤怒”一词：“微信统治世界的时候，你哪里会有发出不同声音的可能性，这个是非常无耻的。你在使用微信的时候，那种审查是世界最全面的，最彻底的。”
 </p>
 <p>
 </p>
 <div>
 </div>
 <div>
  周锋锁愤怒的对象，不但是微信本身，还包括在他看来是为微信辩护的人。
  <br/>
  <br/>
  5位在加州、纽约州等地执业的华人律师两周前针对微信禁令发起了诉讼活动，他们起诉的对象是特朗普总统以及具体实施这一总统令的商务部长。
  <br/>
  <br/>
  他们的理由是，微信禁令限制了美国宪法规定的言论自由，影响到了美国几百万华人的利益。
  <br/>
  <br/>
  参与者之一加州律师朱可亮日前在接受本台记者采访时，也使用了“愤怒”一词：“如果（微信）这种沟通方式没有的话，对我本人的影响是非常大的。”
  <br/>
  <br/>
  但他并不想就微信事件中牵涉到的中美外交和政治问题置评。
  <br/>
  <br/>
  这几位律师为此还领头成立了专门的非营利机构，美国微信用户联合会（U.S. WeChat Users Alliance）。他们在自己的网页上称，这是一次大规模、多司法管辖区协同运作的史无前例的大诉讼。
  <br/>
  <br/>
  但周锋锁说，微信涉及到的已经不是简单的言论自由的问题，“你在那里面（微信）要维护的、要讲的话，你是要服从北京的。从根本上讲，这是中共改变了商业最基本的原则。”
  <br/>
  <br/>
  周锋锁虽然语气平缓，但言辞锐利。他强调，针对微信的行动是一场战争，“在这个中间是没有含糊其辞的余地的。所谓大多数中国人受到伤害，是非常可耻的一种说法。这就是中共的一套说辞，事实上这个国家就是被绑架了。”
  <br/>
  <br/>
  <b>
   同一阵营的纷争
  </b>

 <div>
  周锋锁在说“这中间没有含糊其辞的余地”的时候，在暗指与他观点不同的人权组织“公民力量”创始人杨建利。
  <br/>
  <br/>
  周锋锁和杨建利之间是有精神联系的，这要追溯到1989年六四大屠杀。当年，他们都曾在天安门广场上为中国的民主和自由振臂一呼。这一际遇也同时改变了他们的命运。自此以后，周锋锁和杨建利都长期从事中国人权活动。
  <br/>
  <br/>
  但两天前，这两位民主运动的同仁在参与Youtube上一家美国社媒“China Unscripted”的视频辩论会时，发生了“遭遇战”。在立场上，他们都主张要限制微信；但在具体方式上，杨建利认为要采取更加务实的态度，这在周锋锁看来是留有“中间余地”，他转而强调要在美国全面禁用微信。
  <br/>
  <br/>
  杨建利感到自己有被误解，他在接受本台记者采访时指出，技术上、法律上要全面禁用微信都不太可能。他请教过各方面的专家，如果在美国要全面禁用微信，就不得不建立防火墙一类的设施，但这在美国是不可行的。
  <br/>
  <br/>
  与杨建利合作的美国律师王代时向本台强调，美国宪法规定的言论自由原则是全面禁用微信的根本法律障碍，“事实上，宪法第一修正案甚至保护具有偏向性的言论。我是美国公民，如果我出来说支持中国政府，支持中国共产党，这是第一修正案保障的权利。”
  <br/>
  <br/>
  今年二月份，新冠疫情的吹哨人李文亮医生感染病毒去世后，杨建利主持的人权组织“公民力量”就筹备对微信提起集体诉讼。
  <br/>
  <br/>
  “它是中共的长臂，进行审查、迫害、洗脑，还有间谍功能，这几宗罪我们在（诉讼文书）里面都讲得很清楚了。”
  <br/>
  <br/>
  杨建利指出，在美国现行的法律框架下，有很多方式可以对微信施加严格的限制，“如果我们对腾讯的集体诉讼胜了的话，就成为一个先例，使得每一个被微信封号的人都按照这个先例去告它。因为美国是案例法，我们赢了，他们肯定会赢。这种情况下，微信或者改变它的行为方式，或者它在美国就没办法做生意了。”
  <br/>
  <br/>
  他分析说，还可以仿效对华为的措施，对微信母公司腾讯的特定员工实施签证限制，以及剥离微信在美国的生意；也可以调查帮助微信发展技术的美国公司，并对使用微信的普通用户提出警告。
  <br/>
  <br/>
  杨建利强调，在限制微信的同时，要把对普通用户的损失降到最低。
  <br/>
  <br/>
  <b>
   普通人逆袭 诉讼大战山雨欲来
  </b>
 </div>
 <div>
  <b>
   <br/>
  </b>
  <img class="image-inline captioned" src="../../../../resolveuid/03bee98f-e1bf-41b4-b825-efb159f259ff"/>
 </div>
 <div>
  美国微信用户联合会计划中针对美国政府的诉讼就是这种普通用户的代表。
  <br/>
  <br/>
  参与行动的朱可亮律师认为，就美国政府信息安全考虑来说，可以有打击面小得多的方法来处理微信的问题。
  <br/>
  <br/>
  他告诉本台，诉讼准备已经取得了很大的进展，“我们准备这个周五递交诉状，发起诉讼。在递交诉状的当天，我们还会递交一个临时禁制令的申请。”
  <br/>
  <br/>
  这个律师团起诉的依据是《宪法》和《行政程序法》。后者规定，美国政府在制定政策或法规时，要遵循一定的程序，把公众的利益和诉求考虑进去。
  <br/>
  <br/>
  但王代时律师认为，针对总统令的起诉还为时尚早，“问题是总统令的内容太模糊，没有人知道那到底是什么含义。如果是我来为特朗普政府辩护，我会说，我都还没有解释这个总统令的性质是什么。所以，现在起诉还太早。”
  <br/>
  <br/>
  在另一面，“公民力量”已经筹备半年的针对微信的起诉，很快就要递交法院。王代时是幕后操作人之一，但他卖了个关子，对具体内容暂时保密。
  <br/>
  <br/>
  周锋锁则告诉本台，一旦有人针对微信起诉美国政府，他愿意站出来反对诉讼。
  <br/>
  <br/>
  （本台记者唐家婕对报道有贡献。）（记者：王允；编辑：申铧） 网编：申铧
