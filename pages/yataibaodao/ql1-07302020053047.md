 <p>
  武汉协和医院一年轻护士于本周三（7月29日）从该医院的13楼坠下身亡。事件暴露该医院防护不足，一线护士身处险境。死者家属称，坠楼现场监控探头被告知“坏了”，而死者生前给护士长的留言却看不出她有轻生念头。
  <br/>
  <br/>
  事发本周三上午近11点，武汉协和医院一名心内科护士坠楼身亡，目前警方已展开调查。当天，华中科技大学同济医学院附属协和医院发出通告，称“7月29日上午10：45，我院不幸发生一起坠楼事件。坠楼者为我院护士，经抢救无效去世。对此，我们深表痛惜，并将积极配合家属及有关部门处理善后事宜。我们向逝者表示深痛哀悼，向家属表示深切慰问。”死者父母称，该护士系家中独女，孩子不到两岁，在该医院工作不到五年。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  <b>
   护士父母医院拦路求见院长
  </b>
  <br/>
  <br/>
  武汉协和医院一患者家属高女士接受自由亚洲电台采访时说，她见到死者父母在医院停车场出入口哭诉，要求医院对该名护士的死因，给予合理解释：“昨天护士的爸爸，还有妈妈，家里其他的亲戚就在医院住院部的出入口，挡车出入的栏杆，不让车辆进出。说他们的女儿是抗疫英雄，但是现在被协和医院逼死了，从13楼跳下来。（父母）要求院长出来，院长一直没有出来，所以他们没办法，就在那个地方，当时很多人围观。”
 </p>
 <p>
 </p>
 <p>

  <b>
   不出所料 现场监控探头坏了
  </b>
  <br/>
  <br/>
  据澎湃新闻引述死者家属说，他们在监控探头未看到内容，院方只回答护士跳楼了，监控坏了。对此，武汉网民提出质疑，认为医院方面似乎又要想隐瞒些什么。邱先生对本台说，官方的回应让人猜疑：“我们国家历来就是这样的，出了一点大事情或者关注度高一点，他往往监控不是失灵就会坏了，或者手机储存的一些资料也会没有。为什么这么巧合？我认为没有这么巧的事情。”
 </p>
 <p>
  <b>
   生前留言不会为领导而死
  </b>
  <br/>
  <br/>
  网络流传死者生前给心内科护士长的留言写道，“护士长,我明天不能来上班了,我辞职。我想了一下,明天我们病区的人力应该也是够的,不好意思，我先当逃兵了。我可以做一个英雄而死，但不能为这样的领导班子而死,我为所有的护士感到悲哀，以后我也不会再从事这个职业... 我愿意在我执业的最后一天，为所有的临床一线的护士发声。对不起护士长，我辜负了你的期待。”
 </p>

 <p>
  关注此事的武汉居民王梅对本台说，死者的留言不像要轻生：“各方面消息，包括这个护士的朋友圈已证实此事有问题，她最后的朋友圈留言，（给人）不像是要自杀的感觉。我觉得医院整个医疗系统通过这次疫情，已暴露出他的千疮百孔。我觉得问题是很大的。”
  <br/>
  <br/>
  <b>
   防护不足一线护士身处险境
  </b>
  <br/>
  <br/>
  武汉协和医院作为武汉肺炎定点医院，而该名护士坠楼死亡据说与护理部主任发生工作冲突有关。坠楼护士朋友圈曝出医院黑幕，称护理部主任将医院隔离区全交由护士值班，至少每天要在那里待上8小时，不能吃不能喝不能上厕所。
 </p>
 <p>
 </p>

  王梅说，假如连护士的防护措施及健康难以保障，如何抢救患者：“前线的战士都没有防护，连她们都有危险的话，如何救治病人。这个道理大家都懂啊。从她最后发朋友圈的信息看，他应当要辞职‘我明天不能来上班了，我辞职，我要为自己而活，明天睡到自然醒，看一看下雨还是日出’，这说明她要开始一段新的生活，并没有想要结束自己的生命。她有父母，又是个独生女。”
  <br/>
  <br/>
  <b>
   护士集体要求撤换护理部主任
  </b>
  <br/>
  <br/>
  网民“我叫安然君”转发来信说，在武汉协和医院隔离病房工作的护士集体要求医院撤换“护理部主任刘义兰”，私讯称，“在抗疫时期没有一个护士退缩，在物质匮乏时，刘义兰要求核酸咽拭子采集由临床护士完成，但她们没有经过专业培训，可能无法完成，对此刘义兰却说，已经发了操作影片，学习后照这个搞......”。
  <br/>
  <br/>
  上述事件引发网民议论，有人写道：“监控这么巧又坏了”、“熬过了疫情，没有熬过领导”、“家人该有多伤心啊”，“希望院方能给个说法!”、“每个医院的护理部都是最恶心的”等等，本台记者致电协和医院，但对方多次拿起电话又挂断。
