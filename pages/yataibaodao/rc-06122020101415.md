 <p>
  视频会议软件领头羊Zoom公司6月11日认错，坦言“应中国政府要求”曾关闭美国用户帐号，并终止了一场关于六四的Zoom纪念活动。尽管Zoom声称，未来“不会再让中国政府的要求，影响任何非中国本土注册用户”，但这能缓解争议吗？ Zoom高度配合中国政府，又对美国、尤其学术界有什么影响？
  <br/>
  <br/>
  “我们希望有一天，各个设置障碍、使自己的人民与世界断绝联系的政府会承认，这种行为违背自身利益，也侵害公民及所有人类的权利。” 这是创立于美国的Zoom公司，在关闭多个中国海外民运人士的付费账号引发质疑后，公布详细个案调查报告时，开宗明义的说法。
  <br/>
  <br/>
  Zoom的这段话没点名中国，但承认近来三个帐号及所主办的活动遭关闭与终止，是收到“中国政府”指示。
  <br/>
  <br/>
  总部位于美国的人权组织“人道中国”创办人周锋锁，在办完纪念六四的活动后，帐号一度无预警遭关闭。香港六四活动主办方“支联会”负责人李卓人的账号，更早在5月22日遭屏蔽。八九民运学生领袖王丹，在六四31周年当天透过Zoom主办线上纪念会遭干扰被迫停止。Zoom说，因为“中国政府通知”，这类活动在中国属非法。
  <br/>
  <br/>
  周锋锁和王丹都是位于美国的付费使用者，李卓人则是在中国声称“一国两制”的香港注册。虚拟的网路世界，隐私本来就难有百分之百的安全保证，但一家美国企业听从中国政府的指挥，连非中国本土注册的使用者，都可能活在中国共产党监控审查的阴影下，让人担忧中国负面影响力的无远弗届。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  <b>
   软件平台向中国低头 美学者尤其担忧陆生隐私安全
  </b>
 </p>
 <p>
  华盛顿乔治城大学艾德蒙·华许外交学院教授米华健（James Millward）接受自由亚洲电台采访时就说，新冠肺炎疫情以来，乔治城大学都透过Zoom这一平台来维持教学运作。他教授中国历史，撰写过关于新疆的研究专著。对于中国官方藉Zoom将影响力扩张到美国本土，他用中文讲述自己的“三个担忧”，包括校园学术自由、课堂上学生讨论的自由，以及学生、尤其是大陆学生的隐私与安全。
 </p>
 <p>
  “我很担心也很失望，因为我也比较喜欢Zoom，虽然这并不是一个完美的教书办法，不过，也可以说，这个春天刚开始的学期，要不是有这个平台，我们也没办法上课，但是（Zoom承认）可能会偷听或偷看，我们的学生知道有这样威吓的可能性，这个是会干扰到学术自由的。”
  <br/>
  <br/>
  米华健说，美国今年秋天疫情能否缓解还是未知数，今年九月开学季中国留学生若不能返美上课，乔治城目前规划仍以Zoom平台远距教学。那么，身在大陆的学子、在虚拟课堂上还敢不敢畅所欲言？他说，“对中国政府而言，讲到西藏、台湾、新疆，都可以是国安议题。”
  <br/>
  <br/>
  他建议，美国大学应该联合行动，像是发表共同声明，要求Zoom明确承诺并写明原则规范，不能让美国的学术自由受侵害。
 </p>
 <p>
  尽管Zoom强调，公司未提供任何用户资讯或活动内容给中国政府，也不会允许中国政府的要求，影响境外注册用户，然而，参与“人道中国”Zoom线上纪念活动的澳大利亚90后中国留学生“洞物员”在个人推特上说，在她公开表达追求中国民主运动后，她在国内的父亲遭到中国警方与国安人员调查与恐吓。
  <br/>
  <br/>
  王丹则告诉本台，他再也不会使用Zoom，并发表书面声明指出，Zoom配合中国政府要求的说明，凸显中共“积极主动”进攻民主制度，世界应该警醒。他还指出，将通过法律途径，要Zoom公司承担责任。
  <br/>
  <br/>
  Zoom的创办人袁征是华裔美国人，在山东出生。新冠肺炎疫情以来，Zoom因为使用方便，设计简洁而广受欢迎，但关于Zoom的安全性争议，以及是否配合中国政府审查，一直受到关切。
  <br/>
  <br/>
  袁征接受《华尔街日报》访问时曾说，“中国政府从未要求提供外国用户的数据信息，中国政府部门只关心本地会议”。  事实上，Zoom 2019年9月曾在中国遭禁用，直到Zoom对于中国用户采取实名登记注册后，才开始中国市场的运营。今年五月，Zoom禁止非实名注册的中国使用者发起会议。
 </p>

 <p>
  <b>
   Zoom危机越处理越大 两边不讨好
  </b>
  <br/>
  <br/>
  身在中国国内的金融学者贺江兵说，自己不会使用这类线上通讯或会议软件，是因为隐私顾虑，“在那里参会就像是裸奔”。他不认为Zoom创办人袁征的华裔背景应受关切，因为外国企业想进入中国市场遇到的困境与批评，一直以来都没改变。
  <br/>
  <br/>
  他说，“墙头草嘛！他（Zoom）想双方都讨好，结果可能都得罪了。”
  <br/>
  <br/>
  美国政界对Zoom的说明也多有批评。
 </p>
 <p>
  美国联邦参议员霍利（Josh Hawley）质疑，Zoom所谓的“遵守当地法律”的说法，就是“为中国共产党执行言论审查”；美国前国家安全顾问波顿（John Bolton）也推文指出， Zoom向中共屈服令人感到不安，如此听命于北京的公司，外界都该对其保持怀疑。
 </p>
 <p>
  Zoom今年5月聘请波顿的前任、前白宫国安顾问麦克马斯特（H.R. McMaster）担任独立董事。他当时指出，希望“协助实现Zoom成为世界上最安全的视频通信平台的承诺”。
  <br/>
  <br/>
  Zoom预告，将在今年六月底公布改进自身全球发展政策的透明度报告。虚拟世界的网路主权争夺战，Zoom会是观察风向标。
 </p>
