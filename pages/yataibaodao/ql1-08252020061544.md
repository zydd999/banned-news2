 <p>
  内蒙古政府教育厅下文，从9月1日新学期开始，全区所有中、小学开始接受汉语《语文》教材。官方资料显示，内蒙古，甘肃、吉林、辽宁、青海、四川等6省民族语言授课学校，于今年秋季开始要使用国家统一编制的教材。有人在微信群表达不满，群主很快便被公安带走。
  <br/>
  <br/>
  本台近期曾多次报道内蒙古自治区教育厅公布，9月1日起采用所谓双语教学模式，即小学思想品德课、初中历史、政治等课程采用汉语授课，放弃了传统蒙语教学。到最近官方更是发出正式通知，要求从小学一年级和初中一年级开始，所用《语文》教材放弃蒙古族语，改用汉语。   /&gt;
 </p>
 <p>
 </p>
 <p>
  <br/>
  现旅居美国的蒙古人诺民本周二（25日）接受自由亚洲电台采访时说，很多蒙古族家长不满政府取消母语教学：“据我了解，现在全区各地（民族学校）都要学习汉语，是从小学一年级开始，还有初中一年级的汉语变成语文科，而且像汉族学校的难度一样。”
 </p>

   强制消灭蒙语教学触碰到蒙族人底线
  </b>
  <br/>
  <br/>
  内蒙古教育厅一份内部材料写道，按照中央统一部署, 内蒙古、甘肃、吉林、辽宁、青海、四川等6省区民族语言授课学校，将于今年秋季开始使用三科统编教材。内蒙古作为模范自治区，根据国家教材委员会工作部署, 内蒙语文、 政治、历史三科统编教材分三年逐步实施。今年秋季起，小学一年级和初中一年级先使用统编《语文》教材，2021年小学一年级和初中一年级使用统编《道德与法治》(政治)教材,2022年初中一年级使用统编《历史》教材，并使用国家通用语言文字授课。
  <br/>
  <br/>
  蒙古人诺民说，在民族学校，蒙语文曾被称为语文，但现在被称为“蒙语文课”：“这一次在小学一年级开始教汉语和增加汉语社会科目的行为，确实在海外蒙古人当中引起强烈的反对，他们触碰了蒙古人的底线，也低估了蒙古人保护自己文化的决心。现在不管是在日本的内蒙古人，还是在其他国家生活的内蒙古人，他们积极参与了搜集关于此次行为的文件和报道。同时也在组织各种活动抵抗政府的愚蠢行为。”
  <br/>
  <br/>
  旅居日本的蒙古族忽必斯对本台说：“中国在内蒙古自治区实行的所谓民族语言教育的改革。对于蒙古人的冲击很大。蒙古人在网上纷纷谴责中共的这种做法，都在表示抗议，甚至有蒙古人表示，如果从小学一年级对孩子实行中文教育，他们可以选择拒绝送孩子们去上学。”

  </b>

  <br/>
  <br/>
  近期，内蒙古微信群有人热议有关停止蒙古语授课，被公安封群抓人。忽必斯对本台说：“家长们组成了三个家长群，都在讨论此事。结果今天早晨，公安将这个群主抓走。一个群有500个人，3个群有1500人。”

   海外蒙人认为这等同于文化灭绝
  </b>

 <p>
  旅居德国的南蒙古议会主席席海明对本台说，中国政府此举明显是要对蒙古民族实行文化灭绝政策：“他是要同化和消灭其他民族的文化，这是对人类文明的挑战和践踏，我们蒙古人一定会反抗到底。我们希望国际社会关注，不要坐视一个民族古老的文化被消亡这种情况，在21世纪的世界发生。”

  本台记者致电内蒙古自治区教育厅办公室及通辽市教育局办公室，但始终无人接听。
