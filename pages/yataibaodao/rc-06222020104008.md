 <p>
  瑞士银行（UBS）日前发布的一份针对在中国设厂的制造业首席财务官的调查显示，在新冠肺炎疫情和美国官方政策这两股驱动力的作用下，美中经济、尤其是科技产业脱钩的压力增加。美国半导体行业又如何看待美中经济的加速脱钩？
  <br/>
  <br/>
  瑞士银行旗下的证据实验室（UBS Evidence Lab）针对在中国设厂的两百多家出口制造业首席财务官所作的最新调查发现，新冠肺炎疫情全球大流行，让各国开始反思一些行业如医药器材行业过度依赖中国的安全漏洞，供应链脱钩与去中国化的压力增加。
  <br/>
  <br/>
  瑞银预期，中国制造业的固定资产投资今年将减少百分之五。
  <br/>
  <br/>
  然而，调查也发现，早在2018年美中贸易战开打前，供应链重组已经启动，跨国企业从中国出走的趋势，越来越明显。瑞士银行今年五月公布新冠肺炎疫情期间进行的调查发现，约六成跨国企业高管表示，他们将持续或可能从中国转移到其它国家、减少投资。
  <br/>
  <br/>
  美国福克斯电视台计算，全球生产线撤离或减少投资中国，涵盖的出口贸易额最高恐达七千五百亿美元。
 </p>
 <p>
  瑞士银行中国区首席经济学家汪涛日前在美国乔治城大学主办的一场视频研讨会上则指出，中国外资直接投资（FDI）的数据，包括总金额与占中国国内生产总值（GDP）的占比持续下降，已是多年来持续的趋势，早在美中贸易战升级前就已经发生。另外，生产基地转移或投资分散，不同产业也各有不同的做法。
  <br/>
  <br/>
  她说：“我会形容这是'分散投资'，不是'完全的脱钩'。平均来说，这些专门做出口的受访企业高管指出，大约有三到四成的产能会转移阵地。回答说有超过五成产能会要撤离中国的企业是非常少的，而在美中贸易战开打前，不同产业因为不同原因，早就开始分散投资。”
 </p>
 <p>
  汪涛指出，有鉴于中国庞大的消费市场，她预期，以食品制造业与汽车业为例，在未来十五年，趋势是增加投资。
 </p>
 <p>
  瑞银的调查分析提到，美中经济脱钩更关键的动力源自美国的政策指引，尤其在美国官方主导下的科技业脱钩，瑞银预估，将会造成中国经济下行的压力。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <b>
   各国研究与中国脱钩 习近平功不可没？
  </b>
  <br/>
  <br/>
  关注中国经济与科技发展的评论人士秦鹏则告诉自由亚洲电台，与中国脱钩、甚至是全速进行的脱钩，除了新冠病毒是一个催化剂，中国舆论近来热议、被网友戏称为“总加速师”的中国国家主席习近平在疫情后的作为，也起了很大作用。
  <br/>
  <br/>
  “这一次的疫情，毫无疑问是导致美国全面评估和中国的关系，包括产业链的契合度与转移的必要性，以及增加多样性的需求；我是一直支持习总书记继续干的，因为换别人上来，都达不到他所干的、全世界加速与中国脱钩的效果。”秦鹏说。
  <br/>
  <br/>
  事出必有因，加速脱钩有来由，尤其是在科技产业上，对华为违反美国的出口禁令，陆续出台出口管制限制。然而，美国商务部近来却又宣布，允许美国企业可以和华为分享用于5G行业制定标准的技术。
  <br/>
  <br/>
  商务部长罗斯（Wilbur Ross）当时发声明指出，这是为了“确保美国企业能更全面参与通讯领域的标准制定。”
 </p>
 <p>
  美国半导体产业协会（SIA）在美中科技战以来，一再呼吁美国官方，不要因为针对中国不一致的策略，让美国失去原本领先的行业地位。
 </p>
 <p>
  半导体产业协会负责全球政策的副总裁古瑞奇（Jimmy Goodrich）就说，半导体产业链分为研发、芯片设计、前端制程与后端封装测试，半导体产业中最高端且有价值的研发与芯片设计，美国仍遥遥领先；中国主要负责后半段的生产制造。制造环节是高劳动力密集型、低利润，利润仅百分之十，而前三大芯片制造商又都是外企。
 </p>
 <p>
  然而，中国也是美国芯片行业最大的市场。
  <br/>
  <br/>
  在脱钩与多样化间，美中从行业到官方之间的你来我往，秦鹏形容，这就像是要忍受“长痛还是短痛”的抉择。
 </p>
 <p>
