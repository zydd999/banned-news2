 <p>
  美国上周再次针对中国电信业龙头企业华为实施新的管制措施后，华为戏称，在美国的巨大压力下“还活着”。在美国步步进逼下，华为还能活多久？美国针对华为采许“申请许可制”，又真能堵住现有的法律漏洞吗？
  <br/>
  <br/>
  华为5月18日开始在深圳召开第17届分析师大会（HAS 2020）。在一连四天活动的前夕，美国商务部宣布要修改《出口管理条例》（EAR），使用美国技术或设备的外国半导体业者，未来若要供货给华为及华为子公司海思半导体（HiSilicon）的海外企业，都要先向美国申请许可。
  <br/>
  <br/>
  与此同时美国商务部第五度延长美国企业可以出货给华为的临时许可九十天，今年8月13日到期，预计这次延期将是“最后一次”。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  在深圳，没有炫丽的舞台、灯光设计，也看不到华为创办人任正非出席，华为刻意通过官网上的直播，让外界看到轮值董事长郭平摆出的低姿态，讲述去年在美国巨大压力下，华为这架“弹痕累累的老飞机”还在空中飞翔。
  <br/>
  <br/>
  “我们公司任总常说，华为就像这架千疮白孔的飞机，过去一年，补洞是我们的主旋律，也让我们皮糙肉厚了。好消息是，我们现在还活着。 ”
  <br/>
  <br/>
  在美国一轮又一轮的施压下，华为活得如何？郭平说，华为去年营收达人民币8588亿元，净利则达到人民币627亿元。
  <br/>
  <br/>
  另一方面，在美国五度延长临时许可期间，华为会采取双保险措施。
  <br/>
  <br/>
  郭平说，华为去年一方面扩大投入研发资金达人民币1317亿元，增长近3成；另一方面扩大采购强化库存，金额达人民币1674亿元，增长逾7成。
  <br/>
  <br/>
  华盛顿智库传统基金会亚洲研究中心专研经济与科技的资深政策分析师沃尔特斯（Riley Walters）告诉自由亚洲电台，风口浪尖上的华为短期会受伤，但在5G设备发展上，美国的做法只能拖慢华为，但不能致华为于死地。
  <br/>
  <br/>
  “我想华为是可以活下来的，原因有二。一是华为在美国之前不断展延临时许可期间扩大采购，强化库存；二是在美国压力下，这会警醒中国，一定要以举国之力扶植自己的半导体产业。”
  <br/>
  <br/>
  沃尔特斯还说，美国要加强打压华为，全世界半导体产业也不会好过，因为对许多国家与厂商来说，华为是最大的芯片消费者。在美国的国家安全与企业的盈利上要如何取得平衡？他认为，美国与中国之间的科技战，接下来还会经历一段时间角力。
  <br/>
  <br/>
  他也指出，美国的单打独斗，效果也有限。面对华为，各个国家也有不同关切与考量，像是欧洲联盟就仍没有共识。
  <br/>
  <br/>
  《日本经济新闻》引述知情人士报道，华为与总部位于瑞士的义法半导体公司（STMicroelectronics）合作，开发旗下智能手机的芯片。双方从2019年已开始联合开发，但至今尚未公布结果。
 </p>

  <b>
   打压华为玩假的？美学者批特朗普政府重利轻国安
  </b>
  <br/>
  <br/>
  美国企业研究所常驻研究员史剑道（Derek Scissors）则把矛头指向美国自己。他认为，美国总统特朗普政府和前总统奥巴马一样，在和中国进行商业往来时，美国都是“有钱就能大声，音量高过法律与国家安全”。
  <br/>
  <br/>
  他在四点书面评论中告诉自由亚洲电台，美国把华为列入黑名单，却连续五次临时延期临时许可，华为钻的漏洞，是美国自己挖的。
  <br/>
  <br/>
  “对华为的限制，现在没有一个是真正执行的。商务部出台的这个申请许可制，是他们给自己有权限审核，但不代表一定会拒发许可。我预计一百二十天宽限期后，又会跟之前五度延长临时许可一样（再次延长）。”
  <br/>
  <br/>
  史剑道还特别提到，上周五，包括台积电宣布投资美国，商务部又再次延长临时许可，并推出针对华为的申请出口许可制，这和四月底宣布要对中国、俄罗斯与委内瑞拉的实体，针对可军民两用的设备与技术采取出口管制措施都是有关联的。在他看来，商务部的做法会让一些外国军队受惠，“这背后代表着美国看重商业金钱利益，高过法律”。
  <br/>
  <br/>
  美国商务部4月27日宣布出口管制措施，要防止中国、俄罗斯和委内瑞拉的实体，通过民用供应链获取发展武器、军用飞机或监视技术的美国科技，最终利用到和军事相关的用途上。
  <br/>
  <br/>
  史剑道直言，不论多少共和党参议员接连颁布多少法案，想要维护美国的国家安全，到了主导排案审理顺序的参院共和党领袖那里就止步了。议员们对新冠肺炎、人权、国家安全、知识产权盗窃与法治等议题，可以爱怎么说、就怎么说，“反正和中国商业往来的金钱的声量才最大，有最终主导权”。
  <br/>
  <br/>
  美国企业研究所另一位常驻研究员巴菲尔德（Claude Barfield）撰文指出，“华为不是纸老虎”， 帮助养大老虎的美国现在想造笼子，如果是纸糊的，能关得住吗？
 </p>
