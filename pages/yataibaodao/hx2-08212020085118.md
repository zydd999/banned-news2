 <p>
  在自由的台湾，说话、阅读都享有个人的自由。但是在如今的香港，这个自由已经不复存在。于是，美国在台协会（ＡＩＴ）在台北举行了一场“为自由而读”的论坛，声援香港。香港知名的歌曲词作者林夕、香港铜锣湾书店创办人林荣基在论坛上有感而发。林夕说，没有自由，其他都免谈。
  <br/>
  <br/>
  美国在台协会21日邀请港台文化界人士，讨论言论自由和阅读的关系。邀请来自香港的林夕、林荣基，以及台湾大学名誉教授明居正、新议程研究室主持人周奕成对谈。美国在台协会文化官马明远（Luke Martin）表示，台湾与美国共享价值，香港的自由不断地被侵蚀，而台湾拥有自由，才能像今天这样对话，也才能够保护更美好的未来。

 <p>
  马明远强调，自由与思想密不可分，没有民主就没有思想自由，也因为有了思想自由，才能带动言论自由、宗教自由、新闻自由及和平集会自由。很多人说“先香港后台湾”，他希望在台湾的世世代代，持续拥有阅读自由和思想、言论自由。
 </p>
 <p>
  铜锣湾书店创办人林荣基认为，现在的香港人与过去的犹太人很像。中国把香港摧毁，香港人被中国打压、驱逐，很多人被迫离开，但也有人还没决定去留。不过，留在香港必须要付出代价，用肉身挡子弹的作法并不聪明。像香港“壹传媒”集团创办人黎智英，连他的家人也被以违反香港国安法逮捕。留在香港不是被打死、被自杀、被关起来，就是“被送中”，这些都已经发生。

   林荣基：饿死4千万人共产党都没倒
  </b>
  <br/>
  <br/>
  即便现在盛传习近平和李克强内斗正酣，还要应付洪水、粮荒，面临严峻经济民生问题，以及国际社会抵制北京、香港面临失去国际金融中心的地位，林荣基认为，共产党不可能放过清算香港抗争者。
  <br/>
  <br/>
  林荣基以过去中国在1950年代和1960年代发生的大饥荒为例：“中国发生一个叫大饥荒，饿死了四千多万人，等于是说当时他们经济崩毁。你想一想，四千多万人，等于把台湾两千三百多万人加香港七百多万人，全部饿死，他的政权都没有受到影响。”

 <p>
  林夕说，他生来对自由有过敏症候群，没有自由，其他都免谈。而创作跟自由有绝对的关系。自由跟阅读本身就是互为因果，因为会阅读，才能保持自己拥有被别人抢不走的灵魂、信仰和信念。阅读自由好像呼吸的自由。当香港人感受呼吸开始窒息，已为时已晚 。
  <br/>
  <br/>
  创作华文歌曲歌词逾4千首的林夕提到，如果他是一个没有信念，很容易下跪的人，又何必成为他的粉丝呢？
  <br/>
  <br/>
  林夕传授他读书的方法，他举例孔子学院的傀儡“孔子”这类“权威作者”所著的《论语》等等，他会当是不是“禁书”来自我审查，他不“拜读”，也不“捧读”，他是抱着一个这作者在说什么“五四三”？跟它“对着干”。尤其他提到，所谓政权下建构的“历史书”是最危险的阅读，怀疑每一种观点，才能捍卫思想独立。
