 <p>
  美国国务卿蓬佩奥（Mike Pompeo）周三在捷克国会发表演说，再次批评中国共产党输出自身意识形态对自由社会造成威胁，并表示全球应团结反共；同一时间，台湾的总统蔡英文则在美国智库发表演说，她呼应蓬佩奥捍卫自由民主的努力，声援香港，也以台湾的发展为例强调，民主并不是专属于西方的价值观。
  <br/>
  <br/>
  从捷克国歌唱到美国国歌，捷克参议院周三迎来到访的蓬佩奥发表演说，这位美国最高外交官除了提醒捷克朋友注意俄罗斯试图篡改历史的行为之外，他的十四分钟演说更多聚焦在中国共产党在全世界发动的恐吓式运动、打压自由并企图输出中共影响力。
  <br/>
  <br/>
  他比喻，极权专制政权像暴风雨一样仍未过去，要为这类政权写下墓志铭，需要世界团结行动。蓬佩奥形容，中共的马克思、列宁式的意识形态很顽强，更甚苏联，他更直言批评中共试图控制捷克内政的霸道行径。
 </p>
 <p>
  “现在，他们（中共）不是用坦克与枪枝，而是用经济实力扼杀自由，中共习惯性地报复无辜的人，北京取消了布拉格爱乐乐团到中国的演出计划，只因为布拉格市长贺瑞普（Zdeněk Hřib）努力想深化与台湾的关系；上海取消与布拉格的合作，只因为布拉格与台北缔结姐妹城市。”
 </p>
 <p>
 </p>
 <p>
  <br/>
  <b>
   蓬佩奥声援捷克 批评中共阻止与台交往
  </b>
  <br/>
  <br/>
  蓬佩奥还提到已故的捷克前参议院议长柯加洛（Jaroslav Kubera）。他说，中共驻捷克大使馆知道柯加洛计划访台后，就寄出威胁信，是捷克自由的媒体揭露此事；他更肯定现任议长维特齐（Miloš Vystrčil）这个月底将完成前议长遗愿，访问台湾，他还特别向维特齐说，“这是件好事啊”！
  <br/>
  <br/>
  柯加洛原订今年二月访台，却于一月二十日猝逝。柯加洛的家人后来告诉捷克媒体，他生前接到中国使馆的威胁信，承受沉重压力、身心俱疲。
 </p>
 <p>
  蓬佩奥在演讲中也提到中共在香港、新疆及西藏侵犯人权与迫害自由的行为，他还点名中国电信设备大厂华为是中共帮凶，监控打压中国人民，让秘密警察的幽灵重现。
 </p>

  <b>
   美中非新冷战 中共难对付但非铜墙铁壁
  </b>
  <br/>
  <br/>
  他多次提到前捷克总统哈维尔当年领导丝绒革命的勇气，让捷克在“苏东波民主化”浪潮下终结共产专制政权。蓬佩奥说，现在局势不是新冷战，因为苏联当年是与自由世界壁垒分明，然而，中共已经借由经济发展和国际社会紧密相连，现在对抗中共，确实更困难，也不预期北京在短时间内会改变路线。
 </p>
 <p>
  但他说，就像自己当年派驻北约、作为一名驻守西德的美国大兵一样，在推倒柏林围墙前，没人想到铁幕真会倒塌，“潮流已经改变”，世界团结抵抗中共的压迫，自由社会终将胜利，这也是哈维尔当年所追求的。
  <br/>
  <br/>
  蓬佩奥11日至15日访问捷克、斯洛文尼亚、奥地利及波兰等四国，在这些曾经深受纳粹德国或共产铁幕之害的国家寻求对美国全面对抗中共战略的支持。
 </p>

 <p>
  <b>
   蔡英文忧香港局势 呼应华盛顿团结抗中路线
  </b>
 </p>
 <p>
  同一时间，蔡英文透过事前录制的视频短片，在华盛顿智库哈德逊研究所的研讨会上致词，呼应蓬佩奥的说法，她特别担忧香港局势。
  <br/>
  <br/>
  蔡英文指出，疫情带来的区域政治效应令人忧心。当世界正忙着对抗疫情，自由、民主社会却面临威胁升高，台湾和香港作为华人社会享有自由民主的地方，台湾香港现在的遭遇，特别忧心。
 </p>
 <p>
  蔡英文说，“看到北京背弃对香港人民与国际社会的承诺，摧毁香港这座自由的灯塔，让我们既悲伤又有所警惕......我们深感国际社会有责任发声，并且采取行动，阻止香港自由消逝。我非常肯定美国、英国及其它民主国家的作为，也呼吁更多理念相近国家勇敢站出来。”
 </p>
 <p>
  她还表示，台湾会继续提供香港民众人道援助，但也会以更审慎的态度，处理两岸政策，在维持两岸和平稳定的同时，捍卫台湾的民主自由。
  <br/>
  <br/>
  蔡英文重申，她会以“和平、对等、民主、对话”四原则，维持两岸互动，更强调台湾人民有权利决定自己的未来，这和北京的立场大不相同。
  <br/>
  <br/>
  她说，“台湾人民对我们的民主成就深感自豪，台湾向世界证明，民主并非专属于西方社会的价值观”。
 </p>

  <b>
   台湾国际参与 萧美琴：中国更开放才有助缓解对立
  </b>
  <br/>
  <br/>
  随后参与研讨会问答的台湾驻美代表萧美琴，则详细阐述蔡英文在第二任总统任期，对深化台美关系的愿景。
  <br/>
  <br/>
  萧美琴说，在台湾的国际参与上，她呼吁中国应该抱持“更开放的态度”，才有助于缓解对立态势。
  <br/>
  <br/>
  在台美安全合作上，她则指出，台湾希望借由与美国合作，更加提升台湾军力现代化；在区域与全球议题合作上，尤其这次疫情凸显台湾采取公开透明的防疫方式取得的成效，台湾愿意与国际社会分享经验。
  <br/>
  <br/>
  蓬佩奥呼吁世界要携手合作，必须挺直腰杆、对中国共产党的打压要勇于直言；而与中国仅隔着一道海峡的台湾，则呼应美国，在抗拒中国压迫并吞方面，台湾经验丰富。
