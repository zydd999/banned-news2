 <p>
  《环球时报》总编辑胡锡进在推特发文警告美国，假如不再为中国记者续发工作簽证，届时中方将会采取反制措施，目标包括不再发签证给驻香港的美国记者。
  <br/>
  <br/>
  中共喉舌《环球时报》总编辑胡锡进本周二（4日）在社交平台推特发文表示，假如美方不再为中国记者续发工作簽证，中方已为最坏的情况做好准备，就是所有中国记者都需要离开美国。届时中方将会采取反制措施，目标包括派驻香港的美国记者。
 </p>

 <p>
  北京独立学者查建国认为，胡锡进所说的中方对美采取反制措施，应是停发美国驻华记者的工作签证。他本周二（8日）对自由亚洲电台说，中美两国目前不至于走到停发对方所有记者签证的地步：“减少一部分，还是全部不让进入美国，这个可能还要继续看。可能还走不到所有的记者全部驱逐或者不准进美国了这种地步吧。中国一般现在都是后发制人，对等回应。美国如果全部驱逐（记者），中国肯定要全部驱逐（美国记者）的。”
 </p>
 <p>
  <b>
   胡锡进曝中国在美记者未获签证
  </b>
  <br/>
  <br/>
  互联网周二盛传美国移民局已拒绝全部中国驻美国记者的工作簽证，造成中国外交官和“大外宣”部门措手不及。不过，相关消息有待美国官员证实。周二上午，环球网发表胡锡进的评论，他说，鉴于美方至今拒绝给中国记者续签，中方做好了从美国被迫撤出全部记者的最坏准备，并将猛烈报复。 他说，在驱逐60多名中国记者、无差别将所有中国驻美记者签证缩短为3个月后，美国很可能通过不予签证延期，导致更多中国记者不得不离开美国。
 </p>

  清华大学政治学系前讲师吴强接受本台采访时表示，胡锡进近期的激进言论与中国外交部发言人的表态比较，显得不再靠谱：“具体来看胡锡进的言论包括他此前对（中国）核弹数量的言论，都相当不靠谱。更多程度是雷声大雨点小。（中美）双方都没有意愿，也没有准备迅速脱钩。”
  <br/>
  <br/>
  胡锡进说，据他了解，虽然在美国的中方记者都已按美方要求提交申请，但尚无一人获得延期签证，近40位记者甚至迄今没有收到办理签证延期手续的通知。
  <br/>
  <br/>
  <b>
   美国驻华大使馆拍卖二手货引猜测
  </b>
  <br/>
  <br/>
  山西大学法律学者宋扬对本台说，假如美国以不再续签的方式迫使中国记者离境，假如中国作出报复，那么美中的关系将进一步恶化，可见美国也在进行相关的准备：“结合美国驻华使馆拍卖一些物品，这实际上也是在为全面脱钩作准备。两国不说断交，至少减少美国驻华使馆的工作人员，看起来也是在准备之中，到时候说撤就撤了。”
 </p>

 <p>
  <br/>
  <br/>
  苏州中学前教师潘露认为，《环球时报》总编辑胡锡进眼中的中国驻美记者，不同于普通记者，他们往往带着特殊使命，在美国从事与记者身份不相符的工作。他对本台说：“美国记者绝大多都是自由媒体的记者，不属于受美国政府控制的独立记者；而几乎所有中国去美国的记者其实都是中共记者，属于大外宣，甚至是具有经济或军事间谍的身份的人。”
  <br/>
  <br/>
  美国政府于3月初宣布将大幅削减5家驻美中国官方媒体的中国籍雇员总数上限，从160人调低至100人；名额缩减后，有60名中国籍员工将失去签证，他们如在8月6日签找不到新雇主，就必须离开美国。受影响的5家驻美中国媒体是新华社、中国环球电视网（CGTN）、中国国际广播电台、《中国日报》发行公司和《人民日报》海外版的美国总代理海天发展公司。其后，美国公布新规定，缩短中国记者的簽证有效期至90天，允许申请续期，但新规定不适用于港澳记者。
  <br/>
  <br/>
  另外，本周二，美国驻北京大使馆在顺义区大使馆库房举办密封投标式二手物品拍卖会。卖物品包括住宅家具、电器、电脑，其他有破损物品。当天下午，大批北京市民开车抵达顺义天竺空港工业区天柱路18号，排队等候参加拍卖会。
  <br/>
  <br/>
  舆论认为，美国驻华大使馆此刻买卖二手物品，引发人们对美中关系持续紧张的联想。
