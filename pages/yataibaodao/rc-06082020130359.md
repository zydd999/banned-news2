 <p>
  北约秘书长斯托尔滕贝格（Jens Stoltenberg）6月8日说，中国崛起将从根本上改变全球的势力均衡状态，新冠肺炎疫情也让北约警醒，中国与俄国携手的假信息宣传行动，试图分化北约同盟，北约成员国必须更加团结，并与理念相近的伙伴合作，应对中国威胁。
 </p>
 <p>
  没有二战后的苏联势力横扫欧洲，就不会有北大西洋公约组织（NATO）的成立。 七十多年过去，如今的俄罗斯和中国再次引起北约的警惕。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  斯托尔滕贝格在北约总部所在地的布鲁塞尔，周一连线参与华盛顿智库大西洋理事会（Atlantic Council）主办的活动，他除谈到传统上俄罗斯的军事威胁，也谈到北约对中国与莫斯科携手，在疫情大流行期间，通过假信息宣传战，离间分化北约成员都看在眼里，也警惕在心。
  <br/>
  <br/>
  斯托尔滕贝格说，“中国崛起是从根本上改变了全球原本的均衡态势，激化经济竞争与科技霸权，让开放社会与个人自由更受威胁，对我们的价值观与生活方式构成更多竞争与挑战。因此，我们今天提出“北约2030”（NATO 2030），就是要应对这样的'新常态'。”
 </p>
 <p>
 </p>

 <p>
  <b>
   建欧亚同盟 北约将与日韩澳纽加强合作
  </b>
  <br/>
  <br/>
  中国的“以疫谋私”让北约警醒，但斯托尔滕贝格强调，北约并没有把中国视作“新的敌人或对手”。而应对中俄在疫情上的假信息宣传战，他更强调了自由与独立媒体的重要性，“我们只能用更多事实、战胜假信息，而这需要独立且自由的新闻媒体”。
  <br/>
  <br/>
  作为全世界规模最大的军事防卫合作多边组织之一，北约是为应对苏联的红色势力而生，在冷战时代为防卫欧洲的民主有积极贡献。疫情改变了世界，斯托尔滕贝格说，北约在未来十年，要应对中国崛起的威胁，也必须与时俱进，北约必须“军事上更强大、政治上更团结，并以全球性地更广泛的方式”应对，北约未来十年将强化与亚太国家如日本、韩国、新西兰与澳大利亚的合作。
  <br/>
  <br/>
  他说，“事实是，中国很快将成为世界第一大经济体，他们是世界上国防预算第二高的国家，他们投入大量资源在发展军事现代化，这包括了针对北约传统盟国的导弹系统。他们在网络上的发展也逐步进逼，我们也看到中国在北极圈与非洲大量的基础设施建设的投资，并且与俄罗斯更紧密的合作。这些都会对北约成员国造成安全上的后果。”
 </p>
 <p>
  <b>
   北约七十年团结受考验 中俄趁机搅浑水？
  </b>
 </p>
 <p>
  这是北约在2019年成立七十周年时、首次将中国的威胁写入北约峰会的声明中后，更加完整阐述北约眼中的中俄威胁。不过，目前有30个成员国的北约，作为二战后历史悠久的多边组织之一，也面临内部团结的窘境。
  <br/>
  <br/>
  1949年，英国、法国、美国、加拿大及其它八个西欧国家，在华盛顿签署了《北大西洋公约》，创建了北约；去年，北约伦敦峰会庆祝成立70周年前夕，法国总统马克龙不但直言北约“已脑死”，更称“俄国和中国已不再是敌人”。
  <br/>
  <br/>
  另外，美国总统特朗普上任以来，就对多边组织不满意，他尤其对美国分担多数经费的北约持批评态度，认为对美国不公，北约长期占美国好处。路透社日前报导，特朗普将下令从德国撤军，在德国设有基地的美国，将减少9500名军人，驻德美军规模将减少至2万5000人。
  <br/>
  <br/>
  此前，德国总理默克尔也拒绝特朗普提出6月底到华盛顿召开七大工业国集团（G7）峰会的邀请。
  <br/>
  <br/>
  斯托尔滕贝格在会上缓颊指出，来自挪威的他，仍看到美国对北约的坚定支持：“在挪威，我们看到更多的美国军力的存在”。
  <br/>
  <br/>
  这位前挪威总理强调，欧洲联盟（EU）不能把和平局势视为理所当然，应对中、俄的挑战与威胁，不是北约、欧洲单一国家或美国可以独自应对的。
  <br/>
  <br/>
  他说，在太空、网路、新科技、全球军备控制上，最终都需要开放与自由的社会携手，为世界的自由与民主挺身而出，对抗霸凌与胁迫，“下个十年，我们面对的挑战，哪一国都无法独善其身，挑战更大。”
 </p>
