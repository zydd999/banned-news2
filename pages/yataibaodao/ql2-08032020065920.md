 <p>
  上一个月，中国全国网信办受理网络违法和不良信息举报近1500万件，同比上升超过两成。另外，湖北、四川及新疆，近期有网民因在微信发言，受到行政拘留处罚。
  <br/>
  <br/>
  中国国家互联网信息办公室违法和不良信息举报中心公布的统计数据显示，今年7月，全国各级网络举报部门受理举报1454.9万件，环比增长2.5%、同比增长21.9%。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  中国官方7月下旬曾公布，8月1日起，将集中整治商业网站平台和“自媒体”突出问题，包括片面追逐商业利益，为吸引“眼球”炒作热点话题、违规采编发布互联网新闻信息、散播虚假信息、搞“标题党”等。对于当局公布受理违法和不良信息举报近1500万件。
 </p>
 <p>
 </p>

  <br/>
  河北独立学者吴文浩对本台说，最近半年，在网络上批评中国政府的人，明显减少，如果说造谣，当属有官方背景的人士，而且是在海外说谎：“中国的官方账号无论是在推特，还是在国外其他社交媒体，基本上是假消息的重要来源。中国政府是一个独裁专制的政府。他们基本上是靠假消息洗脑，用虚假的信息和手段来统治。其实他们就是假消息的最初发源地。”
  <br/>
  <b>
   <br/>
   中国全国主要网站受理举报案增36%
  </b>
  <br/>
  <br/>
  新华社本周日引述网信办统计，今年7月，中央网信办违法和不良信息举报中心受理举报9.5万件；各地网信办举报部门受理举报145.7万件；全国主要网站受理举报1299.7万件，同比增长35.8%。
  <br/>
  <br/>
  报道指，在全国主要网站受理的举报中，微博、百度、阿里巴巴、腾讯、今日头条、新浪网等主要商业网站受理量占82.5%，达1072.1万件。
  <br/>
  <br/>
  网络活跃人士朱女士对本台说，被官方调查的网民多，代表不满政府的人越来越多：“发声的人越来越多，反对共产党的人越来越多。他们（政府）不停的在封杀言论自由，他打压言论自由，但是越来越多的人觉醒，这倒是真的。”
  <br/>
  <br/>
  另外，湖北襄樊一男子被指在网络散布谣言称“武汉又有新型病毒”，遭行政拘留。“鄂州公安在线”微信公众号8月2日消息，7月30日，有网民在某微博上发帖称：“又有一种新型病毒在武汉传播，现已死亡十几人，大家千万别到武汉来。”对此，鄂州警方于8月1日将违法人员周某（男，41岁，襄阳市人，在鄂州居住）抓获，周某如实交待了其散布谣言的违法行为。目前，周某已被依法裁决行政拘留7日。
 </p>

  <b>
   组微信群通风报信骂警被捕
  </b>
  <br/>
  <br/>
  据湖南省汉寿县公安局微博消息，7月31日，当地公安局联合侦破查处一起利用微信群对抗城区综合秩序整治案。邓某某、刘某某等5人,通过微信群联络近200名摩托车司机，对官方执法车辆号牌进行暗地拍照发至群里等方式,提前通风报信。邓某某、刘某某等人还多次谩骂执法人员，被行政拘留。
 </p>
 <p>
  网络评论人士宋先生对本台说，现在很多网民都不敢批评政府：“现在网络（管控）是非常严厉，没看到我们都不怎么说话了，也不能随便说。一，可说你造谣，二，可说你寻衅滋事，再有就是扰乱公共秩序。这几个罪名都够我们喝一壶的。现在尽量不说话，等逼急了再说。”
  <br/>
  <br/>
  宋先生感叹道，人长了嘴巴就是用来说话的：“可是人长了嘴巴除了吃饭就是说话，说话是为了捍卫人权，不让人说话了，这个时候就乱套了。”
  <br/>
  <b>
   <br/>
   新疆一维两汉人“传谣”被拘留
  </b>
  <br/>
  <br/>
  据中国官媒报道，7月25日，新疆库尔勒居民阿某、周某及王某在微信群组传播“库尔勒市封城”“哄抢物资”等谣言信息。经与库尔勒市防疫指挥部确认，上述言论为谣言。上述三人被处以行政拘留三日。
