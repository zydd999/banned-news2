 <p>
  中国农业银行、建设银行等国有银行正在测试数字货币的运行。深圳已有超级市场自本周一起鼓励顾客使用电子支付。多位经济学者认为，中国模式的数字货币，实际上是使中央得以全面管控个人财富，也就是回到计划经济年代。
  <br/>
  <br/>
  中国商务部8月14日发布了《关于印发全面深化服务贸易创新发展试点总体方案的通知》，其中公布了数字人民币试点地区。其中提到的数字货币试点城市为深圳、苏州、雄安新区、成都及未来位于河北的冬奥会场。根据中国央视新闻报道，全面深化试点地区为北京、天津、上海、重庆涪陵区等21个市辖区、海南、大连、厦门、青岛、深圳等28个省市。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  <b>
   数字货币有利政府防止逃税漏税
  </b>
  <br/>
  <br/>
  山东大学金融系毕业的学者司令，本周一（17日）对自由亚洲电台说：“中国如此大力推行数字货币，他想更加清晰化组织他的税收财政收入。当前中美关系以及中国外贸形势严重恶化的情况下，他会大力的主抓财政收入，此前很多人使用现金交易来规避税收征管，以此逃税。”
 </p>

  <br/>
  官方的宣传强调，使用数字货币对民众而言“差别不大”，显然是敦促民众接受这一观念。山西商人董永琪对本台说，中国民众一旦使用数字货币，个人利益与隐私都受到侵害：“对于老百姓来说，肯定是弊大于利。大多数人看宣传，并不了解数字货币对于个人隐私的侵犯。所以数字货币就是回到票据时代，回到计划经济之前的准备。”
  <br/>
  <b>
   <br/>
   深圳超市开始限制民众使用现钞
  </b>
  <br/>
  <br/>
  在深圳，部分超市也被要求使用电子支付。一家超市要求顾客尽量使用支付宝或微信支付，如果使用人民币，必须登记个人姓名及银行账号。董永琪说：“在深圳的永辉超市已经出现使用现金结账需要登记。”
  <br/>
  <br/>
  7月20日，日本共同社报道，日美欧七国集团(G7)基本决定就发行央行数字货币展开合作，并拟于8月底在美国举行首脑会议。
  <br/>
  <br/>
  对此，董永琪说，中国的数字货币与西方民主国家的有本质上的区别。他说：“关于民主国家、自由世界的数字货币和我们的数字货币有根本区别，因为自由国家的数字货币就是区块链的性质，是去中心化的，但是我们国家的数字货币是有中心化的就是央行来负责，它是截然不同的两个概念。”
 </p>

  <b>
   西方国家使用数字货币时着重个人私隐
  </b>
  <br/>
  <br/>
  中国经济学者胡星斗对本台说，中国的所谓数字货币，并非真正意义上的数字货币：“它应当被称之为电子货币，它就是货币的电子化。它与数字货币的私密性，可监督可回溯等等具有很大的差别，也就是说数字货币具有个人私密性，其他人是不能够知道的，甚至政府也不能够知道，数字货币只有参与的本人才能知道。”
  <br/>
  <br/>
  海外自媒体“财经冷眼”评论说，中国的数字货币就是为计划经济作准备！其特征是对货币使用和物资分配的控制。可以把数字货币理解为数字时代的粮票、肉票、出门通行证、交通旅行介绍信、大型购买许可证。
  <br/>
  <br/>
  金融学者司令认为，中国政府倘若全面推行数字货币：“交易将完全置于政府的监控之下，有利于政府收入增长。如果数字货币大力推行，有可能是21世纪的公私合营，也就说国民的财富，如果政府失去信用，他完全可以一夜之间变成公有制。”
  <br/>
  <br/>
  司令认为，中国央行作为数字货币中心，民众的货币交易受到监控及追朔。同时防范资本外逃，换汇及金屋藏钱。
