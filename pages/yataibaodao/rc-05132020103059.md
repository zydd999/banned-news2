 <p>
  中国电信设备龙头企业华为遭美国全面制裁后，创办人任正非近来放软身段，称要向美国学习。另一方面，华为近来在美国试图改变形象，本想举办探讨打击假信息的研讨活动，却遭合作方取消。华为在美国究竟砸了多少公关游说费用？为什么难见效果？
  <br/>
  <br/>
  美国知名嘻哈歌手小威廉·詹姆斯·亚当斯（Will.i.am）和有线电视新闻网（CNN）评论员琼斯（Van Jones）这两位非裔美国人社群中的指标人物要在5月13日一起谈“假信息的兴起”。华为美国分公司11日在其官方推特上公布了这个消息后，引起各方注意。但是没想到，活动公布后24小时内，旋即取消。
 </p>

  协办活动的美国全国非裔记者协会（National Association of Black Journalists, NABJ）主席塔克（Dorothy Tucker）回复自由亚洲电台的查询时仅指出，协会12号已公布相关声明，没有多做回应。
  <br/>
  <br/>
  这个记者协会在声明一开始表明，活动取消原因是“有争议的科技巨头华为计划赞助这次网络研讨会”，让原本想聚焦关注疫情期间非裔美国人受到的假信息影响失焦。
  <br/>
  <br/>
  声明强调，协会没签署任何协议，也没有接受过华为的赞助。
 </p>
 <p>
  <div class="image-inline captioned" style="width:680px;">
   <div style="width:680px;">

  <br/>
  <br/>
  华为在美国的分公司为了改变华为在美国的形象，以及改变美国政府的政策，花了大力气进行公关和游说活动。这个被取消的活动就是这种公关行为的一部分。华为在美国游说砸了多少钱？自从华为财务长孟晚舟在加拿大被抓后，华为在美国的游说金额，明显激增，而华为在美国的代理人也更谨慎地为这位顾客提供服务。
  <br/>
  <br/>
  两家美国公关公司锐思博德（Racepoint Global）与博尔森、科恩和沃尔夫（Burson Cohn and Wolfe, BCW）去年开始根据《外国代理人登记法》（FARA），向美国司法部注册代理华为的整合行销业务。
  <br/>
  <br/>
  《外国代理人登记法》是针对外国政府或实体，华为在美国除原本就按照《游说披露法》（Lobbying Disclousure Act, LDA）向国会申报外，去年开始，还按照《外国代理人登记法》登记注册，这也凸显任正非一再宣称“华为是私营公司”的矛盾说法。
 </p>
 <p>
 </p>
 <div>
 </div>
 <p>
  <br/>
  <b>
   在美游说烧钱 华为仍遭冷遇
  </b>
  <br/>
  <br/>
  长期关注政治游说活动的组织负责任政治中心（The Center for Responsive Politics）的资料显示，华为2018年在美游说费用登记仅十六万五千美元，到2019年，爆增到至少近三百万美元。今年前五个月，则已登记达十八万美元。
  <br/>
  <br/>
  截至发稿，华为在美的两家公关公司及华为美国公司都没有回复自由亚洲电台的询问。
  <br/>
  <br/>
  华为砸重金搞公关，但想办一场对抗假信息的活动，都难以顺利举行，遭到美国舆论严厉检视。前白宫副国安顾问麦克法兰（KT McFarland）12日在一场讨论特朗普连任的外交政策活动上，直言表明美国对华为与中国在科技企业发展上的“军民融合策略”的本质怀疑。
  <br/>
  <br/>
  “你要让华为来主导5G科技与通讯基础设施的建设吗？中国的计划是要主导世界的5G网络。果真如此的话，那这就代表中国可以听到我们每一个对话、每一通电话，看到我们的每一个短信和电邮往来。”
  <br/>
  <br/>
  自从孟晚舟遭美国指控违反美国法律，与伊朗进行交易，在加拿大提出引渡要求后，孟晚舟至今被困在加拿大等待引渡审判。另一方面，美国也对华为展开全面封杀。
 </p>

  <br/>
  <b>
   手机软件5G设备到核心芯片 美国致华为于死地？
  </b>
  <br/>
  <br/>
  日前传出美国可能进一步限制台湾晶圆代工龙头企业台积电出货给华为。今年二月以来，媒体纷纷报道，美国要求外国企业出口产品或服务到中国，含有的美国技术的比例需下修至10%。
  <br/>
  <br/>
  《华尔街日报》10日更报道，美国基于国防安全考量，频频要求台积电赴美设厂。另外，白宫还点名英特尔及三星也要将工厂搬到美国。
  <br/>
  <br/>
  在此之前，美国商务部则已下令，禁止华为手机操作系统使用美国谷歌的安卓（Android）；在硬件出口管制上，商务部更将华为列在禁售令实体清单上。美国总统特朗普去年五月签署行政命令，宣布紧急状态，禁止美国企业使用构成国家安全风险的电信设备，都是针对华为而来。
  <br/>
  <br/>
  没有美国软、硬件的供货，华为越来越寸步难行。
  <br/>
  <br/>
  然而，《路透社》6日引述消息人士报导，美国商务部正起草相关法规，允许美国企业与华为合作，共同为5G网路制定标准。
  <br/>
  <br/>
  去年五月华为遭美国商务部列入黑名单后，一些美国科技业与工程师随即停止与华为合作开发5G网路标准，美国业界持续有呼吁，联邦政府应在5G标准上采取更积极行动，不能让华为垄断5G标准的制定。共和党籍参议员鲁比奥（Marco Rubio）等人于4月致信商务、国防和能源部，强调美国非常需要透过相关法规，厘清美国企业什么能做，什么不能做，避免限制美国参与5G标准制定过程，信中也表达美国在全球5G技术领导地位的忧虑。
  <br/>
  <br/>
  科技数据顾问公司Strider创办人雷夫斯克（Greg Levesque）就说，美国的两难是：“在5G上，我们看到美国不断游说欧洲盟邦，不要使用华为的设备；但另一方面，我们确实在5G科技与设备的开发投资不足。我们游说盟邦不要使用中国产品，但我们没有'美国版的替代方案'，只有爱立信（Ericsson）与诺基亚（Nokia）”。
  <br/>
  <br/>
  任正非近来频放风声称，整个华为内部都认真向美国学习，从上到下二十多万员工，听不到一句反美口号，还强调他要求华为员工认真研读美国国防部的文章。任正非说，美国深刻认识到如何打击华为，而他要做的是华为得“理解如何继续把自己变得更好一点”。
 </p>
