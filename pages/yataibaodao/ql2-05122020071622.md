 <p>
  武汉居民杨敏的女儿今年1月中旬在医院感染新冠肺炎，2月上旬离世。近期，杨敏身上背着写有“冤”字的牌子及手持女儿的遗照，要求政府公开疫情真相，还其女儿一个公道，结果本周二她遭到武汉当局的软禁。
  <br/>
  <br/>
  杨敏近期在武汉街头表达诉求，要求政府公开新冠肺炎疫情真相。她说自己是一个年近半百的普通武汉市民,一位新冠肺炎的幸存者，更是痛失24岁爱女的母亲。本周一（11日），杨敏背着一个写有“冤”字的纸牌，手捧女儿遗像，在武汉街头静坐，并展示写有“政府隐瞒疫情真相”、“还我女儿”的纸板，但纸板很快被保安抢走。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  杨敏周二又被当局软禁，不准外出。她说，今年1月16日，女儿田雨曦到协和医院看病时，不幸感染新冠肺炎，2月6号去世。她批评当局早已知道新冠肺炎会人传人，却不通告给市民。如果早知病毒会传染就不会让女儿去医院，女儿也就不会死。她对本台说，由于她被当局软禁，令她情绪极差：“我的孩子情况你查资料，查我的微博吧，上面都有具体情况。 我不想说话，我现在情绪很不好。”
 </p>
 <p>
 </p>

   死者家属公开伸冤被抢维权材料
  </b>
  <br/>
  <br/>
  在一段视频中，4名男子正在抢夺杨敏的伸冤材料。杨敏作出反击：“ （男）干什么？（杨敏）你抢我的东西就是不行，我是中国人这是我的私有财产，你们凭什么抢东西，你们凭什么......”
  <br/>
  <br/>
  杨敏公开表示，公开资料显示，去年12月31日，武汉市已有27例新冠肺炎的感染病例,政府没有向人们发出警示，李文亮医生因在小范围警示身边的亲戚朋友,今年1月3日被训诫。6日，当局又通过电视发出通告称新冠肺炎“未见人传人”，这一切都向武汉市民传达着错误的信息，让大家并不知道危险已经降临。
  <br/>
  <b>
   <br/>
   杨敏要求追究官员责任向受害者道歉
  </b>
  <br/>
  <br/>
  帮助新冠肺炎受害者家属维权，现旅居美国的公益人士杨占青对本台说，杨敏的女儿在武汉封城之前就医，却被感染新冠病毒：“杨母认为她女儿的死和政府瞒报疫情有很大的关系。她去信访局申诉就是要让政府知道其女儿的死因与隐瞒疫情有关。她主要提出两个诉求，第一追究官员的责任，追究他们犯人类的罪行并且给所有受害者道歉。第二，赔偿精神损失和经济损失。”
 </p>

  <br/>
  正在武汉采访的公民记者张展对本台说，她原本约好与杨敏再次见面，但周二早上在武汉市委门口，未等到杨敏，后来得知杨敏被软禁：“早上，我在市委门口等了她，等了五十分钟，她没有来。她说是被社区的人关在小区大门里面，不允许出来。我早上在武汉市委门口等为女儿申诉的母亲杨敏，市委门口停了一辆警车、一辆巡查车，两个巡警摩托车，车里车外有七八个警察，三、四个交警，两个武警。还有认不清的人员。”
  <br/>
  <br/>
  <b>
   当局必须对疫情给出公正的说法
  </b>
  <br/>
  <br/>
  杨敏提出的诉求，包括追究所有官员瞒报疫情的法律责任及反人类罪行，并公开事实真相，向所有死难者家属公开道歉；赔偿家庭的经济损失和精神损失。她说，到现在为止, 那些在疫情初期欺上瞒下，不作为和乱作为的市政府官员，竟然没有一人被追究，受害者的家属不但没有得到一个公正的说法，反而被告知要在政府人员陪同下，低调地私下处理后事。当局如果没有一个公正的说法，死了的人不会同意，活着的人更不会答应。
