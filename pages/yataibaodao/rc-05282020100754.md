 <p>
  美国众议院5月27日以压倒性票数通过《维吾尔人权法案》，美国国会将送交白宫，总统特朗普签署后就会生效。被以“分裂国家罪”判处无期徒刑的维吾尔学者伊力哈木·土赫提的女儿菊尔告诉本台，她心中仍盼着有奇迹出现，和父亲再相见的那一天到来。
  <br/>
  <br/>
  413票赞成、1票反对，议事槌敲下的这一刻，标志着《维吾尔人权政策法案》在美国联邦众议院也过关。继联邦参议院5月14号无异议通过后，众院在不到两周的时间，也以超过2/3的压倒性多数同意，完成立法程序。
  <br/>
  <br/>
  民主党籍的众议院议长佩洛西（Nancy Pelosi）在投票前，也发言正告中国国家主席习近平。
  <br/>
  <br/>
  佩洛西说：“中国国家主席，我们要告诉你，你也许可以在中国国内告诉这些受迫害的人，他们被遗忘了，但事实不是如此。在美国联邦众议院，在跨党派、跨两院的高度共识下，我们本周展现对《维吾尔人权政策法案》的一致支持，捍卫维吾尔人的权利与尊严，不让他们受中国压迫。”
 </p>

 </p>

 <p>
  国会接下来则将法案送交白宫，白宫收到文本后，特朗普有十个工作日可考虑。一般情况下，国会两院高度共识通过的法案，美国总统不会动用否决权；在国会开议期间，就算特朗普不签署，法案也会在十个工作日后自动生效，成为美国法律。
  <br/>
  <br/>
  法案要求行政部门，在生效一百八十天内及日后至少每年都要提交报告，点名虐待、不人道对待或长期监禁维吾尔人的中国官员与外国人士，并通过冻结他们在美资产、拒绝入境美国、拒发或取消签证等措施，予以制裁。
  <br/>
  <br/>
  法案也要求美国总统谴责中国虐待维吾尔人的行为，并要求北京立即关闭新疆再教育营；国务卿还须向国会提交报告，详述新疆维吾尔自治区人权迫害情况。
  <br/>
  <br/>
  美国国务院27号宣布，认定香港不再享有高度自治，不应适用美国所给予的特殊待遇。同一天，《维吾尔人权政策法案》也在众院压倒性多数同意通过，预计已经高度紧张的美中关系也将因为这部法案更加对峙。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <b>
   法案会对中共造成威慑
  </b>
  <br/>
  <br/>
  世界维吾尔代表大会中国事务部主任伊利夏提告诉本台，维吾尔人的处境，当然不会因为一部法律立刻获得改善，对抗中共的迫害，世界必须团结起来，一棒接一棒，这只是开始。
  <br/>
  <br/>
  “美国作为世界政治的引领者，等于是把维吾尔人权问题以正式法律的形式，进入到他的政治议事日程。这也会让维吾尔人的议题进入到国际政治日程中，对中共政权肯定会造成极大冲击，与一定的威慑力。” 伊利夏提说。
  <br/>
  <br/>
  他指出，中共仍奴役维吾尔人，把接受过所谓“职业训练”的维吾尔人送到各省工厂，当廉价劳动力，这些问题还需要国际社会持续关注。
 </p>
 <p>
  <b>
   菊尔·伊力哈木盼望能回故土看望父亲
  </b>
  <br/>
  <br/>
  目前专注写书，要为新疆集中营幸存者留下纪录的菊尔·伊力哈木接受本台访问时则说，美国通过一部这样的法律，对她而言，当然是振奋与鼓舞，更重要的是，这不只是维吾尔人面对的压迫，这是全人类面临的人道问题，尤其看到香港自由急速恶化的处境，她形容中共辩解再多，做法却适得其反。
  <br/>
  <br/>
  “虽然他（中共）一直在说是要抓分裂者、极端者，但是，他其实是逼迫人们团结起来去对抗他。我觉得，这就是搬起石头砸自己的脚的行为。 ”
  <br/>
  <br/>
  菊尔·伊力哈木的父亲伊力哈木·土赫提2014年因“分裂国家”罪被判无期徒刑关押至今，亲友仍未能探视。
 </p>

 <p>
  在北京长大、求学的菊尔告诉记者，因为中共的高压严管，过去，在故土的她甚至没能学习维语；直到踏上美国这片自由土壤，她才找到精神上的自我认同，在异地第一次戴头巾、守斋，练习伊斯兰教的礼拜仪式。这种奇特感受再加上乡愁，使她渴望有一天能在故乡的土地上，演练自己在美国学到的维吾尔文化。
  <br/>
  <br/>
  “我巴不得现在就飞回去，如果我可以的话。我从七年前到现在没有一刻是不想飞回去的。谁想一个人、在一个谁都不认识的国家，一个人度过七年？血亲血亲，断了骨头也连着筋，对不对？怎么样都没办法完全割舍。”
  <br/>
  <br/>
  采访时，菊尔一声长叹，透露她对父亲的思念与期盼再相见：“我真的很希望中国政府能变，虽然很多人会觉得这个希望很渺茫！”
 </p>
