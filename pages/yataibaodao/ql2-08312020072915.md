 <p>
  河南省固始县民族宗教事务局不久前发出通告，警告民众不要受“宗教组织蒙骗、误入私设聚会点、参加非法宗教活动”，并奖励举报者告发。另外，该县一家庭教会举办夏令营活动，最终电脑及投影仪等教会用品被公查抄。
  <br/>
  <br/>
  中国各地政府为打击基督教活动不遗余力，并推出土政策，以现金奖励的方式查处家庭教会。固始县民族宗教事务局8月中旬发出《关于举报非法宗教活动的通告》，称受新冠肺炎疫情影响，该县基督教活动场所关闭近5个月，期间部分信教群众受 宗教组织蒙骗、误入私设聚会点、参加非法宗教活动，对全县正常宗教活动秩序造成较大负面影响。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <b>
   固始县民宗局悬赏举报人500元
  </b>
 </p>
 <p>
  通告称，根据《宗教事务条例》第四十条、第四十一条相关规定，8月20至9月20日，要清查宗教活动场所是否获批准，若不是，则收集图片、视频、声音等资料作为证据，向宗教部门或所在乡镇办事处举报。举报人可得到500元现金奖励。
 </p>

  成都独立作家谭作人本周一接受自由亚洲电台采访时说，四川及成都未见上述规定，可见河南固始县官方新措施是一项“土政策”：“应该是一个‘土政策’，河南省这个地方历来比较左倾，可能是‘左’已经成了风气。以前出现卖血村，很多人患艾滋病的时候，他们怎么没有人去举报？他们这个举报是有选择性的。特别是在亲戚朋友之间举报，道不道德，有没有破坏中国传统文化，目前还不讲这点。”
  <br/>
  <br/>
  固始县信合大道三中北校区一家庭教会近期被人举报，指其从事非法宗教活动，遭到官方人员取缔。该教会负责人王先生对本台说，事发8月23日上午，当地宗教局、公安、城管及街道办等部门近四十人查抄他所在的教会，当时有十多位信徒带着他们的孩子参加学习：“我们是夏令营，还是学习乐理知识，学习钢琴、吉他，最后一个礼拜，我们请这些孩子还有他们的家长在一起搞结业典礼，就被当地三自教会的牧师举报，说我们是非法聚会，把我们的东西都拿走了。”
  <br/>
  <br/>
  <b>
   国保宗教局等多部门联合查封
  </b>
  <br/>
  <br/>
  王先生说，此次带队的是县民宗局，执法人员未有出示法律文件，就把电脑、投影仪等教会用品拉走：“人也不多，不到十个孩子，还有几个孩子家长。国保、宗教局、教育局、文化局，还有城管，三十多人，不到四十人，说我们非法聚会，把我们的音响，桌子板凳椅子，架子鼓，东西全部拉走了。他们把我妻子也带走，他们就拉东西，没给清单，也没有穿警服，也没有出示警官证，宗教局的人也没出示证件，没法律手续，就说我们违法。”

  王先生说，他所在的教会成立六年左右，信徒均为当地居民，他认为家庭聚会并没有违法：“为什么说我们是非法聚会，你说我非法，你没有证件和文件，宗教法说宗教自由，非法不非法不是他说了算。”
  <br/>
  <br/>
  王先生说，周边的教会也曾有过类似遭遇，只是怕事，息事宁人：“其他教会不吭气，怎么会没有。三自教会特别多，乡下十多家三自教会建的教堂花了几百万元，照样不让你开门，谁要是送礼了，你偷偷摸摸开，他们不管你。”
  <br/>
  <br/>
  河南是继浙江之后，宗教信仰受到压制的另一个重灾区。2018及2019年，河南全省有七千多个十字架遭到强拆。今年上半年，安徽省有近一千个十字架遭到强拆。
