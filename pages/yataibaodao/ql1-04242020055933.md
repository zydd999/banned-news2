 <p>
  中共总书记习近平本周一（4月20日）在陕西考察期间，在西安交通大学两度提道“西迁精神”与毛泽东50年前所作的批示枫桥经验。有舆论认为，习近平在疫情下重临陕西，既有“安内”震慑之意，也有“攘外”对抗西方之意，并带有掌控香港，威慑台湾之意图。
  <br/>
  <br/>
  在疫情仍未结束之际，习近平于本周一及周四在陕西省考察。根据官媒报道，习近平周三到西安交通大学西迁博物馆，并向师生发表讲话，两度强调要发扬当年的“西迁精神”，他亦提到当前的疫情，形容疫情为中国带来机遇 ：“大的历史进步都是在一些重大的灾难之后，我们这个民族就是这样在艰难困苦中历练、成长起来的。我也完全相信我们的交大人、我们西部的同志们，一定会在未来的新时代的历史进程中，为中华民族立下卓越的贡献。”
 </p>

  <br/>
  西安交通大学前身是上海交通大学，1955年由上海迁往西安，而上海仅保留一个系，该系发展成为今天的上海交通大学。习近平又前往西北地区最大的制造企业──陕西汽车控股集团，习近平说，国有企业是复工生力军，他又去到西安城内大唐不夜城步行街考察，了解步行街恢复经营情况。
  <br/>
  <br/>
  <b>
   学者指习近平仿效毛泽东遇困难时南下视察
  </b>
  <br/>
  <br/>
  据知情人士对本台披露，习近平原计划4月下旬访问日本，但因日本疫情未解除而取消行程，临时改道赴陕西考察。清华大学政治学系前讲师吴强接受自由亚洲电台采访时说，习近平是在中国面对国际压力之下前往西安，具有战略意义，并且带有毛泽东的斗争策略：“毛主义的一种斗争策略，就好像毛在文革期间或者说从1949年到1976年期间，在中央遇到种种问题的时候，总是选择南下，去湖南或杭州，或去庐山，跟地方要员谈话，联络地方的一些力量。习，我们发现也是采取类似毛主义的手法。”
 </p>

  习近平在陕西省委书记胡和平和省长刘国中陪同下，先后到商洛、安康、西安等地，了解秦岭生态环境保护、脱贫攻坚等情况，对此，吴强说：“在中美关系最紧张的时候，他选择到秦岭应该是类似美国科罗拉多夏延山的这种总统躲藏防御阵地，秦岭其实是中国的夏延山。显示在准备中美关系最糟糕的情况出现，这是他发出很重要的一个很重要的党内战略动员。”
  <br/>
  <br/>
  海外时事评论员文昭在其个人论坛表示，习近平在秦岭特别提到秦岭违章别墅案，2018年中纪委副书记两次去陕西督办，才把这些别墅拆了。其后中共陕西省省委书记赵正永，西安市市委书记魏民洲相继被查办。近期再拿下孙力军（公安部副部长）的同时，习近平再去陕西视察，不是巧合。

  <b>
   分析指习近平国内、外酝酿大动作
  </b>
  <br/>
  <br/>
  文昭分析说，习近平此行似乎是在酝酿大的动作：“在我看来他在清洗公安体系的同时，要出去躲一躲，防止对手暗算反扑，在毛泽东和林彪摊牌的1971年，他坐火车从北京去了湖北，然后又到江西、浙江转了一圈，干啥去？是去见各大军区负责人拿枪杆子，散布对林彪的不满，孤立林彪。毛规习随，所以毛泽东南巡，习近平就西狩（古代皇帝出去打猎）。”
  <br/>
  <br/>
  中国为宣示南海主权，近期接连宣布在海南省三沙市设立2个新的市辖区，同日更出动编队包围美国军舰，又在香港起诉15名民主派人士。对此，文昭说：“习近平清洗公安系统，同时赴陕西是‘安内’，是要压服政敌，有躲避政敌暗算的意识，也有模仿毛泽东南巡的意思，“攘外”这条线就是南海海域的动作和香港最近的异常。这种外围动作最终的目的是台湾。”
