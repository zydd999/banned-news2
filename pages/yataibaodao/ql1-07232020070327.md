  中美关系进一步恶化，在美国政府宣布中方关闭驻休斯敦总领事馆，中国外交部表示将采取反制措施。本周四有关中国可能关闭美国驻华领事馆的消息甚嚣尘上，环球时报的民调显示，66%受访者赞成关闭香港美领馆。
  <br/>
  <br/>
  中美关系进一步恶化。美方周三要求中方在72小时内关闭驻休斯顿领事馆。本周四（23日），中国外交部发言人汪文斌在北京例行记者会上，就前一天美国国务卿蓬佩奥等美国官员指美方关闭中国驻休斯敦总领馆，是针对中方偷窃美知识产权，开展商业间谍活动作出回应。他说：“美方称中国驻休斯敦总领馆从事与身份不符的活动，这完全是恶意的污蔑。美方要求中方关闭驻休斯敦总领馆，严重违反国际关系基本原则，严重违反中美领事条约的有关规定，严重破坏中美关系。这是在拆中美两国人民友好的桥梁。”
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  汪文斌表示，中方必将作出必要反应，维护自身正当权益，但未有提具体反制措施。

  清华大学政治学系前讲师吴强接受本台采访时表示，有迹象显示美国关闭中国驻休斯顿总领事馆代表着双方角力升级:“中国在这种外交报复方面似乎是很难退让，最新的消息似乎已经出来了，中国可能会选择美国驻成都的总领馆作为报复对象，来关闭。在双方现在已经没有任何对话渠道的情况下，这种外交战似乎代替了对话和协商。”
  <br/>
  <br/>
  北京外交圈盛传中国反制美国的措施包括关闭美国驻成都及沈阳的领事馆。据消息人士说，本周三开始，北京将给予美国“足够时间”考虑撤回决定，又称“如果美国一意孤行，将采取反制措施”。早前有外国通讯社指美国驻武汉的领事馆将被关闭。
  <br/>
  <br/>
  不过，美国此前已决定撤出在华外交人员。据美国全国广播公司（CNBC）报道,白宫本周一（7月20日）宣布,总统特朗普签署了一项命令，撤出美国驻中国大使馆工作人员及其家属,暂停发放多种赴美签证，停止发放绿卡。这项命令可能会在2020年12月31日之后延长。

   学者 :香港美领馆地位特殊中方不会贸然关闭
  </b>
  <br/>
  <br/>
  目前，美驻华使领馆分布于北京、上海、广州、武汉、成都及沈阳。在中国境外，美国在香港也有一家总领事馆。环球时报周三晚上在网上民调，有66%人主张关闭美国驻香港总领事馆。
  <br/>
  <br/>
  北京独立学者查建国对本台表示，美中关系持续恶化，非常清晰的表现了双方的冷战升级。至于会否关闭香港美国领事馆，机会不太大:“我想中国马上就会作出对应的反制，这样的话，两国下一步对抗是越来越严重。美国在中国有6家使领馆，中国驻美国（使领馆）是5家。关闭香港的（总领事馆）肯定比关闭内地的哪一个领事馆要严重得多得多，毕竟香港是一个特区，而且香港问题现在这样的敏感”。
  <br/>
  <br/>
  查建国认为，从现在局势来看，中方处于弱势，所以中国不会作出过份强硬回应。他说：“我觉得中国的回应措施，应该还是在对等原则。总得来说，中美现在的博弈，美国应该是强方，属主动进攻方，从总的态势来说，因为中国比美国弱一些，处于守势。”

  当年美苏冷战期间，美国驱逐大量前苏联外交人员，这些人被指在美国“从事与外交身份不相符的事”，被驱逐出境。

  中国一位研究国际关系的学者苏先生对本台说：“冷战期间也没有吧，美国和苏联没有互相关闭领事馆，包括前一段时间英俄之间因投毒案，也是驱逐了俄外交官，也没有关闭领事馆，因为领事馆除了外交事务，还要负责当地本国国民涉外事宜。美国关闭领事馆印象中是二战以后的头一次。”
