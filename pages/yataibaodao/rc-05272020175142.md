 <p>
  美国国务院负责经济发展、能源与环境的次卿克拉奇（Keith Krach）5月27日批评中国共产党终结对香港的“一国两制”，是试图输出“奥威尔式极权统治”的又一例证，美国因此必须针对中共提出反制。
  <br/>
  <br/>
  中国全国人大强行推动并通过“港版国安法”前，美国国务卿蓬佩奥（Mike Pompeo）27日发表声明说，美国认为，香港的现实情况是已不再享有高度自治，他向国会提交的报告认定，香港应不再适用《香港政策法》给予香港的特殊待遇。
 </p>
 <p>
 </p>
 <p>
  <br/>
  克拉奇在斯坦福大学胡佛研究所举办的一场线上研讨会上和前白宫顾问麦克马斯特（HR McMaster）对谈，两人针对香港最新情势与美国的做法有许多讨论，尽管他并未明确表述美国是否将取消香港的特殊地位，但他强调，是中国共产党的做法，逼得美国必须有所回应。
  <br/>
  <br/>
  “你看看香港的情况，他们（中国共产党）在讨论的（立法），就是形同要终结'一国两制'。香港没有了（自治）主权，香港人民失去自由，生活在恐惧之中，受到全面审查。中国共产党设想的就是一个'奥威尔式的国家'，就像是英国作家奥威尔小说《1984》里那样的极权统治，他们想输出这种模式到全世界。”
  <br/>
  <br/>
  不只克拉奇批评，蓬佩奥在稍早的声明中形容，北京强推“港版国安法”的决定是“灾难性”的，中国正按照自有的威权模式，改造自由香港，因此，他必须痛下决定，认定香港不再享有高度自治。
 </p>
 <p>
 </p>
 <p>

 </p>
 <p>
  美国行政部门的评估报告送交国会后，接下来的关键就在美国总统特朗普是否动用职权，透过总统行政命令的方式，终结美国提供香港享有不同于中国的特殊关税地位。这将是紧绷的美中关系里，最新的大国冲突引爆点。
  <br/>
  <br/>
  <b>
   美取消香港特殊地位 伤中共也自伤？
  </b>
  <br/>
  <br/>
  另一个焦点是，一但特朗普中止香港的特殊地位，中国过去透过香港、辗转取得美国敏感的科技与技术的做法，也可能受限。分析认为，这不只将重创香港的经济，在中国与香港有大量投资的外国与美国企业，也会受重创。
  <br/>
  <br/>
  美国国会研究处资料显示，2018年，美国贸易顺差的最大来源地是香港，达310亿美元。
  <br/>
  <br/>
  克拉奇则强调，美国的制裁是针对中国共产党，不是针对中国人民而来。他还形容，美国不怕中国的崛起甚至帮助并让中国能够崛起。然而，美国过去认为，资本主义是能够带来民主化，“中国的作法证明美国过去的想法错了”，美国是时候该拿掉有色眼镜了。他也对美国企业喊话说：
  <br/>
  <br/>
  “企业的责任不只是社会责任，他们还肩负国家安全的责任。因为美国企业是站在全球经济发展的最前线，我们需要他们在前线捍卫并确保美国的资产与经济利益受到保护。”
  <br/>
  <br/>
  克拉奇来自硅谷，在加入国务院前曾创办一家电子签名公司，在电子商务产业有与中国打交道的经验。他也曾任美国通用汽车（GE Motors）副总裁。他说，对于中国政府是如何透过制度的不公平设计，使得在中国市场经营的美国企业设下不公平的竞争环境，甚至窃取美国的科技与知识产权，他自己感触良多。
 </p>
 <p>
  他指出，2049年将是中共建国百年。美国如果对中国的经济侵略还不警醒反击，中国将主导世界，这会危害到美国的国家安全。
  <br/>
  <br/>
  另外，美国的商业团体，包括美国全国商会（US Chamber of Commerce）及美中全国贸易委员会（USCBC）近来接连发表声明，都呼吁中国要维护香港的“一国两制”架构，他们担忧北京的立法行动会损害香港的特殊经济地位。在国务院将报告送交国会后，美国商业界的忧心更难以缓解。
 </p>
 <p>
