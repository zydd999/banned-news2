 <p>
  美国两位学者投书指出，随着中国军事力量及武统台湾意图增强，美国若在“两岸冲突时是否会援助台湾”这个问题上继续暧昧，会增加战争风险。台湾学者丁树范接受自由亚洲电台采访也认为，如果第一岛链上的台湾被中国大陆统一，美国在太平洋的影响力就要大打折扣。
  <br/>
  <br/>
  对美国政府决策层有影响力的防卫事务网站Defense One 10日刊出美国乔治梅森大学安全政策研究中心副主任亨泽克（Michael Hunzeker）与政策与政府学院的博士生戴维斯（Brian Davis）的联名投书，题为《台湾需要的防务改革》，讨论台湾需要的国防改革。亨泽克曾在美军陆战队服役，戴维斯曾是退休的美国陆军外事军官。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  随着中美的日益对立，台美关系加温，亨泽克、戴维斯文章中直言，美国虽然愿意提供台湾训练与武器，也不会无条件协防台湾，华盛顿应该表明对台湾的防卫承诺内容。台湾则该面对国防弱点切实改革，除了改善后备役令人堪忧的实力，更应停止购买昂贵又不切实际的军备。
 </p>

 <p>
  <b>
   习近平恫吓下  美方模棱两可反增战争风险
  </b>
  <br/>
  <br/>
  文章提及，美国长期对台湾保持的战略模糊的作法，也到了应该重新评估的时刻。毕竟习近平对台湾的想法并未含糊其辞、拐弯抹角。解放军经常进犯台湾领海与领空，也表明中国愿意使用武力统一台湾。
  <br/>
  <br/>
  亨泽克今年初在另一篇与杜鲁门国安计划研究员克里斯多佛（Mark Christopher）合著的投书中表示，1979年通过《台湾关系法》后，美国有法律义务向台湾出售自卫所需的武器。而美国却在“两岸冲突时是否会援助台湾”的问题上，故意含糊其辞。他认为这种作法背后的逻辑是，只要继续让海峡两岸猜测它可能干预的条件，美国一方面可以阻止台湾宣布独立，另一方面也可以阻止中国入侵。
  <br/>
  <br/>
  文章提及，在台湾愈多人认同自己是“台湾人”，而中国并吞台湾野心加大。美国过去的模棱两可或许可以依旧阻碍台湾追求独立，可是这种暧昧态度已无法吓阻中国的军事实力。如果华盛顿不能把话说清楚、给予台湾真正的安全承诺，反倒会增加发生战争的风险。因此无论是给予台湾明确的安全保证、或者是要彻底放弃台湾，美国都应该选择更加明确的政策，否则过去的作法已经无法维持现状。
  <br/>
  <br/>
  <b>
   台湾学者：失去台湾美国等于退出太平洋
  </b>
  <br/>
  <br/>
  对此，政治大学名誉教授丁树范接受自由亚洲电台采访也认为：“美国要把有关在台湾利益问题讲清楚，讲清楚台湾在第一岛链有多重要，第一岛链对美国有多重要，把最关键部份讲清楚，习近平就很清楚。”
  <br/>
  <br/>
  丁树范说，以现在美中竞争的这种角度下，第一岛链变得愈来愈重要。“讲白一点，如果第一岛链上的台湾被中国大陆统一 ，美国的影响力就要完全离开太平洋了，我必须坦白一点讲。”
  <br/>
  <br/>
  丁树范进一步表示，美国作为全世界乃至亚洲领导者的信誉度是很重要的。如果台湾丢掉了，他的信誉度就完蛋了。现在中国国力太强了，美国必须要防范，特别习近平领导下极端权威主义，跟美国价值和制度上是绝对冲突的。
 </p>

 <p>
  此外，对亨泽克文章提及台湾应停止购买昂贵又不切实际的军备，丁树范提及，美台双方长久以来各有抱怨。“美国常批评台湾买武器漫天叫价，开一堆清单，看你要卖什么武器给我，随便买不考虑战略。台湾就抱怨说我要的你不卖给我，最后只有列一堆清单，看你要卖什么给我。”
  <br/>
  <br/>
  不过丁树范认为，研发武器单位中科院优先发展的是自制飞弹，像是雄风、天弓系列等，足以证明台湾在国防上是有战略的，因为飞弹对防卫台湾是很重要的武器。
  <br/>
  <br/>
  针对亨泽克和戴维斯的文章中建言，美国应说清楚一旦两岸冲突，会不会派兵保护台湾？前国防部副部长林中斌则认为，“问题不在说，在美国自己的军事能力跟解放军相比，在台湾救援上，现在居于下，在对救援台湾这个地点上屈于不利，说不说、怎么说在其次。“
  <br/>
  <br/>
  <b>
   有分析指美军对解放军胜算低
  </b>
  <br/>
  <br/>
  林中斌提到，一名从中校退伍、现工作于美国智库的Davis Daniel最近发表一篇文章就提供这个观点：“美国能够成功地为台湾防御吗？他引用美国的智库、美国的国防部自己做兵推，发现跟解放军冲突，在对台湾救援这个事情上面都是失败的。他特别强调美国要救台湾是六千公里，解放军到台湾就像弗罗里达到邻近的州一样近。他说台湾应该有这种能力制止解放军接近，美国要从军力方面协助台湾，也要鼓励其他盟友帮助台湾。”
  <br/>
  <br/>
  林中斌说，Davis Daniel文章观点主要说，美国如果跟解放军打，不利。第一，可能会打败；第二，打赢以后就破产了，因为打赢以后要防备解放军反攻，要花更多的钱，随时紧张等他来打。
  <br/>
  <br/>
  林中斌表示，他估计北京不是用军事解决台湾，军事只是中间一环，北京更可能用的是“超军事手段”，例如经济、外交、心理等。
  <br/>
  <br/>
  林中斌说，比起1995、1996年的台海危机 ，解放军在军事力量上提升很多，解放军发展针对航母军舰的东风系列弹道飞弹，和巡弋飞弹，专门对付美国军舰，即便美国过去有实战经验，但今天跟解放军的战争，与以前打中东、非洲的是不同型态的战争。
  <br/>
  <br/>
  <br/>
