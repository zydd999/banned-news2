 <p>
  新冠肺炎让美中关系在跨越四十岁的不惑之年后，充满困惑与猜忌。两个大国怎么会走到这一步？有分析认为，归根结底，还是双方意识形态的根本差距，导致如今出现全面对抗竞争的格局。请听本台记者郑崇生为您整理的大疫下的美中关系大事记。
  <br/>
  <br/>
  1月15日，白宫东厅，美国总统特朗普上任以来念兹在兹说美中关系要公平互惠，现在总算在贸易战打了一年多后有了初步成果，那一天的特朗普，脸上的笑藏不住。而中方代表、国务院副总理刘鹤，镜头拍摄纪录下他在走上缔约舞台时小心翼翼，甚至不敢超越特朗普一步。
  <br/>
  <br/>
  那时的刘鹤，还没回到中国，网上就出现嘲讽他是“刘鸿章”、如同当年签署《马关条约》丧权辱国的李鸿章。然而，中国舆论很快就转移了焦点，因为当刘鹤带着大批北京官员与中国媒体来到华盛顿的时候，被称作是本世纪以来人类最难对付的新冠病毒，已经在九省通渠的中国中部枢纽武汉市，悄然蔓延。

 <p>
  今年1月23日，农历春节前夕，武汉宣布封城，这时候的美国政府，其实已经有部分专业人士知道了武汉出现不明肺炎。
  <br/>
  <br/>
  美国卫生部长阿札尔（Alex Azar）曾在1月28日的记者会上透露，他最早在1月6号就已经向中国方面表明，美国愿意派专家小组前往武汉，协助中国展开疫情调查，但中国没有接受。
  <br/>
  <br/>
  到了1月31日，美国宣布自2月2日起，对中国实施旅游禁令以防疫，并且展开撤侨行动。而美中两国对对方这些举措的解读，已经埋下后来全面互怼的种子。
  <br/>
  <br/>
  2月3日，华春莹批评美国，说美国不但未向中方提供任何实质性帮助，却第一个从武汉撤侨、并宣布对中国公民入境采取全面限制，“制造和散播恐慌，带了一个很坏的头”，她还透露，中国早在1月3日就和美国分享疫情信息，那时的多数中国人还普遍不知情。华春莹借此声称，中国和世界分享疫情资讯，始终公开透明。
 </p>
 <p>
  如果说这与阿扎尔称美国一月初就表达了愿协助中国防疫在时间点上尚说得通，但说美国没有提供实质帮助，则说不通。
  <br/>
  <br/>
  在外交上，美国当时以让事实说话的方式将美国对中国的援助公之于众。 美国国务卿蓬佩奥（Mike Pompeo）在推特上公布，美国的撤侨班机，是满载防疫物资前往中国。大疫当前，美国纳税人对中国人民仍大方援助，却怎么也没想到美国后来疫情爆发，口罩库存远远不足。

  <b>
   大疫下的美中关系 媒体驱逐战二月先开打
  </b>
  <br/>
  <br/>
  到了二月，疫情蔓延下的美中关系开始崩坏，双方拿两国记者开刀，媒体战的你来我往，揭开序幕。
  <br/>
  <br/>
  2月18日，美国国务院将新华社等五家中国在美国的分支机构列为“外国使团”。
 </p>
 <p>
  2月19日，中国外交部宣布，因《华尔街日报》二月初一篇《中国是真正的亚洲病夫》评论文章，北京撤销三名该报驻华记者的记者证，并将他们驱逐出境。
 </p>
 <p>
  到了3月2日，蓬佩奥下令从3月13日起，美方要减少60名中国驻美记者，中美媒体战呈现白热化；到了3月中旬，开始转向社交媒体的信息战，中国外交部发言人赵立坚，一战成名。

  <b>
   假信息战与大外宣启动 战狼三月霸屏造谣美军投毒
  </b>
 </p>
 <p>
  3月12日，赵立坚半夜睡不着觉，在推特上很活跃，推文中、英文并用，刻意片面解读美国专家说法，质疑美军把疫情带到武汉，还称“美国欠大众一个解释”。
  <br/>
  <br/>
  是可忍，孰不可忍。3月13日，美国正式召见中国驻美大使崔天凯，美国国务院亚太助卿史达伟向崔天凯提出了“严正交涉”与抗议。
 </p>
 <p>
  到了3月16日，蓬佩奥和中共中央外事办主任杨洁篪通电话，更讲白了美国强烈反对中国散布“假信息”。
  <br/>
  <br/>
  至此，假信息战加上了口水战，全面在社交媒体开打，中国的五毛加上机器人大军出征，到处掀骂战，同时，媒体战也达到另一波高峰。
 </p>
 <p>
  3月17日，中国外交部宣布，要求《美国之音》在内的五家美媒申报在中国的财务与经营资讯，并且下令《纽约时报》、《华尔街日报》与《华盛顿邮报》驻华记者年底证件到期的人、交还记者证，今后不得在中国大陆与港、澳从事采访工作。
  <br/>
  <br/>
  3月17日这一天，特朗普也忍不住了，在白宫记者室叫板中国。
  <br/>
  <br/>
  特朗普说：“中国在散播假信息，说我们美军把病毒带到中国，这是假的，与其争辩，我宁可讲清楚病毒是哪来的，就是从中国来的，是中国病毒，我觉得这样说很精准....说什么我们美军带病毒给他们，这才是污蔑。”
 </p>
 <p>
  至此，特朗普对中国国家主席习近平的态度，也从疫情初期的肯定与鼓励，转向失望与怀疑。尽管两人3月26日通了电话，但情况未见改善，尤其是特朗普在通话前还签署了《台北法》生效。
  <br/>
  <br/>
  另一方面，三月中，疫情在美国与欧洲迅速传开，这时候，疫情初步减缓的中国，大外宣动作很大，在多个国际社交媒体平台开展“口罩外交”，中国官方媒体急于塑造“全世界都在感谢中国”的大国形象，却避谈中国制造防疫产品的伪劣情况。
 </p>

  <b>
   美国反击：今年四月直指武汉病毒所问题大
  </b>
  <br/>
  <br/>
  赵立坚的“美军投毒说”，让当时才开始遭受疫情重击的美国、对中国的怨怼，升到新高度。他的前辈、曾任外交部发言人的崔天凯则在接受美国媒体访问时试图划清界线，然而，赵立坚四月份重新站上外交部例行记者会蓝厅的讲台，并且“贼喊捉贼”。
  <br/>
  <br/>
  赵立坚4月7日说：“我在个人推特帐户上提出的疑问，是对前一段时间美国一些政客污名化中国的反应。”
  <br/>
  <br/>
  美国则将注意力转向了武汉病毒所。 4月14日，参谋长联席会议主席米利说，病毒属性是否为自然形成尚无定论，仍有不确定性，美国正调查许多情报。
  <br/>
  <br/>
  美国国务卿蓬佩奥四月中也把矛头指向武汉病毒所管理不善与中国隐匿疫情资讯，他更直指中共极权体制不透明，并以染疫病逝的武汉眼科医生李文亮为例，称如果李文亮生在美国，他的专业会受到重视，而不是被打压。
 </p>
 <p>
  蓬佩奥：“美国需要的是可靠的伙伴，当你告诉我们某些事情时，这些资讯应是正确可靠的，不会让我们认为他们有所隐瞒。但是，中国共产党至今仍不让美国与国际社会进入武汉病毒所。”
 </p>
 <p>
  蓬佩奥多次把中国共产党与中国人民分别对待，这或许踩到中共的痛点，也让他成为中国官方喉舌发动“文革大字报式”攻击的对象。
  <br/>
  <br/>
  4月27日，中央电视台一连三天在“新闻联播”重批他散播“政治病毒”、“背负四宗罪”、“突破做人底线”。蓬佩奥选择不随之起舞，但在重新审视美国的对华政策上以及如何与崛起的中国打交道上，他心中也已经有了定见。
  <br/>
  <br/>
  5月20日深夜，白宫发表《美国对中華人民共和国战略方针》报告，并送交国会，特朗普政府认为，美国过去二十年来的“交往政策”（engagement policy）是个错误，必须采取全新方法应对中国的扩张与威胁，对中共不再好言相劝，而是选择直言不讳。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  在华盛顿智库战略暨国际研究中心（CSIS）资深研究员葛来仪（Bonnie Glaser）看来，这就是双方都不愿意再隐藏的意识形态竞争。
  <br/>
  <br/>
  她说：“中国藉由疫情输出所谓’中国模式’比较好，借以诉求’美国主导的体系在衰落’；美国与中国的战略竞争，已经转向了意识形态的对决。”
  <br/>
  <br/>
  在南海和华为问题上，美国都和中国直接对决，而香港问题更成为新的引爆点。
 </p>

  <b>
   庚子年中国两会敲响了香港一国两制丧钟
  </b>
 </p>
 <p>
  因为疫情，中国两会首次延后至5月21日举行，但召开前夕、传出北京将《香港国安法》排上议程，美国国务院批评北京破坏《中英联合声明》。
  <br/>
  <br/>
  新冠病毒让所有国家深陷大疫隔绝，却阻挡不了习近平巩固与稳定权力的脚步，在香港问题上，他逼着特朗普出手。
  <br/>
  <br/>
  5月29日，特朗普说，“中国已将香港的'一国两制'变为'一国一制'，因此，美国启动撤销香港作为独立关税区享有的特殊待遇，美国也将针对扼杀香港自由的中国和香港官员，祭出制裁。我还要宣布，美国启动退出世界卫生组织的程序。”
  <br/>
  <br/>
  <b>
   六月蓬杨夏威夷会谈不欢而散
   <br/>
  </b>
  <br/>
  香港问题在六月份成为美中关系争锋相对的焦点，6月17日，杨洁篪特地赴夏威夷与蓬佩奥会面，双方却不欢而散。
  <br/>
  <br/>
  当天，特朗普签署《维吾尔人权政策法》生效，而美国国务院亚太助卿史达伟后来明白指出，中方在夏威夷会谈上态度“称不上友善积极”。中国随后也宣布，人大常委会开始审议《香港国安法》。
  <br/>
  <br/>
  在《香港国安法》通过前夕，彭佩奥在6月29号的声明中指出，美国决定停止出口美国的军备设施到香港，并将进一步采取措施，限制美国军民两用科技出口香港。
  <br/>
  <br/>
  6月30日，中国人大常委会通过《香港国安法》，香港特区政府当天生效实施。
 </p>
 <p>
  <b>
   七月暑热 难融美中关系因香港新疆南海议题互怼寒意
  </b>
  <br/>
  <br/>
  7月1日，蓬佩奥批评《香港国安法》的域外法权与长臂管辖，连美国人也认为“太离谱”。
  <br/>
  <br/>
  7月2日，美国国会两院火速完成立法程序，通过《香港自治法案》，并赶在7月4日美国的独立日假期前送交白宫。
  <br/>
  <br/>
  同一天，美国五角大楼批评中国7月初开始在南海演习，同时，美国也罕见地派出双航空母舰编队赴南海演练。
  <br/>
  <br/>
  7月7日，美国国务院以未点名的方式宣布，对阻止美国外交官与记者进入西藏的中共官员，祭出对等签证限制。
