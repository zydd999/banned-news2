 <p>
  美国司法部长巴尔（William Barr）周四发表有关对华政策的重要讲话，直指中国国家主席习近平想以“中国方案”取代“美国梦”，更把矛头指向美国企业为了商业利益卑躬屈膝、和中国共产党唱和，这是破坏美国后代子孙的繁荣未来。他提醒美国企业，对于协助外国势力渗透，美国是有相关法律规范的。
 </p>
 <p>
  选择在纪念前总统福特（Gerald Ford）的博物馆发表针对中国的演说，巴尔有备而来、带着讲稿站上舞台。他首先表示，二十一世纪面临的最重要问题是美国对中共全球野心的回应。在将近一个钟头的演说里，他细数中共如何藉由掠夺性的经济政策与不公平的市场准入障碍，从美国制造业技术、好莱坞影视业、教育到美国科技业偷、拐、抢、骗美国的知识产权与研发成果，他还说，中共的威慑利诱已经渗透到美国各领域，也把矛头指向美国企业向中共磕头、是协助输出中共意识形态的帮凶。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  <b>
   向中共磕头？巴尔吁美国机构勿忘美国法治
  </b>
  <br/>
  <br/>
  巴尔特别点名苹果公司，还举例指出当美国逮到盖达组织的恐怖份子，要求苹果协助解密这些犯罪者了手机内部资讯，苹果拒绝配合；当香港追求自由民主的民众上街头游行，苹果却配合中共，将分享即时警力分布的软件下架。
 </p>
 <p>
  巴尔批评说，苹果公司的做法是“双重标准”，他借此提醒美国企业，不要忘记美国是法治社会。他还小秀中文指出，美国可不是中国共产党那套“有关系就没关系”的人治体系，作为美国企业与机构，必须遵守美国法律。
  <br/>
  <br/>
  巴尔说，“美国企业大老板们可能不觉得自己是中共的说客，举例来说，也许你们会认为，要在中国发展就是得靠'建立关系'，才能互惠，这是在中华人民共和国做生意的必备之道，但是，你们要有所警惕，身为一个外国企业，这种做法已经涉及了美国的《外国代理人登记法》 。”
 </p>

 <p>
  <b>
   过度依赖中国有危险 导致中国影响力全面渗透
  </b>
  <br/>
  <br/>
  他表示，虽然美国企业与大学机构可能视自己为“全球公民”或是跨国企业，而非美国机构，但不要忘了，正是因为美国自由发展的企业制度与经济、公平的法治与安全，这些机构才能成功。
  <br/>
  <br/>
  巴尔还说，从迪士尼过去曾投资拍摄关于流亡西藏精神领袖达赖喇嘛的纪录片遭中共封杀，到后来配合中共自我审查而不触碰北京认定的敏感议题，再到美国大专院校开办孔子学院、也配合不谈论有关西藏与新疆的话题，这都是“中国影响力行动”的例证。
  <br/>
  <br/>
  巴尔认为，全球化不一定能带来更多的自由，“和中国共产党一搭一唱”的社会，不会是个热忱好客的世界，自由的市场必须要有自由的贸易与思想交流。
  <br/>
  <br/>
  巴尔在他的开场白中提到，福特1975年访问中国，和当时年事已高的中共领导人毛泽东会面。他说，无法想像经历近半个世纪后，中国成为美国的最大威胁。但新冠肺炎疫情，让世界重新省思过度依赖中国有多危险。习近平展现的作为，是转折点，美国社会必须醒一醒。
 </p>

 <p>
  <b>
   巴尔：中共不再韬光养晦 习近平欲以中国方案取代美国梦
  </b>
  <br/>
  <br/>
  巴尔说，“邓小平当年讲‘韬光养晦'是为争取时间，发展经济改革。现在，中国经济已经占世界生产总额（GDP）的两成了，而习近平的集权程度，是毛泽东以来所未有的，他公开谈论中国要建立比资本主义更优秀的社会主义，要以中国方案取代美国梦。中国不再假装韬光养晦了，他（习近平）认为中国的时代已经到来。”
  <br/>
  <br/>
  巴尔是继白宫国安顾问奥布莱恩（Robert O'Brien）、联邦调查局长雷伊（Christopher Wray）后，近期又一位美国高官对中国亮剑；而作为美国首席外交官，国务卿蓬佩奥（Mike Pompeo）对中国共产党的批评，使他很早就成为北京的眼中钉。
  <br/>
  <br/>
  巴尔在会中指出，演说前，雷伊告诉他，因为日前披露中共的种种非法作为后，自己受到北京抨击；巴尔则自嘲，他不怕成为比雷伊更被北京厌恶的美国官员，还笑说蓬佩奥接下来也有声讨中国的演讲，美国敲打中共的大棒，一棒接一棒。
 </p>
 <p>
  这凸显美国总统特朗普在2020年大选年打中国牌，会是多管齐下；而美中关系全面性的敌对，已经有媒体形容会是“蹦极跳”的失速下滑。
  <br/>
  <br/>
  巴尔说，中国正进行一场经济突袭战，这是整个中国政府与社会精心设计的侵略性作为，他形容“巨龙已经觉醒“，他呼吁美国社会应该思考，未来想生活在一个什么样的社会？若还想保有美国现在的生活方式与制度，就必须重新评估与中国的关系。
 </p>
