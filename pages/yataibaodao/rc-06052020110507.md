 <p>
  六四天安门事件31周年，中国官方仍不允许民众谈论所谓的“那场政治风波”。包括当年的参与者蕫盛坤、马少方与陈云飞，日前遭人带走后虽已经获释，但仍被严密监控。另一方面，在美国的八九学运领袖王丹、美国记者贝书颖，都因谈论或使用了六四相关词汇，遭骚扰或封号。中国政府不让谈六四，世界就会忘记吗？
  <br/>
  <br/>
  “各位朋友大家好，我是董盛坤.....我亲眼目睹了六部口惨案和其它残暴血腥的场面，这一切震撼了我，也深深刺痛了我。”
  <br/>
  <br/>
  5月31日，美国的人权组织人道中国在举办的线上研讨会上播放了这段董盛坤事先预录好的四分多钟的视频。这位当年31岁热血沸腾、上街声援学生的北京胶印二厂工人，如今已是花甲之年。
  <br/>
  <br/>
  蕫盛坤后来被判处死缓，关了17年。他2006年出狱，多年囚禁却没销磨掉一身正气，身在国内，仍不改其志、勇于发声，但中国共产党容不得一点异见的态度。这也没有改变。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <b>
   当年工人挺学生 周锋锁要为苦难一代留纪录
  </b>
  <br/>
  <br/>
  人道中国的创办人周锋锁告诉本台，就在蕫盛坤影片播放当天，他被公安带走。这让周锋锁一直惴惴不安。最新消息是，董盛坤被关5天后，已于六月四号深夜获得释放，但至今仍遭监视住居，不允许他出门。
  <br/>
  <br/>
  对周锋锁来说，当年还是大学生的他，有蕫盛坤这样的北京普通民众声援，如今，说什么都不能不为他发声。
  <br/>
  <br/>
  “对于他们这一代人来讲，他经受的苦难太多了，但知道的人太少，尤其他们还被冠上“暴徒”等等罪名，被隔离被打压。他们是受打压最惨的。”
  <br/>
  <br/>
  除了蕫盛坤，周锋锁指出，包括当年天安门事件的参与者如马少方与陈云飞，也都和蕫盛坤一样，今年的六四被带走几天后才获释放。
  <br/>
  <br/>
  周锋锁说，中共想要世人遗忘，是因为对自己的政权稳定有不安与害怕。越是这样，就越需要把蕫盛坤这些人的声音，以各种方式记录下来。
  <br/>
  <br/>
  对许多中国人来说，六四天安门大屠杀标志着记忆与遗忘的斗争；而对许多关心中国的外国人而言，六四纪念日也使他们亲身经历了中国式监控，无处不在。
 </p>

  <b>
   党长臂伸向美国  美记者改微信密码帐号遭封
  </b>
  <br/>
  <br/>
  美国网络媒体Axios报导中国事务的记者贝书颖（Bethany Allen-Ebrahimain）告诉本台，她有一个以自己美国手机号登记注册的微信帐号。6月4号她试着将账号密码改为咒骂中国共产党的词汇再加上89两个数字。大约45秒后，她的帐号寿终正寝。
  <br/>
  <br/>
  贝书颖说，“我当时应该要屏幕截图的，我没想到会这样，就收到一封信息，它的大意就是告诉我，因为一些所谓的'非法行为'，我的微信号永久被封号.......但不像其他社群媒体，如果有争议发生，可以透过电话联系或是电邮联系，但微信什么申诉机制都没提到。”
  <br/>
  <br/>
  你的密码不只是你的密码。贝书颖的例子显示，使用中国社交媒体，在党管一切的体制里，不只内容受监控，就算是海外使用者，本该加密、并受隐私权保障的个人帐号资讯，中国“老大哥”一直在看着。
  <br/>
  <br/>
  经记者搜索，贝书颖的微信帐号已经显示“此用户不存在”了。她则说，自己没需要、也不打算重启微信帐号。

 <p>
  <b>
   王丹Zoom会议谈六四 两度遭攻击遭扰
  </b>
  <br/>
  <br/>
  8964、535、以及自称“伟大、光荣、正确”的中国共产党英文缩写CCP，都是敏感字词。中国社交媒体的作法，形同主动和海外使用者脱钩；然而，中国官方可能对外国企业的长臂管辖，更让人忧心。
  <br/>
  <br/>
  王丹在六月四号当天主办全球纪念“六四”的网络会议，使用的是最近爆红的视频聊天软件Zoom平台的付费帐号，却两度遭到骇客阻挠会议进行。他告诉本台，他怀疑这是中共主导的破坏行动。对于能在中国国内运作的Zoom公司给出的解释，他不能接受。
  <br/>
  <br/>
  王丹说：“被攻击之后，第二次公布的链接我们就没有公布密码了，仍然被攻击。所以这不是我们公布密码的问题，我觉得这是Zoom公司在试图逃避责任。”
  <br/>
  <br/>
  Zoom的创办人是出生于中国山东的袁征。新冠肺炎疫情以来，这个线上视频会议软件用户数与市值都暴涨，袁征在彭博亿万富翁指数中排行第4，身价今年暴增20亿美元。
 </p>
 <p>
  截至发稿，微信的开发者腾讯公司未回复本台查询。
  <br/>
  <br/>
  Zoom的一名发言人则以书面回复本台时强调，Zoom致力于为用户打造平等、受尊重与具包容性的网络环境。公司对于网络会议受到干扰极其关注，并在需要的时候与执法机关密切合作；Zoom也鼓励用户将这类干扰事件，向执法单位报告，以便他们采取适当措施。
 </p>
