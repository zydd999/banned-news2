 <p>
  视频会议软件ZOOM近来声称，为遵循中国“当地法律”，一度关闭美国付费用户、人道中国创办人周锋锁的帐号。周锋锁告诉本台，美国企业为利益一再配合中国官方“并不奇怪”。但让人担忧的是，中国政府正主导虚拟网络世界的规则建立，以美国为首的自由社会，能有什么霹雳手段反制？
  <br/>
  <br/>
  在美国人权组织人道中国5月31日主办的纪念六四31周年的ZOOM视频会议上，已经高龄八十三岁的天安门母亲群体成员张先玲，透过事先录制好的视频说，为了不愧对于像儿子王楠一样在天安门大屠杀中丧失宝贵生命的中国年轻人，他们坚持追求真相、道歉与咎责的立场，“虽然政府对我们严加打压，但是，我们不害怕，我们要坚持到底，我们所有的通讯工具都被监听、监视。”
  <br/>
  <br/>
  中共的监听、监视，显然无处不在，且跨越到防火墙外。这场活动顺利完成后，主办人周锋锁却发现，自己付费使用的ZOOM帐号在6月7日无预警遭封锁，直到美国网路媒体Axios报道，他的帐号才于6月10日恢复。
 </p>

 <p>
  <b>
   外国企业接连跪舔中共 ZOOM非首例
  </b>
  <br/>
  <br/>
  周锋锁告诉本台，从谷歌（Google）为进入中国市场，曾一度传出要创办“中国审查版谷歌”的蜻蜓（Dragonfly）计划 、到Youtube声称按照世界卫生组织规范，审查用户分享的疫情资讯内容，还有他自己经历的领英（LinkedIn）与ZOOM帐号遭关闭，他见怪不怪的背后，更想知道这些美国企业究竟是以何种方式，配合中国审查。
  <br/>
  <br/>
  他说，“他们是按照北京的要求做了，但这里面我们想知道的细节很重要，他们怎么做的？他们说是要配合当地法律（local law），什么样的当地法律？是他们自己的解释？还是中国给他们的规定？”
  <br/>
  <br/>
  新冠疫情后，ZOOM成为爆红的视频会议软件，在中国也能使用，但是不是为了中国市场。由于ZOOM的创始人袁征来自中国，而且该公司在中国有一个很大的研发团队，所以ZOOM软件的安全性一直受到质疑。他们已经展开自我审查？ ZOOM未来是否会监控用户的开会内容？包括人权、法治，甚至是中国政治体制改革这样的主题，ZOOM是否会将审查范围扩及全球使用者？
  <br/>
  <br/>
  ZOOM的一名发言人在回复自由亚洲电台的电子邮件询问时再次强调，“和其他跨国公司一样，ZOOM必须遵守业务营运所在地的法律规范”；ZOOM作为支持复杂且跨国境对话的视频会议平台，要遵循各国不同的法律法规“非常困难”，也对近来在中国境内、外的会议遭干扰、受到负面冲击感到遗憾。
  <br/>
  <br/>
  在没有提到“中国”的情况下，发言人说，“ZOOM无权改变多个政府反对言论自由的法律”。
  <br/>
  <br/>
  这名发言人还指出，公司正在开发其他功能，让在某个政府封锁境内参与者参与沟通的情况下，能保障其他境外参与者对话。
  <br/>
  <br/>
  ZOOM的说法也证实周锋锁的猜测，他认为，ZOOM会像领英公司一样，开发出“中国墙内、外”两个不同使用版本。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <b>
   反制中国主导互联网规则？周锋锁：推倒网路防火墙
  </b>
  <br/>
  <br/>
  除了周锋锁之外，有媒体报导，香港六四活动主办方“支联会”负责人李卓人的ZOOM账号，早在5月22日，就遭到屏蔽。
  <br/>
  <br/>
  周锋锁对美国企业向中国市场低头的作法，早就多有批评。他认为，1989年后，历任美国政府的纵容，养大中共巨兽；尽管美国总统特朗普政府在面对独裁中共上，已经比前几任做得更多，但现在必须采取霹雳手段，否则越晚处理，代价越大。
  <br/>
  <br/>
  周锋锁指出：“比方说，贸易谈判就应该以（要求中国）拆毁防火墙作为最重要的打破贸易壁垒的目标，而不是说卖农产品；农产品的销售数字再高，都比不上让谷歌Google、推特能进入中国市场（来的重要）。”
  <br/>
  <br/>
  天安门事件31周年、6月4日当天，民运人士王丹主办的ZOOM会议活动也屡遭攻击。王丹告诉记者，ZOOM所谓“遵守当地法规”的说法，根本在说谎。
  <br/>
  <br/>
  “中国哪有一条当地法律是限制网路上纪念六四的？给我举出来...这公司根本在撒谎、在编，中国根本没有这个法律，他只是遵守了中国政府的旨意，不是法律，他是遵守独裁政府的意志。”王丹说。
  <br/>
  <br/>
  经营中国航线的美国航空公司，遭中国要求更改各自官网上对台湾的说明；美国职篮（NBA）高管因为声援香港“反送中”运动，遭中国官媒发动抵制、停止转播；这一次，登记在美国加州的ZOOM，是主动还是被动封锁用户帐号？如果媒体不关注，是不是就不恢复付费使用者的服务？ ZOOM没有回答，留下更多疑问与对中国政府的猜忌。
 </p>
