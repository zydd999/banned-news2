 <p>
  美国媒体《华尔街日报》与《纽约时报》近来分别从中国与美国官方引述不具名的消息来源报导，新冠肺炎疫情失控蔓延全球，主要责任在于中国湖北地方官员的失职瞒报。中国国家主席习近平能借此完美甩锅切割吗？在这个时间点将疫情蔓延全球的责任，指向中国的地方官员，对美中关系正面临全面对峙的紧张态势，又有何影响？
  <br/>
  <br/>
  发源自中国武汉的新冠肺炎疫情蔓延全球，究竟谁该负主要责任？ 《纽约时报》日前一篇发自华盛顿的报导，引述未具名的消息指出，美国情报界认定，新冠肺炎疫情初期，中国湖北地方官员出于政治考虑，对北京中央政府隐匿瞒报；疫情在湖北蔓延后，当北京的中共中央官员在慌忙中要从地方官员那里撬开信息时，仍不忘隐瞒世界卫生组织（WHO），掩盖疫情。
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  报导说，这份含有机密与非机密信息的报告，今年6月陆续在美国的中央情报局（CIA）与其他情报部门间流传开。报告引用证据表明，中国地方官员渎职，似乎是病毒在武汉内、外传播的决定性因素；而不具名的现任和前任政府官员则认为，这份报告没有减少中国的罪责。

   习近平只享权不负责说不通

 <p>
  美国哈德逊研究所（Hudson Institute）的中国问题学者白邦瑞（Michael Pillsbury）认为，“是武汉、还是北京该负主要责任，差别很大”，如果习近平不是主要犯错者，意味着中国高层官员没有在新冠病毒问题上彻底欺骗，这代表美中官员就共同关心的问题开诚布公谈判，仍具备一定基础。
  <br/>
  <br/>
  不过，纽约城市大学政治学教授夏明不认同这样的观点，他说，追究中国内部该由谁负责任，和美中谈判完全两回事。
  <br/>
  <br/>
  这位深知中共体制运作的华裔学者告诉本台，政治学上讲权责相符，不管何种体制都是如此。习近平定于一尊，在中国大权独揽，出了事情却说责任不在他，这逻辑说不通：“现在再要把（责任）推到地方政府，我认为，这完全是错误的。疫情失控就是中共的失误，中共的失误当然是习近平的失误，推责是为了维护习近平的虚荣与权力。”
  <br/>
  <br/>
  按《纽约时报》的报导，美国情报部门仍认定，中国对世界卫生组职隐瞒实情。夏明就说，光这一点，就不是中国地方官员有权决定的，更不要说疫情在全球蔓延开后，中共满世界散播扭曲且有利于自己的宣传论述与假信息，习近平当然要负责。
  <br/>
  <br/>
  对于《纽约时报》的报导，美国国土安全部发言人沃尔托尼斯特（Alexei Woltornist）表示，“不对任何声称的泄露文件发表评论”。中情局则拒绝置评。
  <br/>
  <br/>

 <p>
  在《华尔街日报》的记者笔下，每日睡前走一万步的中国疾病预防控制中心（简称：中国疾控中心）主任高福，则像是个“孤臣无力可回天”的有责无权者。
  <br/>
  <br/>
  报导说，中国疾控中心作为中国国家卫生健康委的直属事业单位，在中国有3000多个地方的疾病控制中心但却鞭长莫及，地方的疾控中心是不听中国疾控中心指挥的。
  <br/>
  <br/>
  报导提到，中国疾控中心过去在美国的培训与帮助下，于2002年成立。在美中关系日趋恶化的情况下，高福这位接受过英国牛津大学与美国哈佛大学训练的“海归派”专家，面临“里外不是人”的窘境。
  <br/>
  <br/>
  在北京，与美国关系密切的中国官员受到更多怀疑。与此同时，一些美国官员则对美国国家卫生院（NIH）和中国合作、高福也参与的全球病毒分类研究的项目（Global Virome Project）有所警惕。美国政府在新冠肺炎疫情爆发后，已经宣布停止拨款。
  <br/>
  <br/>
  <b>
   美中关系回不去了 美全力棒打中国不给胡萝卜
  </b>
  <br/>
  <br/>
  疫情加速了美中之间的不信任，就连科学合作也无以为继，美国亚州协会政策研究所（Asia Society Policy Institute）资深研究员莫德（Richard Maude）曾是澳大利亚的外交部副部长。他形容，美国在外交上一向以“大棒与胡萝卜”的两手策略，但他预测，不论美国年底总统大选结果为何，这场疫情后的美国的中国政策，会是两党都选择出棒打击的局面。
  <br/>
  <br/>
  “我们可以看到，特朗普政府对待中国是更愿意用大棒反击，而不是胡萝卜。他对美中关系之间的摩擦也有更高的忍耐程度，也就是不怕冲突……但是，我想，就算接下来是民主党主政，情况也不会有太多不同。”莫德说。
  <br/>
  <br/>
  疫情下的地缘政治学，中国当然也看得懂。已经派驻华盛顿长达7年的中国驻美大使崔天凯近来在与美国智库的一场非公开会谈后，中国大使馆主动公布资讯指出，北京并没有等待今年11月美国大选结果。
  <br/>
  <br/>
  崔天凯还说，中方愿随时与美国现任政府合作，寻求解决问题的办法，更要坚决防止“麦卡锡主义”回潮，推动中美关系尽快回归正轨，而“后疫情世界”的建设，为中美合作提供难得良机。
  <br/>
  <br/>
  但美中关系回得去吗？全球科学家们都呼吁民众，要尽快适应后疫情时代的生活方式，接受新冠病毒不会瞬间消失的事实。这可以给美中关系什么样的启示？
  <br/>
  <br/>
  自由亚洲电台记者郑崇生华盛顿报导 责编：申铧 网编：郭度
