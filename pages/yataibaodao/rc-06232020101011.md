 <p>
  美国总统特朗普近来受访时指出，他对中国采取的关税措施，远比针对中国迫害新疆维吾尔人权的官员祭出的制裁还严厉。特朗普视贸易为最重要议题，对美国重塑复杂的美中关系有何影响？中国又如何看待？
  <br/>
  <br/>
  6月19日，特朗普接受美国媒体Axios的访问指出，他已经签署《维吾尔人权政策法》，但暂时不会按照这个法律对迫害新疆维吾尔人的中国官员祭出经济制裁措施，因为重大的美中贸易协议中，中国承诺采购的金额高达两千五百亿美元。
  <br/>
  <br/>
  特朗普还强调，他在贸易上对中国祭出关税措施，远比任何制裁都严厉，并解释他在签署《维吾尔人权政策法》前，之所以没先引用《全球马格尼茨基人权问责法》制裁中国官员，是因为“没有一个国会议员要求我这么做。”
  <br/>
  <br/>
  特朗普的说法，引发在野的民主党抨击。众议院议长佩洛西（Nancy Pelosi）透过书面声明指出，特朗普为了贸易协议，不愿对抗北京，令人震惊；在这样的情况下签署《维吾尔人权政策法》“显得格外空洞”。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <b>
   经济胜过人权？美记者揭示对华交往政策历史脉络
  </b>
  <br/>
  <br/>
  在复杂的美中关系中，特朗普最看重贸易议题，标榜要矫正中国的不公平贸易作为，美国利益是最优先考虑。那么，外界如何看待特朗普的这个立场？
  <br/>
  <br/>
  曾派驻北京的《华尔街日报》资深编辑戴维斯（Bob Davis）23号在美国全国新闻基金会（NPF）主办的一场视频会议上回顾美中交往历史，这位美国记者勇于检视美国自己，说大白话。
  <br/>
  <br/>
  “回顾美国与中国交往历史，我们不要只看这一届政府。基本上，美国历任每一届政府都面临人权议题的挑战，但坦白说，经济议题一直都占上风。让我们回想一下美国开启与新中国交往的时代背景。尼克松是在'文化大革命’时期访华的，文革在近代中国史上，是最大规模、最严重的人权侵害，而尼克松却选在那个时刻访问中国。”戴维斯说。
  <br/>
  <br/>
  与中国交往看重经济议题，特朗普不是第一位这么做的美国总统，他只不过更敢于说出来。然而，美国政治体制不会让政府团队中只有一种声音。在特朗普政府里，仍有人对中国的人权问题，高分贝喊话，例如对中鹰派色彩鲜明的白宫贸易顾问纳瓦罗（Peter Navarro）。
 </p>

 <p>
  <br/>
  <b>
   对华鹰鸽派相争 北京旁观坐收渔翁利？
  </b>
  <br/>
  <br/>
  纳瓦罗22日接受福克斯电视台访问时指出，看看中国在新疆的作为，中国还霸凌邻国越南，在南海上撞沉越南渔船；中国还说谎，放任新冠肺炎疫情在全世界大流行，他说，基于这些原因，“美中贸易协议玩完了”。
 </p>
 <p>
  不过，他后来又改口贸易协议还在；特朗普更在华盛顿22日深夜发推文指出，美中贸易协议“完好无损”，希望中国继续遵守并兑现承诺。
  <br/>
  <br/>
  究竟美中贸易战这一轮交锋下来，谁占上风？同样参与视频会议的皮特森研究所（Peterson Institute for International Economics）资深研究员勒夫莉（Mary Lovely）指出，美中互征关税，都是对方的企业与消费者买单。然而，在华美国企业并没有把工厂搬回来，对中国制造业就业的打击也不明显，她还点出了美国单打独斗的困境。
 </p>
 <p>
  勒夫莉说：“中国和美国只是互相脱钩。美国的做法，并没有让世界与中国脱钩。比如美国对中国开征关税的产品，数据显示，中国转而出口到世界其他国家，且占比稳定，没有减少。”
  <br/>
  <br/>
  <b>
   中国国务院参事：认真对美加大采购
  </b>
  <br/>
  <br/>
  人在北京的中国国务院参事王辉耀也参与了全国新闻基金会举办的视频会议。他则认为美中所谓的“全面脱钩”不切实际。
 </p>
 <p>
  中国国务院参事室是具统战与咨询功能的机构。王辉耀强调，中国始终认真对待美中贸易协议的承诺。今年第一季度，中国采购美国农产品的金额已经超过五十亿美元，同比增百分之一百一十以上，其中，大豆采购达三十一亿美元，美猪采购金额则达六亿九千万美元。
 </p>
 <p>
  他指出，美中双方更应该要有“新论述”与“新思维”对待彼此，避免“新冷战”。
  <br/>
  <br/>
  勒夫莉则在会中提醒，中国身为世界贸易组织会员国，在解决国际社会诟病的补贴问题上，是有义务与责任的，中国也该有“新作法”。
  <br/>
  <br/>
  她说，接下来，美国和中国第二阶段贸易协议才是真正难啃的骨头，包括国企改革、中国的补贴政策。她疾呼美国要想真正改变中国不公平的贸易作为，必须对内先有协调的政策，对外拉拢盟邦、打团体战。
  <br/>
  <br/>
  王辉耀顺势指出，这也是为什么中国始终认为维护国际体系的运行至关重要，而中国国务院总理李克强也才因此表示，中国对参与跨太平洋伙伴全面进步协定（CPTPP）持积极开放的态度。
 </p>
