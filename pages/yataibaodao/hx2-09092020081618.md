 <p>
  西藏精神领袖达赖喇嘛近日透过直播向亚洲信徒讲法时透露，已经收到一些台湾朋友的邀请，疫情过后，有可能去新加坡和台湾访问。不过达赖喇嘛特别强调：“这当然也是在北京允许我访问台湾的情况下”。
  <br/>
  <br/>
  受到新冠肺炎疫情影响，过去行程满档到全世界弘法的西藏精神领袖达赖喇嘛，自今年2月停止会客，也取消所有海外访问行程，至今已超过7个月。往年各地信徒前仆后继组团前往印度达兰萨拉聆听他讲法的情况不复见，而转为透过视讯方式进行。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  9月6日，达赖喇嘛应新加坡、马来西亚、香港、越南、泰国、印度尼西亚等亚洲佛教信众共同祈请讲授佛法。主持对话的新加坡信徒提到，希望疫情过后，达赖喇嘛身体安康长久住世，他们要组团再上山到达兰萨拉听尊者讲佛法。
 </p>
 <p>
 </p>
 <p>
  <iframe frameborder="0" height="620" scrolling="no" src="https://www.facebook.com/plugins/video.php?href=https%3A%2F%2Fwww.facebook.com%2FRFAChinese%2Fvideos%2F314871283080118%2F&amp;show_text=0&amp;width=622" width="622">
  </iframe>
 </p>
 <p>
 </p>
 <p>
  达赖喇嘛顺势问了信徒所在地“新加坡”，而后达赖喇嘛说：“因为疫情的缘故，我无法离开我的住所去外面为你们讲法，说不定明年有可能去访问新加坡。我已经收到一些邀请，来自台湾的一些朋友的邀请，从印度达兰萨拉这边先去新加坡，之后再去台湾。”
  <br/>
  <br/>
  不过达赖喇嘛紧接着强调：“当然这也是在北京、中华人民共和国允许我访问台湾的情况下。我会待在新加坡几天，之后再去台湾。”
  <br/>
  <br/>
  <b>
   台湾外交部：在双方认为方便可行下 慎重研议
  </b>
 </p>
 <p>
  对达赖喇嘛预告明年可能访问台湾，台湾的外交部发言人欧江安答覆自由亚洲电台说：“达赖喇嘛是世界重要的宗教领袖，并曾获得诺贝尔和平奖。他在台湾也有许多信众，期待他能再度来台湾弘法。但目前印度疫情仍属严峻，跨国旅行相当困难。以后倘收到申请，在双方都认为方便可行的情形下，我们会慎重研议。”
 </p>
 <p>
  西藏台湾人权连线理事林欣怡，在接受自由亚洲电台采访说，听到达赖喇嘛公开回应明年规划访问台湾，很受鼓舞。
 </p>
 <p>
  林欣怡认为：“我们政府、外交部或许就用平常心看待这件事，他来真的就是加分的事，而且他愿意来，我自己觉得不用还要研议、慎重什么的，都已经来过那么多次，而且每次来对台湾影响都是非常正面，他也是很有智慧的人 我们就敞开心胸欢迎他来就是了。”
 </p>

 <p>
  <b>
   刻意提及“如果北京允许” 援藏团体：达赖喇嘛展现慈悲
  </b>
  <br/>
  <br/>
  林欣怡提到，2009年八八风灾，达赖喇嘛到台湾祈福，抚慰许多台湾人心，他现在只有宗教领袖的身分，在佛教上影响力非常大，如果他愿意来，再多困难，相信他有智慧克服。
  <br/>
  <br/>
  林欣怡：“法王他是很慈悲，不想造成大家困扰，他总是会非常愿意回应他的信徒、回应期待他的人。他会加上那句‘北京允许’的话，他是不希望带给任何人政治上的困扰，对我们来讲的确非常期待。”
  <br/>
  <br/>
  <b>
   在台藏人：不管中共开不开心 全世界在看台湾
  </b>
 </p>
 <p>
  西藏台湾人权连线理事长札西慈仁接受自由亚洲电台采访表示，“我觉得台湾政府应该要请达赖喇嘛尊者来台湾，全世界在看台湾的民主发展。我也在台湾很久，我知道台湾人民非常支持，非常相信人权、自由、民主。那达赖喇嘛尊者在世界各地，所有的事情都是为了世界和平而已，我们做什么，中共还是会不开心，我们一定要做，为了人权、自由、民主，我们继续会努力做这些，不管中共开不开心。”
 </p>

  在台西藏人福利协会理事长丹增南达接受自由亚洲电台采访表示，身为西藏人很希望法王来台湾，但最重要的是中国那边有什么反应，他比较担心这个。如果法王个人他很想来，是因为在台湾有很多佛教徒。
  <br/>
  <br/>
  连续12年在台北举办“净韵三千”音乐会，为达赖喇嘛祝寿的台北市甘丹东顶显密佛学研修协会创会会长刘金钗，9日接受自由亚洲电台采访，证实有邀请达赖喇嘛访问台湾。
 </p>
 <p>
  <b>
   佛教徒团体递出邀请 “台湾疫情安全”
  </b>
  <br/>
  <br/>
  刘金钗：“今年疫情开始和他生日的时候，法王都有这想法，说他想来台湾，他这次又再次提出来，我觉得他有很大的需求，我希望他可以很顺利来到台湾，有很多信徒欢迎他来，希望因缘具足，做弟子的我们会多祈求，希望明年他的行程可以到台湾。”
  <br/>
  <br/>
  刘金钗说，该会每年举办庆寿会之前，都会到达兰萨拉邀请达赖喇嘛访问台湾，今年疫情没办法去，在今年7月透过视讯邀请达赖喇嘛，他才会提到明年要来。
  <br/>
  <br/>
  <b>
   达赖喇嘛首次赴台在李登辉任内促成 已11年无法入境
  </b>
  <br/>
  <br/>
  达赖喇嘛驻台代表达瓦才仁9日接受自由亚洲电台采访说：“李登辉总统去世时，就有很多台湾朋友想邀请达赖喇嘛来，有人可能做了邀请动作，但是因为疫情来不了，达赖喇嘛可能就说，台湾有一些朋友邀请，所以希望疫情过了有希望到台湾去。”

  达瓦才仁表示，邀请达赖喇嘛访台的很多，达赖喇嘛也算是给邀请者的一个回应。
  <br/>
  <br/>
  达瓦才仁说：“邀请人很多，有宗教团体，佛教、基督教、学校、NGO（民间社团）好多人都邀请过，有些人口头邀请、有人发邀请函到基金会，更多直接向达赖喇嘛办公室邀请。经过基金会传过去有十几个吧！”
  <br/>
  <br/>
  达瓦才仁表示，最关键还是台湾政府怎么去评估中国大陆的反应，有时台湾政府甚至不能够完全按照自己的意志决定。
  <br/>
  <br/>
  达赖喇嘛流亡印度逾60年，85岁的达赖喇嘛曾在1997年、2001年及2009年，李登辉、陈水扁、马英九总统任内三度访台，至今已11年没有访问台湾。2009年马英九执政，台湾发生重大风灾，达赖喇嘛受民进党南部县市邀请，远赴被土石流活埋近500人的小林村祈福，当时马英九避而不见，时任民进党主席的蔡英文则到法会见达赖喇嘛。不过，蔡英文上台4年多，达赖喇嘛仍无法入境。
