 <p>
  2017至2019年，新疆当局大规模抓捕维吾尔族、哈萨克族等少数民族穆斯林，并将他们判刑入狱。据多位被捕者家属说，他们的亲人根本就没有犯罪，他们有的是商人、农民，甚至是出租车司机。新疆伊犁地区一个村庄的村民举行生日聚会，结果21对哈萨克族夫妻共42人被捕。一年后，妻子无罪获释丈夫被判刑。
  <br/>
  <br/>
  现旅居土耳其的新疆维吾尔人约买尔.艾木杜拉和许多维吾尔族人的命运相似，他在新疆的两个哥哥在2017年大抓捕中，先是被送入所谓的职业技能教育培训中学，其后被判重刑。他说，自己的家乡在新疆库尔勒：“2013年，我与父亲一起去沙特，我父亲两个月后在沙特去世。然后我就在沙特麦地那上学，但每到假期都会回国探亲，2016年9月最后一次离开家乡去沙特，2017年7月来土耳其伊斯坦布尔。”
 </p>

  今年30岁的约买尔说，他的大哥是一个遵纪守法的商人，一直教育他不做不合法的事，且不参与任何非法活动。他二哥买买提.艾木杜，今年37岁，也是一位房地产商人。2017年却被送到培训中心，最近听说被判了15年。

   两名维吾尔商人被判刑15及25年

 <p>
  <br/>
  约买尔说，新疆当局重判其家人，又切断他和亲友的联系：“从2017年10 月份开始，我跟库尔勒的亲戚们联系不上了，他们都没有用之前所用的手机号、微信等通讯方式。我母亲叫海妮莎汗.塞麦提 ，今年60岁，是个不能照顾好自己的、坐轮椅的残疾人。现在关于她的情况什么都不知道。”
  <br/>
  <b>
   <br/>
   伊犁哈萨克族41人聚会半数被判刑
  </b>
  <br/>
  <br/>
  新疆哈萨克族也有不少人同样遭遇牢狱之灾。现居住在哈萨克斯坦阿拉木图的哈萨克人斯纳提对本台披露，2018年2月6日，他在新疆伊犁察布查尔县的叔叔加纳提别克.别克苏力坦等21人和他们的妻子共41人，进行生日聚会时被抓，其后妻子获释，丈夫被判重刑：“41人都是同学，同学聚会的时候，他们21人的老婆被关了一年，一年以后放了。她们的老公有的判18年，有的12年，还有的判21年。”

  斯纳提的叔叔加纳提别克是在法院未开庭审讯的情况下，被判刑13年。斯纳提说，被捕的41个人住在该县新疆生产建设兵团捷仁布拉克镇67团7连：“1988年出生的加纳提别克没或释，他开出租车，我们也不知道理由，抓的人很多。”
  <br/>
  <br/>
  在生日聚会上被捕的41人，除了他们的妻子获释，其余21名男子，4人因健康状况获释，17人被判刑。
