 <p>
  美国总统特朗普簽署行政命令，禁止美国企业或个人与TikTok母公司“字节跳动”及WeChat母公司腾讯进行交易。上述消息本周五（7日）令香港股市开盘大跌超过七百点，微信母公司腾讯股票急跌近一成。
  <br/>
  <br/>
  美国总统特朗普当地时间本周四簽署两项行政命令，由发布的45天后开始，分別禁止一切与微信拥有者腾讯，以及TikTok的母公司“字节跳动”有关的交易。该命令提及，微信及TikTok都会主动收集美国用户的资讯，可能被中国共产党获取，所以要针对两个程式采取积极行动，保护国家安全。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  <b>
   罗列TikTok及微信的漏洞
  </b>
 </p>
 <p>
 </p>
 <p>
  特朗普在行政令中写道，微信会自动撷取大量使用者讯息，这些数据得以让中国共产党取得美国公民的个人资讯，微信還使得中國可以在其公民訪美時對其進行監控；TikTok则是被用于散播假讯息，让中共从中受益，强调美国“必须采取强势行动来保护国家安全”。他表示：“这种数据收集行为可能会使中国共产党获得美国人的个人及专有讯息，还有可能允许中国追踪联邦政府雇员的位置……建立起用于敲诈的个人讯息档案，并进行企业间谍活动。”
 </p>

  <b>
   美中通讯及数据很快脱钩
  </b>
  <br/>
  <br/>
  广州商人王爱忠周五对本台说，美国将封锁中国应用程式和通信公司，这是美中数据脱钩的表现，但他提出了问题：“这是数据脱钩，美国既要从经济层面，包括贸易、企业，然后到现在的互联网数据脱钩。当前这个行政命令影响会很大，因为他涉及到美国的公民与企业不能跟中国“字节跳动”公司或腾讯一类企业有业务交易，这就涉及到中国的业务问题。”
  <br/>
  <br/>
  王爱忠举例说，在中国销售的美国苹果手机，未来是否不得使用中国应用程式：“苹果公司的苹果手机应用商店需要提供给中国大陆的很多应用程序，这都涉及到交易，如果这些也属于被禁止的内容，那么等于中国大陆的手机用户不能用微信一类的应用程序，可能会造成很大的影响。我认为这个执行的可能性不是太多，最终可能会在美国本土限制使用。”
  <br/>
  <b>
   <br/>
   字节跳动发声明将诉诸法院
  </b>
 </p>
 <p>
  “字节跳动”周五发表声明，表示对美国总统最新颁发的这项行政命令没有遵循正当法律程序“感到非常震惊”，并指该行政命令会破坏全球企业对美国法治承诺的信任。声明称，如果美国政府不能给予公正的对待，他们将诉诸美国法院。中国外交部发言人汪文斌当天表示，有关的企业按照市场原则和国际规则，在美开展商业活动，遵守美国的法律法规，美方借口国家安全，频繁滥用国家力量，无理打压非美国企业，这是赤裸裸的霸权行径，中方坚决反对。
  <br/>
  <br/>
  毕业于山东大学的学者司令认为，美国此举表明要彻底解除与中国的联系。他对本台说，美国国务卿蓬佩奥不久前在一次演说中，已经明确的表明要改变过去四十年多年与中国进行接触及对话策略：“对中国，看他的行动，而不再看他的言论。因为中国在与美国合作问题上，还没有展现出诚意。这也是美国政府向中国施加如此巨大压力，全面制裁中国在美的红色资本公司，全面限制中国军工企业渗透美国通信及科技领域的重要原因。”
 </p>

  中国早前高调宣布北斗三号全球卫星导航系统建成，目前已经覆盖亚洲和欧洲部分地区。司令对此表示：“中国已经完成了北斗卫星导航系统的全球组网部署，虽然美国国家航空航天局表示对北斗星通讯对人类作出贡献方面表示欢迎，但是美国的外交政策已经表明对于这个微信导航系统在全球窃取各地方位置数据，以及北斗系统和5G网络融合的时候，给世界各国特别是自由世界带来前所未有的威胁。”
  <br/>
  <br/>
  特朗普簽署行政命令，禁止美国企业及资金与腾讯等交易，拖累香港股市开盘大跌逾七百点。恒生科技指数急挫百分之4， 腾讯跌近一成，京东跌百分之三，阿里巴巴跌半成。
  <br/>
  <br/>
  <b>
   学者分析新冷战将持续数十年
  </b>
  <br/>
  <br/>
  北京独立学者查建国对本台说，中美新冷战将是一场十年到几十年的持久战。但因其本质是意识形态价值观之争、是政经体制之争、是不同生活方式之争,因而结局一定是零和而不是双赢。他说，对于中国，美国手中的牌还很多：“这两三个月里美国肯定会连连出招，步步惊心，极限施压，美国大选以后，对华基本态势不会变，可能节奏、形式有点变化。”
  <br/>
  <br/>
  查建国表示，中美新冷战双方会有攻有守有斗有和，斗而不破,破而不裂地长期共处，但其结局一定是中共爱说的一句话，不是东风压倒西风，就是西风压倒东风。
