 <p>
  中国首次延后召开的两会将于5月21日开幕。中国国家主席习近平会因为疫情危机会在会上招致党内批评与挑战吗？外国观察人士认为几率不大，并指出美国总统特朗普的表现，帮助习近平摆脱困境。
  <br/>
  <br/>
  2020年5月的北京，特别不一样。因为新冠肺炎疫情，俗称“两会”的中国政治大事、全国人民代表大会与中国人民政治协商会议全国委员会延迟近两个半月将在五月底召开。疫情加上两会，让北京的气氛更紧绷。
  <br/>
  <br/>
  英国杂志《经济学人》北京分社社长任大伟（David Rennie）19日参与华盛顿智库卡内基国际和平研究院（Carnegie Endowment for International Peace）一场网络研讨会时，分享了他几个月来经历的“中国式防疫”。生活在北京，官方以防疫之名， 能轻易取得个人行踪资讯，外国人也同样受管理。手机里的健康绿码，是他出入住家小区的必备通行证，到处都要量体温。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <b>
   习近平摆脱治理不善困境 英驻华记者：特朗普帮的忙
  </b>
  <br/>
  <br/>
  疫情改变了包括中国在内世界各国人民的生活，欧美仍在艰辛对抗疫情，试图缓和确诊与死亡人数的曲线。任大伟说，中国现在一切从严防疫，追求的是“消灭曲线”。在他看来，中国国家主席习近平已经度过政治上会动摇根本的时期，把习近平“扶上马、送一程”的，则是华盛顿。
  <br/>
  <br/>
  “本来习近平可能要面对的公众质疑包括：我们（中国）怎么搞砸了中美关系？中国在世界怎么这么不受欢迎？我们在疫情初期怎么没处理好？但是，我很遗憾必须说，是美国总统特朗普的防疫表现，帮习近平大忙，帮习近平摆脱了困境”。任大伟说。
  <br/>
  <br/>
  他指出，中国媒体作为官方的宣传工具，叙事内容都重复说是中国人民的卓越牺牲，帮助世界争取防疫时间，但西方民主制度的国家却失败了，中国共产党以此辩称，一党专政是适合中国的正确道路。任大伟说，这当然不是事实，民主台湾的防疫成功就是最好例证，但是，在中国强烈管控舆论，有计划宣传所谓西方民主的失败，是有一部分的中国民众认可这种说法。
 </p>

  <br/>
  <br/>
  <b>
   美作者佛里曼：华盛顿自弃领导力 助长中国民族主义
  </b>
  <br/>
  <br/>
  同场参与讨论的美国知名作者佛里曼（Tom Friedman），一向不掩饰对特朗普的批评。他说，美国的领导力过去来自于协调、整合及强大的科学基础，而这一次新冠肺炎疫情中，现任政府完全没有展现美国的强项，就更不要说美国在改革世界卫生组织上采取的做法，凸显特朗普厌弃多边主义的一贯立场，也给中国民族主义增添柴火。
  <br/>
  <br/>
  “如果我是美国总统，我会请新加坡帮忙、与世界卫生组织合作，主导研究疫情流行起源。新加坡是受中国与西方国家都信任的国家，可以做得好，美国可以借此团结并领导全世界，对抗中国官方的疫情与病毒起源论述，这能让中国的改革开放派在内部更有筹码。然而，特朗普反其道而行，变成美国独自对抗中国，让中国的民族主义团结在习近平周围。”
  <br/>
  <br/>
  佛里曼不只对特朗普不满。他对中国钻世界贸易组织的漏洞，将国家资本主义养成巨大怪兽也多有批评，他说，他支持特朗普勇于点出中国在世界贸易体系中的问题，但不支持特朗普解决这个问题的方法，例如一上台就直接撕毁了让中国官方曾经很紧张的《跨太平洋伙伴关系协定》（TPP）。
  <br/>
  <br/>
  卡内基国际和平研究院资深研究员米勒说（Aaron David Miller），新冠疫情成为二战后改变世界的最大的一场危机，后疫情时代，势必将影响整个地缘政治格局。
  <br/>
  <br/>
  <b>
   全球领导力真空 中国能补缺？
  </b>
  <br/>
  <br/>
  在派驻北京前曾驻华盛顿的任大伟形容，他观察，特朗普并不是美国传统典型的对中鹰派，对中国的人权与民主并不在意。贸易上，他则是房地产商人思维，认为美国就是豪宅，但被各国强占，富裕的中国，更以便宜的租金租房子、占美国好处，华盛顿眼中只看见自己。
  <br/>
  <br/>
  但他说，现在的世界格局不是1940年代的二战后，美国留下的世界领导力真空，中国无法接下重担，“中国不具备道德上的全球领导力”。
  <br/>
  <br/>
  任大伟和佛里曼都说，全球领导力真空导致的问题，已在这次疫情中浮现，但两人短期间都看不到有替代国家能担起这个责任。佛里曼指出，美国对外的领导力需要有内部团结为基础，在这一点上，他是悲观的。
 </p>
