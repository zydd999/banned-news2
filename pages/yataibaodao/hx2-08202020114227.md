 <p>
  网上盛传中国驻基里巴斯大使唐松根在当地接受走过“人体地毯”欢迎仪式的照片，引发各界批评。不过，自由亚洲电台专访马英九主政时期，台湾派驻基里巴斯大使何登煌时他表示，自己在基里巴斯也曾经历过这个仪式，是当地人对客人的最高礼遇。
  <br/>
  <br/>
  去年9月20日，基里巴斯和中华民国断交，结束16年邦谊，并与中华人民共和国复交。中国驻基里巴斯大使唐松根今年5月履新。网上流传一张照片，显示他在基国马拉凯环礁（Marakei），踩在一排趴在地上的当地人的身上，当地人扶着他走过所谓“人体地毯”。这张照片招致网民的批评。
  <br/>
  <br/>
  其实，唐松根并不是第一个在基里巴斯受到这种特殊仪式欢迎的外国客人。自由亚洲电台采访到2009年5月至2012年1月，马英九主政时期，担任中华民国驻基里巴斯大使的何登煌，他表示，自己派驻基里巴斯约1年后，2011年6月受邀访问基国规模约第四大的外岛BEKIN，也接受过这种被当地人称为最高的礼遇。
  <br/>
  <br/>
  何登煌：“我本人也被安排过这非常特别的礼遇。一个部长跟我讲，基里巴斯这种礼遇只有对特殊的人，我是第二个，第一个是英国女皇，她那时去访问。”

   何登煌：被他们崇敬的人踏过肩 让他们感到荣耀
   <br/>
  </b>
  <br/>
  何登煌提到，基里巴斯曾是英联邦成员国，英国女皇当时等于是他们国家元首。
  <br/>
  <br/>
  何登煌：“我看报导说中国大陆新大使也有这种待遇，（被人批评）这不人道。我当初也不肯，他们说你一定要，不然当地人会非常失望。这是相当于元首礼遇，是表示他们对某人的非常欢迎和推崇的方式。”
  <br/>
  <br/>
  何登煌提到，那时趴卧在地的都是大人、男性，如果是小孩他不会踩。“一排下来每一个卧倒，你要踩在他肩膀和背部，大约有上百个人，从下飞机一直走到会场，都是大人、男性，都躺着，当时我说不能这样走，太没有人性。他说不能，你不这样，他们会很难过。两边都有人帮忙扶着慢慢走过去，很长，一、两百公尺。”
  <br/>
  <br/>
  当地人如何看待这个仪式？何登煌说：“他们会感到非常受到重视、荣耀。一个大使、元首能够踩在他们背上，是对他们最崇敬的人物的特殊的礼遇，联合国秘书长去访问也没有这种礼遇。”
  <br/>
  <br/>
  何登煌提到，当时在基里巴斯有欧洲、新西兰、澳大利亚等国大使，古巴还设有医疗团在当地，他们的大使都没有受到这种礼遇。“我相信中国可能有给他们特别的什么东西，不然他们不会这么做。”
  <br/>
  <br/>
  为何基里巴斯人用这个仪式来欢迎他？何登煌说，基里巴斯人口13万左右，有30多个岛，他可说“有求必到”，只要当地人邀他到任何一个岛，不论坐船、坐飞机，他都会尽量陪他们去。很多地方没有房子，也跟他们一起露天而睡，就这样培养感情，从来没有花什么大钱，就一些小钱捐助学校买课桌椅，从头到晚日夜陪他们。

   何登煌：“基国人感谢我的奉献”
  </b>
  <br/>
  <br/>
  何登煌说：“他们知道没有一个大使像我这么亲民，陪他们跳舞、陪他们喝酒、陪他们一起工作、教他们养鸡、养猪、煮中国菜，打成一片。再穷的地方我都会去。他们还有一个学校用我的名字（命名）。我估计中国去一定把它改掉。我说，不要，要用台湾、中华民国作为学校的名字，他们说不、一定要用你的。”

  受到“人肉地毯”的礼遇，何登煌说，外交部始终不知道这件事，台湾前后任大使都没有受到这个礼遇。“没有跟外交部汇报。我不要宣扬自己，这个事情外交部不知道 。”

  何登煌话锋一转提到，他当时就知道中国大陆要拿台湾六个南太平洋岛国。第一个会是所罗门，第二个会是基里巴斯，因为索罗门拥有丰富的矿藏林业。而基里巴斯具有非常重要的战略地位，能监控美国飞弹试射、人造卫星发射。

  <b>
   台湾特使团前往基里巴斯 并未见过人肉地毯
  </b>
  <br/>
  <br/>
  2018年2月台湾原住民族委员会主委、阿美族人夷将‧拔路儿Icyang‧Parod，以总统特使身分率团访问瑙鲁和基里巴斯，随行的有民进党籍立委、阿美族人Kolas Yotaka，和国民党籍立委、赛德克族孔文吉。三位都是原住民族，出访被台湾原住民族视为“兄弟姐妹”的南太平洋友邦。自由亚洲电台20日采访三人都表示，不曾听过有“人肉地毯”这种仪式。
  <br/>
  <br/>
  回忆出访基里巴斯，夷将拔路儿说：“还是有他们高阶官员接待，我还因为是总统特使，代表总统晋见他们当时马茂总统，我们有进他们总统府去看，但是他们也没有用这样的方式来接待。最多就是用歌舞，南岛民族国家跟我们是同一个语系，都是用南岛语族的歌舞迎接。”
  <br/>
  <br/>
  对于中国大使接受“人肉地毯”的欢迎式，夷将还说：“实际为什么要这样接待，我不便表示意见，即使他们主动这样接待，中国官员你作为泱泱大国大使，你应该在第一时间就要婉拒。”
  <br/>
  <br/>
  孔文吉也说：“怎么可能这样欢迎？我们一切都是官方的接待，两年前我们特使团没有这样。我还第一次听到什么人肉地毯，我们绝对没有，也不准许这样做，非人道、不可理喻。”
  <br/>
  <br/>
  当时以立法院南岛国会议员联谊会会长随团出访基里巴斯的Kolas Yotaka表示，她不了解当地有这样的仪式，不过，如果当地真有这样的仪式，要尊重当地的文化。再者，这是两国之间的活动，外界也无从置喙。此外，外交官要尊重、了解当地文化，即便要共同进行对方的仪式，也要得宜，这是基本的对太平洋岛国的尊重。

  <b>
   台湾看法两极
  </b>
  <br/>
  <br/>
  Kolas Yotaka提到，2018年出访，外交部带了一个表演团体要去联谊，但是突然在台湾出访团落地前一、两天，基里巴斯发生重大船难，非常多人丧生。
  <br/>
  <br/>
  Kolas Yotaka：“特使团即便带表演团浩浩荡荡，准备了多少舞码，要跟大家友善连结，一旦发生这样的事，只能说停就停，这是政治判断，反而要去哀悼，用不一样方式建立外交关系。”
  <br/>
  <br/>
  Kolas Yotaka说，即便对方仍客气表示 “可移至室内”举行，但台湾特使团还是感受到这是不宜进行的活动，所以决定取消，一切从简。
  <br/>
  <br/>
  Kolas Yotaka以此强调，两国外交关系其实是可以讨论的。那是一个什么样的仪式？以及在当地的意义？一般在什么情况下举行？适不适合用在欢迎一个外国大使参与？而且是在没有拖掉鞋子的情况下进行？ 先遣部队有没有先了解？这些外交工作都需要先沟通、先做足功课。
  <br/>
  <br/>
  Kolas Yotaka说，她作为南岛民族一分子，现在的确很多评论，西方人觉得有些南岛的仪式很残忍。但如果是岛上的族人自己提出的仪式，还是要尊重当地人。作为外交官要注意的是有没有用善意，以当地人真正尊敬而且可接受的方式，完成这仪式。

  <b>
   台湾外交部：全世界自有公评   中国外交部：客随主便
  </b>
  <br/>
  <br/>
  台湾外交部发言人欧江安20日在例行记者会回应中国大使踩踏基里巴斯“人肉地毯”的新闻时表示：“相关照片和影片大家都看到了，我想我们的震惊程度跟大家都是一样的。这个不是我们认识的基里巴斯，我们也不认识基里巴斯有这样的欢迎作法。在台湾我们也不会这样对待我们的友邦跟人民。至于中国如何对待他们的新友邦和人民，全世界自有公评。”
  <br/>
