 <p>
  美国国务院6月10日向国会递交《2019年国际宗教自由报告》，再度批评中国打压宗教自由。国务卿蓬佩奥指控中国想借美国社会近来的种族平权游行而指责美国，根本是混淆视听，美中两国对待人民所享有的自由的方式，“毫无可比性”。
  <br/>
  <br/>
  “美国绝非完美国家，但我们始终精益求精，试着改进。” 蓬佩奥（Mike Pompeo）在公布《2019国际宗教自由报告》的记者会上的开场白，摆明是要说给北京听。
  <br/>
  <br/>
  他更主动提到，中国近来利用美国社会因非裔弗洛伊德（George Floyd）之死而引发的骚乱和抗议示威，试图为自己的独裁政权擦脂抹粉；美国人民享有言论自由、法治与和平集会权利，中国人民根本都没有，无法相提并论。
  <br/>
  <br/>
  美国记者在会上质问蓬佩奥，上周在白宫前发生的集会示威，遭执法人员强势清场，媒体记者甚至遭人身攻击。美国是否还有道德高度关注世界人权情况？蓬佩奥直面挑战，没有回避，还再次提起1989年的六四事件。
  <br/>
  <br/>
  他说，“像现在，你们能直接质疑我，但那些国家不允许记者进行采访工作......我们这里发生弗洛依德这样的悲剧，政府必须出面回应民众诉求，包括地方执法部门及司法部，都特别针对执法行动提出改进方案......31年前的天安门广场，有上千人遭屠杀，但他们（中国）仍在压迫记者，中国人民无故消失，美中两国根本就不相同。 ”
 </p>
 <p>
 </p>
 <p>
 </p>
 <p>
  <b>
   中共侵害香港自治 宗教自由恐遭波及
  </b>
  <br/>
  <br/>
  国务院今年报告中以一百一十五页篇幅详述中国迫害宗教自由的斑斑劣迹，包括政府主导下大规模关押维吾尔人、镇压西藏人、打压佛教徒、法轮功与基督教徒，被列为美国“特别关注”的国家，更针对新疆、西藏、香港与澳门各有专章介绍。
  <br/>
  <br/>
  值得注意的是，这份报告是在中国提出制定“港版国安法”之前撰写，却已表达中共收紧对香港的管治，恐危害宗教自由。
  <br/>
  <br/>
  报告指出，尽管香港基本法及相关法律都明确规定，保障宗教自由与宗教集会权利。但一些外国宗教团体认为，中国若进一步侵害香港自治，香港的宗教自由前景令人担忧。
 </p>

 <p>
  <b>
   报告关注新疆再教育营
  </b>
  <br/>
  <br/>
  关于新疆， 在世界各国的关注与压力下，中共去年底声称已经关闭所谓的“再教育营”，不过，美国国务院负责宗教自由事务的无任所大使布朗巴克（Sam Brownback）在记者会上，反驳中共说法。
 </p>
 <p>
  他说：“我们没有证据显示，维吾尔人已被释放。就算是真的，他们仍活在中国创造的'虚拟警察国家'中...... 我们最担忧的是，当维吾尔人离开监禁营时，他们还得面对摄像头监控，社会信用体系的紧箍咒，这些都是压迫，尤其当他们想要实践自己的信仰时，官方甚至藉由手机的移动通讯系统监控。 ”
 </p>
 <p>
  报告点名，中共持续以三股邪恶势力、“种族分离主义、宗教极端主义与暴力恐怖主义”的名义，迫害新疆穆斯林的信仰与宗教自由；美国政府估计，2017年4月以来，中国政府在新疆专门建造的拘留营，已关押超过100万维吾尔人及其他少数民族。
  <br/>
  <br/>
  而台湾持续建立对穆斯林友好的环境，受到肯定。
  <br/>
  <br/>
  报告引述中国回教协会秘书长马超彦表示，台湾每年增设两成穆斯林友善酒店及清真认证餐厅，越来越受到穆斯林旅客欢迎。
  <br/>
  <br/>
  不过，报告也指出，台湾仍未改善家庭帮佣与看护不适用《劳基法》的现状，导致主要来自印尼与菲律宾约二十五万的在台外籍移工，无法在周末参加穆斯林或天主教的礼拜仪式。美国对这一涉及外籍劳工权益的议题，表达关切。
 </p>
