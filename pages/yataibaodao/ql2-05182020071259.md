 <p>
  近期武汉各小区排队等候核酸检测的居民担心受到感染，批评当局安排不当，容易造成交叉感染。另外，吉林省吉林市本周日凌晨起，所有个体诊所、门诊部暂停营业。
  <br/>
  <br/>
  根据武汉市疫情防控指挥部的要求，当地开展全员新冠病毒核酸筛查“十天大会战”，全市1100万人将接受核酸检测。据当地居民说，该市盛世东方小区有六千多人，居民排队轮候数小时，到下午才发现检测人员只带了600多个检测试剂，并且检测结果上面连标签也没有，分不出来是谁的，还每次把20多个的检测结果丢一个瓶子里。愤怒的业主打了市长热线后有领导过来了解情况。
 </p>

  当地居民本周一上传一段视频，指工作人员做事不规范。该视频显示，接受核酸检测的唾液标本，未独立封存，而是把几个人的标本共同存放在一个塑料瓶内。一居民手持两个装有多个棉签的塑料瓶，她说：“在一个盒子里面，没有编号，全部放在一个盒子里面，谁是谁的，都不知道。这种（测试）做了有什么意义，装在一个瓶孜里，你们看一下。”
 </p>
 <p>
  <br/>
  <b>
   标本棉签混装一个塑料瓶子
  </b>
  <br/>
  <br/>
  武汉居民李书平对本台说，部分检测站确实存在这种情况：“这个是不规范的。我们小区做，每一个人有个瓶子，上面有条码。每一个人都有一个条码，共有两个条码，一个条码在登记表上，第二个条码贴在装棉签的瓶子上，每一个一个瓶子。问题最关键的不是在这里，测试一回，他起不了任何作用。因为我们的核酸测试盒成功率百分之三十左右。”
 </p>

  另有居民抱怨一家三口排队等候三个小时，他们可能暴露在污染环境中，因此担心被传染。一位女士批评政府做事不动脑筋：“你知不知道本来我们这些人平时是不出门的，你突然要我们都出门，扎堆啊，检测过程中间，如果交叉感染怎么办，检测你要张嘴巴，医护人员在短时间内，她怎么样去换手套，消毒，还有人咳嗽，打喷嚏，你怎么样及时消毒，这些你都没有想到，一千多万人做检测，脑残，脑残的决定。”
 </p>
 <p>
  青山区居民孙涛对本台说：“武汉市的核酸检测是有点乱。昨天礼拜天上午，我也去做和核酸检测。我当时也提出了不少批评意见，乱糟糟的，只能用三个字来形容。”
 </p>

  <br/>
  <b>
   网民估算武汉有五、六十万人感染
  </b>
  <br/>
  <br/>
  有1100多万常住户籍的武汉人，究竟有多少是无症状感染者，最近互联网上流传一篇笔记，说大约有二十分之一人，也就是有五、六十万名无症状感染者。
  <br/>
  <br/>
  有武汉居民分析说，当局要求当地居民都进行核酸检测，若非打算处理一批不合格的核酸检测剂，以便废物利用，并附上其现金价值出售，就是疫情有死灰复燃之势。
  <br/>
  <br/>
  <b>
   中国副总理抵吉林全面封城封闭管理
  </b>
  <br/>
  <br/>
  另外，除东北三省、湖北省武汉，上海也新增一本土病例。《新京报》报道，吉林省新增的2例确诊病例，均为先前吉林市丰满区和舒兰市确诊病例的密切接触者。

 <p>
  <br/>
  继吉林市上周三（13日）宣布封城、关闭边界后，本周日零时起，吉林全市所有个体诊所、门诊部暂停营业，不得开展任何诊疗活动。所有发热病人一律到发热门诊就诊。该市丰满区全面实施严厉的封闭管理、地毯式排查。
  <br/>
  <br/>
  吉林一网民本星期一说：“现在又是吉林省，现在是吉林市，新冠疫情昨天起，吉林市已经采取了全面的封控措施，禁止任何聚餐，火车站停运，出市区必须持有检测报告等。不到一周时间，吉林市新增确诊病例已超过20例。”
  <br/>
  <br/>
  中国国务院副总理孙春兰已于上周三从黑龙江抵达吉林，当局担心疫情向辽宁扩延。北京及吉林省政府派出500余人，已在17日陆续抵达吉林市，投入抗疫。
