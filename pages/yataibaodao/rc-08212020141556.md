 <p>
  美国总统选举已经进入最后冲刺阶段。民主党全国代表大会（Democratic National Convention,  DNC）周四晚间刚落幕，前副总统拜登接受提名、正式成为民主党总统候选人，将挑战现任总统特朗普。另外，民主党通过新的《党纲》，除了对中国的各项作为直言不讳，与4年前的《党纲》相较，民主党这次在关于台湾的议题上拿掉了“一中政策”，只提《台湾关系法》。这释放出了什么讯号？中国外交部又如何接招？
 </p>
 <p>
  <br/>
  <br/>
  美国政党的政治大事、民主党与共和党的全国代表大会每四年举行一次。在2020这一年，见不到万人齐聚、气球彩带齐飞的热闹场面，这全拜新冠肺炎疫情之赐，让两党都只能改在虚拟的网路世界举行。周四傍晚进行接受提名演说的拜登，在批评特朗普疫情防控失职失能时，提到中国。
  <br/>
  <br/>
  拜登说：“我们将生产我们美国所需要的医疗用品和防护设备，而且，我们就在美国制造。这样，我们就再也不需要受到中国和其他国家的摆布。”
  <br/>
  <br/>
  外交政策在美国的总统选战里是冷门话题，多数美国民众关心的仍是和民生较直接相关的，如医疗保险、就业与社会安全保障的退休金制度。拜登接受提名的演说当晚，没有针对中国政策详细表态，不过，《民主党党纲》今年有些修正，周四晚间通过的内容引人注意，连北京都关心。
  <br/>
  <b>
   <br/>

 <p>
  与2016年《民主党党纲》相较，在台湾议题上，民主党今年删除了“一中政策”（One China Policy），只提《台湾关系法》，强调“将遵守《台湾关系法》，继续支持在符合台湾人民意愿和最大利益的情况下，以和平方式，解决台湾海峡两岸议题”。
  <br/>
  <br/>
  关于香港，《党纲》强调，“民主党会起身捍卫港人的民主权利，全面执行《香港人权与民主法》，包括制裁损害港人自由的官员、金融机构、企业及个人。2016年的《党纲》，没有提到香港。
  <br/>
  <br/>
  民主党新版本《党纲》还说，将信守《维吾尔人权政策法》。
  <br/>
  <br/>
  新版本《党纲》概述民主党的对华政策，除批评中国的不公平贸易行为，也谴责中国操纵货币及窃取知识产权；民主党会团结全世界的盟友，以清晰、强硬及一致的方式，抵抗中国在经济、安全及人权领域上引人关切的行为，但民主党不会采取不利己的单方面关税措施，陷入“新冷战”的陷阱。
  <br/>
  <br/>
  《党纲》也提到，中国带来的挑战主要不在军事，但民主党仍将坚决回应中方任何的“威慑及侵略”，致力倡导全球的航行自由承诺，抵御中国在南海的“军事恫吓”。
  <br/>
  <br/>
  <b>
   党纲仅具象征性　美学者预期选后会重申一中
  </b>

 <p>
  和中国共产党是刚性的列宁式政党不同，美国两党都是柔性政党，不论在国会运作或取得行政部门的执政权后，政党对党员没有严格的党纪约束的成文规定，党员可来去自如，政党机构本身只是个选举机器，更不用说每四年公布一次的党纲，仅具象征意义。
  <br/>
  <br/>
  华盛顿智库战略暨国际研究中心资深研究员葛来仪（Bonnie Glaser）告诉本台，“若拜登胜选，预期拜登的执政团队就会重申'一中政策'，因为只有'一中政策'才能有更弹性的解释与运用空间。”
  <br/>
  <br/>
  <b>
   赵立坚老调重弹提一中原则
  </b>
  <br/>
  <br/>
  而中国外交部发言人赵立坚在例行记者会上对此的回应，也仅是老调重弹。
  <br/>
  <br/>
  他说，“台湾问题是中美关系中最重要、最敏感的问题。'一个中国原则'是中美关系的政治基础，也是国际社会的普遍共识。我们敦促有关方面切实恪守'一个中国原则'和中美三个联合公报规定，慎重妥善处理涉台问题。”
  <br/>
  <br/>
  <b>
   共和党沿用旧党纲　批习近平重返毛泽东路线
  </b>
 </p>

 <p>
  当晚待在白宫的特朗普也没闲着，他透过推特发文批评对手拜登从政多年，却什么都没做，“光说不练”，更立刻推出针对拜登的负面选举文宣，还点名提到中国。
  <br/>
  <br/>
  广告词提到：“乔·拜登，在华盛顿政坛47年的从政资历，却是47年的失败，签署灾难般的贸易协定，把美国原有的就业机会奉送给墨西哥与中国。”
  <br/>
  <br/>
  美国共和党全国代表大会（Republican National Convention）将于24号开跑，也将在线上举行。不过，共和党仍未公布活动行程安排细节。特朗普日前透露，有意在白宫发表接受提名的演说，若真落实，将创下近代美国总统选举史上先例。
  <br/>
  <br/>
  共和党6月已决定今年《党纲》沿用2016年版本。
  <br/>
  <br/>
  《2016年共和党党纲》（2016 Republican Platform）首度纳入美国的对台“六项保证”。在中国议题上，共和党当时就展现强势立场，呼应了特朗普这4年对北京采取的姿态。
  <br/>
  <br/>
  《共和党党纲》指出，中国领导人要重返“毛泽东路线”。尽管美国仍欢迎中国留学生、观光客和投资，让他们亲身体验美式民主运作，但共和党反对中国政府控制下的学术和文化行为，并呼吁美国各界与中国的渗透行动划清界线。
  <br/>
  <br/>
  共和党当时也谴责中国在西藏和新疆的“文化灭绝”行径、“野蛮控制人口”，并以“举国之力且充满敌意”的方式占领美国企业。
  <br/>
  <br/>
  1982年，共和党的里根政府对台湾提出的“六项保证”是指：一、美国不同意在对台军售上，设定结束期限；二、美国不同意中共提出在对台军售上、要事先与北京咨商的要求；三、美国无意在台北与北京之间扮演任何调停人的角色；四、美国不同意修改《台湾关系法》；五、在台湾的主权问题上，美国的立场没有改变；六、美国无意施压中华民国与中共进行谈判。
  <br/>
  <br/>
  <b>
   共和党沿用旧党纲　骂奥巴马文字殃及特朗普
  </b>
  <br/>
  <br/>
  不过，共和党沿用旧党纲没有修正，引发美国媒体调侃。
  <br/>
  <br/>
  《纽约时报》指出，共和党当年许多批评前任总统奥巴马的文句，现在应用到特朗普身上，依旧适用。
  <br/>
  <br/>
  共和党的2016年党纲写明：众所周知，互联网的存活岌岌可危，最大的凶险就源自白宫，目前的白宫主人发动攻势，想让互联网臣服于政府官员，国内国外皆然；“现任总统”及其行政单位导致国债巨幅增加、伤害美国与盟邦之间的邦谊。
  <br/>
  <br/>
  共和党的这份党纲将延用至2024年。
