 <p>
  美中关系越来越坏，太平洋两端的国家，要如何在惊涛骇浪中平安自保？台湾陆委会副主委邱垂正6月19日说，台湾的自由民主需要美国在内的国际社会的坚定支持，但他也说，美国持续维持与中国共产党的接触交往，极为重要。
  <br/>
  <br/>
  在华盛顿眼中，太平洋再大，现在显然很难容得下美中两国的和平相处。美国总统特朗普星期四就放话，强调两国仍可能“完全脱钩”。北京方面从学界到官方忙着灭火，但华盛顿这一头，关注美中关系的学者们，几乎没有人认为，太平洋的惊涛骇浪，能在短时间平稳下来，海洋两端的国家该如何是好？
  <br/>
  <br/>
  在华盛顿智库战略暨国际研究中心（CSIS）19日举办的一场视频研讨会上，邱垂正就以英语谈到身处汪洋中的台湾观点。
  <br/>
  <br/>
  “毫无疑问地，（美国与国际社会）坚定支持民主台湾与台湾维持区域稳定的政策，就是对抗中共极权专政与侵略扩张的最好做法，同时，我们认为，美国与国际社会持续维持与中国共产党的接触交往，也极为重要。”邱垂正说。
  <br/>
  <br/>
  他强调，台湾的总统蔡英文在就职演说中重申“和平、对等、民主、对话”，不只是台湾在两岸关系上的原则，台湾的两岸政策也符合区域国家的最大利益。
 </p>

  <br/>
  <b>
   美中秀肌肉 台湾压力山大
  </b>
 </p>
 <p>
  在北京眼中，台湾涉及中国的核心利益，总强调台湾是美中关系里最敏感的议题。
  <br/>
  <br/>
  美国国务卿蓬佩奥（Mike Pompeo）日前与中共中央外事办主任杨洁篪在夏威夷闭门会谈后，中国外交部立刻以发言人赵立坚书面答记者问的形式，通篇强调两人会谈的重点包括台湾、香港与新疆议题；美国国务院的简短声明中则强调，双方关系必须是全面互惠，并重申美方看重处理新冠肺炎疫情，要有完全的透明度与信息分享。
 </p>
 <p>
  美中两国外交官各说各话的同时，两国的军方近来在台海上空也密集互秀肌肉。中共频繁派军机扰台后，美国罕见展现高调姿态，日前刻意公布海军运输机穿越台湾领空的航迹讯号，台湾天空一时间成为美中互相亮剑的舞台。
  <br/>
  <br/>
  同场参与讨论的台湾愿景基金会执行长赖怡忠与前白宫国安会亚洲事务主任何瑞恩（Ryan Hass），都提到中共总书记习近平可能拿台湾当缓解内部压力的出口。
  <br/>
  <br/>
  何瑞恩说，“随着中共2022年二十大的召开将至，疫情的蔓延显然会使中国经济减速，美中关系又显得失控，这些都是习近平的弱点；如果习近平在内部真有反对者，我认为，他们会批评习近平，在处理两岸关系上，他交不出完整的成绩单。”
  <br/>
  <br/>
  赖怡忠则指出，“中共对台的军事准备由来已久，我会形容这像是'大象走路'般，他步步进逼、无法避免。我认为，明年会是非常关键的时刻，可以来观察中共在对台政策上，究竟为了他期望的'统一'，愿意付出多少代价。”
 </p>
 <p>
 </p>
 <p>
  <br/>
  <b>
   中国最大的挑战是台湾的民意
  </b>
  <br/>
  <br/>
  台湾的在野党国民党19号公布两岸新论述，肯定“九二共识”过去的贡献，并提出4大支柱的建议，建构和平稳定的台海新关系，向中国大陆喊话强调坚持中华民国主权与保障自由、民主、人权。
  <br/>
  <br/>
  华盛顿智库史汀生中心（Stimson Center）中国项目主任孙韵在会中分析，尽管中国对国民党谈自由民主会有不满，但她不认为中国现阶段会把国民党归类为“反统促独”。
  <br/>
  <br/>
  她说，在两岸关系上，对中国更大的问题不是国民党路线怎么调整，而是在于怎么让台湾的民意转向愿意支持统一。她认为，“这非常困难”，尤其是香港议题上，中国推出“港版国安法”的做法显示，“中国自己破坏了'一国两制'的承诺，这对台湾民意有负面影响。”
 </p>
