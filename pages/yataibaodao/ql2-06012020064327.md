 <p>
  江西景德镇一家庭教会，上周被当地宗教局取缔，公安带人将教会用品搬走。另外，广州广福家庭教会，近期被公安上门检查及安装摄像探头，并警告不得聚会。
  <br/>
  <br/>
  上周一（5月25日）景德镇市浮梁县蛟潭镇一家庭教会被该县宗教局下令取缔。当天上午，蛟潭镇公安，镇政府等人进入教会拆除台前墙上写有“神爱世人”的牌子，又清理教堂内的凳子及教会用品。本台收到一段两分多钟的视频，显示了信徒与警察争论的过程，其中一女子与一名自称是蛟潭派出所的男警官对话，众信徒在一旁拍摄：
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  “这是我们这里所有人的家。你们这里因为没有通过民宗局的同意，没有办理合法的手续，所以你们这里是个非法的据点，现在我们依法予以取缔。
 </p>

  <b>
   清理家庭教会信徒难再聚会
  </b>
 </p>
 <p>
  在场的信徒要求公安出示证据，公安回答：“我是蛟潭派出所的吴宗昆（音） 这是我的警号，你们如果有什么意见，可以到县公安局投诉我，我们是依法在这里办事。”
  <br/>
  <br/>
  该名吴姓公安的身后，约两百平方米的教堂内用品已被清空，做祷告用的椅子被抬到墙边堆成一排，正面黑板上方的横幅和标语全被撕下。公安对信徒说：“你们这个场所没有经过民宗局的登记，属于非法宗教场所 所以我们依法予以取缔”。
  <br/>
  <br/>
  一位信徒立即回应该名吴姓警官：“那我也在这里说 你们没有执法证，也没有执法文件就随意把我们这里的东西拆了。”
 </p>

  <b>
   九江等地多家教堂及十字架被拆
  </b>
 </p>
 <p>
  本台记者致电浮梁县民族宗教事务局办公室，但始终无人接听。据当地信徒说，蛟潭镇有多个聚会点，信徒数百人。而这次被官方取缔的教会有信徒一百多人。
  <br/>
  <br/>
  除此之外，江西九江，南昌上个月有多个教堂或十字架被强拆。据对华援助新闻网引述多位信徒说，4月3日，九江市都昌县一名政府官员要求该县三自教会负责人拆除十字架。10日，抚州市一处有约二十年历史的三自教堂被拆毁。官员指该教堂无土地证，没有批文等。16日，南昌新建区西山镇政府官员责令当地三自教堂负责人自行拆除十字架，竖立五星红旗。

  <b>
   广福教会门口被装摄像探头
  </b>
 </p>
 <p>
  在广州，广福家庭教会上周三被4名警察搜查。该教会牧师马超对本台说，当时他不在现场，但有同工被盘问：“上个礼拜来了几个警察，就说要看看有没有人非法聚会，要他们出示证件，他们说你又不是房东，又不是你租的房子，你让马超来，我再出示证件。吴牧师看不到他警服上的警号，用手扒了扒他衣服上的警号，他说吴牧师袭警，要吴牧师出示身份证，录像，刷脸等。”
  <br/>
  <br/>
  马超说，他发现教会楼下，门口及走廊被安装的摄像探头：“他们装了摄像头，把我们的电弄没了。一个是专对着我们教会的门，第二个是对着我们上来的楼梯，第三个是对着我们小区进来的门口，共装了三个。整个春庭花园小区有几十个楼梯，其他地方都没有。”
 </p>

 <p>
  <br/>
  广福教会吴牧师对本台说，公安警告他们不准聚会：“他说我们在调查，在了解事情， 你有什么不满可以去投诉。有一个警察身上佩戴视频，我就用手把小视频划了一下，他说你不可以碰我，你们不可以在这里非法聚会，我说没有聚会，我们在这里聊天。他就拿着手机在屋内四处拍照，诗歌本、圣经等都拍了照片。”
  <br/>
  <br/>
  广福教会成立于2005年3月，在广州有多个分会场，有一千多位信徒。但自2012年开始，该教会受到骚扰，聚会被禁止。2012年3月，马超购入位于白云区的房屋作为聚会场所，被公安指涉嫌从事非法聚会而断水停电，其后租用酒店聚会，被公安阻挠。该教会曾有信徒涉印刷福音单张被判
  <br/>
  刑。
