 <p>
  有关朝鲜最高领导人金正恩的健康状况究竟如何，目前传说纷纭。日本媒体报道指金正恩已成植物人，韩国官员说，金正恩仍然在世，并且情况良好，而美国官员則說媒体有关报道“都只是推测”。不过，消息人士对本台披露，金正恩健康确实“出现严重问题”，朝鲜不披露只是争取时间。
  <br/>
  <br/>
  上周一开始，有关金正恩病危甚至死亡的消息引发各国媒体关注。有报道说金正恩做心血管手术时死亡，也有报道指出，金正恩成为植物人。上周六（4月25日）中午，中国官媒新华社援引朝鲜官媒《劳动新闻》的消息称，金正恩向支援三池渊郡建设的工作人员和劳动者表示感谢。
 </p>
 <p>
 </p>
 <p>
  <br/>
  朝鲜中央通讯社周日则报道，俄罗斯共产党中央委员会主席久加诺夫，4月24日（上周五）向金正恩发去贺电，祝他身体健康、幸福快乐。不过，总部设在华盛顿的朝鲜研究机构“北纬38度”公布了一张卫星照片，一辆疑似金正恩的私人专列，停在朝鲜东部海滨度假胜地。“北纬38度”说，这并不能说明金正恩就在当地，也不反映他真实的健康状况。
 </p>
 <p>
 </p>

  一位接近朝方的商人李先生本周一接受自由亚洲电台采访时表示，朝鲜官方需要时间来处置一些与金正恩有关联的问题：“他的身体状况可能是真的出了问题。有两种可能，第一种可能是他和中国联合放出烟雾弹，是为了迷惑国际社会，第二种可能是他本人真的不行了。现在释放假消息，是为了无论是给中国，还是朝鲜方面，作出一些部署和调整，以争取时间。”
  <br/>
  <br/>
  《韩国中央日报》本周日报道，韩国国会外交统一委员会委员长尹相接受采访时透露，韩国国会已经超过一周时间没有收到有关金正恩健康状况的任何报告，应该是有异常征兆。此前他曾介绍，金正恩可能动了心血管手术。而韩国总统文在寅的国安特别顾问接受美国媒体访问，指首尔政府相信金正恩在世，状况良好，未发现任何可疑活动。美联社报道引述要求匿名的美国政府官员说，对近期有关金正恩健康状态的传闻，“这些情报都无法改变美方认为（传闻）纯属推测的判断”。
 </p>
 <p>
  <br/>
  <b>
   金正恩曾长期不露面 中国边境列车停驶不寻常
  </b>
  <br/>
  <br/>
  朝鲜族商人李先生说，2014年秋天，金正恩曾有40天未公开露面，此次金正恩即使一个月不露面，也有可能：“即使一个月没出来，我觉得也不排除这种可能，他一个月不出来也不会出什么大事，他说他生病住院中，你也没办法。如果说他脑部神经和血管供血不足或暂停供血几个小时，真的有可能造成神经坏死，而坏死以后，神经是不可再生的。”
 </p>

  对于一周来，有关金正恩的状况众说纷纭，一位不久前接触过朝方人士披露，金正恩的健康状况确实“出了严重问题”，指金“很可怜”。但该名人士出于个人安全考虑，不愿多说。
  <br/>
  <br/>
  朝鲜族商人李先生则认为，金正恩遭受此次打击，命不久矣：“我觉得不管他们（朝鲜）说不说，这个人（金正恩）肯定活不了多久。”
  <br/>
  <br/>
  上周一（20日）传出金正恩病危消息后，周二起，中朝边境地区的中国铁路出现异动。官方公告列出丹东、延吉、珲春等靠近朝鲜的列车暂停。周五，沈阳局集团发出通告，自4月28日至5月20日，北京、长春、吉林、沈阳、大连、哈尔滨等大城市，前往中朝边境城市丹东及珲春，临时停运，涉及20多趟列车。

  关注朝鲜局势的时事评论人士郭宝胜对本台说：“各地前往丹东的列车停驶，那边（朝鲜）肯定有大事，朝鲜来人或中国一些重要的人物要去朝鲜，或者是中国的军列要去朝鲜。停驶时间由4月21日至5月21日，说明中国政府认为朝鲜局势稳定最少需要一个月时间。”
  <br/>
  <br/>
  郭宝胜说，丹东地区的新冠疫情已经趋于稳定，他相信中国当局此举与近期朝鲜局势有关。此前，美媒CNN和NBC都曾报道，金正恩因心血管手术失败而生命垂危，即使能保住性命，也可能会残疾，丧失身体活动的能力。而路透社上周六援引三位知情人士爆料称，北京当局已向朝鲜派出一支包括医学专家在内的团队，商议金正恩的健康，其中包括中共对外联络部的高官。本台曾于上周二（21日）率先报道，中国有一支由专家组成的医疗队正在待命，计划随时赴朝鲜提供协助。
