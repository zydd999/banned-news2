 <p>
  北京海淀区上周六出现防空袭海报，当局要求民众听见警报后立即听从指挥，躲避空袭，此举引来网民嘲讽。在重庆，有居委会开始登记在中国边防地区服役的军人家属名单。有学者认为，美中两国不会全面开战，但不排除局部“擦枪走火”的可能。
  <br/>
  <br/>
  随着美中两国关系持续恶化，中国开始散播一种战争氛围。在北京，中断近五十年的预防空袭海报，上周六出现在北京海淀区。网上流传的一段视频显示，官方将宣传挂板挂在铁栅栏门上，挂板内容提醒居民“听到警报后如何做到快速进入人防工程......”等。不少居民对这类标语感到恐慌，以为美国空军会袭击北京城。

 <p>
  孙滨认为，当局此举并不是真的担心美国空袭北京、上海等大城市，而是制造紧张气氛:“在全国营造这种紧张气氛。当局经常靠营造紧张气氛控制民众。民众稍微有一点反抗，会顺势给你戴一顶帽子，说你支持美国等等。”
  <br/>
  <br/>
  官方挂出防空袭海报，有人赞成，也有人不屑一顾。网民李先生对本台说，民众极力撇清与官方关系：“网络上民众的反应也很有意思，在微信也有。网民把它制成图片，上方是防空知识，下方写上‘现在都高科技定点清除，斩首，跟我没啥关系’。”
  <br/>
  <br/>
  <b>
   胡锡进再提核导弹震慑美国
  </b>
 </p>
 <p>
  《环球时报》总编辑胡锡进本周日在微博发文表示，中国必须做好应对下一步美国更为极端对华冒险的准备，提出“抓紧把足以震慑美国疯子的更多核导弹造出来”，称不能排除中美在南海、台海擦枪走火的可能性，而中国拥有更强大的核武库，是把美国嚣张态度，压制在安全线之下的关键砝码。
 </p>

  北京独立学者查建国对本台说，中美新冷战已全面打响，他认为两国不会发生大规模战争，但是有可能在南海“擦枪走火”：“美国对朝鲜、对伊朗这样的小国家动武都要三思而行，很难下决心，何况对中国这样的大国，但是擦枪走火是有可能的，尤其在南海。南海如果擦枪走火，影响也会非常大。”
  <br/>
  <br/>
  胡锡进曾于5月提出，中国需要在较短时间内，将核弹头数量扩大到千枚水平，包括至少有100枚东风41型战略导弹等，他强调要抑制美国的战略野心和对华冲动。
  <br/>
  <br/>
  查建国表示，当前官方宣传防止空袭，防范核武器等，均属官方所谓底线思维的范畴，即估算可能出现的最坏情况。
