 <p>
  美国在1979年转向承认中华人民共和国，接触交往四十年来究竟做对了什么？在哪些方面可以做得更好？美国纽约大学法学院资深教授孔杰荣（Jerome Cohen）和美中关系全国委员会会长欧伦斯（Steve Orlins）曾在1980年前往北京协助中国培训商业法专业人员。他们5月6日在一场视频会议上回顾往事，展望未来，各有不同的关注。
  <br/>
  <br/>
  今年七月，孔杰荣即将过九十岁生日。他正在整理自己有关中国的往事。 1980年6月，美中关系正常化后，中国在已故领导人邓小平的领导下开启改革开放；四十年过去，孔杰荣这位中国法律制度建设的见证者、参与者，经历过几代中国领导人，他对邓小平有较高评价。
 </p>
 <p>
 </p>
 <p>
  <br/>
  <b>
   孔杰荣：邓小平有弹性富想像力 习近平绝不妥协
  </b>
  <br/>
  <br/>
  孔杰荣：“和现在的中国领导人习近平相比，邓小平是不一样的领导人。邓小平有令人惊讶的弹性与想像力。看看他对待'一国两制'的态度，他展现的令人难以置信的弹性；看看他在钓鱼台问题上，怎么与日本打交道。邓小平说，这一代解决不了的问题，先搁置分歧、开展合作；在美中关系正常化上，他也没有因为美国'对台军售'的问题悬而未决而改变（接触交往的）立场”。
  <br/>
  <br/>
  孔杰荣说，如果胡耀邦不那么早走、赵紫阳没有在六四前下台、朱镕基能更上层楼，现在的中国，会是不一样的光景。
  <br/>
  <br/>
  他形容，和邓小平相较，习近平是个“寸土不让、绝不妥协”的人。习近平加深了中国共产党统治的独裁极权，让中国社会全面倒退。
  <br/>
  <br/>
  孔杰荣耶鲁大学毕业后，就专注于亚洲国家的法治与人权研究。从当年戒严时期的台湾、韩国，到改革开放后的中国，他一直关注亚洲国家的法治与人权改革，直到1989年天安门大屠杀发生，他才从香港返回美国，在纽约大学任教并且成立纽约大学亚美法研究中心（US-Asia Law Institute），致力促进亚洲的法治建设。
 </p>
 <p>

  <br/>
  而毕业于哈佛法学院的欧伦斯回忆往事，则谈到当年美中关系正常化、他不到30岁的年纪能有幸参与美国与中国建交的法律起草，建立美国与台湾“非官方关系”的法律架构，至今仍念念不忘。“看看现在台湾的社会、经济、民主与法治发展，美国当时根本难以想像。”
  <br/>
  <br/>
  <b>
   拥抱猫熊没有错？欧伦斯：美国人民企业也受惠
  </b>
  <br/>
  <br/>
  欧伦斯后来投身商业法，在美中经贸往来与投资中，为多个美国不同企业提供咨询，担任代表，和中国官方有长期往来与接触。
  <br/>
  <br/>
  他仍认为，美国过去与中国的接触交往政策没有错。而中国的法治发展，与他当年到北京街头看到的文革标语大字报相比，他的评价较为正面。
  <br/>
  <br/>
  “当年美中关系正常化，美国也是考虑到亚洲不能再有一次越战，而因为接触交往，我们得以协助中国从建立基本的法治开始，中国也才没有陷入混乱与战争。”
  <br/>
  <br/>
  欧伦斯著重经贸方面两国的来往。他说，美中的接触交往，美国人民也受益，“问问波音公司的员工就知道，他们的薪资收入是增长的”。
  <br/>
  <br/>
  对于经济起飞后的中国，人权与法治建设却陷入倒退，欧伦斯则指出，中国的各种发展史一直都有高低起伏，尽管他同意习近平管制更严，但他不认为美国能以经济制裁的方式，要求中国在法治与人权上做得更好，“我怀疑这能对习近平起到的效果”。
  <br/>
  <br/>
  他认为，关键还是在中国人民对于共产党统治的看法。
  <br/>
  <br/>
  对此，孔杰荣则说，美国必须承认中国人民在追求言论自由与法治上遇到的障碍，中国人民要如何才能决定自己的命运？他认为，外部关注仍然重要，像是美国能运用自己的《全球马格尼茨基人权问责法》，追究并制裁中国迫害人权者，并且要持续关注香港、公开批评新疆再教育营的人权迫害。
  <br/>
  <br/>
  自由亚洲电台记者问，如果能重回1980年代，他们两人愿多做些什么或改变什么？孔杰荣说，他会选择留中国久一点，和当年邀请他做国际经济法律讲习的北京市外经委副主任肖秧有更多接触，进而培养更多个较有改革开明思想的肖殃。
  <br/>
  <br/>
  欧伦斯仍聚焦更多参与中国经济发展：“如果能时光倒流，我会在上海置产，我一直很后悔没在上海买栋房”。
 </p>
