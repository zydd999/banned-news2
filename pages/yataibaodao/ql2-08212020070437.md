 <p>
  美国财政部本周公布的月度国际资本流动报告显示，今年6月份，中国抛售93亿美元国债，成为抛售美债最多的国家。有分析认为，虽然中方抛售金额不多，但似乎在向美方发出某种警告。
  <br/>
  <br/>
  美国财政部本周二公布了月度国际资本流动报告。目前，中国美债持仓总金额维持在1万亿美元上方，仍是美债第二大海外持有国。
  <br/>
  <br/>
  北京大军智库经济咨询公司主任仲大军认为，中国抛售美国债券93亿美元是向美方发出某种暗示。他还说，最近几年，中国政府不断增持美国国债，已超过一万亿美元：“好像是1.05万亿元，这两年减持了约两千亿美元，减持的速度不算太快。因为从两国关系如此恶化的情况来看，如像俄罗斯的做法，俄国干脆是把数千亿美债全抛没了，但是中国不想跟美国的关系搞僵。”
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  山东大学毕业的金融学者司令说，此次中国抛售美债的数额不算大，意在试探：“这似乎是一种试探性举动，表明对美国正在发出警告，中国现在仍然是实际上美债第二大持有国。只不过它要用象征性的抛售行为作为政治信号和作为外交政策信号的传递，它的政治意义大于经济意义。”
  <br/>
  <br/>
  中国国家外汇管理局本月上旬公布，中国7月末外汇储备3.15万亿美元，前值3.11亿美元。目前，中国外债余额约两万亿美元。另据官方公布，在扣除外债后, 目前的外汇淨额仅为9600多亿美元。但是中国每年要购买外国产石油、天然气、铁矿石、零部件、农作物及食品，耗资数千亿美元。
 </p>

  <b>
   抛售美债数量“温和”留回旋余地
   <br/>
  </b>
  <br/>
  仲大军说，中美两国关系恶化，中方此次抛售的美国债券金额相对比较“温和”：“由于美国一步一步的这么施压，中国有时候也得做点动作，不断给你抛一点美债。中国的做法还是比较温和，抛93亿美元，不算什么事。中国和美国还是要保持友好的关系，不想把事情做绝。中国采取比较温和的态度，在美债方面不得不提防一些。”
  <br/>
  <br/>
  仲大军说，目前美元存在贬值风险，世界各国都在抛售美元，投资者持有美元的比重持续下降，这对美国国债有负面影响。
  <br/>
  <br/>
  对于网传中国正抛售美债，美元信用快速损耗等传闻，司令认为，美元信用是稳定的。他说：“穆迪、标准普尔、惠誉这三家主流信用评级公司对于美债的评级维持在一个相当高的水平，没有调低它的信用评级。所以从这个角度来讲，美元资产债务从全球金融视察远期估值的方向来看，仍然十分稳定。”
  <br/>
  <br/>
  中国央行党委书记、银保监会主席郭树清8月16日在《求是》刊文表示，在以美元为主导的国际货币体系中，当前美国这种前所未有的无限量化宽松政策，实际上也消耗着美元的信用，侵蚀着全球金融稳定的基础，会产生难以想象的负面影响。
  <br/>
  <br/>
  不过，司令分析说，倘若美中关系进一步恶化，中国可能会将美债当成与美方谈判的筹码，随时抛售美债。但是，美国也有很多反制措施。
