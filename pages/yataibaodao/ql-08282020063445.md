 <p>
  内蒙古政府教育厅日前下文，从9月1日新学期开始，全区所有中、小学一年级开始，语文教材禁止用蒙古族母语教学，引发全体蒙古族人抗议。本周四（8月27日）扎鲁特旗的学生家长拒绝带子女去学校报到，更有人发起9月1日在全区多个城市中心广场集会，当局高度紧张。
  <br/>
  <br/>
  9月1日起，内蒙古民族学校将采用所谓双语教育模式，要求从小学一年级和初中一年级开始，所用《语文》教材放弃蒙古族语，改用汉语。当局此举招致蒙古族人强烈不满。本周四（27日）通辽市民族学校的学生家长拒绝把自己的子女带到学校报到。关注事件的蒙古族学者忽必斯周五对自由亚洲电台说，蒙古族人很气愤：“昨天（27日）在（内蒙古）东部的通辽市扎鲁特的蒙古族学生家长，一个都没有送孩子去学校，拒绝送孩子去学校。今天（28日）扎鲁特蒙古族实验小学校方发出一个对没有送孩子上学的家长的通知书，没有学校的印章，没有校长签字。”
 </p>
 <p>
 </p>
 <p>
  <br/>
  <br/>
  忽必斯说，该通知书明显是威胁拒绝送孩子去学校的家长：“大意是没有送孩子去学校的家长，后果将是您的孩子被认为，你们的孩子将回原户口地学校。通辽市教育局通知各家长，根据通辽市教育局文件，今天8月27日，我校已经正式开学，您没有送孩子来学校，耽误了学生受义务教育的权利。”
 </p>

  <b>
   蒙古族为延续民族文化而抗争
  </b>
 </p>
 <p>
  锡林郭勒盟正蓝旗居民哈斯朝鲁对本台说，他们必须要为延续蒙古文化而进行抗争：“以前从小学三年级开始上汉语课，现在要从一年级开始上汉语课。而且把汉语当成重点课，蒙语当副科，这样慢慢的我们民族的蒙语就消失了，所以我们在这个问题上有意见。”
  <br/>
  <br/>
  锡林郭勒盟正蓝旗的蒙古学校教师周四也发起了签名行动，要求停止“双语教育”。旅居美国的蒙古族人诺民对本台说，内蒙各地蒙古族教师也发起签名行动，要求当局停止所谓双语教育，但受到压力：“他们（官方）给学生家长，还有老师施加压力，威胁说孩子将来不能考大学，把你们的孩子入黑名单等，恐吓。现在各个地方人们的情绪很高涨，很生气。”
  <br/>
  <br/>
  <b>
   倡议9月1日在内蒙古十多各城市集会
  </b>
  <br/>
  <br/>
  与此同时，有蒙古人在网上发起反对《双语教育》的集会倡议，呼吁内地蒙古族人于9月1日（星期二）上午九时，在内蒙古呼和浩特市新华广场、包头市昆都仑区万达广场、乌海市万达广场、海拉尔市成吉思汗广场以及通辽市哲里木广场等十多个地点及各旗县政府门前，进行集会抗议。

  对此，诺民说：“家长聚会就是为了反对第二套蒙语教学方案。他们还是要保持以前的第一套蒙语教学方案。
  <br/>
  <br/>
  倡议发起此次集会者认为，因《双语教育》政策违反中国《宪法》和《民族区域自治法》，当天的横额和口号是《反对双语教育》、《捍卫学习母语权利》、《行使自治权力》以及《尊重宪法》。大会要求参与的学生尽量穿校服，其余人士尽量穿着民族服饰。
  <br/>
  <br/>
  本台致电内蒙古扎鲁特旗政府信访办，但无人接听。南蒙古新闻脸书账户发布的一学生家长致电内蒙古政府信访办，询问为何在9月1日采取汉语教材授课，接听电话的工作人员拒绝承认有这项新规定。她说：“我刚才听了您说的这个话以后，我到内蒙古自治区教育厅官网上看，我把今年他们所发的所有文件都看了一下，没有发现有相关的文章。”
  <br/>
  <br/>
  这位学生家长质问工作人员：“问题就在这里，我刚才跟你说的没有正式的文件，就是没有，但是口头相传就落实下去了。工作人员抢话：如果还没有开学的话，第一它没有文件证明，第二，学生并没有实际接受到这种教育。我觉得您现在来提这方面的顾虑，稍微早了一些。您说的也有可能是存在的，我会向我们跟这个事情有关的司局反映一下，这个情况。”
  <br/>
  <br/>
  另外，内蒙古自治区教育厅周四发布《内蒙古教育督导规定（征求意见稿）》，公开征求意见的公告。自8月27日起，为期一个月。有蒙古族人说，面对强大的舆论压力和国际媒体的关注，当局的态度处于摇摆不定，但后续发展仍不明朗。
