 <p>
  新冠肺炎疫情在全球蔓延之际，中国借“口罩外交”，宣传中国展现大国担当与全球领导力。但受援助的欧洲国家，真会买单“中国方案”？受疫情冲击最为严重的两个欧洲国家，意大利与西班牙的驻美大使们，分享他们的看法。
 </p>
 <p>
  意大利驻美大使威力奇欧（Armando Varricchio）和西班牙驻美大使卡巴纳斯（Santiago Cabanas）都是经验丰富的资深外交官，两人4月2日在美国梅瑞迪恩国际中心（Meridian International Center）举办的一场网络研讨会上，讲述当前欧洲应对新冠肺炎疫情的形势，他们都表达大疫当前，国际协调合作的重要性，也都对美国提供的协助表达感谢。
 </p>
 <p>
  <br/>
  身处华盛顿这个全球政治中心，两人对疫情当下、全球领导力谁领航这样的话题，也有备而来。自由亚洲电台在会中问到，欧洲国家如何看待中国试图借着疫情蔓延输出所谓的“中国模式”，标榜“中国的全球领导力”？威力奇欧的外交官风范，充满意大利人有话直说的直率个性。
  <br/>
  <br/>
  威力奇欧：“我们（意大利）并不天真，我们清楚得很。是的，中国政府和中国企业正提供意大利亟需的医疗器材，但同时，我们完全了解，我们不该接受中国借由所谓‘软实力’公众外交，试图输出的那一套论述。”
  <br/>
  <br/>
  他强调，也就是因为中国这样的宣传行动，更彰显全球合作协调的重要性。他说，大疫当前，义大利对所有的国际合作持开放态度，但意大利对自身的价值观与利益心知肚明，不会陷入这种短期的战术赛局陷阱。
  <br/>
  <br/>
  事实上，中国办口罩外交，在意大利引发不少反感。就有国会议员质疑，部分医疗用品是意大利向中国采购，北京却说成是捐赠。
  <br/>
  <br/>
  另外，包括捷克、西班牙与土耳其都指出，向中国采购的检测试剂品值欠佳，西班牙从深圳一家名为易瑞生物公司购得的试剂盒，准确度只有30%，远低于厂商声称的80%。
  <br/>
  <br/>
  对此，两位外交官在会中并未过多着墨，而曾经历过威权政体的西班牙，对威权体制与民主体制的危机管理模式展现的不同特色，非常警醒。
 </p>

 <p>
  <b>
   大疫让威权复辟？
  </b>
  <b>
  </b>
  <b>
   西驻美大使吁民主展现善治效率
  </b>
 </p>
 <p>
  卡巴纳斯不断强调欧洲联盟（EU）此刻展现团结的重要性，同时，也必须有解决问题的能力，尤其在应对疫情过后的经济危机上，民主若不能展现善治的效率与团结，会给欧洲民粹主义加添柴火。
  <br/>
  <br/>
  “疫情过后可能加剧贫富差距，我们得现在就着手准备，否则会出现很危险趋势，也就是在欧洲，民族（？）主义与民粹主义会声势更猛，质疑民主体制没有效率，这会让威权主义复辟。当然，并不尽然如此，只要我们现在就认识到问题。”
  <br/>
  <br/>
  卡巴纳斯还说，中国特殊的社会结构在应对危机上是否真展现更高效率？中国的软实力及中国的专制治理模式是否更吸引人？他认为，这会是全球长期将持续论辩的课题。
  <br/>
  <br/>
  他和威力奇欧都强调，西班牙和意大利在对抗疫情时，要确保人权、民主与自由这些两国长期信仰的价值不能被牺牲。
  <br/>
  <br/>
  根据美国约翰霍普金斯大学统计，欧洲受疫情冲击最严重的国家是意大利及西班牙，分别都有超过10万起确诊病例，死亡病例也都已超过1万人。法新社则统计，全球新冠肺炎确诊人数已达94万例，其中，欧洲就占了超过50万人、逾半数；全球死于武汉肺炎人数则达4万7836人，欧洲病逝患者占3万4571人。
 </p>
 <p>
  <br/>
  自由亚洲电台记者郑崇生华盛顿报道   责编：申铧   网编：洪伟
 </p>
