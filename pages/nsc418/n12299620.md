<div><p>
 【大纪元2020年08月01日讯】斯坦福大学胡佛研究所（
 <span class="s1">
  Hoover Institution
 </span>
 ）
 <span class="s1">
  7
 </span>
 月
 <span class="s1">
  30
 </span>
 日发布新报告称，美国与中国学者及研究机构之间的合作，直接促进了中共的
 <span class="s1">
  “
 </span>
 军事现代化
 <span class="s1">
  ”
 </span>
 。报告指出，中方研究人员涉嫌隐瞒与中共国防项目的真正关系。
</p>
<p class="p1">
 报告还认为目前迫切需要新的方法来识别和管理风险，并提出了一个综合的政策建议计划，旨在协调美国对开放和全球化研究的承诺与维护美国国家安全和经济竞争力的必要性。
</p>
<h4 class="p1">
 国防七子和美研究员合作的
 <span class="s1">
  254
 </span>
 篇
 <span class="s1">
  “
 </span>
 大作
 <span class="s1">
  ”
 </span>
</h4>
<p class="p1">
 据美国之音报导，周四（
 <span class="s1">
  7
 </span>
 月
 <span class="s1">
  30
 </span>
 日）发布的这份报告名为《全球参与：重新思考研究企业的风险》（
 <span class="s1">
  Global Engagement: Rethinking Risk in the Research Enterprise
 </span>
 ）。研究人员在一个由中共政府支持的公开学术数据库
 <span class="s1">
  “
 </span>
 中国知网
 <span class="s1">
  ”
 </span>
 中找到
 <span class="s1">
  254
 </span>
 篇论文，这些论文是由
 <span class="s1">
  115
 </span>
 家美国大学和政府实验室的研究人员，与人称
 <span class="s1">
  “
 </span>
 国防七子
 <span class="s1">
  ”
 </span>
 的七所和中共军方有关联的中国大学和研究所人员共同撰写。
</p>
<p class="p1">
 论文涵盖了从化学、光电（
 <span class="s1">
  photovoltaics
 </span>
 ）、材料科学到海军工程等一系列主题，胡佛研究所的报告指出中国科研人员涉嫌隐瞒他们与国防项目的真正隶属关系。
</p>
<p class="p1">
 <span class="s1">
  “
 </span>
 国防七子
 <span class="s1">
  ”
 </span>
 分别是北京理工大学、北京航空航天大学、哈尔滨工业大学、哈尔滨工程大学、西北工业大学、南京航空航天大学和南京理工大学，这
 <span class="s1">
  7
 </span>
 所大学中有
 <span class="s1">
  4
 </span>
 所在美国商务部的出口管制实体名单上。
</p>
<p class="p1">
 七子中最高产的是哈尔滨工业大学，
 <span class="s1">
  254
 </span>
 篇合著论文中有
 <span class="s1">
  106
 </span>
 篇与哈工大相关，哈工大的合做伙伴则包括哥伦比亚大学等共
 <span class="s1">
  63
 </span>
 所美国科研机构。
</p>
<p class="p1">
 报告称，
 <span class="s1">
  “
 </span>
 与中国（中共）的军事发展合作不符合美国的国家利益
 <span class="s1">
  ”
 </span>
 ，这种大量合作可能助长中国（中共）作为美国军事对手的实力。
</p>
<p class="p1">
 报告建议美国研究机构扩大与中国合作伙伴的合作审查和尽职调查，建立共同的道德和伦理标准，以防止可能有助于威权政府或违反民主价值观的合作。
</p>
<h4 class="p1">
 新风险评估及管理范式
</h4>
<p class="p1">
 胡佛研究所在报告中称，无论是美国政府，还是美国大学和国家实验室，
 <span class="s1">
  “
 </span>
 都没有充分管理与外国实体进行研究接触所带来的风险
 <span class="s1">
  ”
 </span>
 ，
 <span class="s1">
  “
 </span>
 为了弥补这一缺陷，研究界应采用一种新的、主动的风险评估和管理范式，这种范式以操作安全（
 <span class="s1">
  OPSEC
 </span>
 ）为基础，并通过能力成熟度建模（
 <span class="s1">
  capability maturity modeling
 </span>
 ）加以实施
 <span class="s1">
  ”
 </span>
 。
</p>
<p class="p1">
 美国智库兰德公司的高级防务专家何天睦（
 <span class="s1">
  Timothy Heath
 </span>
 ）也认为，美国的学校和研究机构应该实施更有效的监督和保障措施。他对美国之音说：
 <span class="s1">
  “
 </span>
 应该更仔细地审查与中国（中共）军事研究人员合作的提议，并准备拒绝任何可能对美国国家安全构成威胁的合作。
 <span class="s1">
  ”
 </span>
</p>
<p class="p1">
 报告指出，机构应制定战略全球参与风险评估和管理计划（
 <span class="s1">
  GERAMP
 </span>
 ），并设立全球参与审查办公室（
 <span class="s1">
  GERO
 </span>
 ）。
</p>
<p class="p1">
 胡佛研究所还建议研究机构改变范式，采用操作安全（
 <span class="s1">
  OPSEC
 </span>
 ）作为评估和管理外国参与风险的管理范式，通过全球参与成熟度模型（
 <span class="s1">
  GEMM
 </span>
 ）正规化和优化其内部能力。
</p>
<p class="p1">
 此外，报告还建议建立一个政府资助的实体，以支持更好的决策。
</p>
<h4 class="p1">
 如何应对风险？
</h4>
<p class="p1">
 《国家评论》（
 <span class="s1">
  National Review
 </span>
 ）曾发表文章说，中国（中共）在保送部队军官到海外深造，
 <span class="s1">
  “
 </span>
 利用西方民主国家的开放性和学术自由，搜集情报或甚至进行间谍活动，以推进中国（中共）的技术发展和军事竞争。
 <span class="s1">
  ”
 </span>
</p>
<p class="p1">
 近年来，有军方背景的中国学者盗窃美国技术和研究成果，威胁美国利益及国家安全的案件频频发生，如何应对这些风险成为了一大难题。美国
 <span class="s1">
  6
 </span>
 月已宣布禁止与中国（中共）军方有牵连的中国公民持学生学者签证进入美国。这项禁令涉及持
 <span class="s1">
  F
 </span>
 类学生签证和
 <span class="s1">
  J
 </span>
 类访问学者签证前来美国就读研究生以及从事研究生与博士后研究的中国公民，攻读本科专业的中国学生不在此列。
</p>
<p class="p1">
 国防部高级分析师斯托夫（
 <span class="s1">
  Jeffrey Stoff
 </span>
 ）在周四就胡佛研究所报告举行的一场讨论会中表示
 <span class="s1">
  , “
 </span>
 报告中有很多例子，可能还有更多的是故意混淆和歪曲他们的背景，他们从事的工作，甚至他们工作的地方
 <span class="s1">
  ”
 </span>
 。因此，政府和学术界都有责任对一些项目进行更详细的调查。
</p>
<p class="p1">
 斯托夫还说：
 <span class="s1">
  “
 </span>
 我们并不总是清楚发表的研究是否是基础性的，这需要更仔细地检查，但他们可能会做更多的应用。
 <span class="s1">
  ”
 </span>
</p>
<p class="p1">
 德州农工大学首席研究安全官加马什（
 <span class="s1">
  Kevin Gamache
 </span>
 ）博士在讨论会上说，高校和研究机构在风险应对中扮演着非常关键的角色。
</p>
<p class="p1">
 他说：
 <span class="s1">
  “
 </span>
 在谈到降低风险时，其中一部分涉及到一个持续的过程，以确保我们知道是谁在从事研究工作
 <span class="s1">
  ”
 </span>
 。
</p>
<p class="p1">
 他还认为降低这些风险是国家层面的当务之急，
 <span class="s1">
  “
 </span>
 一所大学在这方面所起的作用即使不超过国务院的作用，也是一样多的，如果你接受（降低风险）是一个持续的过程，那么大学绝对可以发挥作用
 <span class="s1">
  ”
 </span>
 。
</p>
<p class="p1">
 胡佛研究所高级研究员戴雅门（
 <span class="s1">
  Larry Diamond
 </span>
 ）则在讨论会上表示，周四发布的这份报告并不是要排斥任何国家的人。这份报告是有关了解风险的，这样人们就可以采取适当的缓解措施。
</p>
<p class="p1">
 曾担任联合国安理会反恐委员会执行局副主任的纽黑文大学教授斯托弗尔（
 <span class="s1">
  Howard Stoffer
 </span>
 ）告诉美国之音，
 <span class="s1">
  “
 </span>
 我们需要小心，不要提供可能被中国某些人使用的战略军事信息，这些人不是亲美国的，会对美国非常有攻击性
 <span class="s1">
  ”
 </span>
 ，
 <span class="s1">
  “
 </span>
 我认为每一次合作都需要审视，这个合作真的会伤害我们，还是会帮助我们。如果合作是双方双赢的话，我们应该这样做。
 <span class="s1">
  ”
 </span>
</p>
