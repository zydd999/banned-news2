 <p>
  我们本专栏的上篇文章《赵立坚“美军投毒”论源于中共“首席军事理论专家”》刊登和播发后的次日，这位“首席军事理论专家” 杨承军即又发新论，中共当局专为“祖国完全统一”而开设的“中国台湾网”介绍和转载杨承军这篇文章时，为夺人眼球，特意以《说透了！“武统”台湾什么时候开始？解放军专家权威解读》为耸动标题。
  <br/>
  <br/>
  该文开篇即如此分析当前台海局势的“严峻程度”，说是：当前，新冠疫情导致国际局势及台海局势发生深刻变化。因民进党当局种种冷血作为，在网络上激起强烈民愤，“武统”声音再次水涨船高。
  <br/>
  <br/>
  这个“中国台湾网”对杨承军的具体介绍是：“火箭军退休研究员、中国核战略专家杨承军”。
  <br/>
  <br/>
  而我们本专栏的上篇文章被一些网站转载刊登后，有网友在跟帖中质疑笔者：凭什么把这个叫杨什么的“封为首席军事理论专家”？对此仍然还有兴趣的读者和听众都请上网一查，无论是中共内宣还是外宣媒体，有好多家都给这个杨承军罗列了如下一堆头衔：“火箭军参谋部退休研究员，大校，教授，军事运筹学博士研究生导师；享受国务院特殊津贴；国家安全委员会、国家智库高级研究员，国家城市安全战略研究所副所长，全军军事学学科评议专家，装备理论专家，《中国军事百科全书·战略卷》终审专家，核战略专家，核军控专家，导弹技术专家，军史专家，首席军事理论专家”。比如，最初刊登杨承军文章的“凤凰军事”也是这么样介绍他的。
  <br/>
  <br/>
  如上头衔中，“首席军事理论专家”的“首席”二字，从字面上讲当然是“第一”的意思。虽然从常理上讲，无论是哪个门类里的专家或者领袖，不一定设有“首席”的基础上，也还有“次席”一说，但“首席”肯定只是一个。比如中共当局的司法体制下，“首席大法官”就是最高法院的院长，而副院长们都只能是一级或者二级“大法官”。
 </p>

 <p>
  如此说来，无论中共当局的御用“军事理论专家”有多少，他们中那个“首席”应该不是自封的，而是当局，也许是习近平亲自册封的。
  <br/>
  <br/>
  而杨承军最新的这篇文章的原始标题是《武力统一有决战决胜把握但疫情不是时机》，文章的第一段是：“当前，有人提出应趁着美国和台湾抗疾、美四艘主力航母及亚太军事基地发现疫情之机实现国家统一；也有人讲了一些貌似，‘义正词严’的大话、空话，指责现在的任何说法、做法都不对；还有人说美国的撤侨撤资、军机战舰巡航、贸易战等是马上要对中国发动战争的信号。对于战争说法我不这么看，特朗普是唯利是图的商人，但不认为这些举动就意味着即将对华发动战争，因为他们当前还没有此战略需求，没有这样的精力和余力，更没有做好向中国宣战的准备……。”
  <br/>
  <br/>
  但是，杨承军话锋一转，即刻列出了“务实做好武统准备”的大体工作内容，诸如“科学决策战后治理，以及恢复经济，处置罪犯、部署驻军、稳定治安等，切实深化、细化对台斗争”。
  <br/>
  <br/>
  那么，当前趁源自中国武汉的新冠病毒引发的美国和世界之危，这一“千载难逢的大好时机” ，“一举实现祖国完全统一”是否是中共当局的目前考量之一？杨承军似乎认为，这取决于“对当前和今后一个时期台海局势的六点战略判断” 。
  <br/>
  <br/>
  不过，他的这六点判断中的第一点，“台湾当局必将更加孤立” 中的具体分析内容，没有一句能够佐证他的结论“台当局必将深陷内外交困境地” 。而他六点判断中的第二点，“美国干预程度加深”，恰恰证明了台湾不再会被孤立。
  <br/>
  <br/>
  在杨承军的第二点分析内容里，他认为虽然美国对台湾局势的干预程度加深，但不会为台与大陆宣战。
  <br/>
  <br/>
  杨承军认为，虽然上个月特朗普签署了《台北法案》- 在1979年美国《台湾关系法》基础上升级的美台实质政治关系的长臂管辖法律，加上以前的《台湾保证法》、《台湾旅行法》、《亚洲再保证倡议法》及《国防授权法案》等，确定了对台军售正常化，把台纳入军事演习，重新确认对台湾承诺；批准与台高层互访，全面提高对台支持力度；公然挑战联合国“一个中国”国际法及中美三个联合公报……，但是，“我一旦启动武统，美国、日本对台支持将集中体现在情报信息和武器装备上，不会为了台湾对中国宣战，因为他们会顾虑战争成本和后果”。
  <br/>
  <br/>
  言下之意，如果中共当局决心近期内对台动武，美国直接出兵干涉的可能性已经不存在。
  <br/>
  <br/>
  杨承军所分析的第三点是，“民进党台独纲领不会改变。民进党对外完全依赖美国，竭力扩大所谓生存空间；对内不顾百姓利益，排除异己，全力打压和清剿国民党；继续推进去中国化进程；提出将太平岛租给美军使用；不惜重金购买美国军火；最近蔡英文甚至还公然叫嚣‘日本是我们的坚强后盾’。”
  <br/>
  <br/>
  如上内容再加上第四点的“国民党不会有任何作为” ，甚至“竭力摆脱与大陆的连接；不断背弃孙中山、蒋介石的理念……”，和第五点的内容“台岛对‘一中’的认同感下降……”，都是意在说明：“和平统一”已无半点可能，“武统”是必然，剩下的只是时间和时机问题。
  <br/>
  <br/>
  对于过去多年来,中共当局为实现将来“和平统一” 所做出的那些“努力”和“善意”，杨承军在这篇文章里给予了全盘否定。他认为：“台湾当局把我们提出的‘和平统一，一国两制’视为软弱可欺，反而更加变本加厉、肆无忌惮的加速推进‘台独’进程。多年来，大陆对台湾展现了太多善意、给予了台商多种优惠扶持，把其农产品等给予免税包销，力图通过感化降低敌意，然而他们把得到的红利用在了购买和发展武器装备。多年来，我们把实现祖国统一的希望寄予台湾人民，然而随着岁月流逝，我们等来的是国民党放弃‘一中’、民进党的投敌叛国和岛内社会对国家的背弃。”
  <br/>
  <br/>
  杨承军在文章中还说：“对于台湾，虽然我们进行了多年的军事斗争准备，虽然具有决战决胜的把握，但在完善作战预案、实现武统进程和后续施政治理上，还存在着深化、细化和优化的空间”。
  <br/>
  <br/>
  他的这个所谓的“决胜的把握”，是否在习近平的心中也已经是“百分之百”暂且不论，但应该是证明了习近平当局在对台武统的问题上，目前已少有打不赢的顾虑，而只需要在“战后治理”问题上考虑周全。

 <p>
  就“必须适时宣布对台武统时机”的问题，杨承军笔下的“武统时机” 共六点：1、台当局公然宣布独立时；2、台当局组织独立公投、使祖国面临分裂时；3、台岛有外军部署时；4、台重启核武研发时；5、台军使用军事手段攻击大陆时；6、岛内发生大规模动乱时。
  <br/>
  <br/>
  细分析下来，所谓“台当局公开宣布独立时” 的假设可以永远不会成立，因为“中华民国一直是主权独立国家”是事实，也是台湾朝野共识。谁上台也不会有所谓“公然宣布独立”，因为搞一次所谓的“公然宣布独立”，等于是承认自己的中华民国过去一直不是主权独立国家。
  <br/>
  <br/>
  所以，杨承军这里说的所谓“台当局公然宣布独立”，应该是暗指台湾当局抛弃中华民国国号， “另起炉灶”。但笔者百分之百相信，无论是蔡英文还是下届中华民国总统，无论他或她是民进党还是什么党，改国号的可能基本没有。
  <br/>
  <br/>
  既然不需要改国号，那么所谓“独立公投” 的前提也就不存在了。至于杨承军所谓“使祖国面临分裂时” 这句话，无疑是他全文中最滑稽的一句。从一九四九年至今，杨承军口中的“祖国” 不一直都是处于“分裂”状态吗。如果不承认一直都是处于“分裂”状态，何来“争取早日实现祖国完全统一”之说。“武统”也好，“文统”也好 ，“分裂”不都是“统一”的前提吗？
  <br/>
  <br/>
  至于如上所谓“台军使用军事手段攻击大陆”，以及“台重启核武研发”两条，也都是不可能发生的假设，毫无意义。对于“岛上有外军布署”和“岛内发生大规模动乱”的假设，倒是有必要分析几句。
  <br/>
  <br/>
  现在的中华民国有效统治区域内，确实已经没有“外军布署”，但与中共内部所谓“趁机武统台湾”的“呼声高涨”，外部自由世界也出现了趁机推动美国与台湾建立正式的、国与国外交关系的呼声，也就是趁机让“两个中国”成为美国国策的意思。但这仅仅只是民间呼声而已，在可以预见的时间范围内，美国两党无论谁上台，“维持台海现状”的政策应该不会有根本的改变。
  <br/>
  <br/>
  至于“岛内发生大规模的动乱”，也许会成为中共当局未来悍然对台动武的最可能借口。因为中共当局很可能会先用某种或者多种非军事手段在台湾岛内制造动乱，继而以“平乱平暴”为武统借口。
  <br/>
  <br/>
  还需要特别分析的是，作为中共习近平当局的“首席军事理论专家” ，杨承军这篇文章里透露出来的一个外界媒体似乎还没有具体分析到的细节是，中共当局目前已经在沙盘上推演过的武统手段，已经把导致岛内军民大量牺牲当成了必要后果。虽然他口口声声“以最小的牺牲和代价完成祖国统一大业” ，但内文中尤其强调了，“特别要掌握城市巷战、攻坚战、摧毁地下工事及联络中断情况下的行动要领……” 。
  <br/>
  <br/>
  那么，城市巷战也好，攻坚战也好，即使届时中共军队自己可以现实“以最小的牺牲和代价”，但导致的结果肯定是届时的守方 - 台湾军民的大量死伤。届时的台湾甚至可能会出现因为同仇敌忾，殊死抵抗而被屠城屠乡的惨状……。细想下去，真的是不寒而栗。
  <br/>
  <br/>
  仔细阅读下来，虽然这位杨承军的文章漏洞不少，逻辑上也谈不上严谨，但他文章中透露出来有关“战后治理”的内容，应该不会是他自己的一家之言。
  <br/>
  <br/>
  众所周知，自邓小平时代开始的对台“文统”宣传，一直都是承诺“国家实现统一后，台湾可作为特别行政区，享有高度的自治权，并可保留军队……”。而杨承军如今透露出来的所谓“战后治理”的重要内容之一则是，“必须尽早确定台岛驻军”，甚至都包括了陆军的布署应不少于齐装满员的两个集团军规模，部署一定规模的火箭军常规部队和信息部队……，之类的细节。
  <br/>
  <br/>
  要知道，中共习家军的陆军在“调整组建”后，九百六十万平方公里的土地上总共只分布了从七十一至八十三这十三个集团军。实现“武统”之后，居然在小小的，总面积只有3.6万平方公里的台湾岛上驻守两个集团军，而且还要“齐装满员”，真是要多可怕有多可怕。
