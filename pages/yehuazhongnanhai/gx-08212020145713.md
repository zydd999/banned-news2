 <p>
  昨天读到一则关于“中南海高层内斗”的分析报道文章，大致内容是从8月18日到20日，连续三天，习近平都在安徽勘查洪水灾情，受到党媒大篇幅报道……。 20日这一天，习近平在安徽听取军方的防汛救灾汇报；同一天，李克强前往洪水重灾区重庆进行勘灾，李克强穿着胶鞋走进灾区，满脚泥泞，与习近平的一身干净、背景干净的勘灾场面，形成鲜明对比。
  <br/>
  <br/>
  但是，李克强的行程，中共党媒却完全没有报道。不论是新华社、人民日报、央视等主要党媒，都只有习近平的身影，李克强仿佛人间蒸发一样。只有在隶属国务院的“中国政府网”里，才发布了这项消息。
  <br/>
  <br/>
  到本文截稿为止，笔者对如上信息核实的结果是，无论是新华社还是人民日报，还是没有关于李克强重庆之行的半句内容。今天（八月二十一日，星期五）的人民日报头版头条是《习近平在听取军队参与防汛救灾情况汇报时强调 发扬连续奋战作风 切实完成防汛救灾后续任务》；头版二条（报眼位置）是《前7月我国对一带一路沿线国家投资逾百亿美元 同比增长28.9%，占同期投资总额17%》；头版第三条是人民日报自己的“走向我们的小康生活”系列报道《山东推动农村各项产业融合发展，为乡村振兴夯实基础 产业兴旺 日子红火》；头版第四条是《李克强签署国务院令 公布修订后的&lt;中华人民共和国预算法实施条例&gt;》；头版第五条是汪洋主持政协会议的报道……。

  但是，国务院自己的官方网站的“总理 ”栏目里，已经刊登了四篇报道李克强从本月二十日开始的重庆之行，分别 是《李克强来到重庆看望慰问受灾群众：勉励大家重建好家园》，内容为“李克强8月20日上午11时许飞抵重庆，一下飞机马上转乘火车、汽车，一路辗转来到洪水灾情严重的潼南区双坝村……”；《李克强重庆考察为何专门来到这个水库》，内容为“重庆近日遭遇百年一遇的洪水。李克强8月20日下午专程来到重庆潼南区，察看正在建设的大石桥水库……”;《李克强来到重庆磁器口 用实际行动支持商家带动消费》，内容为：“李克强8月20日下午来到位于嘉陵江畔的重庆磁器口。这里昔日是繁华的水陆码头，现在商铺云集。此次超强洪峰过境重庆，磁器口不少商铺和房屋被淹，有的低洼处仅露出屋顶。总理来到江边察看汛情，并一连走访了几家商铺和住户察看受灾情况……。离开磁器口前，总理临时拐进一家麻花店掏钱买了三袋麻花，用实际行动支持商家、带动消费。”
  <br/>
  <br/>
  该网站“总理” 栏目里，关于李克强重庆之行的最新一则报道的标题是《李克强曾送这家公司6个字，此次重庆考察再赠3字》。
  <br/>
  <br/>
  如上国务院官网的四篇李克强重庆之行的报道都配有数张内容较为自然和生动的现场图片，其中两张是李克强脚穿高筒水靴趟泥涉水，与民同苦。
  <br/>
  <br/>
  有外界分析文章据此认为：中共党媒只报道“党内一把手”的消息，堂堂国务院总理的消息却刻意“视而不见”，这个现象非常不寻常。从中可以嗅出的重要信息就是“习李分裂台面化，李克强遭孤立”。因为北戴河会议之前，李克强已经出现多次与中央“唱反调”的公开言论，包括“6亿人口月收入不到1000元人民币”、“推动地摊经济”等等，当时已经被外界质疑习李双方走向分裂争斗……。所以，李克强被党媒封杀，也意味着习近平方面对李克强下了重手，要在舆论战线上“边缘化”、“孤立化”李克强，让他成为弱势总理，也意味着李克强在党内的派系争斗里，目前落居下风。
  <br/>
  <br/>
  其实，有没有一个所谓的“北戴河会议”，或者说习近平上台之后，特别是被确立为“一尊”之后，还有没有必要特别在北戴河“例假”期间召开党内重要会议另论，自李克强在二零零七年的中共十七大上与习近平同进政治局常委，但却排名在习近平之下那一天起，他李克强就再无可能占了习近平的“上风”，从未“上风”过，何来“落居下风”之说。
  <br/>
  <br/>
  至于香港有媒体近日所谓北戴河会议上李克强“带头逼习下台”的说法，就更是无稽之谈了。
  <br/>
  <br/>
  记得去年中共“两会”期间，因为李克强在朗读《政府工作报告》时几度大汗淋漓，被我们外界解读为是“做奴隶生还怕做不稳”……。

  对此点评最到位的文章，当属香港《苹果日报》刊登的资深传媒人吕月的《封口大会从政治局封起》文中说：李克强的报告，一方面必须聚焦经济和社会发展的重大问题，另一方面必须关注涉及人民群众切身利益的重点问题。作为总理，这些都是压在他身上的担子，回避不掉。中国政法大学法学院副院长何兵在微信有一段感言：“我的法律水准比申纪兰大妈高吧，可人家有权审议刑诉法，我只能在网上喊两嗓子，喊多了还被禁言，什么道理？——因为人民大会堂坐着的是两个人，主子和奴才。”
  <br/>
  <br/>
  从另一角度看，习帝自然是扬鞭的主，克强大人看来也只有驾辕的份。
  <br/>
  <br/>
  笔者当时也曾为文分析：所以说，人大会议上与台下代表和台上除习近平而外的所有“领导成员”们噤若寒蝉形成强烈反差的李克强大汗淋漓，除了身病的原因还有心病的原因，那就是做奴隶都怕做不稳。
  <br/>
  <br/>
  李怡先生在他的《“不愿做奴隶”的奴隶》杂文中分析到：中国国歌是掌权者“理念的标志”，这理念就是：老百姓必须“做稳奴隶”，做一个一边唱着“不愿做奴隶”而实际在做奴隶的奴隶。
  <br/>
  <br/>
  习近平和李克强分别接班总书记和总理职务后，并肩站在党代会和人代会还有政协会的主席台上，共同高唱“起来，不愿做奴隶的人们……”已先后有几十次了。每每在这个时候，他李克强内心很可能想的都是我这“自愿做奴隶的人”，还是无时不刻地忧心着这”奴隶“的位置是否还能稳坐下去，直到两届任满……。
  <br/>
  <br/>
  所以，笔者去年还特别为另一篇相关文章取标题为《伸中指暗抗习近平？借个胆儿给李克强他也不敢》。
  <br/>
  <br/>
  至于这次李克强的重庆之行，截止本文完稿为止还没有被中共新华社和人民日报给以报道，严格地说是还没有被人民日报和新华社给以及时报道，唯一的原因就是绝不能让他抢了习近平的“风头”。
  <br/>
  <br/>
  过去已经有过多次的先例就是，如果李克强和习近平在相同的时间到不同的地方展示他们的“亲民”举措，那么央级媒体对李克强的报道 -- 特别是新华社和人民日报纸面媒体上对李克强的正式报道，首先一个规定就是时间上必须错后。所以说，李克强的这次重庆行踪日后应该还是会被新华社和人民日报简单报道一下的，或许是明天 ，或许是再迟……。但无论是哪一天，人民日报和新华社对李克强这次重庆之行的被报道内容，绝不可能像我们在国务院官网“总理”栏目里读到和看到的那些，称得上是图文并茂的较为生动的内容。

  有兴趣的读者和听众可以分别到人民网、新华网和国务院官网上对比一下，对李克强此前两次分别到山东和贵州考察的报道内容。区别立见高下。
  <br/>

  今年早些时间曾听到国内记者朋友说，几年前中宣部向 人民日报传达了内容为“报道李克强同志的新闻不能上头条”的王沪宁同志指示。所谓“王沪宁同志指示”的具体内容中，还特别强调了即使是当天的人民日报没有报道习近平同志的活动，李克强也不能上头条。
  <br/>
  <br/>
  笔者先从国务院官网上核对了李克强这次考察重庆之前的两次外出考察时间和地点，然后上网查对了人民日报纸面版的相关报道。
  <br/>
  <br/>
  该报七月七日纸面版的头版头条是“走向我们的小康生活”系列报道，《河南省栾川县仓房村咬定生态脱贫，由卖竹子到卖风景 生态饭才是长久饭》；头版二条（报眼）是《习近平同阿根廷总统费尔南德斯互致信函》；头版第三条是“在危机中育新机 于变局中开新局”系列报道《五月接待游客量环比增长逾五成 丽江旅游业 转型拓新路》；第四条才是《李克强在贵州考察时强调 突出做好重点民生工作 推动改革创新增强发展动力》，没有配图片。
  <br/>
  <br/>
  六月三日的人民日报纸面版的头版头条标题是《习近平主持专家学者座谈会强调构建起强大的公共卫生体系为维护人民健康提供有力保障》，内容很长，占了头版全部版面的五分之三强；头版二条（报眼）是《大江奔流千帆竞——写在&lt;长江三角洲区域一体化发展规划纲要&gt;实施一年之际》，文章很长，头版报眼上只刊登文章前面的一小部分便“下转第二版；头版三条是标题为《紧紧依靠人民——论坚持以人民为中心的发展思想》的“本报评论员”文章。
  <br/>
  <br/>
  如上三篇占了六月三日头版全部版面的绝大部分，只在该版右下角一块很小的地方刊登了三个政治局常委的活动报道标题，分别 为《李克强在山东考察》，《汪洋同老挝建国阵线中央主席通话》和《韩正在海南调研》。三篇的报道内容都是在第二版，都没有配发照片。
  <br/>
  <br/>
  比较之后，笔者相信无论是明天还是明天之后的哪一天，虽然李克强的重庆之行总要会被人民日报和新华社“透露”一下，但文字报道内容肯定是简而又简。而国务院官网上李克强趟泥涉水的亲民图片，人民日报纸面版的编辑们应该是不敢使用的。而李克强本人也许更不希望被人民日报的读者拿这样自己的这些亲民照片，与习近平视察的官样文章做比照。
  <br/>
  <br/>
  正如我们前面提到的那篇分析文章所说：目前洪水的重灾区是在长江上游的四川、成都、重庆一带，洪锋才刚刚通过当地，所以李克强选择前往重庆灾区；但习近平却是到长江下游的安徽去，很显然，习并没有真的想要面对灾区与灾民。习近平这三天在安徽勘灾的新闻，新闻画面都是一片风平浪静、岁月静好，完全看不见灾区的杯盘狼藉，也看不见沾满泥巴的胶鞋。说穿了，与其说是勘查灾情，不如说是一场灾情维稳的政治真人秀。
  <br/>
  <br/>
  而且，尴尬的是这次的勘灾行程还被网友发现，一名抱着孩子的受灾民众，竟然是当地公安局的治安大队副队长假扮的。换句话说，这几天的勘灾活动，只是一场精心安排的真人秀。

  <br/>
  如此对比之下，假如人民日报的纸面媒体上刊登出李克强在重庆趟泥涉水与民同苦的图片，岂不是又在把他李克强架在火上烤？
