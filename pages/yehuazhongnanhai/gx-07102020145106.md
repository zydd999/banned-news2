 <p>
  在《夜话中南海》栏目的上两篇文章《港版国安法的出台程序到底有没有违反基本法？》和《是北京当局违法了基本法还是全世界都被基本法忽悠了？》里，都介绍了有香港民主之父之称的李柱铭先生强烈谴责中共港版国安法在全国人大常委会层面的立法程序就是违宪，因为《基本法》是香港的小宪法，当中规定只有香港的立法机构能够制定法律。
  <br/>
  <br/>
  本人曾经是香港基本法起草委员会成员之一、当年也曾经和已故支联会主席司徒华一起出席《中英联合声明》签署仪式的香港民主党创党主席李柱铭先生，七年前悼念刚刚去世的英国前首相撒切尔夫人时曾对记者发表看法说：当年代表英国政府和北京谈判，撒切尔夫人已尽力而为。
  <br/>
  <br/>
  对于当年有港人不满时任英国首相“铁娘子”撒切尔夫人在就香港问题与中国谈判期间节节退让，未有捍卫港人最大利益，李柱铭说出他的看法。他说，得悉当年英国想以“主权换治权”，但邓小平最终坚拒任何让步。他个人认为，英国政府当时无法有任何作为，“难道像对阿根庭一样开战吗？”李先生反问说。
  <br/>
  <br/>
  按照李柱铭先生的说法：如果当年提出让香港独立，中共肯定是不给的，“她撒切夫人做的是尽量把细节写出来，希望透过这样给香港人多些保障……。”
  <br/>
  <br/>
  而香港民主党的另一任党主席，而且也是民主党的首位女性党主席刘慧卿女士则与李柱铭先生的看法相佐。
  <br/>
  <br/>
  英国BBC中文网站昨天刚刚刊登的《BNO：“国安法”下英国新政会不会导致香港人移民潮》一文中回顾说：1984年12月，《中英联合声明》墨迹未干，时任英国首相撒切尔抵达香港。她刚刚签署的这份文件中列明，香港人在1997年7月1日后虽可保留英国政府颁发的英国国民（海外）护照（BNO），但不会享有在英居留权。在香港，一位记者就主权移交后的港人利益向撒切尔提问道，“你答应将五百多万人交给一个共产党独裁政权，这在道义上讲得通吗？还是说在国际政治中，道德的最高形式确实就是本国利益？”

  撒切尔当时冷静回答道：“英国为香港尽了最大努力，我们做了一切可以为你们做的事情，香港每个人都对这项协议感到满意。”
  <br/>
  <br/>
  当年提问的记者名叫刘慧卿，后来成为香港政坛民主派的重要人物。去年卸任立法会议员后，她担任香港民主党国际事务委员会主席至今。
  <br/>
  <br/>
  早在七年前撒切尔夫人去世的次日，当时已经是香港民主党主席的刘慧卿这样回忆自己在记者会上，当面质问撒切尔夫人的经过：“我就问她说，首相二天前和中国签署协议，答应将超过5百万人交给一个共产政权，这样做在道义上说得过去吗？还是在国际政治层面，最高的道德境界是维护你国家的利益？她当然很生气，说我什么都做了，为你们香港人做最多，所有香港人都满意，你是唯一的例外。”
  <br/>
  <br/>
  七年前的刘慧卿还指责说：一九八九年北京六四屠城后，香港人心惶惶，但统治香港百年的英国连居留权都不肯给港人，“最简单是这样，她（英国首相撒切尔夫人）都不肯做，我觉得她这样很过分……。近日（中共涉港官员）乔晓阳讲话，他都知道很多香港人憎恨共产党，很害怕，那你（英国政府）要给他救生艇。在1984年当时罗德丞他都办公司帮人们移民，有这个问题，所以撒切尔夫人和代表英国政府的人做得不够。”
  <br/>
  <br/>
  前文中提及的英国BBC中文网站昨日报道文章的标题，《BNO：‘国安法’下英国新政会不会导致香港人移民潮》中的“BNO”是英国国民（海外）护照的英文缩写，该类护照是1987年7月1日起签发。
  <br/>
  <br/>
  英国国民（海外）国籍于1997年9月30日截止登记，日后不论任何情况都不会接受新登记。英国国民（海外）身份终身有效，可终身续领英国国民（海外）护照。
  <br/>
  <br/>
  根据英国移民规章，本护照持有人访问英国无需入国许可或签证。英国国民（海外）护照持有人拥有免签证入境英国6个月的权利。2015年3月，英国外交大臣告知下议院，英国政府不打算为英国国民（海外）扩展其免签待遇。
  <br/>
  <br/>
  BBC的报道文章中介绍，当年当面质问撒切尓夫人的香港青年女记者名叫刘慧卿，多年来，她一直争取扩大BNO持有人在英国的权利，如今终于等到开心的一天。

  “过去大部分（香港）人对英国政府没什么期望，多年来大家都觉得即使香港的情况变差了，英国也不会做什么帮香港人，他们最紧张的是自己的经济利益，”刘慧卿对BBC中文讲道，“如今我很高兴，其实它（英国）一早就应该这么做了。”
  <br/>
  <br/>
  BBC的报道文章评论说：从当年刘慧卿眼中撒切尔“居高临下”的回应，到如今向数百万香港人伸出友好之手，英国政府的转变让包括刘慧卿在内的许多人大感意外。
  <br/>
  <br/>
  本月之前，对于拥有BNO护照的香港人来说，他们手上的这本英国国民海外护照只是一份旅行证件，让他们在前往英国时可以最多一次性停留6个月，但不能凭此在英国工作或学习。目前香港约有30万人持有这种护照，英国驻香港总领馆说，约290万人有资格拥有这种护照。
  <br/>
  <br/>
  但英国的“BNO新政”这次看上去是有备而来。本月1日，中共当局悍然宣布港版国安法正式生效，英国外相多米尼克·拉布随即在英国国会表示，“港版国安法”违反《 中英联合声明》，准许符合申请英国国民（海外）护照（BNO）的香港人及其家属在英国居住、读书或工作5年，5年后可以申请定居，再居住12个月后登记成英国公民。这种方案将不设人数上限，省去目前外国移民在英定居时的大部分限制要求。拉布称，这项安排是“特殊、专门的”，是英国在兑现对香港人的“历史承诺”。
 </p>

  中英两国在1984年签订的《中英联合声明》中指，香港在主权移交后享有高度自治权，除外交和防御等事项，其享有的权利和自由50年不变，直至2047年。而7月1日开始实行的香港《国安法》则允许北京直接在香港设立机构处理与“国家安全”有关案件，北京派遣的人员与机构均可不受香港规定限制。英国政府认为，北京强推《国安法》表明，中国没有信守在联合声明中做出的“对香港人的承诺”。
  <br/>
  <br/>
  此举引发了北京的愤怒。中国驻英大使刘晓明本周表示，英国此举“严重干涉中国内部事务”，违反了英国在联合声明中做出的不给予BNO持有人居英权的承诺。而在过去，北京曾多次强调，《中英联合声明》已经“过时无效”，是没有任何现实意义的“历史文件”。
  <br/>
  <br/>
  那位貌似战狼的中共现任驻英大使在他的视频讲话中，还煞有介事地自称引用的是美国国家安全顾问布热津斯基（Zbigniew Brzezinski）的话，“如果我们将中国变成敌人，那中国就会变成敌人”，借此警告英国政府：一切违背14亿中国人意愿的做法都注定要失败。他说，当主权平等和互不干涉的原则被侵犯时，关系就不可避免地会遭遇“挫折，甚至倒退”。
  <br/>
  <br/>
  不过，仔细聆听了这位刘大使的这番讲话，感觉他更为在意的并非是英国决定为拯救港人而改变BNO政策一事，而是英国对华为的政策。因为英国首相约翰逊在上周指出，中国现在或许会被视作“敌对国家供应商”，这明显会被认为是一次严重的冒犯。所以这位大使表示，中国官员从未这样形容过英国。
  <br/>
  <br/>
  这位大使在讲话中也表示了希望英国重新考虑，向英国海外公民BNO护照持有人提供公民身份的决定；同时也还说了一句，“我们姑且看看事态发展，我们需要视乎英方真正采取的行动，以决定我们相应的措施。”
  <br/>
  <br/>
  但他的这番话，居然就被部分外界媒体解读成了“中国驻英大使称不排除禁止港人离境移居英国”。
  <br/>
  <br/>
  相关分析报道文章中说：英国答允给予差不多300万港人一条成为英国公民的路径，进一步恶化与中国之间的关系。中国上周宣布在这个前英国殖民地实施一本全面性的国家安全法，约翰逊形容这本国安法是“明显而且严重牴触”1984年，伦敦与北京所签署的《中英联合声明》。尽管中国威胁英国此举将有“严重后果”，但英国政府迄今仍坚持对港人提供避难港的计划。
  <br/>
  <br/>
  不过英国外相拉布承认，假如中国禁止港人离开香港，英国政府恐怕是无能为力阻止。职是之故，计划能否落实似乎要看北京下一步的举措。

  当然，谁也不能排除疯狂的习近平当局日后对港政策会更疯狂的可能，但仅从逻辑上判断，习近平当局下决心出台港版国安法的过程中，肯定是对吓跑不少香港人做足了心理准备的。中共决策者们自然会因此，而回想起三十一年前的“六四”镇压的外交后果是什么。其他西方国家不说，仅美国对一九九零年四月之前入境美国，其中也包括许多六四镇压之后是以合法身份离开中国进入美国的中国人发放的“六四绿卡”就达八万之多，因此获益的许多人都是当年的公派留学生。而一九九零年四月后陆续离开中国，进入美国之后获得政治庇护者更是不计其数。
  <br/>
  <br/>
  不过，一九九零年四月之前进入美国，在当时的美国政府宣布了“六四绿卡”申请时限之时人在美国但并未提出申请，日后按期回国者也不是没有。最典型的，就是当时分别在美国不同大学里做访问学者的王沪宁和陈希。两位正好是习近平眼下党务工作的第一助手和第二助手：一个是现职中共中央政治局常委，一个是现职中共中央政治局委员和书记处书记。
  <br/>
  <br/>
  有当年的“六四绿卡”之鉴，习近平当局在决策出台港版国安法的过程中，肯定也已经都预料到了即使没有一个BNO的事先存在，无论是英国还是美国，无论是加拿大和澳大利亚还是法国，甚至包括日本及台湾等，都会为一批因恐惧港版国安法而背井离乡的香港人提供人道援助的。
  <br/>
  <br/>
  那么试想，一九八九年六四镇压之后，美国和西方国家的政治庇护政策都没有导致中共当局全面禁止国人出国，那么现如今仍然还在继续打着“一国两制”幌子的中共当局应该也不会在港版国安法出台后，要求香港政府禁止港人离港。在此前提下，只禁止持有BNO护照者离港的可能性就更小了。
  <br/>
  <br/>
  恰恰相反，所谓“留港不留人”的说法，很可能就是中共当局为出台港版国安法预设的、仍可以从政治和经济层面，特别是经济层面承受的底线。甚至还可以设想，随着港版国安法的出台，不想留或者不敢留的一批港人离乡出走，也许是正中中共中央当局和香港林郑当局之下怀。后续的分析内容，留待下次节目向听众们做详细的分析和介绍。
