  <strong>
   1.引言： CCP的红线问题
  </strong>
  <br>
   如果一个国家的执政党是代表国民利益的，那么国土问题、统一问题都是红线，但是中国共产党（CCP）只代表自己的利益，稍微了解历史就知道，在CCP的眼里是没有国家民族这些概念的。比如CCP在赣南、闽西武装割据时期所建立的政府叫苏维埃政府，宣传口号是武装保卫苏联，并且鼓动边疆少数民族独立，也鼓吹台湾独立。这些人是中国人吗？说到底就是一伙强盗找靠山，唯恐天下不乱。CCP获得政权后，中国边边角角的国土被随便送；到江首领时期更是承认了被俄罗斯占有的大片领土（不少于150万平方公里，列宁曾经都承认中俄之间订立的是不平等条约）。所以CCP关切台湾并不是因为领土问题，而是因为意识形态问题。自由民主在华人社会的存在反衬出CCP专制统治的不合理性，现在的香港是个明证。如果CCP足够强大，那么东亚、东南亚甚至更广阔的地方也不能自由民主，周恩来曾经不是说可让东南亚一夜变色，强盗足够强大就变成了魔鬼。这就很清楚了，CCP口中的祖国统一实质上是专制延伸。既然这样，没有国家、民族、国民支撑的红线也不能称之为红线了，也就是跟强盗力量强弱随时可以改变的界限，也可以说只是心里的界限。那么，地球人须要衡量一下现在的CCP到底强不强。
   <br>
    <br>
     <strong>
      2. 首先看人的素质
     </strong>
     <br>
      大陆人有史以来没有这么弱智过，没有最基本的判断力，不知能否跟无脊椎动物相比。这是CCP将近一百多年来洗脑的结果。洗脑的目的是想把人们变成无脊椎动物，以便专制统治，洗着洗着把CCP自己的后代也洗成了脑残。别看会写些党八股就以为有智识，世界都进入智能时代了，那些写手还在用马列教条，这本身就是弱智的表现。说得更深入一点，大陆的语言系统已经发生很难逆转的改变，语言对人的文化思想具有根本性的影响力。看那些八股文、马屁文和马屁语就知道大陆的堕落有多严重。台湾没有拍马屁的环境，所以有个叫林毅夫的兵哥从金门游到对岸，摇身一变成了高层的智囊，开口是“某某的英明领导”，闭口是“我们经济学家”。此人前段时间在厦门又重复五年前说辞，中国五年后会进入发达国家收入水平。  愚顽的领导只能配马屁精。
      <br>
       <br>
        大陆的状况令人唏嘘，这种弱智和堕落是整体性的（多一个林毅夫也不要紧）。外界都以为大陆在发展，实际上是在鼓经济泡沫，竭泽而渔，转移财富。谁都想逃离这个国家，但只有有权势和有财力的人才能逃离出去。可笑的是，被管控的大陆舆论一致性地批判文明世界的腐朽和堕落。大陆什么事都做不成，打仗照样不行，打仗首先是看人，其次才是看武器。
        <br>
         <br>
          先看领导层，中国各方面现在被一批年轻时经历文革的人所掌控。文革十年没大学，恢复高考后，极少数人上了大学，这些人有思想的通常坐不到重要的岗位上（不是指官职的大小），不会思考的人仍然跟没上大学一样。这是体制设置的结果。所以，总体上看，掌控中国的这批人（50多岁到70多岁）智识短浅，思想简单粗暴，可把大陆现阶段的政治称作红卫兵政治。红卫兵是
          <span href="https://www.secretchina.com/news/gb/tag/毛泽东" target="_blank">
           毛泽东
          </span>
          阶级斗争的工具，这代人也算是牺牲品，当然后来混得一官半职的人可能不自知。红卫兵的特点：一是崇拜领袖，对领袖着魔般的思想依赖致使学习和与人交流的能力散失；二是爱斗争，从斗争中获得满足感，被领袖的魔力所控制，斗争起来往往非常残酷，甚至六亲不认；三是思想空虚，一个人的头脑不被知识和理念所占据，而是被恶魔所控制，那么所作所为必是违背常理常识，以至违背人性，所以会产生思想矛盾和思想空白。
          <br/>
          <br/>
          台湾人没有经历80年代以前大陆接连不断的政治运动，也许体会不到那种地狱般的感觉。而现在大陆正在回到，许多方面已经回到文革时期，有的人称之为文革2.0版。为什么会这样？因为现在掌控中国的这批人就是红卫兵的思想水平，年轻时思想已经被固定（以前也有人称中国是巨婴国），知识和思维能力都太欠缺，什么都不会，只会斗争，只会沿着毛的老路走。因为头脑空白，实在是没东西，偶尔说些看起来合时宜的话，自己也不知道是什么意思，完全与文明世界脱节，使用一套流氓语言系统自说自话。恐怕地球人还会以为中国高层会为经济的滑落担忧，其实他们最担忧的是手中的权力。所以在他们看来闭关锁国比开放还好，像现在的金三胖或以前的毛，被人山呼万岁多过瘾。
          <br/>
          地球人都以正常的思维推测中国大陆和大陆的领导层，结果误判。那为什么这批人又会上台呢？就是因为如前所述的体制。到了这个时候这个年龄段的人占大多数，像胡耀邦、赵紫阳这样的人现在也有，但当时二人都下来了，现在这类人就更难上去了。这也是中国帝制的铁律，打天下，守成，腐败堕落，失去江山；到了朝末妖魔鬼怪就出现了，只会把最差的人推上去。两年多时间，大陆社会竟然滑落得这么快。把大陆当做现代的正常国家根本是天大的误判。这样的领导层怎么可能有战斗力！一遇到问题就会混乱不堪，就像红卫兵武斗一样，即贪生怕死又表现得斗志昂扬，打不过对手就作鸟兽散。心智根本不成熟，国家大事也会成为儿戏。
          <br/>
          <br/>
          那么，军队的情况又如何呢？俗话说，兵熊熊一个，将熊熊一窝；国家领导层如此，部队哪能有出色的表现！江一上台，以腐败治国著称，高层的默许和军队的独立性使得军队的腐化堕落社会一无所知，军队成为国家的大蛀虫之一。买官卖官，明码标价；参与经商，倒卖军需品和军用资源；虚增和贪污军备物资的研发和生产经费；参与走私等等，腐败不亚于各级政府。已被曝光的军头只是冰山一角，整个军队就是一个烂泥塘。所以，军费年年增高，武器装备的技术支撑却仍然靠偷窃和模仿。
          <br/>
          <br/>
          从功能上看，共军完全是党卫军，在集权的顶峰时期，甚至成为个人的禁军，如在毛时期。集权必然导致腐败，军队非国家所有，军队的腐败也自然而然产生。那么，军队有作战任务的时候，就有为谁而战的问题。一支部队只有当其为百姓，为民族，为国家而战的时候才最有战斗力。如果是为了派系，为自己利益而战那成了内战了。而这几十年，军官正是纨绔子弟扎堆的地方，部队一直是肥水横流的地方，别说有可能在部队里升官的情况，就是参军做普通士兵也有不少人须要向征兵单位（武装部）送礼，以便转业后能有一份工作。这样的军队，一遇到战事，人人都想逃避。别看一些军官天天叫嚣战争，他们还以为有那么多炮灰可用，而他们本身都是贪生怕死之徒。举个例子，现在时不时出来叫嚣一下的罗某应该是比较大的军官了，曾经对越作战的时候（当时可能是连长什么的）被其老爹从前线曲线调回。如果说远些，当年推翻国民政府也是靠欺骗农民兄弟做炮灰的。为什么说是骗，看看现在农民的生存状况，土地仍然是CCP的（不是打土豪分田地吗），在泡沫经济面前度日艰难。所以说，有没有战斗力首先问一问，从军官到士兵，为谁而战？对个体而言，战争最重要的代价是生命，在生命可能瞬间消失的情况下，生命的价值问题就自然被提出。CCP忽悠国人去做炮灰有没有当年那么容易？
          <br/>
          <br/>
          这就是为什么战争不能只盯着武器看。再说，共军的武器装备先是模仿苏联（包括苏联解体后从乌克兰大量购买武器和引进那里的武器专家），后靠偷窃美国等国的技术，而其自己的研发能力却十分有限，性能如何外界不是很清楚，但内涵专家应可估算得到，没准用起来错漏百出。看那仿造的航母，做靶子还差不多。核弹是能吓人，而且本来就是用来吓人的，使用核弹意味着也在威胁自己。美国本土不易受到轰炸是CCP不敢使用核弹的保证。历史地看，“会叫的狗不咬人”这句话基本正确，想想看现在谁更怕，所谓的红线只是心理防线，CCP保政权，魁首保个人权力才是实质。
          <br/>
          <br/>
          <strong>
           3. CCP是怎样的存在
          </strong>
          <br/>
          人性之善恶如硬币的两面，人类在发展的过程中不断努力地把恶的一面按下不让其露出。离我们最近的上个世纪人世间出现过许许多多惨绝人寰的恶行，至今人类的正义力量基本上消灭了这些施恶者。然而，其中一个共产恶魔现在仍然危害中国大陆，并且正在施害于全球，这是一百多年来生命力最强的邪恶力量。马克思写《共产党宣言》的时候，开篇就自称共产党是幽灵，其后对苏联和中国来说真如恶魔附体。苏联是解体了，但中国大陆仍然掌控于魔掌之中。
          <br/>
          <br/>
          恶魔的魔杖是一种思想武器，叫做马克思主义，它很强的欺骗性激起了人们的破坏欲望，而且它经过俄罗斯和中国后连续发生两次大变异，形成更具破坏性的恶魔变种。这就是CCP的由来，成为马克思主义变种毛泽东思想的载体。马克思主义宣扬世界大同，去除国家和民族观念，但从巴黎公社维持七十二天到后来三个共产国际的解散，历史已经做出选择，商品经济充分发展的地方，共产主义留不下来。因为有最基本的两点可看出其结果是走不通的，一是计划经济对资源配置是不合理的并且必然导致资源垄断，二是共产主义抹去了不同国家和民族之间历史、文化、生活习俗等方面的差异性。发生了两次大变异，马克思主义到俄罗斯后增加了暴力成分变成列宁主义，到了中国之后又增强了欺骗功能形成毛泽东思想。是这两个落后的、专制思想根深蒂固的国家才能保留和延续了马克思主义并使其变异。此后，这些思想武器的危害不仅表现出违反社会的发展规律，而且在反智、反人性、反社会的道路上越走越远。从成立的那天起CCP即已拥有恶魔基因，一路打打杀杀，问题是夺取政权后仍然这样。在CCP的潜意识里谁都可能是敌人，大到文明大国，小到草民个体。朋友、亲人是；同事、战友是；上司是，下属也是。这种恶魔只有通过打打杀杀才能保持其邪性，给所谓的社会主义国家造成的人间灾难罄竹难书，其破坏力之大连其自身都难以生存。所谓皮之不存毛将焉附，被寄生的机体都要崩解了，它自身也就不存在了。苏联解体前，中国大陆到70年代末也几近崩溃，那种状态是物质和精神两方面都极度贫乏，人们近乎绝望，是改革开放让CCP苟延残喘至今。
          <br/>
          <br/>
          认为改革开放是CCP的顶层设计，那是绝对的错误，这种政治僵尸就是僵死了也会把自由思想，把文化教育，把政治、经济勒得死死的，也就把百姓勒得死死的。是在谁（包括CCP）也活不了的情况下，由底层自发推动的上层稍微松点缰绳而产生的一些变化而已，这就称作改革开放了。后来加入世贸恐怕它自己都没想到能获得如此巨大的好处。一阔脸就变，霸气十足，本性立刻暴露；不守文明规则，吃相难看，对文明世界如闯入瓷器店的蛮牛。做个比喻，基因这东西很难改变，没有营养则机体萎缩，有了营养则机体会长出它本来的样子。所以CCP又重拾那骗人的把戏，不忘初心，牢记使命，命运共同体等等，跟以前要解放全人类是一样的意思；同时也更有条件像以前渗透国民党一样渗透到民主法治国家和众多国际组织。更令地球人愤怒的是绑架14亿人宣扬国家恐怖主义，公开会见塔利班组织让其在北京街头晃悠，提示美国911“殷鉴不远”，暗示偷袭珍珠港事件，炫耀核弹头。这种没是非、反人类的恶魔如何还能在这个星球存在。
          <br/>
          <br/>
          在资源方面，人、财、物、技术等等，什么在CCP眼里都是工具，从技术到人才，从文化资源到土地资源，所有东西CCP只看它有否利用价值。这就是为什么CCP在中国几十年、上百年（从其活动算起）从来不是在培养人才，而是一直在驯化统治工具和迫害人才，使中国人变得更加无脑和顺从。本来就没有国家和民族的观念，这里没有人才和技术，可以到国外去偷，去挖，通过各种手段让世界上的财团和资本、政客、有影响力的媒体为自己服务，如此侵蚀正常的世界次序，如果做大了，那就是解放全人类了。有思维能力和开拓型的人物上不来，而把当年的一班红卫兵推上来，这就是CCP的基因决定的。如果不是这个在台上，而是唱红打黑的那个上来，那结果也差不多！
          <br/>
          <br/>
          认为大陆这40多年来发展了，这又是天大的错误。首先，政治一点没变，在大陆走向民主之前，大陆的所有问题都是政治问题。然后，教育文化改变了没有？这方面的腐朽堕落不知要花多少时间才能说得完；科技发展了没有？大陆没有科技人才，可以说以前还是民国时期留下的一些老本在起作用，CCP要的是又红又专的砖头，不需要科学思想和创造性思维，这40多年来打开国门更可以从全世界偷技术和挖人才了。被地球人津津乐道的是中国的经济，GDP全球第二，本土砖家早就宣布某些方面已经超过美国，多少年后又会全面超过美国。大陆的GDP是怎么构成的？一是来自世界工厂，二是来自房地产。谁都能看得出来，大陆的经济是粗放的，脆弱的，靠廉价的劳动力也好意思说经济强大，靠卖地和堆砌钢筋水泥也敢炫耀GDP第二！现在的烂尾楼（包括各种园区）数不胜数，就不知西藏有没有了，这又跟粮食安全有关。
          <br/>
          现在的大陆已是千疮百孔，只要稍微了解，对这种邪党恶政就不是怕了，而是愤怒。
          <br/>
          <br/>
          <strong>
           4.台湾的关键时刻
          </strong>
          <br/>
          从国际格局看，现在逐步形成不对称的两极，以美国为首的多数国家代表文明和正义，日益孤立和垂死挣扎的CCP代表野蛮和邪恶。这么明显的对垒，谁都能做出正确的选择；但利益的诱惑和恐吓也可能导致一些国家和地区迟疑。就台湾来说，历史的和地理的原因，跟其他地方不一样，台湾不得不做出明确的表态，稍微迟疑，成本会升高。CCP再弱，捶台湾都不是问题，就像北朝鲜被国际制裁会拿韩国撒气一样；CCP比现在还强，则可能威胁到东亚和东南亚；再强可称霸全球。这是马列的基因所带来了，所以把CCP当做中国人也是误判。
          <br/>
          <br/>
          百年来受CCP的影响，大陆人羡权拜金，狭隘自私，暴力和仇视世人，一点不像原来的中国人；相反，台湾人因为很好保存中国的文化传统，更像中国人。与大陆人相比，台湾人算是幸运的；与香港人就不用比了，现实摆在那里；与日本人、新加坡人相比可能会觉得有那么一点不好的感觉吧，始终有一只魔爪在头顶上罩着。台湾人属于中国人，这是改变不了的事实；处于什么样的社会制度，那是另一方面的事。自从CCP在中国产生以来，它首先和主要欺负的是中国人，所以作为中国人都逃避不了当下要怎么做的问题。
          <br/>
          <br/>
          几个月前，香港被誉为柏林墙所在地，自从大陆的港版国安法推行，柏林墙移到了台湾海峡，表明讲道理是没用的。不知台湾人有否压力。无论如何没得选，历史走到这一步，加上处于地理位置上的关键点，台湾注定是世界的关注点。事有凑巧，突如其来的疫情凸显了台湾社会的成熟度和发达水平，特别是不容易受骗这一点让世界刮目相看。顺理成章，台湾与多国互动，提升了国际地位。
          <br/>
          <br/>
          然而，瓶颈终会到来，台湾人和民国政府是如何看待自己在国际上的地位以及台湾与大陆的关系的？估计多数台湾人希望与大陆分家，可能民进党也有这个倾向，这是人之常情，大陆与台湾现在各方面的差距实在很大。国民党在两蒋后越来越像南宋小朝廷，没勇气，没骨气，有些党员帮助CCP只能是为虎作伥，跟统一毫无关系，只是个人获得了好处而已；但估计有少部分国民党党员希望在大陆恢复中华民国，这一点大陆的百姓翘首以盼。
          <br/>
          <br/>
          不管出于何种考虑，提升台湾的国际地位必须亮出正统身份，这一点是共识吧！中华民国取代帝制，总统加五院分权体制，国民有选举权，这是现代正常的国家；CCP推翻国民政府没有法理依据，是无道伐有道，且是极权体制，至今还压迫和剥削大陆百姓。所以，恢复联合国常任理事国席位，与多国恢复以前的邦交正是时候。统一、独立这些事美国这样的国家恐怕不会插手；但是维护台湾不受侵犯，维护台湾自由民主的社会制度，这还用说？美国一直是这样做的，这既是美国自视为它的大国责任，也是维护它自身安全的需要。所以，老是担心CCP动武未免太小心了。
          <br/>
          <br/>
          一周前，民国外长吴钊燮在回答质询时说，台湾现时不寻求与美国的全面外交关系，并说先自助而后才能求到人助。这番话引起多方评论。如果没有美国的帮助，台湾的压力是很大的，而且可能会像香港一样，在压力下先一国两制而后再变成一国一制。美国这么明显的帮助和热心，民进党政府却不敢接受，这反映出民进党心虚和讨巧的心态。小国寡民是老子倡导的一种理想社会，但在残暴面前无法偏安一隅；韩国人也很幸福，但一想到北边的流氓兄弟顿时失色 。如果这时候是国民党执政没准反而能大刀阔斧。台湾在维护自身的利益、安全和国际地位方面，建议向以色列学习。台湾先恢复国际地位，才更有利于跟邪党恶政抗衡，光复大陆还是独立等等都是以后的事。这样做对台湾，对大陆，对全世界都是有利的，唯独会让CCP害怕。 人有时候是被迫推上历史舞台的，如果1949年没有国民党退守台湾并守住，那么台湾的今天也许跟海南岛一样，是很平静；历史选择了台湾让今天的台湾不平静。
          <br/>
          <br/>
          另一方面，很明显，僵尸即将僵死，共产主义运动已经走到末路，暴力都使用过了，骗人也骗不下去了。同时，社会主义国家都具有王朝或帝制特征，所以也就逃不过王朝更替的铁律。苏联解体有外部的压力，但根本的原因是内部的腐败。一个专制体制的建成，它的走向就指向灭亡那个终点；外部的压力往往对灭亡的时间起调节作用。而今中国，腐败更不用说，终点的表现是机器无法运转了。然后，国际局势的逆转，这是外部压力。CCP的滔天罪恶让大陆百姓深受其害，持续的倒行逆施，终使现在全世界都无法容忍。
          <br/>
          红线是谁设的？依据是什么？
