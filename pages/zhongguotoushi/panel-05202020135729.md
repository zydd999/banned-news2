  <b>
   一、本次世卫大会背景
  </b>
  <br/>
  <br/>
  1） 世卫视讯大会—习近平在谭德塞安排下率先发表讲话，自夸“公开透明负责任”。特朗普决定不致辞。
  <br/>
  <br/>
  2） 台湾参会议题——谭德塞推给世卫大会委员会作决定，台湾未获邀函。
  <br/>
  <br/>
  3） 特朗普長信斥責世卫总干事，限期30天整改，否则美国退出。特朗普称世卫是中共的“傀儡”，特朗普指责中共试图掩盖疫情以及世卫未能追究北京的责任。
  <br/>
  <br/>
  4） 澳大利亚的牵头，欧盟草拟、俄罗斯、日本、尼日利亚等122个国家签署要求启动对新冠病毒来源及蔓延的“独立调查”。
  <br/>
  <br/>
  5） 投票结果：世卫大会194国一致同意，尽快独立调查世卫疫情应对。

 <p>
  世界卫生组织（WHO）的194个成员国，包括美国和中国在内，于周二（5月19日）的世界卫生大会上无异议通过一项决议，要求对全球在新冠病毒大流行病疫情上的反应进行“独立评估”，同时也包括调查世卫本身在这场疫情中的作用。
  <br/>
  <br/>
  这一决议案获得与会194个成员国的一致同意。决议案由欧盟提出，要求世卫的调查应独立和完整，包含每一个时间段的评估，且应尽快公正展开，以期提高应对大流行病的全球预防能力。
  <br/>
  <br/>
  决议案内容是什么？
  <br/>
  <br/>
  决议案呼吁对国际反应进行一次“公正、独立和全面的评估”。
  <br/>
  <br/>
  决议案将重点放在世卫“与Covid-19大流行病有关的时间表”上。世卫因迟迟未宣布新冠病毒为全球紧急卫生状态、导致全球未及时预警已备受全球批评。
  <br/>
  <br/>
  该决议还呼吁各国确保“透明、公平和及时地获得”任何新冠病毒的治疗或疫苗，并敦促世卫调查“病毒来源和（病毒）向人类的传播途径”。
  <br/>
  <br/>
  世卫面临压力：
  <br/>
  <br/>
  美国总统特朗普称世卫是中共的“傀儡”，并暂停对世卫的资助。美国是世卫的最大捐助国。
  <br/>
  <br/>
  此外，特朗普还指责中共试图掩盖疫情以及世卫未能追究北京的责任。
  <br/>
  <br/>
  美国卫生部长亚历克斯·阿扎尔（Alex Azar）在周一的大会讲话中提出了严厉批评。
  <br/>
  <br/>
  他说：“我们必须坦率地说，本次（疫情）爆发失控的一大主要原因是：世卫组织未能获得世界所需要的信息，而这种失败导致许多人丧生。”
  <br/>
  <br/>
  美国认为，世卫忽略了台湾关于疫情严重性的预警，并推迟宣布疫情大流行。
  <br/>
  <br/>
  美国是遭新冠病毒肆虐的重灾区，周一的疫情致死人数已超过9万人。
  <br/>
  <br/>
  虽然欧盟、英国、澳大利亚和新西兰在内的国家没有像美国那样批评世卫，但他们也一直在推动对世卫如何处理大流行病进行调查。
  <br/>
  <br/>
  欧盟发言人维妮妮·巴图·亨里克森（Virginie Battu-Henriksson）说，作为审查的一部分，有几个关键问题是需要回答的。
  <br/>
  <br/>
  她说：“这种流行病是如何传播的？其背后的流行病学是什么？这对于我们继续避免另一种这种流行病至关重要。”
  <br/>
  <br/>
  这份决议案比较敏感的部分是，对世卫组织总干事谭德塞提出11点要求，特别是要求他应与世界动物卫生组织（OIE）、联合国粮食及农业组织（FAO）以及各国密切合作，通过科学合作的“实地考察”，调查人畜共通传染病毒的源头及向人类的传播途径包括中间宿主的可能作用。
  <br/>
  <br/>
  决议案草案同时要求，谭德塞与成员国协商，在适当时机尽早就国际社会应对疫情、世卫行动及其疫情时间表，逐步展开一个公正、独立、全面的评估，检视从WHO协调下的COVID-19国际卫生应对措施中获得的经验和教训。
  <br/>
  <br/>
  世卫的应对：

  <br/>

 <p>
  世卫组织负责人谭德塞（Tedros Adhanom Ghebreyesus）已经同意将对世卫本身的大流行处理进行审查，但他同时驳回了对世卫进行大范围改革的建议。
  <br/>
  <br/>
  他说，将“尽早”进行一项独立评估，评估可以吸取的教训、并提出任何建议。
  <br/>
  <br/>
  他呼吁各国强化世界卫生组织的作用。
  <br/>
  <br/>
  不过，大会通过的决议案对世卫组织应对大流行及其时间顺序采取何种具体审核仍含糊不清。
  <br/>
  <br/>
  外界怎么看中共回应？
  <br/>
  <br/>
  去年12月，在中国武汉市发现首例新冠病例。
  <br/>
  <br/>
  中国国家主席习近平周一在大会上再次重复说，中共对疫情采取“公开、透明”的行动，同时罕见改变立场说支持调查病毒起源、并承诺未来2年对世卫认捐20亿美元。
  <br/>
  <br/>
  但中国外交部发言人赵称中国赞成的是在疫情过后，在世卫主导下的对疫情的评估，不是澳大利亚所说的独立调查。
  <br/>
  <br/>
  但全票通过的独立调查决议，特别是指出世卫组织本身也是被调查评估的对象，这就打脸了赵立坚。
  <br/>
  <br/>
  习近平周一的公开表态是中共应对疫情对外宣传的一部分，旨在转移病毒源自中国武汉、全球追责中共隐瞒疫情造成世界大范围损失的声音。
  <br/>
  <br/>
  白宫国家安全委员会发言人约翰·尤利奥特（John Ullyot）周一在一份声明中回应说，中方承诺注资20亿美元“是一个象征（令牌，token），旨在转移注意力，越来越多的国家呼吁——要求中国政府对其未履行义务承担责任”，因为中共没有“说实话并警告世界即将发生的疫情”。
  <br/>
  <br/>
  尽管中国驻澳大利亚大使成竞业已经对澳大利亚发出贸易威胁，不买他们的大麦和牛肉，但是澳大利亚像他们的牛一样倔强，总理莫里森说：原则问题是不能拿贸易做交易的。显然不惧怕中国的贸易制裁。
  <br/>
  <br/>
  澳洲外长佩恩（Marise Payne）表示，她乐于见到这项由欧盟草拟、澳洲支持的“全面性”动议。她还说，动议获得各国广泛支持很振奋人心。
  <br/>
  <br/>
  最新版动议也呼吁世界卫生组织（WHO）秘书长谭德塞（Tedros Adhanom Ghebreyesus）“启动程序，逐步进行公正、独立、全面性评估，包括在适当情况下使用现有机制，检讨世卫协调各国因应COVID-19时学到的经验和教训”。
  <br/>
  <br/>
  自去年底中国武汉爆发疫情以来，全球已有超过460万人确诊染疫，逾31万人不治。
  <br/>
  <br/>
  中国的态度刚开始是坚决拒绝对病毒源头进行调查，中方发言人明确表示反对这种有罪推断式的调查，但是后来中方发言人改变了先前的拒绝调查态度，表态同意来调查。
  <br/>
  <b>
   <br/>
   二、北京 机关算尽太聪明，反误了卿卿性命
  </b>
  <br/>
  <br/>
  中共绞尽脑汁，竭力加强全球大外宣以及在背后游说运作，但是谁知却反而恶化了其孤立的处境。
  <br/>
  <br/>
  世态炎凉，各国百态；北京五味杂陈，未曾料到：
  <br/>
  <br/>
  1.       世界各国一边倒的站在了美国一边，包括以前关系不错的国家。这是令官方和吃瓜群众都意想不到的事情。
  <br/>
  <br/>
  2.       俄罗斯竟然早早就加入到这个阵营中来，这是令中国高层感到吃惊的事情。毕竟中国在俄罗斯遭到美欧经济制裁濒临崩溃时，是中国拉他们一把，签订了106美元一桶的大单，并提前付款，可以说这笔大单占了俄罗斯当年整个GDP相当重的比例份额。
  <br/>
  <br/>
  3.        中国周边国家一律站到了中国的对立面上去。日本、韩国、印度、土耳其、马来西亚、乌克兰、白俄罗斯、新西兰、澳大利亚等，这是很多人没有想到的事情。
  <br/>
  <br/>
  4.        美国居然没有站出来挑大梁，而是退居幕後。相反加拿大的总理特鲁多却强硬批评中国：“我认为，很明显，围绕新冠疫情的起源和早期行为，各国有很多问题，尤其是中国”。（他派来中国拉医疗货物的大飞机白跑一趟，空载而归让他颇多恼火。）
  <br/>
  <br/>
  5.       没想到非洲的穷哥们本来都是中国的好朋友，不想这次却是集体反水。尼日利亚成了带头大哥，第一个向中国递交追责索赔的国家就是尼日利亚， 而且是狮子大开口，一下就要索赔2000亿美元。非盟也是加入索赔队伍里的急先锋，真是令人头疼，看来用金钱买来的友谊靠不住。
  <br/>
  <br/>
  6.       更没想到的是世卫组织总干事谭德赛同志竟然急速转弯，也站在了那些要求调查病毒源头的一边。更让人想不到的是，他现在竟然主动地要求研究台湾出席世卫大会事宜。譚書記成為强国心口永远的痛！
  <br/>
  <br/>
  7.       欧盟向世卫组织提案，对病源展开独立调查。这也是继澳大利亚之后又一国家联盟发出的独立调查请求，拥有27个国家的欧盟在世界上話語權僅次於美國，難於置若罔聞。
  <br/>
  整个非洲联盟及其成员国也不甘落後，早已加入。
  <br/>
  <b>
   <br/>
   三、无力抗拒，就加入；并追随大流，伺机搅浑水
  </b>

 <p>
  在新冠疫情全球大流行背景下，中国被推到风口浪尖。
  <br/>
  <br/>
  没有一个国家出来为中国站台，厉害国孤独站在中央，状如被告。茕茕孑立，形影相吊。
  <br/>
  <br/>
  倘若迎头相撞，抗拒调查，北京必然失去了一切道义支撑，等价于直承有罪。
  <br/>
  <br/>
  枪打出头鸟澳洲，以加徵大麦以钜额关税相要挟。不料反而激起澳洲更激烈反弹。
  <br/>
  现在北京紧急改换策略，不能正面反对，不敢与提案国对立，那就乾脆顺势入伙，成为它早先说的“反北京的政治操作”大潮中的一员，然后在其中离间、挑唆、转移调查的区域、目标及方式，混战一场，以便金蝉脱壳。
  <br/>
  <b>
   <br/>
   四、新冷战的首次重要遭遇战
  </b>
  <br/>
  <br/>
  中国在本次世卫大会已经面对多重风险。最关键点是，以国际独立调查为起点，会不会最终形成目“定格”——明确指向中国，并朝着追责、追偿的方向发展？
  <br/>
  <br/>
  这次大会在一定程度上拉开了新冷战——世界秩序重组和变革的序幕，预演了中共对抗美国为主的新西方的新模式：从北京精心佈置的开幕式企图掌握主导话语权，曲折排挤抗疫模范国台湾的谋略，到最后在疫情调查的提案中兵败如山倒，孤立无援， 预示了习政权在国际政治中聪明反被聪明误的基本谋略和策划方式，大会的博弈也预示了疫情之后世界秩序的未来。
