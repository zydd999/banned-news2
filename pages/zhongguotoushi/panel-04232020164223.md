  <b>
   一、山雨欲来风满楼
  </b>
  <br/>
  <br/>
  何谓“珍珠港时刻”？
  <br/>
  <br/>
  A. 中共方面：（突破底线，以攻为守，准备与美国及国际社会全面对垒）
  <br/>
  <br/>
  1）香港大抓捕（15位民主派）事件 &amp; 公然违反基本法，中联办直接出手干预立法会
  <br/>
  <br/>
  （中联办日前发声明谴责民主派瘫痪立法会内会，干预香港内部事务，涉违《基本法》第22条。中联办发稿称不受该条文约束，港府则在6小时内三度出稿，与中联办统一口径，推翻了以往说法。
  <br/>
  <br/>
  民主派立法会议员（20日）召开记者会，指不担心释法，因为现时中联办自行定义《基本法》第22条比（人大）释法更差。香港一国两制将荡然无存。）
  <br/>
  <br/>
  何以选择此时动手？
  <br/>
  <br/>
  1、大疫期间，港人无法上街抗议集结，各国集中精力抗疫，无暇顾及香港
  <br/>
  <br/>
  2、制造恐怖气氛，意图重新强行通过2003年失败了的23条立法
  <br/>
  <br/>
  3、影响即将来临的香港立法会选举
  <br/>
  <br/>
  香港两大事件之意涵：表明习已经破罐破摔，准备一步步抛弃“一国两制，港人治港”，接受可能到来的美国《香港人权与民主法案》的制裁，即使丧失香港的特别关税地位， 从而在经济上遭遇重大打击也在所不惜，他的底线只是保住他的权位、政权。
  <br/>
  <br/>
  2）辽宁号出巡“第一岛链”事件：
  <br/>
  <br/>
  辽宁号前往西太平洋“第一岛链”（指东亚北起日本群岛经琉球群岛，台湾，南至菲律宾的链形岛屿带之间海域）航行，对台湾和美国传达的信息十分浓厚。当前美国在印太地区部署的罗斯福号、里根号、卡尔文森号和尼米兹号4艘现役航母，全因为新冠疫情感染严重，暂时停泊于各军港内。北京趁机出行，有浓厚的机会主义耀武扬威意味。美方以出动B52 重型轰炸机出航应对之。
  <br/>
  <br/>
  中共并有意让世界知道它正在重启非法的核试验，有意暴露其核基地，提醒美国及西方它是拥核国家。
  <br/>
  <br/>
  3）收缴国民护照事件
  <br/>
  <br/>
  中共收紧社会管控，令不少民众想方设法逃离中国。为防止民众卷款出逃，公安部出入境管理局下发内部文件，按省、直辖市，一、二、三县市步骤，逐步收回公民因私个人护照，无法收回的情况按取缔处理。（这是一种类似战前准备措施）
  <br/>
  <br/>
  4）中国央行正在逐步发行“数字货币”
  <br/>
  <br/>
  中国央行开始在四个城市试点本土研发数字货币，标志着一家全球主要央行在通往首个电子支付系统的道路上迈出了具有里程碑意义的一步。中国四个大城市——深圳、苏州、成都和北京的卫星城雄安——正在进行数字货币的内部测试，以完善数字货币的功能。
 </p>
 <p>
  数字货币将空前强化中共对中国人的全面掌控，几乎丧失了全部隐私。
  <br/>
  <br/>
  5）国保总负责人孙力军落马事件
  <br/>
  <br/>
  堪比王立军出逃美领馆，权斗白热化的信号。另，与当前抗拒全球追责有关，据传是因为泄露了病毒研究所资料。此案并非中纪委抓的，而是中央警卫局派人夜里抓的，连秘书司机等人都抓了，董卿也失去自由。事发突然，亦不合正常程序，令中直机关人心惶惶。
 </p>
 <p>
  估计还会有后续事件发生。
  <br/>
  <br/>
  6）中共发言人赵立坚甩锅美国及其失败的事件
 </p>

 <p>
  B.  美国及西方方面：（脱钩与追责）
  <br/>
  <br/>
  1）欧美国家形成基本社会和政治共识：中国隐瞒早期疫情，后又传输给WHO及各国，致病毒蔓延世界，酿成大祸。调查武汉病毒所将成为全球焦点。
  <br/>
  <br/>
  2）越来越多国家意识到，因为中国不按西方规则行事，不透明从而风险和不确定性都极大。所以必须改变对中国的依赖，逐步斩断在中国的供应链。各国须携手找到与中国打交道的新办法。
  <br/>
  <br/>
  3）中国与欧美国家的关系日益紧张，战狼外交陷入国际孤立。因武汉病毒对各国的巨大伤害，华为5G、网路安全、南海台海局势等领域都充满对北京不利的变数。
  <br/>
  <br/>
  4）若疫情过去，西方各国将进入追责阶段。（美国哈里斯最新民调指出， 90%共和党人和67%民主党人认为，中国政府应对疫情负责，只有22%共和党人和34%民主党人相信中国公开的疫情数据准确。两党支持者普遍认为，中国不能信任，疫情对中国国际声望是一次巨大打击。这种负面态度，不仅是决策者和精英，普通民众也如此。
  <br/>
  <br/>
  5）非欧美国家民众也可能跟随问责。实际上，中共传统上的“友邦”非洲国家因其留学生在疫情期间在中国所受的种族歧视与侮辱已经联合起来抗议中共了。
  <br/>
  <br/>
  6）追责与索赔阶段，才是真正的“珍珠港时刻”。届时，习政权将会受到内外夹击。可能骤然出现真正的惊涛骇浪。习近平不得不走向闭关锁国以应对。而国际巨大压力将转化为国内浪潮，酝酿惊心动魄争斗。
  <br/>
  <br/>
  <b>
   二、    全球合并同类项大疫过后，国际关系变化，世界开始重组。全球开始合併同類項运动。
  </b>
  <br/>
  <br/>
  对中共及其体制巨大的不信任导致世界的非共化潮流（即  去中国化）。将出现一个选边站的潮流，就是合并同类项。
  <br/>
  <br/>
  两种同类项：
  <br/>
  <br/>
  美、英、法、日、澳大利亚、印度、埃及、德国、瑞典、丹麦、以色列……
  <br/>
  <br/>
  中、朝鲜、伊朗、俄？
  <br/>
  <br/>
  从上述划分可推测习何以需要准备闭关锁国了。
  <br/>
  <br/>
  世界认识到，与共产党国家（不透明，信息封闭国家, 不履行国际承诺，不遵守条约义务，欺骗世界的国家，穷兵黩武扩张影响力，破坏国际基本秩序的国家）是不能真正和平共处的。其风险系数大到不可接受，鸿沟大到无法共处。
  <br/>
  <br/>
  所以，去中国化实质上是非共化。
 </p>
