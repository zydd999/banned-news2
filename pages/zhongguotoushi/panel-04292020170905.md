 <p>
  <br/>
  主持人：陈奎德
  <br/>
  座谈人：夏业良教授，经济学家与政治评论家
  <br/>
  <br/>
  4月26日，中国人民银行宣布，数字货币在苏州落地流通，将于下个月在中国四个城市（深圳、苏州、成都和雄安），开启央行数字货币的封闭测试。
  <br/>
  <br/>
  这将是首个大型中央银行直接发行数字货币。
  <br/>
  <br/>
  <b>
   一、中国发行数字货币（DCEP）
  </b>
  <br/>
  <br/>
  1） 何谓“数字货币”？
  <br/>
  <br/>
  中国人民银行数字货币研究所所长穆长春对央行数字货币的定义，“其功能属性与纸钞完全一样，只不过是数字化的形态”。
  <br/>
  <br/>
  电子现钞，电子钱包，交易时也无需网络（不像支付宝等）。发行数字货币即印钞。
  <br/>
  <br/>
  相当于纸币的电子化，具有金钱价值的媒体不是纸币，而是电子数据。手机软件或是IC卡等专用电子钱包中的CBDC数据本身就具有金钱价值。支付无需经由央行或商业银行的帐户，电子钱包之间的的数字数据可以即时转移。只要手机有电，两个手机碰一碰，就能把一个人数字钱包里的数字货币，转给另一个人。
  <br/>
  <br/>
  央行数字货币将采取双层运营体系，上层是央行，对商业银行投放数字货币并对货币做信用担保，再由商业银行负责向公众发行。
  <br/>
  <br/>
  2） 中国发行的数字货币与国际上认知的数字货币的区别
  <br/>
  <br/>
  一般的数字货币概念：  脸书的加密货币Libra     区块链   比特币   去中心化
  <br/>
  <br/>
  （2019年6月脸书宣布推出数字钱包的加密货币Libra后，中国央行大大加快了其研发中国数字货币的步伐，与Libra争夺数字货币市场。）
  <br/>
  <br/>
  Libra作为虚拟货币，建立在区块链的概念之上。区块链的概念由中本聪在2008年金融危机后提出，其本质是去中心化的数据库，也就是货币与政府的管控需要分离。
  <br/>
  <br/>
  宾夕法尼亚沃顿商学院(Wharton School of Business) 教授凯文·沃巴赫(Kevin Webach) 说，从这个意义上来看，中国的数字货币使用了部分区块链科技，然而它不能被称为一种虚拟货币。
  <br/>
  <br/>
  “中国央行的这种数字货币将依赖区块链的部分加密安全技术以及分布式账簿技术的元素”，沃巴赫教授说，“然而由于这并不是像比特币这样使用无许可的区块链进行供应，而是由中央银行进行统一发行管控，它最多只能说是受到了虚拟货币的启发”。
  <br/>
  <br/>
  3） 中心化还是去中心化？
  <br/>
  <br/>
  中国的数字货币将不会有真正的去中心化，反而会进一步巩固国家对金融领域的整体管控。
  <br/>
  <br/>
  匿名使用现金的选项完全消失，人们的隐私会暴露在政府的监控之下。你能够通过透明的区块链来查询款项。这个透明不是对所有人，而只是对中央政府”。
  <br/>
  <br/>
  因此，中国的数字货币是高度中心化的。这也是它与去中心化的比特币的根本区别。
  <br/>
  <br/>
  类似于古巴的比索。（货币双轨制）
  <br/>
  <br/>
