   一、                 背景
  </b>
 </p>
 <p>
  特朗普政府上任之前，印度太平洋地区国家与美国、中国的关系：
  <br/>
  <br/>
  大体上，安全靠美国，经济连中国
  <br/>
  <br/>
  印度太平洋地区国家一向把中国当作经济伙伴，美国则是其安全依靠。
  <br/>
  <br/>
  <b>
   二、                 新冠肺炎大流行后，印太地缘政治的变化
  </b>
  <br/>
  <br/>
  特朗普政府上任后，美中关系开始后恶化。2020年以来，从中国武汉爆发的新冠肺炎蔓延全球，美中关系如自由落体坠落，印太国家面临选边站的问题。
  <br/>
  <br/>
  在刚刚结束的阿斯彭安全论坛上，来自印太不少国家的领导人表达了各自的态度。
  <br/>
  <br/>
  印太各国态度一瞥
  <br/>
  <br/>
  印度
  <br/>
  <br/>
  印度前国家安全顾问希夫尚卡尔·梅农（ Shivshankar Menon ）：我们越来越反对中国人
  <br/>
  <br/>
  中国正试图成为海上强国，正在进入印度洋。南中国海对印度和美国都至关重要，航行自由符合印度的利益。在这些方面，美国和印度之间在海上安全方面可以做得比以往任何时候都要多，基本上在安全和国防领域的合作。”
  <br/>
  <br/>
  很久以来，美国及其盟国一直试图与印度建立更加紧密的军事和经济联系，以制衡中国在印太地区的野心。新冠疫情以及印度最近与中国在喜马拉雅边界的军事对抗事件让，这样的想法变得越来越真切起来。
  <br/>
  <br/>
  6月初，印度与澳大利亚签署了一项重要的国防协议，允许两国军方使用对方的军事基地。
  <br/>
  <br/>
  7月有报道说，预计印度还将邀请澳大利亚参加它与日本和美国进行的马拉巴尔海军演习，以加强所谓的四方机制--澳大利亚、日本、美国和印度--对抗中国预计将在该地区部署的海上力量。
  <br/>
  <br/>
  在经济方面，与其他印太国家不同，新德里试图减少对中国经济的依赖。印度甚至愿意切断与中国的重要经贸联系来报复中国，以迫使中国在边界 争端中让步。
  <br/>
  <br/>
  6月29日，印度政府宣布，出于“安全”考虑，禁止包括TikTok和微信在内的59款中国应用，认为这些应用从事的活动有损印度主权、国防、国家安全和公共秩序。据《印度经济时报》称，印度另外还起草了一份包含275款中国应用程序（APP）的清单，准备对它们进行审查。
  <br/>
  <br/>
  今年4月，印度通过了一项立法，要求任何来自中国实体的投资都必须通过政府批准，这对中国来说是一个挫折。
  <br/>
  <br/>
  另有报道说，印度或将加征进口关税并限制中企在印的投资和业务，并考虑阻止中国电信巨头华为进入其市场建设5G网络。
  <br/>
  <br/>
  印度也在积极推动国际影响力。6月18日，印度当选为联合国安理会非常任理事国。5月，印度赢得了世界卫生组织执行委员会主席的席位，并在会议上迅速支持调查新冠病毒起源，而这是中国激烈反对的。
  <br/>
  <br/>
  两国军方高层8月2日举行了第5轮谈判，但是这次谈判并没能打破边境对峙僵局，双方甚至也没有发表声明。
  <br/>
  <br/>
  中印在喜马拉雅山的冲突将新德里推向了华盛顿，而中国将失去一代印度人。

   东盟十国（汶莱 柬埔寨  印度尼西亚  老挝  马来西亚  缅甸  菲律宾  新加坡  泰国  越南）与美、中的关系大同小异，我们谈三个比较典型而重要的国家：越南、菲律宾、 新加坡。
  </b>
  <br/>
  <br/>
  <b>
   越南
  </b>
  <br/>
  <br/>
  为了共同对抗中国，曾经结下血海深仇的美国与越南关系如今空前良好。报道指出，美国与越南和解主要拜中国所赐。
  <br/>
  <br/>
  越南现在是东亚地区最强烈反对中国扩张主义的国家，双方在南海僵持不下。对于越南而言，中国是历史上的占领者，也是长期的威胁；美国则提供了缓冲，并且是潜在的保护者。
  <br/>
  <br/>
  从两国建交起，美越双方已大大加强了军事联系。今年3月，第二艘美国航母在越南停泊。
  <br/>
  <br/>
  现在美国参议院正在审议的“太平洋威慑计划”（Pacific Deterrence Initiative）可能会为其亚洲盟国提供多达69亿美元的美国军事援助，其中就包括台湾和越南。
  <br/>
  <br/>
  近年来，美国的支持也帮助越南在亚洲取得更大的发言分量。越南目前是东南亚国协（ASEAN）的年度轮值主席，并且在接下来的两年内担任联合国安理会的非常任理事国。
  <br/>
  <br/>
  在经济上，近年来越南与中国也有激烈竞争，在美国主导的“去中国化“潮流下，逐步与中国脱钩的各国企业相当大一部分转移去了越南。
  <br/>
  <b>
   <br/>
   菲律宾
  </b>
  <br/>
  <br/>
  菲律宾最近公布在南沙群岛有争议岛屿上扩建设施，将加剧同北京的主权争议。南沙有争议岛屿军事化加剧，成为美中军事对抗的潜在热点。
  <br/>
  <br/>
  今年2月，菲律宾宣布要终止与和美国签订的《访问部队协议》；但在6月初，菲律宾外长表示要暂停协议终止过程，为美菲军队可以继续进行军演保留方便之门。
  <br/>
  <br/>
  菲律宾总统杜特尔特对美中的态度有重要转变，现在倾向离中趋美。

   新加坡
  </b>
  <br/>
  <br/>
  外长：世界进入多极化，选边站不适用
  <br/>
  <br/>
  新加坡外交部长维文（Vivian Balakrishnan）8月 5 日也参加阿斯彭安全论坛。新加坡与美国和中国都有着特别的关系。 美国是新加坡最大的投资者，而中国是新加坡最大的贸易伙伴。在被问到是否担心日渐紧张的美中关系时， 维文表示， 新加坡深感担忧。
  <br/>
  <br/>
  维文说，对亚洲各国来说，最可怕的两件事是：中美之间会爆发冲突；美国放弃亚洲。
  <br/>
  <br/>
  新加坡总理李显龙（Lee Hsien Loong）7月底在美国智库大西洋理事会发表演讲时，也表达了类似的观点。他说，亚洲国家希望同时与美中保持友好的关系。他还说，如果美中迫使亚洲国家作出选择的话，预期中的“亚洲世纪”岌岌可危。
  <br/>
  <br/>
  <b>
   日本
  </b>
  <br/>
  <br/>
  酝酿有年的习近平访日计划因疫情搁置，后又因美中关系迅速恶化而泡汤。
  <br/>
  <br/>
  日本加入美国主导的与英、澳、印联合的南海军演。
  <br/>
  <br/>
  五眼联盟与日本双方同意，日本加入五眼联盟。
  <br/>
  <br/>
  <b>
   台湾
  </b>
  <br/>
  <br/>
  近日美国卫生部长阿扎正式高调访问中华民国台湾，代表特朗普总统向台湾及台湾总统致意。
  <br/>
  <br/>
  台湾事实上处于印太战略地带的枢纽，成为美国的准盟友。
  <br/>
  <br/>

   香港
  </b>
  <br/>
  <br/>
  中共以港版国安法为名大抓捕，再次下重手摧残港人自由，逮捕黎智英及其二子、以及众志创党成员、“学民女神”周庭等，引发西方舆论愤怒。
  <br/>
  <br/>
  香港（一传媒）《苹果日报》股票两天飙升12倍！股票成选票，世界史上奇观。中共塑造了一位香港的政治领袖。
  <br/>
  <br/>
  红色恐怖下的香港，现已成对抗中共的前沿。

  <b>
   澳大利亚
  </b>
  <br/>
  <br/>
  澳大利亚总理： 与中国保持经济联系，同时维护印太战略平衡
  <br/>
  <br/>
  澳大利亚总理斯科特·莫里森（ Scott Morrison通过视频参加了阿斯彭安全论坛，他说： “我知道美国的存在是非常受欢迎的，因为它提供了稳定。我们只想看到，问题得到管理，野心得到管控，使其不至于破坏该地区的稳定，这是我们的国家利益和我们地区许多国家的利益。”
  <br/>
  <br/>
  澳洲前总理陆克文（Kevin Rudd）近日撰文称，美中之间发生武装冲突的可能自韩战后有可能再现。在被问到是否赞成陆克文的观点时，莫里森说，美中之间爆发战争不再是不可想象的。
  <br/>
  <br/>
  自澳洲倡言对新冠疫情实施国际调查后，北京极为恼怒，以经济制裁相威胁，澳洲并未妥协，而是更坚持自己的立场， 成为五眼联盟对抗中共的中坚力量。
  <br/>
  <br/>
  莫里森说，澳大利亚希望与中国保持经济联系，因为长期以来，这是一种互利的关系，但是这种关系也必须是互相尊重的关系。他敦促美中和平解决分歧，维护旨在加强地区和全球稳定的共同规则。
  <br/>
  <br/>
  莫里森说，澳大利亚对中国的看法并非“天真”。他说，澳大利亚正在积极采取措施保持印太地区的战略平衡。
  <br/>
  <br/>
  他在阿斯彭安全会议上说：“当务之急是在印太地区建立持久的战略平衡，这需要美国在这个地区的持续存在以及志同道合的国家调整其利益，更加团结，更一致，更经常地行动。”
  <br/>
  <br/>
  澳大利亚目前正在与日本、印度、越南以及东南亚和太平洋国家接触，以建立新的合作网络。莫里森7月1日还宣布将大幅度增加军费，加强澳大利亚在印度太平洋地区的军事力量，以应对中国在该地区的军力的增长。
  <br/>
  <br/>
  在新冠疫情、香港《国安法》、维吾尔人权、台湾、南中国海、网络安全等诸多问题的立场上，澳大利亚也逐渐与美国趋于一致。

  <b>
   结论
   <br/>
   <br/>
   综上，在美中大对决中，没有一个印太地区国家是清楚地站在中共一边的。虽然有些国家仍希望维持与中国的经贸关系，但总起来说，目前印太各国对美中的倾向，大体上可作八二开。但倘若美中热战开打，北京不会有一个盟友，连巴基斯坦恐怕也会作壁上观，从最近它与日本的勾搭即可看出。
  </b>
