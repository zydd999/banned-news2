  <b>
   一、                    美中对抗的意识形态转向
  </b>
  <br/>
  <br/>
  意识形态门槛——共党禁入
  <br/>
  <br/>
  拟议中的禁止中共党员进入美国的法案引发中国内部轩然大波，无论最后出台的版本有何修改，都将产生长远影响：
  <br/>
  <br/>
  引发中国内部分化——国人疏离中共，引发中共内部分化——党官疏离习派
  <br/>
  <br/>
  这是把中国与中共区分开，把中国国民与中共党员区分开的一个分水岭事件，将终止中共绑架中国国民的历史。 事实上，中共的统治从未获得中国人民的授权。

  <b>
   二、                    美国“Deep State”高官的拒共演讲确立了战略方向
  </b>

 <p>
  白宫国家安全顾问罗伯特·奥布莱恩专门论述了中国共产党的意识形态及其全球野心, 指出习近平是斯大林的继承人。
  <br/>
  <br/>
  美国防部长埃斯珀称，……中国希望改写二战以来，包括中国在内的世界各国都从中受益的国际秩序，“中国的崛起不会让美领导人担忧，但中国共产党统治下崛起的中国令人忧虑。"
  <br/>
  <br/>
  司法部长威廉·巴尔说，中共正在以全社会的方式发动经济闪电战，试图取代曾有世界“民主兵工厂”之称的美国领导地位，以使中国成为世界“独裁兵工厂”。
  <br/>
  <br/>
  美国联邦调查局局长克里斯托弗·雷阐述中共如何通过邪恶，甚至是违法的行为来追求其野心。
  <br/>
  <br/>
  美国国务卿蓬佩奥更是不遗余力指出，中共不代表中国人民。他还将在下周，总结美国和自由世界面临的危险。
  <br/>
  <br/>
  上述论述代表美国当下的深层次支配情报 、国防、外交、产业等核心力量对中共的认识，这种认识甚至超越了特朗普总统个人的政治利益和偏好，主宰了对中共政策的方向，代表了美国对华政策的总体倾向——意识形态转向乃至军事转向。
  <br/>
  <br/>
  目前的倾向使得新冷战越来越接近旧冷战的一些基本特征了：意识形态、军事、情报的对抗逐渐占据重要地位。在经济领域，金融战将日趋主导。
  <br/>
  <br/>
  <b>
   三、                    冷战成围殴
  </b>
 </p>
 <p>
  <b>
   <br/>
  </b>
  <img class="image-inline captioned" src="../../../../resolveuid/85f83ca1-dff4-46fe-9b2e-616e67902c68"/>
  <br/>
  1）美国与西方：后发制人。
  <br/>
  <br/>
  导火索是习近平执意推出港版国安法。之后，国际社会基本上动员起来了。英国、日本、澳大利亚、印度、欧盟都纷纷发表声明并诉诸行动：
  <br/>
  <br/>
  特朗普总统签署了《香港自治法》和取消香港特殊关税区地位的决定。
  <br/>
  <br/>
  欧盟27国发表声明，成员国将采取一致性措施应对涉港问题；英、澳等国出台力度极大的对港移民法案。
  <br/>
  <br/>
  美国将最先进的五代战机F35对日一次性售出105架，该批战机将和日本最新建造的数艘轻型航母配套。以色列目前仅仅只有20多架F35，就已经在中东如入无人之境，多次轰炸叙利亚和伊朗战略目标。因此，日本如果装备这批战机，从军事实力上来说，绝对是亚洲第一。
  <br/>
  <br/>
  英国、澳大利亚拒绝华为；英国、澳大利亚和加拿大已经宣布中止或暂停与香港的引渡条约。美国预计会在未来几个星期里采取同样行动，新西兰和芬兰等国也在做出评估。
  <br/>
  <br/>
  本周美国务卿蓬佩奥公开挑明：“中国在南海的领土声索完全不合法。美国与东南亚盟友站在同一阵线。”  这是一箭五雕：护裁决，驱中共，联东盟，通航路，争资源。
  <br/>
  <br/>
  蓬佩奥还重申了对日本和对菲律宾的安保条约。美国的"尼米兹号"和"里根号"航母群已经在7月4日从菲律宾海进入南海，在那里举行双航母海上军事演习。这是自2014年来，美国海军在南海首次举行双航母军事演习。英国派出"伊丽莎白女王"号航母战斗群赶往南海助阵。日本、澳大利亚也将加入联合军事行动。中共的海军处于极大地劣势地位。
  <br/>
  <br/>
  美国海军尼米兹号航空母舰打击群星期一（7月20日）与与印度海军在印度洋安达曼—尼科巴群岛附近海域进行合作演习。
  <br/>
  <br/>
  菲律宾要求中国执行南海仲裁结果。
  <br/>
  <br/>
  印度号召取消联合国常任理事国席位，连续增兵边界。……
  <br/>
  <br/>
  2）后发制人体制与习近平式的闪电战的比较
  <br/>
  <br/>
  短处：初期因效率低，后发制人者会率先付出一定的代价（如珍珠港、如香港国安法）
  <br/>
  <br/>
  长处：1、道义优势      2、达成共识后，爆发力强，且极难逆转
  <br/>
  <br/>
  <b>
   四、                    西方盟国与中伊轴心同盟
  </b>
  <br/>
  <br/>
  1）与第一次冷战比较
  <br/>
  <br/>
  当年美、苏各有自己的国际阵营，资本主义阵营与社会主义阵营，尚可对峙
  <br/>
  <br/>
  如今，美国西方加印度的阵营势力更盛。而中共，举目无盟，情急之下去和自己也投票赞成制裁的西方大敌伊朗结盟，缔结邪恶轴心国，不仅得罪美国，也得罪了众多西方和中东国家，还包括俄罗斯、以色列这种过去立场暧昧的国家。
  <br/>
  <br/>
  2）  是群殴而非两大阵营
  <br/>
  <br/>
  所以，有评论家认为与其说有两大阵营，不如说是群殴更合乎实际。实力更不对等，在南海、台海等处，爆发“热战”的可能性比第一次冷战时更大。
  <br/>
  <br/>
  简言之，在意识形态与军事上，美国已经完成了对中共的战略包围圈。
