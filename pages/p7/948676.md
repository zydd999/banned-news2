 <p>
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/晋国" target="_blank">
    晋国
   </span>
  </strong>
  的太史屠黍，心明眼准，看相精稳，但守口如瓶，经常沉默寡言。这位晋国
  <strong>
   太史
  </strong>
  屠黍，早已看到晋国混乱，知道晋公骄奢，不讲求道义。因此，他就携带晋国的法典，回到周朝所在地居驻。
  <br>
   <br>
    有一天，周威公（即周威王）专门找到他，他躲不过，只有应对。
    <span id="hideid" name="hideid" style="color:red;display:none;">
     <span href="https://www.secretchina.com">
     </span>
    </span>
   </br>
  </br>
 </p>
 <p>
  周威公召见他，问道：“如今，天下的国家很多，哪个国家，会首先灭亡?”
 </p>
 <p>
  他沉默良久。周威公又问了一次。他答道：“晋国先亡。”
  <br>
   <br>
    周威公问：“何以见得？”
   </br>
  </br>
 </p>
 <p>
  他说：“我不敢坦白地向晋国公进谏，我只暗示晋公，有些上天所表现的奇异现象，如日月星辰的运行，都不合轨道…他却说：‘怎么会这样?’我又暗示他：朝廷的人事，多不适当，百姓十分怨恨。晋公仍不重视，却说：‘这对我国有什么损伤呢?’我再暗示他：邻国不服我们；贤良的人，都不来帮助我朝。他却说：‘这对我晋国，未见有什么损害！’这是他不明白国家能生存和灭亡的道理。所以我说：‘晋国先亡。’”
  <br>
   <br>
    过了三年，晋国果然灭亡。
    <br>
     <br/>
     周威公又召见屠黍问道：“哪一国，会接着灭亡?”
     <br/>
     屠黍答道：“中山国!”
     <br/>
     <br/>
     周威公问他是什么缘故？他解释说：“天生人，是要人有分别，有分别是人的义理，也是人所以异于禽兽的地方。如此，君臣上下的关系，才能建立。可是，中山国的君臣，是拿白天当做黑夜，拿夜晚继续白天的事情，男女生活浪漫、猥亵，日夜淫秽，从无止息之时，他们荒淫昏乱，从事喜乐，现在歌声都很悲哀，而国君却不知道这些是坏事情，这就是亡国之兆，所以我说：‘下一个是中山国!”
     <br/>
     <br/>
     过了两年，中山果然灭亡。
    </br>
   </br>
  </br>
 </p>
 <p>
  周威公又召见屠黍，问道：“此后，哪一国，又会跟着灭亡呢?”
  <br/>
  晋国的
  <span href="https://www.secretchina.com/news/gb/tag/太史" target="_blank">
   太史
  </span>
  屠黍，不回答。周威公继续追问下去，他才说：“是贵国!”
  <br/>
  <br/>
  周威公因而恐慌！召请国内年长、有道之士，除去苛令三十九项。再将这些施政情形，告诉屠黍。屠黍说：“即使这么做，也只能到你为止了。”
  <br/>
  <br/>
  周威公死后，九个月无法安葬，周朝就分裂为二。
  <br/>
  <br/>
  （上述这事，出自《吕氏春秋》卷十六先识览；《说苑》卷十三权谋。译文参阅林品石注译《吕氏春秋今注今译》下册、卢元骏注译《说苑今注今译》）
  <br/>
  <strong>
   <br/>
   【解析】
  </strong>
  <br/>
  屠黍之所以预言晋国第一个灭亡，是因为他来自晋国，对晋国的乱象，了如指掌的缘故。
  <br/>
  <br/>
  中山国不及晋国那么严重，所以亡在晋国之后。该亡的都亡了之后，就轮到周朝了。因为周朝也重蹈了前人的覆辙。即使周威公做了“亡羊补牢”的工夫，仍不能保证周朝不会瓦解。可见，这是历史的必然。屠黍是一位深具“史眼”的政治观察家。
 </p>
 <p>
  <strong>
   【诗曰】
  </strong>
  <br/>
  威公召相问，
  <br/>
  屠黍感神伤；
  <br/>
  不明生之道，
  <br/>
  三国遂衰亡。
  <br/>
  <strong>
   （以上是笔者，从《察典》书中，整理写出）
  </strong>
