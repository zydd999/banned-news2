  </strong>
  <span style="font-family:Comic Sans MS,cursive">
   克勤于邦，克俭于家，则旺。(《尚书•大禹谟》)
  </span>
  <br>
   【今译】勤俭治国，勤俭治家，国家和家庭就能富裕、兴旺。
   <br>
    <br>
     <strong>
      【古训2】
     </strong>
     <span style="font-family:Comic Sans MS,cursive">
      为之者疾，用之者舒，则财恒足矣。(《礼记•大学》)
     </span>
     <br>
      【今译】做事勤快，开销迟缓，那么财物总会常常充裕。
      <br>
       <br>
        <strong>
         【古训3】
        </strong>
        <span style="font-family:Comic Sans MS,cursive">
         货悖而入者，亦悖而出。(《礼记•大学》)
        </span>
        <br>
         【今译】用非法手段获得的财物，也会被别人用不正当的方式拿去。
         <br>
          <br>
           <strong>
            【古训4】
           </strong>
           <span style="font-family:Comic Sans MS,cursive">
            不取于人，谓之富；不屈于人，谓之贵。(《孔丛子》)
           </span>
           <br>
            【今译】不取得别人的东西，就叫富有；不屈从于别人，就叫高贵。
            <br/>
            <br/>
            <strong>
             【古训5】
            </strong>
            <span style="font-family:Comic Sans MS,cursive">

            <br/>
            <strong>
             【古训6】
            </strong>
            <span style="font-family:Comic Sans MS,cursive">
             见利思义。(《论语•宪问》)

   不义而富且贵，于我如浮云。(《论语•述而》)

  <br/>
  【今译】靠不正当的手段，去获取富贵，我（孔子自称）是绝对不干的！

  <br/>
  <strong>
   【古训8】
  </strong>
  <span style="font-family:Comic Sans MS,cursive">
   不义而疆，其毙甚速。(《左传•昭公元年》)
  </span>
  <br/>
  【今译】依靠不义的手段强盛起来的，其灭亡一定很迅速。

 <p>
  <strong>
   【古训9】
  </strong>
  <span style="font-family:Comic Sans MS,cursive">
   良贾深藏若虚，君子有盛德若无。(《大戴礼记•曾子立言》)
  </span>
  <br/>
  【今译】聪明的商人总是把钱藏得很严密，好像身上不名一文；高尚的人总是很谦虚，不在别人面前显示自己。
  <br/>
  <br/>
  <strong>
   【古训10】
  </strong>
  <span style="font-family:Comic Sans MS,cursive">
   不学问，无正义，以富利为隆，是俗人者也。(《荀子•儒效》)
  </span>
  <br/>
  【今译】不学习，缺乏正义感，只求财富兴隆，这是庸俗的人啊！
  <br/>
  <br/>
  <strong>
   【古训11】
  </strong>
  <span style="font-family:Comic Sans MS,cursive">
   治国常富，乱国常贫。(《管子•治国》)
  </span>
  <br/>
  【今译】安定的国家，总是富强的；纷乱不安的国家，人民总是贫困的。
  <br/>
  <br/>
  <strong>
   【古训12】
  </strong>
  <span style="font-family:Comic Sans MS,cursive">
   惰而侈，则贫；力而俭，则富。(《管子•形势》)
  </span>
  <br/>
  【今译】懒惰又奢侈，生活就会贫困；勤劳而节俭，生活就会富足。
