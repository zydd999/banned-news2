 <p>
  【石正丽和她的团队花了十三年时间，最终确定了SARS病毒起源于蝙蝠。“不管南方还是北方，中部还是西部，只要听说有蝙蝠的地方我们都会去，足迹遍布了我国28个省市，像大海捞针一样。这样一做就是十多年的时间。” ——《一席》YiXi】
  <br/>
  <br/>
  武汉肺炎，在某些人的预料之中但出乎绝大多数人意料之外，演变成一场不折不扣的全球大瘟疫！
  <br/>
  <br/>
  追责、朔源、索赔、问罪等行为模式，日趋一日地在灾难深重中展示着风暴眼般的时尚新闻头条效应。
  <br/>
  <br/>
  诸如此类的头条新闻中，一个女人的名字时隐时现，她是： 美国微生物科学院院士、武汉病毒所新发传染病研究中心主任、武汉P4实验室副主任石正丽。
  <br/>
  <br/>
  一月下旬封城开始，相关武汉病毒所过失泄漏或人为散播的小道、谣言、阴谋论、假新闻，搅拌着形形色色的真相调料，夹带着与石正丽这个名字相关的种种绘声绘色故事，幽灵般的在坊间大小新闻媒体家喻户晓地传播着。
  <br/>
  <br/>
  这位嘴角浮漾着笑纹，看似性情中人的世界知名中国科学家，用 “闭上你们的臭嘴！ ”等激烈措辞，作出毫不妥协的高调反击。此后的事态发展一度看似趋于缓和。
  <br/>
  <br/>
  但到了四月，全世界几乎每一个国家都被这突如其来的瘟疫陆续击倒，石正丽和武汉P4病毒所的故事，高攀上诸如纽约时报华尔街日报华盛顿邮报等顶级媒体的头条版面后，情况开始出现变化。
  <br/>
  <br/>
  自由时报4月26日实时新闻综合报导：“前美国总统顾问班农24日在直播节目《战情室》透露，有位来自武汉P4实验室的资深研究员，目前已成功从中国出逃，武汉P4实验室是新冠病毒起源地，该名科学家近日会在他的直播曝光；班农并称，「中国必须知道，你们的实验室中有很多人，不管P2或是P4实验室，出来的人都已遍布全世界，他们都会发声，因为他们都知道实验室的黑幕，你们一直在说谎」。
  <br/>
  <br/>
  班农表示，病毒由武汉实验室泄漏的说法由来已久，坊间抱持这种观点的人也不在少数，现在将有武汉P4实验室的科学家现身说法，相信很多人都相当感兴趣，之后等着看内部人员何时出面曝光手中信息。”
  <br/>
  <br/>
  报道中班农先生对这位“出逃者”的详细身份背景语焉不详。
  <br/>
  <br/>
  但接下来的一天后RFI法广一条爆棚新闻似乎告诉了我们一些相关答案：“随着外界对武汉P4实验室安全问题的日益关注，一位在疫情爆发前夕在武汉实验室工作的法国专家René Courcol 被推上舆论前台。法国记者《法国与中国的危险关系》一书的作者Antoine Izambard先生向法广表示，这位学者是他参观武汉实验室时遇到的唯一的一位法国专家，他受法国外交部的委任负责武汉实验室的安全监督工作。这位专家今天因此成为舆论的关注焦点，但是他以职业规定为由拒绝接受包括法国世界报在内的所有媒体的采访。”
  <br/>
  <br/>
  <b>
   班农先生的“他”莫非是他？
  </b>

 <p>
  RFI的这篇报道接下来援引了当地世界报4月25日发布的有关武汉实验室的调查。两位调查作者是巴凯（Raphaelle Bacque）和佩德罗莱蒂（Brice Pedroletti）。根据这个调查，去年12月武汉公卫危机爆发时，武汉P4实验室的传染中心负责人石正丽陷入焦虑和害怕中，她说自己好几夜没合眼，反复回想自己的每一项研究，每一个动作，不停问自己是不是从我们那些实验室泄露的？
  <br/>
  <br/>
  她告诉《美国科学》月刊的记者珍妮·丘（Jane Qiu）：“的确这使我头脑乱了，闭不上眼睛。”
  <br/>
  <br/>
  石正丽向法国世界报调查作者的上述心情流露，间接证明了一个极为重要的事实：武汉P4病毒所在本次瘟疫爆发之前，已经拥有着让今天的她 “焦虑和害怕得夜不能寐阖不上眼的” 与COVID-19完全相同的新冠病毒活体样本。
  <br/>
  <br/>
  以上是外媒的一次综述。
  <br/>
  <br/>
  一年多前，北京著名的网络名人讲坛【一席】，邀请石正丽作了一场名为 “追溯SARS源头”的主题演讲。
  <br/>
  <br/>
  在萨斯远去长达16年后2018年的下半年，这场马后炮式的演讲注定无法激起任何波澜，并理所当然的立马被这个快餐型失忆时代迅速遗忘。
  <br/>
  <br/>
  但在该演讲被武汉肺炎夹带石正丽大名全盘激活后的YouTube视频里看，这场从头到尾黑着个脸恍若乌云一团的不祥演讲，越瞅越像一个精心计算后的剧情预告，引导性主题自始至终围绕着果子狸，野生动物，广东野味交易市场，蝙蝠等一年后华南海鲜市场冤假错案下惨遭灭门的飞禽走兽。
  <br/>
  <br/>
  显然，她是预感到了些什么和为她认定不久之后的 “新冠病毒大概率爆发” 做足了备案的！

  演讲视频中最让人触目惊心的是：讲台上那绘声绘色的投影图片显示，在荒山野岭悬崖峭壁与蝙蝠零距离肌肤之亲中，包括石正丽在内武汉p4所的一众科研人员，没有配戴任何防护装置。而这一切，都发生在2002年被认定由动物传染人进而人传人的萨斯冠状病毒瘟疫之后。
  <br/>
  <br/>
  接下来的演讲，石正丽轻描淡写地一笔带过了蝙蝠与她和她的庞大团队那长达十多年亲密接触史中直接将病毒传染给人的极大可能性。
  <br/>
  <br/>
  这场原本落寂却在今天翻盘成网红的演讲，在YouTube留下上千条几乎一面倒的跟贴，在此摘要几条以醒受众：

  <br/>
  <br/>
