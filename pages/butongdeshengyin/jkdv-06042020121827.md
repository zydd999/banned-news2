 <p>
  <b>
   【受“世界必须立即按照我的意愿运转”的自恋驱使，巨婴总希望事情能尽快实现，越快越能满足其自恋，而不能皈依事情本来的道理，由此打造无数到位细节，让事情浑然天成。——《巨婴国》武志红】
  </b>
  <br/>
  <br/>
  习近平的灭贫大业，是新三年，旧三年，如今到了最后一道缝缝补补又三年的肉紧关头。
  <br/>
  <br/>
  他急了：
  <b>
   我们不能一边宣布全面建成了小康社会，另一边还有几千万人口的生活水平处在扶贫标准线以下，这既影响人民群众对全面建成小康社会的满意度，也影响国际社会对我国全面建成小康社会的认可度。
  </b>
  <br/>
  <br/>
  2020，神奇的年份。
  <br/>
  <br/>
  这一年，悠悠中华五千年文明史中不可或缺的穷困故事，被一只巨手一笔勾销，天下无贫。
  <br/>
  <br/>
  这一年，邓小平“上世纪末达成全面小康社会”的未竟夙愿，由习近平同志亲口宣告，如期实现。
  <br/>
  <br/>
  北京时间6月1日，中共领导人习近平在中共机关报《求是》杂志发表文章《关于全面建成小康社会补短板问题》，习近平在文章中表示：
  <b>
   中国已经基本实现全面建成小康社会目标，
  </b>
  成效比当初预期的还要好。
  <br/>
  <br/>
  可突如其来的，那位一向如履薄冰唯唯诺诺的弱势总理，竟然在上周末人大答记者问时大鸣大放了起来：
  <b>
   “有6亿人每个月的收入也就1000人民币。”
  </b>
 </p>
 <p>
  <b>
  </b>
  <br/>
  <img class="image-inline captioned" src="../../../../resolveuid/551531d7-78f7-42c6-a4b7-7c0bfd7e674d"/>
 </p>
 <p>
  快闪两面人斗胆口出狂言，短板大实话掀起千重巨浪。
  <br/>
  <br/>
  果不然，中央喉舌《经济日报》立马急急刊出中共御用经济学家李实专访，为总理的“六亿一千论” 缓颊：“6亿人平均月入1000元左右是“平均数”，不仅包括在就业、有收入的人口，也包括无就业、无收入的人口，如老人、儿童、学生等被赡养人口。”
  <br/>
  <br/>
  以下是第一种“平均数论”的解释性分析：
  <br/>
  <br/>
  即便如此，如何平均？从超级富豪到新生婴儿，国家统计局是否有，或中华人民共和国过去任何年代曾经有过这么一种奇怪的统计数据？退一步问：同样的统计方式，除了1000元这条线，还有没有月入500元，乃至月入300元诸如此类直接涉及贫困人口的平均数据？如果有，那是几个亿？
  <br/>
  <br/>
  接下来第二种“劳动人口论”，听上去就严重了：
  <br/>
  <br/>
  中国官方人口统计数据的年龄结构看，2018年中国大陆0-15岁（含不满16周岁）总人口为2亿4860万人，占总人口比重的17.8%；16-59岁（含不满60周岁）总人口为8亿9729万人，占总人口比重的64.3%；60周岁及以上总人口为2亿4949万人，占总人口比重的17.9%。
  <br/>
  <br/>
  如果总理的“六亿一千论”指的是劳动人口（按理应该这么算），那么，中国14亿总人口扣除0-15岁未成年人近2亿五千万，去掉60周岁以上退休人口同样近2亿五千万，总数五亿。也就是说中国的9亿劳动者中，远远超出半数的人在月收入1000人民币或以下的相对贫困线上苟且挣扎着！
  <br/>
  <br/>
  万能的网民制作了一集【月入不过万就是贫困人口，胡歌说出了90后的烦恼】YouTube节目：
  <br/>
  <br/>
  澎湃新闻·澎湃号·湃客《中国贫困人口简史》：2020年贫困人口消失，只是绝对贫困人口从统计口径上消失了，相对贫困人口依然长期存在。这种相对贫困，同样会显示出强烈的地域落差。比如城乡之间，城市和农村低收入户的收入倍差，从2013年的3.4倍，提升到了2017年的4.3倍。贫困人口消失了，但农村相对城市正在变得越来越穷。
  <br/>
  <br/>
  以此类推，即便按照国家拟定的最新脱贫标准年收入4000元乘以上述仅仅2017年的4.3倍，相对农村贫困人口，城市贫困人口的标准应该在17000元左右（三年后的今天应该涨到20000元了吧？），分摊到月，李克强总理“六亿一千论”，不但所言不虚，而且尚有余地。
  <br/>
  <br/>
  2020，神奇的年份。
  <br/>
  <br/>
  毋论总书记发起的精准脱贫运动逾期后庄严宣布本年度中国农村地区已然全歼贫困，和口口声声提前宣布“我国已经基本实现全面建成小康的社会目标。”
  <br/>
  <br/>
  抑或中国国家总理在这一年全国人大悍然而庄重的寻衅滋事：免谈小康！中国人民（劳动者）的大多数，正无奈地挣扎在贫困线上。
  <br/>
  <br/>
  党内斗争，终于迎来了人人看得见摸得着的白热化和公开化，以至于这“脱贫”“小康”两造鸳梦，惶惶然流于家喻户晓笑料般的空谈式黄粱。其作祟因子，被心理学家认为基础于一种严重的精神疾病：
  <b>
   全能自恋型人格障碍。
  </b>
  <br/>
  <br/>
  追求国富民康，于情于理原本无可厚非。但限时限日，不惜大规模数据造假，“下军令状”、“提头来见”、“保证完成任务”、“不达目的誓不罢休”，这便是愚昧，其怪诞甚至荒谬的亚健康心理，终将被世人耻笑于一时。
  <br/>
  <br/>
  年前因不明原因被凤凰卫视腰斩的时尚脱口秀节目【锵锵三人行】，其主持人窦文涛近来似乎推出了一款改头换面的新品种【圆桌派】。【不同的声音】择其某集主题“高配隐形贫困人口” 嵌入本节目结束部分。讨论者为窦文涛、武志红、蒋方舟、梁文道：

  <br/>
  <br/>
  生命是为了更好地成为自己，而不是成为更好地自己，因为，你自己本身，就是最好的。
