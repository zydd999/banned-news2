  </b>
  <br/>
  <br/>
  上周末以来，一直在回味着苗德岁教授那段隽永的箴言：“病毒一词原本就包含了两重性：一方面是给予生命的物质，另一方面代表了致命的毒害。病毒的确是致命的，但同时又赋予了这个世界不可或缺的创造力。因此，创造与毁灭又一次完美地结合在了一起......”
 </p>
 <p>
  是的：
  <br/>
  <br/>
  2020年，病毒带来了前所未有的全人类范围内的死亡和恐惧，同时又以其“不可或缺的世界级创造力”，奇迹般的为地球催产出席卷苍生的一腔温情——居家隔离。
 </p>
 <p>
  因此：
  <br/>
  <br/>
  我们惊喜的读到了方方日记。
  <br/>
  <br/>
  终日饭局不断沉迷声色犬马恣情纵欲的京沪广CEO们，开始乖乖的宅在家中，看一本闲书，喝一杯清茶，作一次略带冥想功能的社交距离客厅散步。 令人难以置信的是：“他找到了安慰，因为日常平庸的手段暂时停顿了”。
  <br/>
  <br/>
  受益于大规模人为活动的终止，自然界得以悄悄启动被摧残殆尽后自我修复的功能。
  <br/>
  <br/>
  迈阿密海滩，大海令人惊艳地呈现出从未见过的艳蓝，让人感动。
  <br/>
  <br/>
  优胜美地空荡荡的行政大楼前，山猫和土狼在人行道上优哉游哉，无人打扰。
  <br/>
  <br/>
  旁遮普人有幸在家中膜拜喜马拉雅山巅的雄姿，这30 年来印度从未出现过的景色。
  <br/>
  <br/>
  中国的碳排放量比去年同期减少 1 亿公吨，米兰和罗马的二氧化氮较之往年下降约45%，巴黎也大幅下降了54%......
  <br/>
  <br/>
  被誉为目前在世 “最伟大的哲学家”的斯洛文尼亚左翼学者齐泽克，一个月前在抗疫隔离状态中完成了新着《大流行病！新冠病毒撼动世界》PANDEMIC! Covid-19 Shakes the World》。他写道：
  <br/>
  <br/>
  在现今的严峻环境之中，我们不应去追寻甚么精神上的真实感以作慰藉，而应该无耻地认同自己的一切征状（以拉冈精神分析的术语来讲）。为了防止心理崩溃，我们可以拥抱任何能够安定我们日常生活的一切小仪式、方程式或怪癖，在家中尽情看电视或电影，可以是灾难片或无聊的罐头笑声喜剧，或是 YouTube 上重大战役的纪录片。
  <br/>
  <br/>
  齐泽克认同他的朋友所讲：现在人们在家工作，已经不如以前那样「我工作因为我可以拿到钱，然后去度假」；现在没有人能肯定未来我们还有机会去度假，甚至仍存在钱这回事。现在所谓的世界是这样的概念：我们有地方可住，有水与食物等基本需要，有他人的爱与真正重要的事去做。
  <br/>
  <br/>
  齐泽克希望：即使疫情过后，我们仍能拥抱这种非异化的、体面的生活态度。
  <br/>
  <br/>
  当然，目前旅居在不为人知瘟疫不算太严重的美国某州的【不同的声音】节目主人公，退休的中国大陆高级历史教师颜智华，也是在居家隔离期间那百无聊赖又百感交集的极为复杂抗疫心绪下投书本台【反馈】部门，进而主动加入【居家隔离故事会】“创刊号”的讲述的。
  <br/>
  <br/>
  打自一年多前【不同的声音】播出《点滴的输液管 妈妈的鬼门关》后，该集颇受听众欢迎节目的受访嘉宾，退休后旅居美国纽约的高级中学历史老师颜智华，便风尘仆仆地踏上了他“海外上访”的不归路，节目播出10天后的2018年10月14日，因妻子惨死于疑似中药静脉滴注造成的重大医疗事故而悲愤交集的颜智华老先生，在纽约寄出了第一封给世界卫生组织的“投诉中国卫生部纵容医院（特别是公立大医院）医生谋财害命”的控告信，半年不见回音后，第二次投诉并批评了世卫组织违规不作为，但至今从未接获谭德赛们的任何回复与信息。（按照联合国规定应该回复的）。
