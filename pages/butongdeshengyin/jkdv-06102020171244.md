  人民公敌习近平？
  <br/>
  <br/>
  上周以来，【反习】这首陈年老调，重弹起一时，蔚为风气。
  <br/>
  <br/>
  蔚为风气是由于，三位“反贼”多多少少都是和他“说得上话的人”：身为校长，中央党校他的蔡霞大教授；身为球迷，前国足他的郝海东明星中锋；身为皇上，现总理他的李克强二当家。
  <br/>
  <br/>
  点将开始。
  <br/>
  <br/>
  首先推出郝海东。六四事件31周年纪念日前夕，流亡美国的中国富商郭文贵与美国白宫前首席战略顾问史蒂芬•班农(Steven Bannon)通过网络直播，宣布成立“新中国联邦”。这一新闻事件最具轰动效应的是，郭文贵请来了前中国国家足球队明星球员，“中国足球先生”郝海东，宣读长达近三十分钟的“新中国联邦宣言”。
  <br/>
  <br/>
  第二员“虎将”，是中国人家喻户晓的李克强总理。上周两会打烊前，一向唯唯诺诺的这位二当家，突然之间无预警无厘头的打出一套“反习”乱拳“六亿一千论”+“地摊经济学”，着实让沉醉在“天下无贫”和“全面小康”全能幻梦中打盹的总书记好一阵子缓不过劲来。
  <br/>
  <br/>
  而蔡霞这一路红纵的反习代表人物有任志强，及其背后的王岐山，舆论界的胡舒立，工商界的樊立勤，以及以阿拉善SEE富豪企业家为主的八年前“扶上马走一程红二代”拥习派”同路人。
  <br/>
  <br/>
  蔡霞那近日来流传甚广的二十分钟“某女士讲话”反习言论，本节目在下半部分精剪后播出。
  <br/>
  <br/>
  除此之外，各分支机构中著名的一度刷屏的反习分子，据非常不完全统计：
  <br/>
  <br/>
  人大经济学家向松柞（去年11月抛出标题为【炮轰习近平终身制及复辟文革】的公开演讲。）
  <br/>
  <br/>
  高校公开反习的代表人物有：北大的郑也夫，清华的许章润，复旦的葛剑雄，上海的张雪忠，台湾的李家宝（留学生），中央党校的蔡霞等等等等。
  <br/>
  <br/>
  中南海离休老人反习群组可以说基本上是一个庞大而边界不清且人数很难统计的既得利益统一战线，包括江泽民派，温家宝派，党内斗争主导的选择性反腐运动受害政治对手的幸存党羽（两面人），邓朴方为核心的30年改革开放成果和理念的受益者捍卫者等不一而足。择选其中不同于上述人等的另类代表人物，已故的毛泽东秘书李锐病床上一段珍贵的声音：“没想到他文化程度这么低, '毛病不改，积恶成习'。”
  <br/>
  <br/>
  红二代企业家中最著名的反习思想家任志强的北大演讲：“我们应努力的站起来，把面前的墙推倒，建立民主制度！”。
  <br/>
  <br/>
  任志强最具影响力的反习文章《一位剥光了衣服也要坚持当皇帝的小丑》摘要朗读：

 <p>
  复旦教授葛剑雄“骂人不吐脏字儿”的反习言论：
  <br/>
  <b>
   <br/>
   万维博客 吉歌 警世恒言：不反习必然天下大乱 （文章发表于习近平2018年修宪之前）
   <br/>
   <br/>
   核心提示：中国政治经济的结构性危机越来越深重，如果不坚定反习的新时代理念，天下大乱这只魔鬼的降临将势成必然。反习才能经济着陆成功，反习才能政治改革成功，反习才能维护世界和平。
   <br/>
   <br/>
   一、美国反习，东南海才得以和平
   <br/>
   <br/>
   奥巴马时代美国奉行绥靖主义，但对中国却相对强硬。美国首次将钓鱼岛纳入安保条约，美日韩澳进入“东海识别放空区”，巡航南海等动作，沉重打击了习近平的军国主义企图。
   <br/>
   <br/>
   川普当政后制定了全年南海航行计划，多次进入12海里，三大航母及舰队云集西太平洋。通过更加强硬的动作，有效遏制了习近平南海扩展的野心。
   <br/>
   <br/>
   人们绝对不可低估原教旨势力的非理性，希特勒、萨达姆、伊斯兰国、金正恩的疯狂等就是最好的说明，王岐山的“杀二十万保江山”也绝非戏言。试想，如果美国不坚定反习，如今钓鱼岛、南海非常可能已经战火纷飞和一片疮痍。而国际经济也难免受到沉重打击，国际秩序也将天下大乱。
   <br/>
   <br/>
   二、李克强反习，暂时维系中国经济
   <br/>
   <br/>
   李克强是胡锦涛选定的接班人，被习近平夺取总书记大位，内心的不满具备历史依据。
   <br/>
   <br/>
   头五年就任总理，李克强本想以大干一场，施展自己寒窗苦读的抱负，却被习王的权力抵制和金融腐败所破坏，导致“克强经济学”全部走板变样。李克强内心的愤怒可想而知，但为大局着想而处处忍让......
   <br/>
   此外，党内新老常委及各级党员也暗中纷纷反习，“反习公开信”事件、向郭文贵输送材料等都是说明，虽然在习王极左势力强劲之下作用有限，但内在的力量仍然顽强和广泛。
   <br/>
   <br/>
   三、郭文贵反习，防范了法西斯复辟
   <br/>
   <br/>
   郭文贵虽然口头不反习，甚至内心不愿反习，但其行为构成实质反习。通过郭文贵的长期爆料，习近平想当皇帝的野心已经受到了沉重打击，不过仍有残余。
   <br/>
   <br/>
   四、反习符合现实，习代表邪恶落后势力
   <br/>
   <br/>
   习近平继位以来，可以说一事无成。除了妄图复辟个人崇拜搞得牛皮哄哄，他没能拿出任何实质成绩。
   <br/>
   <br/>
   经济上，股市崩盘，楼市走向极端，一带一路变成口号和跃进，大量的对外投资蕴藏巨大风险，大把向西方列强撒钱引发大量民怨。权贵资本拼命向银行借债购买海外企业，以转移资产和洗钱。
   <br/>
   <br/>
   政治上，习近平大搞任人唯亲，反腐成了党内斗争工具。
   <br/>
   <br/>
   外交上，习近平狂妄自大，严重错判世界形势和中国地位，盲动出击，结果处处受到挤兑，让中国形象受到严重损害。
   <br/>
   <br/>
   习近平、王岐山和栗战书都是典型的极左原教旨红色权贵，他们目前以反腐为名的清除异己的运动，其实质是为了最大限度地集权，意在更加贪腐，甚至试图贪天！维持这种深度腐败200年，已经成为他们的内在目标。俄罗斯的普京、朝鲜的金正恩成为他们效仿的对象。一旦攫取无限权力成功，中国人民将陷入类似毛时代的万劫不复状态。
   <br/>
   <br/>
   由于中国经济已经和全球经济紧密相连，一旦中国大乱，世界也会受到巨大的冲击。
   <br/>
   <br/>
   五、全人类反习，世界才能和平进步
   <br/>
   <br/>
   总体看，习近平对外谄媚送钱，腐蚀人类正义；对内搞坏经济，镇压改良力量；他正在掏空和摧毁邓小平改革开放以来的各种成果。中国政治经济的结构性危机越来越深重，如果不坚定反习的新时代理念，天下大乱这只魔鬼的降临将势成必然。
   <br/>
   <br/>
   反习是历史和新时代的客观要求，具备十分真实的现实基础，也完全符合民意，符合国内国际的大气候。反习已经成为人类共识，也是大格局大潮流，无谓阻挡只有自找麻烦。人们绝不可在“等三年”的幻梦中无所作为，宣传正义、推动革命才是最切实的工作。
   <br/>
   <br/>
   反习才能经济着陆成功，反习才能政治改革成功，反习才能维护世界和平。必须对邪恶势力保持高压，改良才可能具备空间。必须认识到，人民的武力革命不可或缺，也是和平的切实来源和真正保障。口头可以不反习，但内心和行动必须反习！无论是中国人还是世界人，都应该坚定站在反习的统一战线上，为维护中国及世界的长治久安作出贡献。
   <br/>
   <br/>
   在最后播出蔡霞教授反习谈话之前，让我们回味一段难忘的日子。2016年底，轰动一时的雷洋命案以情节轻微不起诉涉案警察尘埃落定后，举国上下一片群情沸腾，短暂的思想解放言论小阳春中，当时的蔡霞写下一段非常有名的文字：“雷洋案目前这个结局，彻底粉碎了权力残存的最后一点点公信力，粉碎了所有正义之人的最后一点善良的幻想和希望。从此以后人心将与权力彻底分道扬镳，中国人的老话：先礼而后兵，此后的局势走向，和平之路将极为渺茫。寒冬冰封大概是不可能春风化了的，而将是地火融了的！让我们在历史的的隧道里共同坚守我们内心的亮光。”
  </b>

  中华人民共和国71年建国史，或中国共产党100年党史，从来没有出现过像今天这样的尴尬局面：一位在任党国一把手，完完全全咎由自取亦步亦趋一条道走到黑地坠入了“人民公敌”的悲惨境地。目前的现状就是，上至达官贵人，下至贩夫走卒，全民怼习，蔚为风气。习近平“人民公敌”的准漫画形象，正由他自己本人，一笔一划的，快速融入闹剧般的闭幕式“点睛”大舞台。
