  <b>
   综合外媒报道：休斯顿中国总领馆本周二晚间发生火警，多名领馆人员在大院中用明火大量焚烧文件，浓烟滚滚，邻居报警。
   <br/>
   <br/>
   据休斯顿市多家电视台报道，消防局在接获火警报告后赶到位于Montrose Blvd 3417号的中国总领事馆。
   <br/>
   <br/>
   一位不具名的休斯敦警员对当地的一家电视台透露，这家领馆所在地，以及有不少领馆雇员居住的一座公寓，将于周五下午四点清空。
   <br/>
   <br/>
   美国国务院发言人奥特加斯(Morgan Ortagus)星期三在一份电子邮件声明中表示，美国下令关闭中方驻休斯顿总领馆，是以此保护美国知识产权和私人信息不受威胁。
   <br/>
   <br/>
   奥特加斯表示，根据《维也纳公约》规定，外交人员必须“尊重所在国的法律和法规”，并且“有义务不干涉该国的内政”。她强调，正如特朗普政府不容忍中方不公平贸易、盗窃美国就业机会和其他恶劣行为一样，“华盛顿不会容忍中华人民共和国侵犯美国主权，以及恐吓我们的人民”。
   <br/>
   <br/>
   中共党报人民日报旗下的环球时报则在星期三（2020年7月22日）称，美国政府突然要求中国在72小时内关闭休斯顿总领事馆，称这是华盛顿一个疯狂的举动。之后，中国外交部发言人汪文斌说，美国7月21日突然要求中方关闭驻休斯敦总领馆，称这是“美对华采取的前所未有的升级行动”，“中方必将做出正当和必要反应。”
   <br/>
   <br/>
   本周四（23日），中国外交部发言人汪文斌在北京例行记者会上，就前一天美国国务卿蓬佩奥等美国官员指美方关闭中国驻休斯敦总领馆，是针对中方偷窃美知识产权和开展商业间谍活动作出回应。他说：“美方称中国驻休斯敦总领馆从事与身份不符的活动，这完全是恶意的污蔑。美方要求中方关闭驻休斯敦总领馆，严重违反国际关系基本原则，严重违反中美领事条约的有关规定，严重破坏中美关系。这是在拆中美两国人民友好的桥梁。”
   <br/>
   <br/>
   汪文斌表示，中方必将作出必要反应，维护自身正当权益，但未有提具体反制措施。

  <br/>
  <br/>
  事件发生得非常突然，且在扑朔迷离的诱因中不断释放极具爆炸性的新闻元素。在星期四【不同的声音】本周节目连网前，YouTube中国问题名人堂海量的相关评论，依然是“八仙过海，各显神通。”
  <br/>
  盛名之下的【江峰漫谈】，观察角度一向独特，其言下之意是休斯顿中国总领事馆事件是纵火在前，驱逐在后，领馆火灾报警在晚间8-9点左右，美国国务院的驱逐声明则在凌晨时分发出。
  <br/>
  <br/>
  但美国国务院的驱逐理由“保护美国知识产权和私人信息不受威胁”，让不少观察人士不由自主地联想到几乎是在同一天发生的重大事件：
  <br/>
  <br/>
  美国司法部本周指控两名中国公民通过网络攻击，盗窃与新冠病毒疫苗和国防安全有关的机密信息。
  <br/>
  <br/>
  根据本周二公开的庭审记录，美国政府指控中国公民李啸宇（Li Xiaoyu）和董家志（Dong Jiazhi ）两人在2014至2020年间，作为中国国安部招募的特工，依照中方国安部门提供的电脑软件后台信息发起黑客攻击。受害者包括香港抗议民众、达赖喇嘛办公室、中国基督教公益组织等，遍布美国、英国、德国、澳大利亚和比利时等国家。
  <br/>
  <br/>
  美国政府还指控上述两名中国公民在多次黑客攻击行动中，盗窃武器设计、药品信息和软件代码等。
  <br/>
  <br/>
  美国司法部负责国家安全事务的助理部长迪默斯（John Demers）指出，中国政府为换取黑客盗窃的知识产权信息，不惜为网络罪犯提供庇护，并且已经成为这一可耻俱乐部的一员。
  <br/>
  <br/>
  值得注意的是，起诉状明确列指的数家被盗公司或机构所在地，包括被关闭的中领馆所在地德克萨斯州的一家航天科技相关企业。
  <br/>
  <br/>
  流亡美国的维吾尔青年Inty较为详细地谈到了他所看到的美国司法部的起诉与休斯顿中国总领馆被关闭事件之间的联系。

  <br/>
  稍后进入节目的两位油管知名人士文昭、江峰，他们都多多少少地涉及了本周这一轮美中外交关系历史上的重大危机，及其引发的 “纵火、踢馆、断交、热战”之间的推理，想象，抑或猜测。
