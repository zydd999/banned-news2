 <p>
  中国人大常委会法工委即将就制止餐饮浪费展开立法研究，分析认为，一旦立法“刑事化”，严重浪费食物者随时会被训诫或者遭行政拘留。
  <br/>
  <br/>
  本台以《贯彻习近平指示 “红卫兵”再现上海》为题做出以下报道：
  <br/>
  <br/>
  <b>
   习近平一声令下，严禁食物浪费成为各地政府的迫切任务。上海对于贯彻习近平指示精神也不甘后人。市精神文明建设委员会周日（16日）发通知，说要以刚性的制度约束、严格的制度执行、强有力的监督检查、严厉的惩戒机制，大力整治“舌尖上的浪费”。
   <br/>
   <br/>
   而具体来说，就是建立一套举报投诉机制，重点加强对商务宴请、婚丧嫁娶红白事等餐饮浪费行为的监管，透过“媒体、人大代表、政协委员等各种社会力量开展监督，及时劝导制止攀比摆阔、奢靡浪费等不良风气，如果被检举的人坚决不改，会遭到”批评曝光”。
  </b>
  <br/>
  <br/>
  习近平主导的走火入魔全民反浪费运动一周怪现象之拍案惊奇：
  <br/>
  <br/>
  某酒店为制止餐饮浪费专门设立了巡检员岗位，监督是否浪费和光盘：“这个鸡屁股切除部分有点大了，这么多连着肉的部分都切除扔掉太浪费。”
  <br/>
  <br/>
  某酒店推出这样一条规定：“倡导光盘行动”以来，酒店对服务员的考核标准由剩菜率不能超过20%修订为15%，如果超过标准，会对服务人员进行处罚。
  <br/>
  <br/>
  因此导出一段火了的YouTube视频：某饭店老板在原本司空见堆满残汤剩菜的圆台面前愁容满面地吃着客人留下的剩饭，还命令员工一起过来吃。
  <br/>
  <br/>
  河南日报的大河网署名刘哲者居然言之凿凿地向政府部门献策献计：能不能实行举报制？对举报属实者有奖。能不能像执行“八项规定”那样，组织对饭店明察暗访，发现浪费现象对食客和饭店均予处罚？能不能对落实节约的饭店予以补偿性奖励？能不能像如今不少地方涌现的“红白理事会”那样，规定婚丧嫁娶宴席的规格？能不能推行诸如“N－1”（点菜数是吃饭人数减1）、卖“小份菜”“半份菜”这样好办法？能不能推行标准化的小饭碗、小菜盘？相应的，能不能把可操作的各种措施，通过人大立法确立为法律......
  <br/>
  <br/>
  从小到大，共产教育体制中充斥着的反浪费红色洗脑，通常几乎百分之百夹带着一阵又一阵对西方富裕生活的极端仇视：
  <br/>
  <br/>
  “三毛流浪记”，饿得皮包骨的流浪儿在餐馆的玻璃窗外看着脑满肠肥满脸横肉的万恶资本家一个人面对一大堆鸡鸭鱼肉大快朵颐，浪费着食物。
  <br/>
  <br/>
  新华社日复一日的国际新闻中，罪恶的资本主义世界，大老板宁愿浪费成吨的牛奶倒进大海，也不愿意赊给饥寒交迫的工人阶级。
  <br/>
  <br/>
  “贪污和浪费是极大的犯罪。” 毛泽东在中华苏维埃第二次代表大会上说的这句话，86年来成为中国共产党在形形色色政治运动中整人的“刑法”，数不清的中国人因为“浪费罪”被整治，被批判，被认罪。
  <br/>
  <br/>
  今天习近平重拾这一无法无天弊句的牙慧，让病入膏肓的新冠中国再一次的坠入全民走火入魔。
  <br/>
  <br/>
  让我们体验一下YouTube里展现的当下中国大陆反浪费运动的癫狂程度：
  <br/>
  <br/>
  <b>
   浪费从来不是犯罪，节约不见得有理。
  </b>
  <br/>
  <br/>
  我们听一听美国、日本这些发达西方国家人民对“浪费”的诠释：
  <br/>
  <br/>
  胡立德（David Hu）是佐治亚理工学院机械工程与生物学系教授。是全世界获得过最多幽默科学奖的科学家。同时，他也是FOX新闻“十大资源浪费排名榜”上唯一囊括三项排名的美国人，“浪费科学家”的雅号因此不胫而走。他在麻省理工大学获得数学博士学位，26岁两次登上Science杂志封面。他研究动物们如何巧妙利用物理原理在不同环境中适应并进化。 他研究的课题包括：松鼠和蜜蜂谁的毛更多？苍蝇为什么老在那儿搓手呢？狗甩干自己身体的速度到底有多快？为什么大象和狗尿尿的时间一样长？这些研究大部分被福克斯电视台的节目制作人认为是对科研经费的巨大浪费，但事实上却在不断的以其充满惊人想象力的发明创造进一步改善着人类的生活。
  <br/>
  <br/>
  【不同的声音】网海拾贝，撷取颂赞浪费的一节节美轮美奂，断章取义如下：
  <br/>
  <br/>
  <b>
   一分钟不浪费，大错特错。持此观念生活，人生充满负罪感。另类观念是：高效利用碎片化时间，心安理得地浪费时间。
   <br/>
   <br/>
   心理学家迈克尔·古特里奇（Michael Guttridge）说过：“浪费时间是为了给你的电池充电，恢复状态。”
   <br/>
   <br/>
   这个世界永远有风景，但只有懂得浪费人生的人才会看到 ——松浦弥太郎
   <br/>
   <br/>
   “即使是疯狂看电视也会是一种愉悦的体验，只要你放松去享受它。研究发现，对于那些带着负罪感把自己叫做“沙发土豆”的人来说，看电视反而不那么愉快。一天结束的时候，我们所有人都应该花点时间去翻翻杂志，逛逛街，或者什么都不做。我们应该享受这些时间，并且应该在心理上正确看待它们：这样“浪费”时间很好。”
   <br/>
   <br/>
   享年82岁的歌德几乎“浪费”一生的光阴，耗时60年完成了《浮士德》。
   <br/>
   <br/>
   男作家：“我已经出版了339部小说了，请问你出版了几部？”
   <br/>
   <br/>
   “我只写了一部。”
   <br/>
   <br/>
   “什么名字呢？”
   <br/>
   <br/>
   “《飘》”女作家平静地说。
   <br/>
   <br/>
   “我一生最重要的时间都浪费了。浪费时间与等待，占去生命的大部分。看似调侃生命的徒劳，实际上是彻底抛弃外物，投进时间纯粹而无穷的河流里。浪费时间也分几层。像在笼子里熬时间，剩下的就只有时间。人活在这种状况，不看书，什么都不做，人会慌，会思考，才能令你活一天，过一天。” ——明周 【著名行为艺术家专访】《谢德庆：我一生最重要的时间都浪费了》
  </b>
  <br/>
  <br/>
  <b>
   当你质疑自己浪费时间的时候，你就已经在双重浪费时间了。
   <br/>
   <br/>
   生活中有无数种幸福，能毫无压力地浪费时间，是其中的一种。能够时不时心安理得地干这种事，是一种能力。只有碌碌无为的人，才会对浪费时间有无法抑制的恐慌。
   <br/>
   <br/>
   我们的目标不是不浪费时间，而是可以不带负罪感地浪费时间。
   <br/>
   <br/>
   陈道明是中国最具气质的男演员，演艺界的一股清流，而他的这份独特一部分来自他的“不务正业”。在演艺生涯和世俗声名达到巅峰的时候，陈道明“躲”了起来。他更喜欢待在家里读读书、写写字、弹弹琴、下下棋，或者是给妻子女儿做点针线活儿。闲暇时和妻子同坐窗下，一个绣花草，一个裁皮包，窗外落叶无声，屋内时光静好。
   <br/>
   <br/>
   很多人可能认为他这样佛系的生活是在浪费时间，可陈道明却不以为然：“不做无为之事，何以遣有涯之生？
   <br/>
   <br/>
   “生命就应该浪费在美好的事物上。”这是台湾黑松汽水的一句广告词。生命从头到尾都是一场浪费，你需要判断的仅仅在于，这次浪费是否是“美好”的。后来，当我每做一件事情的时候，我便问自己，你认为它是美好的吗？如果是，那就去做吧，从这里出发，我们去抵抗命运，享受生活。什么都不干，浪费时间其实并不可怕。假期在既没有工作也没有玩耍的纠结中度过，那才是真正的浪费，周末忙着浪费时间的感受其实挺好的——吴晓波送给18岁女儿的一段话
   <br/>
   <br/>
   真正的浪费是获得世界的赞誉却在追求这种赞誉中丢掉了灵魂——圣经 参太16：26
   <br/>
   <br/>
   比起热闹的市中心，我更喜欢宁静的住宅区。一个转角一面墙，都是另一种风景。东京的美令我流连。不需要觉得好像失去了珍贵的时间，因为在东京，这些都不重要了；不怕因为放慢步调而浪费一个东京的夜晚，因为这浪费也是一种享受——Fliper 矢名作者
   <br/>
   <br/>
   用淡泊写意人生，用安然葱茏时光。把光阴浪费在最美好的事情上 ——【原木之美】
  </b>
  <br/>
  <br/>
  节目的最后，我们借势欣赏一首由台湾歌手林宥嘉演唱的美妙歌曲《浪费》。歌中唱道：
