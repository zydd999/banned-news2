  <br/>

  <br/>

  <br/>
  这座城市，自被官家贴上封条之后，便开始极具中国特色地向神州，向邻国，向全世界几乎每一个角落夜以继日不停歇地散发着亿万万无以计数的问号状毒株，而其自身，却令人惊艳般描摹出洁身自好和出污泥不染般匪夷所思的绝对优良医疗数据。
  <br/>
  <br/>

  <br/>
  <br/>
  2020年4月2日，强大舆论倒逼之下，李文亮被湖北当局评定为“烈士”。

  <br/>
  <br/>
  本周，她的名字突然出现在与武汉千里之隔的上海，当地警察出示给家属的逮捕通知书上的罪名是烈士的反义词：寻衅滋事。
  <br/>
  <br/>

  <br/>
  <br/>
  因而，他们对李文亮们与生俱来原罪般坚如盘石的刻骨仇恨，绝不是一个贻笑大方虚妄至极的“烈士”虚名掩盖得了一丝一毫的。
