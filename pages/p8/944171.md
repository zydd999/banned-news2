 <p>
  只是看起来简单好料理的鸡蛋，从保存到烹调都有不少需要注意的地方。若不小心的话，影响口感事小，危害身体健康事大。
 </p>
 <p>
  以下就列出常见的鸡蛋5大错误处理方式，快来看看你犯过几种错误？
 </p>
 <p>
  <strong>
   1
  </strong>
  <strong>
   、在碗缘、锅边或是灶台边
   <span href="https://www.secretchina.com/news/gb/tag/打蛋" target="_blank">
    打蛋
   </span>
  </strong>
 </p>
 <p>
  很多人打蛋时，习惯沿着碗缘或是锅边敲打，这种方式看来好像方便又好施力。但这动作不但容易让
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/蛋壳" target="_blank">
    蛋壳
   </span>
  </strong>
  外面的细菌，透过裂缝跑进鸡蛋里面，还可能让破裂的细碎蛋壳掉进蛋里。
 </p>
 <p>
  正确的方式是选择在平坦的台面上将鸡蛋敲开，然后将裂缝处朝上面向自己，再剥开蛋壳，把蛋打入碗中。
 </p>

  <strong>
   2
  </strong>
  <strong>
   、蛋洗过后才冷藏保存
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 <p>
  因为蛋壳上有许多
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/气孔" target="_blank">
    气孔
   </span>
  </strong>
  ，加上还有一层“护膜”可以防止细菌进入。所以洗蛋时的水温很重要，必须精准拿捏才不会把护膜洗掉。只是一般人很难掌控清洗鸡蛋的水温，只会随便用清水冲洗鸡蛋，如此一来不但会把护膜洗掉，鸡蛋也容易因此变质，缩短保存期限。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 </p>

 <p>
  <strong>
   3
  </strong>
  <strong>
   、把全部的蛋打进一个碗里
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 <p>
  有些时候，当料理需要将多颗蛋混在一起时，不少人会贪图方便，直接用一个碗把所有的蛋都装在一起，这种做法看似没错，但其实是有风险的。因为在打蛋时，我们很难确定蛋是否完好，有可能发生一颗“坏蛋”毁了一碗“好蛋”的状况发生。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  正确的处理方式，应该分别拿一个小碗跟大碗，每次先打一颗蛋在小碗里，确认鸡蛋没有坏掉之后，再倒进大碗中，就会非常保险了。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  <strong>
   4
  </strong>
  <strong>
   、蛋在水滚时才下锅
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 </center>
 <center>
  <ins class="adsbygoogle" data-ad-client="ca-pub-1276641434651360" data-ad-format="fluid" data-ad-layout="in-article" data-ad-slot="3646767294" style="display:block; text-align:center;">
  </ins>
 </center>
 <p>
  氽烫食材时，等到水滚了再把食材放入锅中，是一件很合理的事。只是煮水煮蛋并不是氽烫，若等到水滚才下锅，由于内外剧烈的温差，很容易让蛋壳产生裂缝。如此一来，里面还没熟的蛋白，就会沿着裂缝渗出来，可能水煮蛋没煮成，反而煮出一锅蛋花汤。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

  </div>
  <div>
   <center>
    <div id="div-gpt-ad-1589559869784-0">

   </center>
  </div>
 </center>
 <p>
  正确的做法是在冷水时就下锅，这样可以让鸡蛋跟水同步加热，不但能避免蛋下锅后，手被热水喷溅烫伤的状况，蛋壳也比较不会有裂缝产生。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 <p>
  另外，选用温度跟室温差不多的蛋，或是在煮蛋的水中加入少许盐，也都可以让蛋在烹煮时，蛋壳较不容易裂开。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

  <strong>
   5
  </strong>
  <strong>
   、大火炒蛋
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  很多料理习惯采用大火快炒，但鸡蛋并不适合用这种方式烹调。因为蛋的水分容易因为高温流失，导致炒出来的蛋偏干硬。所以想让炒蛋的质地湿润软嫩，就要以中小火慢慢拌炒。有兴趣的读者，也不妨加入牛奶、起司片或是酸奶油等，让鸡蛋增添香气，口感也更绵软。
