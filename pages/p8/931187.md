 <p>
  很久以前，在长白山下住着一个年轻
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/猎人" target="_blank">
    猎人
   </span>
  </strong>
  ，他枪法本来就不怎么好，可在打猎的时候还总是挑东拣西的，大的呢打不了，小的呢又看不上，每次打到的猎物都很少。看着其他猎人每天都能打到许多猎物，他很是
  <strong>
   嫉妒
  </strong>
  。碍于
  <strong>
   面子
  </strong>
  ，他总是对妻子说：“我打到了大动物，可是我一个人扛不动。”
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  一次他又这么说，他妻子说：“下次我跟你一起去，两个人可能抬动吧。”
  <br>
   <br>
    他不愿意，妻子又接着说：“我不打扰你。等你打完了，我帮你抬回来。”
   </br>
  </br>
 </p>
 <p>
  没办法，他只好让妻子同行。说来也巧，刚一进山，他们就看了一只大野猪，猎人端起枪来瞄了半天，可是开枪却打到了大石头上。野猪给惊吓到，对着他们夫妻就冲了过来，幸好有个猎人正经过这里，一枪把野猪打死了。
  <br>
   <br>
    <strong>
     “
    </strong>
    哎呀，都怪那块大石头，不然一只小野猪算得了什么呢。”年轻猎人埋怨着说。
    <br>
     <br>
      <strong>
       “
      </strong>
      呵呵，年轻人，走路栽了跟头只能说明走路的人没有看清楚，又怎么能怪不会动的石头呢；同样地，打猎要看枪法，栽了跟头可别怪石头啊。”
      <br>
       <br>
        <span href="https://www.secretchina.com/news/gb/tag/俗语" target="_blank">
         俗语
        </span>
        ：
        <strong>
         “
        </strong>
        栽了跟头，怪石头”比喻犯了错误,不要一味强调客观情况,要从主观上找原因。要有承认“我错了”的勇气，要有担当和改过的勇气。
        <br>
         <br>
          <strong>
           “栽了跟头，怪石头”的四类人
          </strong>
         </br>
        </br>
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  有一种人，犯了错，却不肯承认是自己的问题，而是在第一时间，找个替罪羊，怪罪别人。要是找不到可迁怒的人，那门前的
  <span href="https://www.secretchina.com/news/gb/tag/石头" target="_blank">
   石头
  </span>
  、树叶，都能成为他怪罪的对象。而且找出的理由，也是无奇不有——有荒诞无稽的，有逻辑霸道的；有引人义愤填膺的，有让人匪夷所思的。怎么有利于推脱就怎么讲。
  <br/>
  <br/>
  这种人，比较典型的有四大类：
  <br/>
  <br/>
  <strong>
   1、缺乏责任心
  </strong>
 </p>
 <p>
  这类人没有担当，缺乏责任心。一旦出问题，就千方百计地，想着如何脱身，极尽能事地把一切责任堆到别人头上，找人来给他背黑锅。这种人，不能与他合作，也无法开展合作，因为他时刻都想着逃避责任，时刻都想找个人垫背。
  <br/>
  <br/>
  <strong>
   2、没有良心
  </strong>
 </p>
 <p>
  这类人像只乌龟，遇事就缩头。要是犯了错误、闯下祸端，他马上绞尽脑汁地想着如何摆脱。不管找什么理由，强词夺理也好，丧尽天良也罢，凡是能让自己撇得干净的借口，他都找得出，也都做得出。在推脱责任方面，他可以不择手段、毫无人性。
  <br/>
  <br/>
  <strong>
   3、死要
   <span href="https://www.secretchina.com/news/gb/tag/面子" target="_blank">
    面子
   </span>
  </strong>
 </p>
 <p>
  这类人喜欢怪罪于人，是因为特要面子。他很好强，自尊心极强，也非常敏感，不肯在别人面前示弱，更不肯承认自己会犯错误，尤其是犯低级错误。因此，只要一犯错误，他立马就会想着如何销毁证据、转嫁焦点。
  <br/>
  <br/>
  <strong>
   4、居心险恶
  </strong>
 </p>
 <p>
  这类人城府深、狡猾、阴险、自控力强。往往是将好事揽到自己身上，将坏事推给别人，理由总是那么堂皇，又是那么无可辩驳。这类人做事诡异、缜密。做事之前，就已布局好了后路，一旦搞砸了，马上就能顺手拉上几个垫背。
  <br/>
  <br/>
  喜欢怪罪别人的人，常常会把污秽搅到无关的人头上。因此，极少有人愿意靠近他。因为，就算追究起来，证明了自己清白，但也臭了自己，烦了自己，累了自己，破坏了自己的心情。
  <br/>
  <br/>
  其实，人非圣贤孰能无过？出了错误，要有知错、认错、改错的态度，接受批评，认真配合，逐步搞清真相，别让同样的石头绊倒两次；而不是不从自身找原因，强调客观，极力推诿。在任何场合，做人都要有担当。
