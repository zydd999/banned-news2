 <p>
  近年来许多地方的天气变得越来越极端，要不是久旱不雨，就是豪雨成灾。而下大雨
  <strong>
   淹水
  </strong>
  时，往往会造成许多生命财产的严重损失。价值不菲的
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/汽车" target="_blank">
    汽车
   </span>
  </strong>
  一旦被水淹过，那在水退去后，心急的车主也会想要寻求挽救的方法。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  那么如果遇到
  <span href="https://www.secretchina.com/news/gb/tag/泡水车" target="_blank">
   泡水车
  </span>
  ，有什么方法可以补救，将损失降到最低呢？
 </p>
 <p>
  <strong>
   1
  </strong>
  <strong>
   、千万不要发动车子
  </strong>
 </p>
 <p>
  这点是最重要的，因为车子若经过
  <span href="https://www.secretchina.com/news/gb/tag/淹水" target="_blank">
   淹水
  </span>
  地区导致熄火，或是在豪雨来袭时遭到水淹，那在积水退去后，一定不要发动车子。否则如果
  <strong>
   发动机
  </strong>
  已经有进水，那么一启动车辆，就可能造成进一步损坏，严重时甚至整台车会因此报销。
 </p>
 <p>
  <strong>
   2
  </strong>
  <strong>
   、初步估计车辆受损程度
  </strong>
 </p>
 <p>
  在确认周围没有安全疑虑的情况下，车子受损的程度要赶快进行检查。可以先了解汽车周围的水深，注意要把泥土和杂物留下的水位线证据保留下来。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  如果水位没有淹到车门底部，那造成严重损坏的可能性较低。但如果水位较高，甚至到了仪表盘的水平位置时，那么整台车的零件估计都已经受到影响了。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

   3
  </strong>
  <strong>
   、拍照并联系保险公司
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 </p>

  <strong>
   、尽快把车子拖去车行或是维修公司
  </strong>
 </p>

 </p>

 </p>
