 <p>
  美国总统奥巴马27号访问了核爆地广岛向核爆遇难者慰灵碑献花，悼念包括核爆受害者在内的全体战争遇难者。奥巴马发表感言表示应实现无核武世界。
  <br/>
  <br/>
  美国总统奥巴马27号在结束了伊势志摩峰会会议日程之后下午5时许搭乘直升机从山口县的美军岩国基地飞抵广岛直升机机场，对广岛进行访问。
  <br/>
  <br/>
  1945年，美军在广岛投下人类历史上第一颗原子弹，造成14万人死亡后，期间经历了12任总统，奥巴马是第一位到访广岛的在职美国总统。
  <br/>
  <br/>
  奥巴马在和平公园与前来迎接的日本首相安倍晋三及日本外相岸田文雄汇合后，首先参观了展示原爆受害者遗留物及照片等的广岛和平纪念资料馆。奥巴马在参观了原子弹轰炸资料馆后表示：“我们必须正视历史的责任”。此后，在广岛、长崎两地市长及原爆受害者的见证下，奥巴马向核爆遇难者慰灵碑献花并发表了17分钟的演讲。
  <br/>
  <br/>
  奥巴马在讲演中表示说:“71年前世界改变了。我为了悼念包括女性及儿童等超过10万名日本人、许多朝鲜半岛出身的人以及美国战俘而来到这里。”
  <br/>
  <br/>
  在此之上，奥巴马还表示，“我们应该有勇气摆脱必须拥有核武器的恐怖理论，应该致力实现无核武世界。”
  <br/>
  <br/>
  在奥巴马发表讲演之后，日本首相安倍晋三发表了感言。安倍表示说：“美国总统亲眼见证原子弹轰炸造成的现实，坚定打造无核武世界的决心，给坚信‘无核世界’的全世界的人们带来了希望。广岛以及所有日本国民都对这次历史性的访问期盼已久，我对此访表示衷心的欢迎。这次访问为日美两国的和解、信赖与友谊翻开了新的历史篇章，我对总统的决断和勇气表示深深的敬意。”
  <br/>
  <br/>
  奥巴马在发表感言后，与日本核爆受害者团体协议会代表坪井直进行了交谈。坪井向奥巴马表示，“我们不责怪也不憎恨美国”。奥巴马还紧紧拥抱了为确定在广岛核爆中遇难的美军战俘等的身份而尽力的核爆受害者森重昭。
  <br/>
  <br/>
  据日本政府称，奥巴马发表感言后向安倍表示“能来广岛真是太好了，今天仅仅是个开始”，强调将努力实现废核。
  <br/>
  <br/>
  就奥巴马访问广岛一事，中国外交部部长王毅27号表示：“广岛值得关注，南京更不应忘记。受害者值得同情，但加害者永远不能推卸自己的责任。”
  <br/>
  <br/>
  （特约记者：南洲）
