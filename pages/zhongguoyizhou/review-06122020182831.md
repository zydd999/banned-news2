 <p>
  1 香港反送中运动于6月9日届满一周年。当天，香港各区均举行不同形式的抗议纪念活动。大批市民前往湾仔、中环、太古、九龙湾、观塘等商场，高喊“光复香港，时代革命”口号。
  <br/>
  <br/>
  值得注意的是前一天，中国国务院港澳办副主任张晓明在香港特区政府举办的一场网上研讨会上强调，香港现在的最主要问题源于政治，就是“反对派及其背后的外部势力企图把香港变成一个独立或半独立的政治实体，变成一个反华反共的桥头堡，变成外部势力一枚牵制和遏制中国发展的棋子”，同时将此作为“中央出手处理香港国家安全立法问题的事实依据”。
  <br/>
  <br/>
  美国《华尔街日报》9日的报道提到，中国政府一直将香港抗议的源头归结于经济和社会问题，张晓明这番不同于以往官方说辞的言论，标志着北京方面对香港“骚乱”的判断发生了转变。
  <br/>
  <br/>
  2 美国国务卿蓬佩奥星期六（6日）发表声明，抨击中共利用美国的抗议活动进行下流宣传，再一次暴露了中共的真实面目。
  <br/>
  <br/>
  蓬佩奥谴责中国“使用弗洛伊德的死，来洗白中共剥夺中国人民最基本的人权”。他说，“就像历史上其他的独裁政权一样，只要谎言能够服务于政党对于权力的渴求，那么谎言就没有下限”。蓬佩奥认为，“这种可笑的宣传手段不能愚弄任何人”，“中共通过宣传上的努力，把弗洛伊德死后美国人的行为，与中共对基本人权与自由的否认，混为一谈，这应当被视为一种欺诈”。
  <br/>
  <br/>
  蓬佩奥在声明中说，美国与中国共产党（CCP）之间的对比再明显不过了。比如在中国，教堂被烧毁时，袭击几乎肯定是由中共指挥的；在美国，教堂燃烧时，纵火犯受到惩罚，正是政府将消防车、水、援助和慰藉带给了信徒。在中国，当公民持有与中共不同的意见时，共产党将他们囚禁在再教育营中，而当人们、例如香港和台湾的人们要拥抱自由亦遭到打压；相反，在美国，即使在骚乱中，我们也表现出对法治、透明和不可剥夺的人权的坚定承诺。
  <br/>
  <br/>
  3 就在中国各地纷纷响应总理李克强推动地摊经济之际，北京官方、中国官媒突然出现不太同调的讯号。
  <br/>
  <br/>
  五月底，中国总理李克强在全国人大会上大力提倡“地摊经济”，推出鼓励失业人士做小贩谋生的保民生政策。“地摊经济”在中国大街小巷热议不过一个多礼拜，却接连遭到北京市官方及中央级媒体"打脸"。
  <br/>
  <br/>
  6月6日，《北京日报》先刊出一篇题为《地摊经济不适合北京》的文章，引述北京城管执法部门称将"依法严格查处占道摆摊设点违法行为"。7日及8日，央视则连续两天发出热评文章: 《“地摊经济”不能一哄而起》、《一线城市不宜推行“地摊经济”》。
  <br/>
  <br/>
  这些批评声音与李克强在6月1日走访山东烟台考察时大赞“地摊经济”是“人间的烟火，是中国的生机”的画面形成强烈对比。
  <br/>
  <br/>
  4 中共中央和国务院机关近日发布一份内部文件名为《中央和国家机关党员工作时间之外政治言行若干规定（试行）》，要求中共党员必须旗帜鲜明严守政治纪律和政治规范。其中提出十九个“不准”，比如在“任何场合”，坚持政治底线不能碰，政治红线不能越。不准说不同意见，特别是“不准发表偏离“两个维护”的言论，搞任何“低级红、高级黑”；不准点赞；不准浏览“反动网站”；不准“收听收看境外反动电台和电视节目”。

  5 六四事件31周年纪念日，流亡美国的中国富商郭文贵与美国白宫前首席战略顾问史蒂芬•班农(Steven Bannon)通过网络直播，宣布成立“新中国联邦”。这一新闻事件最具轰动效应的是，郭文贵请来了前中国国家足球队明星球员，“中国足球先生”郝海东，宣读长达近三十分钟的“新中国联邦宣言”。
  <br/>
  <br/>
  6 高雄市民6月6日以超过93万的票数，将高雄市长韩国瑜罢免下台，票数还超越了韩国瑜当初当选得到的89万票。韩国瑜指控民进党政府二次执政，集中“罢韩”国家队，使尽全力要把他拉下台。而“罢韩”发起人则指出，这场罢免选举是要向中国共产党、习近平说不。
  <br/>
  <br/>
  对于韩国瑜指责“罢韩”国家队一事，蔡英文总统并没有正面回应，她在个人脸书写道，今天这个结果，是给所有政治人物最大的警惕，人民可以赋予我们权力，当然也能够收回。
