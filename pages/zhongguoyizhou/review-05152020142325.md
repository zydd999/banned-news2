 <p>
  1 中国前新闻工作者张贾龙涉嫌“寻衅滋事“案周三（13日）完成庭审，择日宣判。控辩双方就张贾龙去信美国前国务卿应否被问罪争持不下。律师坚持为他进行无罪辩护。触动当局神经的是4年前张贾龙写给时任美国国务卿克里的一公开信，会面时张贾龙批评了中国的网络审查制度和言论控制。
  <br/>
  <br/>
  张贾龙被捕前经常在网络发文，曾报道毒奶粉案等敏感事件。根据起诉书，张贾龙自2016年以来在推特上发布了大量诋毁党、国家和政府形象的虚假信息，应当以寻衅滋事追究刑事责任。
  <br/>
  <br/>
  2 德国之声星期五（5月8日）援引德国《明镜周刊》的报道说，德国联邦情报局掌握的信息显示，武汉疫情爆发后，中国曾在最高层面敦促世界卫生组织推迟发布全球警告。
  <br/>
  <br/>
  报道指出，中国国家主席习近平在1月21日与世卫组织总干事谭德塞（Tedros Adhanom Ghebreyesus）的通话中，请求该组织推迟发布有关武汉病毒人传人的信息和大流行病警告。
  <br/>
  <br/>
  3 美国联邦参议院在美东时间5月11日以一致通过法案，要求美国务卿拟定策略协助台湾重获“世界卫生组织”（WHO）观察员身分。这也是继参众两院在今年三月通过《台北法案》后，美国再次采取具体行动力挺台湾参与国际组织。
  <br/>
  <br/>
  世界卫生大会(WHA)即将在18日召开，台湾至今仍然没有收到邀请函。世卫法律顾问索罗门（Steven Solomon）5月4日于记者会上表示，台湾能否参与世卫大会，或获观察员身份出席，视194个会员国的态度，世卫秘书处无权决定。
  <br/>
  <br/>
  而美国国务卿蓬佩奥在5月6日的新闻发布会中则表示：“我呼吁世界卫生组织总干事谭德赛邀请台湾参加本月的世界卫生大会，他本人有权力这样做，而且他的各位前任已经多次这样做了。”
  <br/>
  <br/>
  美中经济暨安全检讨委员会12日以「北京的致命游戏：疫情期间将台湾排除在世卫之外的后果」为题，发布报告探讨世界卫生组织（WHO）排除台湾带来的影响。该报告指出，世卫因政治因素排除台湾损害全球卫生，台湾信息分享遭抑制，世卫应对又不够实时，造成人命丧失。
  <br/>
  <br/>
  对此中国外交部发言人赵立坚指出，该委员会发表的报告一贯歪曲事实、充满偏见，并再度重申台湾参与世卫活动需要按照一中原则处理。
  <br/>
  <br/>
  4 国际民调机构皮尤中心最新一项社调结果显示，66%的台湾民众认为自己是“台湾人”，28%的台湾民众认为自己既是“台湾人”也是“中国人”，而认为自己是“中国人”的台湾民众仅有4%。
  <br/>
  <br/>
  皮尤中心表示，上述结果与其他多项民意调查相符，显示台湾只认同自己是“台湾人”身份的人数正在上升。
  <br/>
  <br/>
  美联社的报道说，皮尤中心的调查结果凸显中国试图掌控台湾的意图将面临挑战。
  <br/>
  <br/>
  5 美国联邦调查局和国土安全部星期三发表联合声明，指控网络攻击美国科研机构试图盗窃新冠病毒研究的黑客组织具有中方背景，并且警告政府卫生主管部门和科研人员予以警惕。
  <br/>
  <br/>
  声明指出，美国联邦调查局针对近期一系列入侵美国研究机构的网络攻击事件进行调查后发现，具有中方背景的黑客“试图识别和非法获取与医疗单位及病患有关，涉及知识产权和与公共健康信息的新冠病毒疫苗研究、治疗方法和检测技术。”
  <br/>
  <br/>
  <br/>
