 <p>
  1 五月二十二日开幕的中国全国人民代表大会将审议有关香港“维护国家安全”的议案。
  <br/>
  <br/>
  中国政府绕过香港特区立法会对香港实施国家安全法的提案，已经纳入全国人大会议议程，预计将于下周通过表决。文件披露，中央会根据需要在香港特区设立“相关机构”。相关人士认为：北京强推行这项法律，胁迫香港入伙“一国一制”，预计未来香港会有严重性不逊于1989年的大抓捕行动。
  <br/>
  <br/>
  苹果日报22日星期五最新消息：「港版国安法」降临香港，正在北京人大会议开幕仪式的特首林郑月娥下午发表声明，指香港是中华人民共和国不可分离的部份，维护国家主权、安全和发展利益是对香港特别行政区的宪制要求，是香港特别行政区政府的职责所在，也是关乎全体香港居民的切身利益。声明又指，香港将全力配合全国人大常委会尽快完成有关立法，履行维护国家安全的职责。当晚上8时半，紧急返港的林郑再在政府总部会见传媒开腔响应。
  <br/>
  <br/>
  而中共的政协会议则在早一天的周四（5月21日）开幕，出席开幕式的约2100名全国政协委员都戴了口罩。不过，大会主席台上，在前两排就坐的中国国家主席习近平等党和国家领导人，以及政协领导人均没有戴口罩。在主席台后排就坐的其他人则戴着口罩。全国政协主席汪洋在开幕式上做报告时，也没有戴口罩。
  <br/>
  <br/>
  本届中共两会，创下两项史上之最，一是会期最短，只有八天；二是外媒记者最少。成为中共史上最短命、对外最封闭的两会。
  <br/>
  <br/>
  2 台湾总统蔡英文、副总统赖清德在520宣誓就职。蔡英文在就职演说中强调，2300万台湾人是生死与共的命运共同体，期许上下一心，当一个光荣的台湾人。最受瞩目的两岸关系部份，蔡英文重申“和平、对等、民主、对话”八字箴言，不会接受“一国两制”。蔡英文承诺遵照中华民国宪法、两岸条例，维持台海和平稳定现状，在国家建设工程上，她宣示立法院将成立修宪委员会凝聚共识。
 </p>

 <p>
  蔡英文5千多字、25分钟的第二任就职演说，“重中之重”是以新冠肺炎台湾的表现作为主旋律。第一部份“作为共同体的台湾”，她感谢所有台湾人对政府的信任，让世界看到台湾在最不安的时刻也能保持公民美德。“国家的光荣感，生死与共的共同体，这一段记忆，将会存在我们每一个人心中，团结的感觉，就是这个样子。”
  <br/>
  <br/>
  此前的星期二，美国国务卿蓬佩奥预祝台湾总统蔡英文连任就职典礼成功。蓬佩奥在声明中表示，美国与台湾针对印太地区局势具有共识 – 一个保障各方享有法治、透明、繁荣、安全的共识。而近期应对新冠疫情更提供了一个契机，让国际社会了解到为何台湾的抗疫模式是值得效仿的榜样。蓬佩奥表示，蔡英文总统以大幅票数胜选连任显示已赢得台湾人民的尊敬、钦佩和信任，而蔡英文总统带领台湾民主的勇气与意愿也鼓舞了区域和世界。蓬佩奥还强调，美国长期将台湾视为世界一股良善力量及值得依赖的伙伴，美国对台湾的支持是跨党派且全体一致，而近期生效的《台北法》更将增强美台整体关系。他相信在蔡英文总统带领下，美台关系将持续茁壮成长。
  <br/>
  <br/>
  就此台湾外交部当天回应表示，美国国务卿首次声明祝贺台湾总统就职的举动意义重大，充分彰显美国对台湾关系的重视，对于台湾民主的鼎力支持，也反映美台关系的高度互信友好。未来台湾政府将在既有良好基础上，与美国持续携手合作，共同推动深化美台全球合作伙伴关系。
  <br/>
  <br/>
  据台湾非政府组织“台湾人公共事务会”（FAPA）发布的消息，十个海外台湾人社团于台湾正、副总统就职日当天，集资在美国《华盛顿时报》刊登全版广告，庆祝台湾的民主，并呼吁美台建交。
 </p>
 <p>
  <br/>
  3  世界卫生大会5月18日以网络视频方式召开，被世界公认为新冠肺炎防疫模范的台湾，连续四年被拒之门外。
  <br/>
  <br/>
  欧盟的27个成员国以及非洲国家集团、俄罗斯、加拿大等122个国联合提案，要求世卫组织对冠状病毒大流行的最初反应，进行“公正、独立及全面的评估”。
  <br/>
  <br/>
  中国国家主席习近平周一在世界卫生大会开幕式上讲话中承诺，中国将在两年内提供20亿美元，用于国际社会抗击冠状病毒。
  <br/>
  <br/>
  美国总统特朗普则在周一晚间通过推特公开一封致世卫总干事谭德赛的信件，警告如不在30天内做出改革，美国将考虑永久终止资助，并且退出世卫组织。
  <br/>
  <br/>
  4 继2月陈秋实、方斌等人被警方控制之后，中国知名异议人士、公民记者张展上周也在武汉被来自上海的便衣警察从宾馆带走，连夜送到上海羁押。上周五（15日），上海市公安局浦东分局以张展“涉嫌寻衅滋事罪”，对其刑事拘留，现羁押在浦东新区看守所。
  <br/>
  <br/>
  张展曾因去年九月在上海街头举伞要求结束社会主义制度，要求共产党下台，被黄浦公安分局以“涉嫌寻衅滋事罪”刑事拘留65天后获释。
  <br/>
  <br/>
  张展于2月上旬左右抵达武汉后，在社交媒体推特发布武汉疫情情况，并介绍了疫情患者或死者家属的维权情况。张展失联前曾向朋友说，5月13日晚她会在汉口火车站广场进行YouTube直播。第二天便与外界失联。
  <br/>
  <br/>
  5 中国国家卫健委官员星期五（5月15日）在国务院新闻办公室举行的发布会上暗示，中国在今年的1月3日曾指示实验室销毁病毒样本，但是强调这样做是为了实验室的生物安全。
  <br/>
  <br/>
  在此之前，美国国务卿蓬佩奥5月6日在美国国务院的新闻发布会上说：“中国本来可以避免全世界数十万人的死亡，可以避免世界陷入全球经济萎靡，他们是有选择的。但是中国掩盖了武汉的疫情爆发，中国卫健委1月3日下令销毁病毒样本。”
  <br/>
  <br/>
  纽约时报也在星期六的报道中说，中国卫生官员间接表示，在疫情早期中国一些实验室销毁新冠病毒样本，中国官员强调这是出于生物安全需要。
  <br/>
  <br/>
