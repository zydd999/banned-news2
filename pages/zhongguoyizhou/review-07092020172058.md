  1 相对7年前3员蹩脚便衣入夜后以嫖娼之名悄悄将雷洋置于死地，中国的国家安全部门本周一出动了10余辆警车，20多名特警，团团围住清华园一位教授的宅邸，抓了一个“嫖客”！
  <br/>
  <br/>
  这或许是中国警方“扫黄史”上一项让人啼笑皆非的吉尼斯纪录。
  <br/>
  <br/>
  本台报道：近年来多次公开批评政府的清华大学法学院教授许章润被指涉嫌在四川嫖娼，周一（6日）早上被警方从北京寓所带走。许章润的朋友：“ 来了十多辆车，看到二十多个人进入了许先生的家。”
  <br/>
  <br/>
  近期许章润一直留在北京，但据了解，他太太接到四川警方电话，说许章润涉嫌在四川嫖娼被带走。日前，许妻7日已获警方通知，教授会在12日获释回家。
  <br/>
  <br/>
  半个月前，这位名满天下的法学教授写下或许是他最后一篇严厉抨击中国政府乃至习近平皇权的讨伐檄文：《践踏斯文 必驱致一邪魅人间》。

 <p>
  2 综合消息：6日晚间，香港政府公布《港区国安法》第43条实施细则，该法授权警方要求发布者或网路平台服务商移除危害国安的讯息，违者罚款监禁；警务处长有权向外国或台湾政治性组织或代理人索取涉港活动资讯，包括个人资料、资金、资产等，而整个程序不需法律程序，保安局长批准即可。
  <br/>
  <br/>
  《港区国安法》生效后，港府与警方被赋予更大权力截取通讯资料，甚至可要求平台供应商交出使用者资料。在脸书、谷歌、推特等多家科技公司纷纷表态拒绝后，美国科技龙头微软和近日十分受欢迎的视讯软件Zoom 也随之宣布，暂不配合港府索取用户个人资料的要求。
  <br/>
  <br/>
  港区国安法43条实施细则列明，保安局长可授权警方，规定发布危害国安信息或内容者，及相关网络服务商移除相关内容；甚或限制、停止任何人连接到该平台或相关部分；否则警方可向裁判官申请手令强行检取电子器材，移除该信息；亦可向法院申请手令，授权警方要求有关服务商提供身分纪录或协助解密。违例者，可被判罚款$100,000及监禁两年或半年等。
  <br/>
  <br/>
  上述法令让不少港人和国际分析人士忧心忡忡：中共互联网防火墙不久后或将杀入香港！
  <br/>
  <br/>
  3 新上任的美国全球媒体总署首席执行官迈克尓·帕克(Michael Pack) 最近在接受媒体采访时表示，中国将是美国对外广播的重点。
  <br/>
  <br/>
  据美国之音新闻网消息，帕克日前接受华盛顿时报的专访时表示，美国政府需要强化对外新闻报道，推广美国价值观，并抗衡虚假信息。
  <br/>
  <br/>
  他表示，要公平呈现美国的思想和制度，讲好美国故事，尤其是针对中国，因为美国正受到来自中国的信息攻击。
  <br/>
  <br/>
  据悉，帕克两周前接管的全球媒体总署下辖美国之音和自由亚洲电台等对外广播机构，该总署每年接受数亿美元的国会拨款用于对外广播。
  <br/>
  <br/>
  4 联合国基金会主席兼首席执行官库森斯星期二证实，在美国总统特朗普近期宣布退出世界卫生组织的决定后，联合国已收到美方通知，按照提前一年的规定，美国将于明年7月6日完成退出世卫组织的程序。
  <br/>
  <br/>
  面对国际压力，世卫组织日前宣布本周将派出国际专家团前往中国武汉，针对新冠病毒起源进行调查。
  <br/>
  <br/>
  5 南中国海这两天热闹非常。中国7月1日至5日在此进行军事演习。美国海军也投桃报李，4日起在南海进行“双航母”联合军演。
  <br/>
  <br/>
  美国海军派出尼米兹号航空母舰（USS Nimitz CVN-68）和雷根号航空母舰（USS Ronald Reagan CVN-76）“双航母”于7.4美国国庆日在南中国海进行操演。美国空军则派出上述两艘航母所属的航舰打击小组和B-52H轰炸机和超级大黄蜂轰炸机。
  <br/>
  <br/>
  《华尔街日报》报导，这两艘航母在周末这场近年来美国海军在南海最大规模的军演里，完成了数百架次的战机、侦察机与直升机起降。
