 <p>
  1 六月二十二日，重庆市水文监测总站历史首次发布綦江洪水红色预警。受降雨影响，预计綦江流域重庆段将出现超历史洪水，此次红色预警是重庆市水文监测总站1940年建站以来的首次红色预警。受降雨和上游来水影响，重庆綦河已超保证水位1米，沿江公路被淹，铁轨冲断。
 </p>
 <p>
  官媒把重庆汛情的信息放在不显眼的位置，如同两周前华南水灾的处理手法一样。网民披露，重庆市公安机构已经紧急通知，如果发现在网上或公开场合发布对汛情“不负责任”的消息，一经查实，立即逮捕法办，绝不姑息。
  <br/>
  <br/>
  2  本台报道：长江流域汛期开始，三峡大坝可能溃坝的传闻再起。
  <br/>
  <br/>
  "宜昌以下跑，最后说一次"。中国社交网站上谣传着中国建筑科学研究院研究员黄小坤的"示警"发文。另一谣言，则称“官媒证实三峡大坝变形，水利部：做好防大洪水的准备工作”。
  <br/>
  <br/>
  据报，黄小坤本人已否认发文。中国官媒《环球日报》22日也发文引述中国专家说法，称三峡大坝不存在溃堤风险。
  <br/>
  <br/>
  三峡大坝真有可能溃堤吗？旅德水利专家王维洛回答本台提问：“风险永远存在，中国在世界上是溃坝率最高的地方，在世界的平均水平一倍以上。”
  <br/>
  <br/>
  3 由香港“二百万三罢工会联合阵线”和“中学生行动筹备平台”发起的6月20日全香港首次罢工罢课公投，结果不如人意。
  <br/>
  <br/>
  这次计划中的罢工罢课目的是为了抗议在香港实施国安法。在日前结束的中国人大常委会，审议了香港《国安法》草案，其中规定北京要在香港设立维护国家安全公署。
  <br/>
  <br/>
  据香港电台报道，主办方于6月21日凌晨宣布，罢工投票人数总计九千人，没有达到6万人投票的目标；其中只有6个工会达到罢工的门槛，也没有达到跨界联合罢工的人数要求。罢课投票参与人数有约一万人，虽然有八千人同意罢课，但其中实体投票只有3000人，不足5000人门槛，所以也未能启动罢课。
  <br/>
  <br/>
  另据台湾中央社报道，香港壹传媒创办人黎智英认为，港版国安法有可能虎头蛇尾，因为中国终要和美国和解，而且中共内部权力斗争正日趋白热化。
  <br/>
  <br/>
  4 今年2月对习近平公开发表“劝退书”的民主人士许志永，日前被中国警方正式批准逮捕。
  <br/>
  <br/>
  许志永为北京大学法学博士，因发起新公民运动和法律维权于2013年被首次逮捕，判有期徒刑，2017年出狱。
  <br/>
  <br/>
  去年底，许志永在厦门与20名维权律师和人士聚会，遭到警方追捕，他因此在国内逃亡。2月份许志永在广州被警方带走。
  <br/>
  <br/>
  据“中国公民运动网”报道，许志永的二姐在6月20日接到山东警方的通知，称许志永已经被批准逮捕，逮捕通知书将寄到北京的二姐家中。逮捕的事由尚不清楚，报道认为，可能与厦门聚会有关。
  <br/>
  <br/>
  厦门聚会的另一位核心人物丁家喜也在本周三被宣布正式逮捕。
  <br/>
  <br/>
  5 折腾两周的北京新冠肺炎第二波疫情表面上有所控制。据官方公布中国内地本周二（23日）增12例新冠肺炎，其中北京7例及河北2例本土病例。另外3例由境外输入，分别在上海有2例、陝西1例。与外国相比，中国的疫情数字明显给人“控制疫情有方”的印象，但是由民间发出的信息披露，当局正不惜代价把居民楼大门封死，阻止被感染者离家。
  <br/>
  <br/>
  境外社交媒体推特消息说，北京某小区因爆发严重疫情，为阻止疫情扩散，干脆将出现疫情的单元楼门完全焊死，不让里面人出行。网友质疑，这样做恐怕造成人道灾难，一旦有火灾或其它灾难，里面的人就是死路一条。
  <br/>
  <br/>
  网络活跃人士王爱忠对本台说，他所看到以及了解的北京疫情，并非官媒报道的那样让人乐观：“看起来情况并不是官方媒体公开报道的那么乐观，比如报道的案例不多。如果说仅仅是官方报道的数据，并不需要采取这么严厉的措施，甚至封闭小区。”
  <br/>
  <br/>
  网民发布的一段视频和图片显示，北京市海淀区永定路70号院520号楼爆发严重疫情，穿橘黄色制服的工作人员，在穿白色防护服人员的协调下，把一栋住房的单元楼门完全焊死，以阻止楼内居民出行。
  <br/>
  <br/>
  <br/>
 </p>
