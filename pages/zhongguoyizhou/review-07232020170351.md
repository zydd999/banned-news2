 <p>
  <br/>
  1. 中国政府本周三称，美国星期二突然要求中方关闭驻休斯敦总领馆。美方在证实这一消息时说，此举是为了保护美国的知识产权和美国人的私人信息。
  <br/>
  <br/>
  一位不具名的休斯敦警员对当地的一家电视台透露，这家领馆所在地，以及有不少领馆雇员居住的一座公寓，将于周五下午四点清空。
  <br/>
  <br/>
  另外，中国驻美国休斯顿总领事馆星期二晚上发生火警，消防人员到场，但被拒绝进入。一些照片显示当时领馆人员在大院中用明火烧文件，浓烟滚滚，邻居报警。美国国务院发言人奥特加斯(Morgan Ortagus)星期三在一份电子邮件声明中表示，美国下令关闭中方驻休斯顿总领馆，是以此保护美国知识产权和私人信息不受威胁。
  <br/>
  <br/>
  奥特加斯表示，根据《维也纳公约》，外交人员必须“尊重所在国的法律和法规”，并且“有义务不干涉该国的内政”。她强调，正如特朗普政府不容忍中方不公平贸易、盗窃美国就业机会和其他恶劣行为一样，“华盛顿不会容忍中华人民共和国侵犯美国主权，以及恐吓我们的人民”。
  <br/>
  <br/>
  “关闭(中国)更多使领馆的可能性一直存在。” 特朗普总统22日在白宮记者会上说，“大家都看到在美国下令关闭后，中方人员在领馆內放火，我猜是烧毁文件或纸张，我想知道是怎么回事。”
  <br/>
  <br/>
  美国国务卿蓬佩奥周三在一场记者会上说，美方指控中国驻休斯敦总领馆“加快了间谍活动”，并指出此举是为了保护美国公民、国家安全和经济，但他并未透露更多细节。
  <br/>
  <br/>
  2. 美国司法部指控两名中国公民通过网络攻击，盗窃与新冠病毒疫苗和国防安全有关的机密信息。
  <br/>
  <br/>
  根据周二公开的庭审记录，美国政府指控中国公民李啸宇（Li Xiaoyu）和董家志（Dong Jiazhi ）两人在2014至2020年间，作为中国国安部招募的特工，依照中方国安部门提供的电脑软件信息发起黑客攻击。受害者包括香港抗议民众、达赖喇嘛办公室、中国基督教公益组织等，遍布美国、英国、德国、澳大利亚和比利时等国家。
  <br/>
  <br/>
  美国政府还指控上述两名中国公民在多次黑客攻击行动中，盗窃武器设计、药品信息和软件代码等。
  <br/>
  <br/>
  美国司法部负责国家安全事务的助理部长迪默斯（John Demers）指出，中国政府为换取黑客盗窃的知识产权信息，不惜为网络罪犯提供庇护，并且已经成为这一可耻俱乐部的一员。
  <br/>
  <br/>
  值得注意的是，起诉状明确列指的数家被盗公司或机构所在地，包括被关闭的中领馆所在地德克萨斯州的一家航天科技相关企业。
  <br/>
  <br/>
  3. 中国外交部网站日前公布，习近平外交思想研究中心7月20日在北京成立。
  <br/>
  <br/>
  中国外交部长王毅在成立仪式上揭牌并讲话。他强调，习近平外交思想是新中国外交理论建设划时代的重大成果。当前世界大变局向纵深演变，中国应该以习近平外交思想作为根本的行动指南，开创中国特色大国外交的新局面。
  <br/>
  <br/>
  4. 1999年7月20日，在时任中国最高领导人江泽民的主持下，中国警方在全国范围内抓捕法轮功信仰者，两天后的7月22日，中国政府宣布取缔法轮功。
  <br/>
  <br/>
  在中共迫害法轮功组织及其成员21周年纪念日，美国国务卿蓬佩奥发表7.20声明，谴责中国政府自1999年以来，一直试图清除法轮功的存在，禁止法轮功成员出于信仰的缘故修炼功法。
  <br/>
  <br/>
  蓬佩奥敦促中国政府立即停止对法轮功成员的迫害，释放被拘押的法轮功成员。
  <br/>
  <br/>
  5. 日前遭当局扣查、其后被清华大学开除的法学院前教授许章润获释后首度发声，向全体清华校友发公开信。他形容现在的中国 “风雨飘摇”，但深信“极权必败，自由终将降临”。他强调只要自己还活着，就会继续发声。
  <br/>
  <br/>
  许章润的公开信以中英文撰写，并夹杂文言文。他表示，这次被清华大学开除公职，对他的教学生涯造成致命一击，无法维生。许章润在信中写道，虽然自己年近六旬，视力也有所下降，但他相信有能力可以继续工作，计划今后以写作谋生。
  <br/>
  <br/>
