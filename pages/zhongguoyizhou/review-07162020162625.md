 <p>
  美国总统特朗普周二（14日）现身白宫玫瑰园，宣布已签署行政命令，中止美国给与香港的特殊待遇。
  <br/>
  <br/>
  特朗普：“香港待遇将与中国大陆一样，没有特权，没有特殊经济待遇，没有出口敏感技术。”
  <br/>
  <br/>
  特朗普并确认，经由美国国会参众两院通过的《香港自治法案》成为法律，向剥夺香港自由的人士和实体追究责任，实施冻结资产和限制入境等制裁。
  <br/>
  <br/>
  特朗普：“所有人都看到发生了什么事，这不是好的局面。港人被剥夺自由，剥夺权利，随着香港事态发展，我认为香港已无力与利伯维尔场竞争，我怀疑有很多人会离开香港。”
  <br/>
  <br/>
  中国外交部发言人华春莹周三下午作出回应，警告美国如果进一步落实制裁措施，中国必会报复。
  <br/>
  <br/>
  2 上周末7月11日—12日香港民主派就2020年9月的香港立法会选举举行内部初选，超过60万港人参加了投票，比组织者最初期待的17万超出3倍以上，这是迄今为止选民人数最多的一次，外界评论认为显示了港人争取自由的决心。
  <br/>
  <br/>
  美国国务卿蓬佩奥发表官方声明，“热烈祝贺香港的泛民主派成功举行初选。超过60万港人在首次全港范围的泛民初选中参与投票。他们的热情清楚的表明，面对中共努力扼杀香港自由，他们仍希望发出自己的声音。”
  <br/>
  <br/>
  初选日深夜中联办发表声明，称这次选举有“外部势力”介入，“处心积虑，策动谋划，举行这次所谓‘初选’，是对现行选举制度的严重挑衅，是对立法会选举公平公正的严重破坏，是对其他拟参选人合法权利和正当利益的严重损害。”
  <br/>
  <br/>
  3 星期一7月13日，是诺贝尔和平奖得主刘晓波逝世三周年忌日，由刘晓波生前好友成立的“晓波助澜会”发表声明，当中特别提到当下的香港。
  <br/>
  <br/>
  声明说，刘晓波生前曾经说过，中国需要300年才能变成香港今天这样，谁料刘晓波身后的香港，居然加速内地化与共产化。作为中国唯一一个言论自由的堡垒城市，作为中国唯一可以公开纪念六四的城市，香港正在失去对言论自由的保障。声明说，这恐怕是刘晓波都无法想象的。
  <br/>
  <br/>
  声明写道，刘晓波生前多次表达过他对香港的热爱，他一生的理想，就是让中国内地民众能像香港民众那样，有免于恐惧的自由，有免于禁言的自由，过上真正开放的生活。而今日的香港被摧残至此，声明说，“刘晓波一定会和我们一样，为此痛心不已”。
  <br/>
  <br/>
  4 刚刚被释放回家的清华大学教授许章润日前对外表示，清华大学以“道德败坏”为由，革除了他的教职、开除了公职。
  <br/>
  <br/>
  许章润于7月6日，被四川警方以“嫖娼”为由，从其北京的家中带走，引起外界广泛震动。舆论广泛指责中国政府这么做，是政治迫害。许章润被警方拘押6天后，于7月12日早晨被释放回家。
  <br/>
  迫害许章润政府行为的升级，与不久前《方方日记》海外出版后被唆使的对作家的全民大围剿如出一辙！教授被抓几天前的6月底，位于纽约的博登书屋出版了许章润文章合集《戊戌六章》，和《方方日记》一样的，这本书在亚马逊购物网站上同步出售。
  <br/>
  <br/>
  5 中共再一次展开学习党史、新中国史等“四史”政治思想教育，要求青少年学生学习“四史”后，向成年人讲解中共党史。
  <br/>
  <br/>
  本周二（14日）北京的《青年报》报道，为了让“四史”教育在青少年中达到入耳、入脑、入心的效果，华东师范大学、钱学森图书馆开始了新的探索。华东师范大学最近将“四史”教育融入“新时代宣讲团”宣讲内容中，通过融入，形成学生为骨干队伍向成年人社会宣讲。
