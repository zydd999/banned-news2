 <p>
  1 纪念六四天安门事件31周年，美国国务卿蓬佩奥6月2日会见六四参与者王丹、苏晓康、李恒青和李兰菊等人。
  <br/>
  <br/>
  此外，蓬佩奥国务卿2日就香港政府不批准举办维园6.4纪念烛光晚会一事在推特（Twitter）上发文称：“不让港人发声和选择，和大陆民众一样。所谓的‘两制’到此为止。”
  <br/>
  美国国务院在6月3日举办在线“六四”烛光晚会，纪念在31年前的牺牲者。
  <br/>
  <br/>
  2 五月二十八日，中国总理李克强在中外记者会上的发言令人震惊：“我们人均年收入是三万元人民币，但是有六亿人每个月的收入，也就是一千元。一千元在一个中等城市，可能租房都困难。”
  <br/>
  <br/>
  两天之后，中共领导人习近平在中共机关报《求是》杂志发表文章《关于全面建成小康社会补短板问题》，习近平在文章中宣告，中国已经基本实现全面建成小康社会目标。
  <br/>
  <br/>
  3 全国人大会议闭幕记者会回答中外记者提问时李克强总理肯定了地摊经济。
  <br/>
  <br/>
  中国政府网，李克强总理6月1日在山东烟台考察时再次强调，地摊经济、小店经济是就业岗位的重要来源，是人间的烟火，和“高大上”一样，是中国的生机。
  <br/>
  <br/>
  地摊经济曾在解决中国就业问题上发挥不小的作用，但过去数十年，地摊业被严重妖魔化，伴随着“城管”这个游走法律边缘不光彩职业的应运而生。
  <br/>
  <br/>
  4 全美掀起反警察暴力与种族歧视的活动，然而，中国官方媒体选择性地不报道美国多地警察放下警棍加入示威者行列，甚至有一整个分局的警察以单膝跪地的方式声援示威者。中国公安有可能陪民众一起跪求公义吗？
  <br/>
  <br/>
  非裔男子乔治·佛洛伊德（George Floyd）遭明尼苏达州明尼阿波利斯市警察萧文（Derek Chauvin）暴力执法致死后，“黑人的命也是命！”（Black Lives Matter）这个口号，再次大声呐喊出来。
  <br/>
  <br/>
  中国外交部发言人华春莹却在推特上幸灾乐祸地模仿弗洛伊德的语气：我不能呼吸 (I can’t breathe)。
  <br/>
  <br/>
  5 五月二十九日美国总统特朗普在美中关系的讲话中特别提及世卫组织，宣布正式退出世卫组织，并表示，可能会把资金投入新的相关组织。
  <br/>
  <br/>
  世界卫生组织秘书长谭德塞态度突然大转弯，于6月1日赞誉美国对全球健康的贡献与慷慨，并发出希望美国与WHO继续合作的呼吁。
