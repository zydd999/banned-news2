 <p>
  1 备受关注的“港版国安法”，在香港民间以及国际社会的反对声中，在北京的全国人大闭幕式中高票通过。
  <br/>
  <br/>
  中国全国人大会议周四（28日）下午闭幕，并表决《关于建立健全香港维护国家安全的法律制度和执行机制的决定》(草案)，也就是授权人大常委会为香港制定“港版国安法”。工作人员：“赞成，2878票，反对，1票，弃权6票，宣读完毕！通过！”
  <br/>
  <br/>
  虽然港府多次强调很多香港市民支持“港版国安法”，港岛民间组织“民间记者会”近日超过37万人民调结果，98.6%受访者反对“港版国安法”，一个星期以来，香港反对“港版国安法”的街头抗争活动也一直持续。
  <br/>
  <br/>
  人大表决“港版国安法”前夜美国国务卿蓬佩奥发声明，表示香港自治不再，不保证可以继续享有1997年前的待遇。而目前“港版国安法”细节仍须由人大常委会决定，在美中角力下会否仍有转弯馀地尚待观察。
  <br/>
  <br/>
  2 受到全球瞩目的华为首席财务官孟晚舟引渡案，加拿大法院周三裁定双重犯罪成立，这不代表孟晚舟即刻会被引渡到美国，但意味着她将继续待在加拿大完成司法诉讼流程。
  <br/>
  <br/>
  加拿大卑诗省法院法官霍姆斯（Heather Holmes）在长达二十三页的裁决书中表示，孟晚舟引渡案所要求的双重犯罪要素成立。
  <br/>
  <br/>
  这个案子涉及的不单纯是法律层面，此案深深牵系着加拿大和中国的关系，尤其是中国当局在孟晚舟被加方拘捕后，以违反国家安全为理由逮捕了两名加拿大人 - 康明凯(Michael Kovrig)和斯帕弗(Michael Spavor)，过去中方不断敦促加方尽快释放孟晚舟，称这是加中两国关系的最大障碍。
  <br/>
  <br/>
  3 美国国会众议院5月27日以413票赞成1票反对，表决通过《维吾尔人政策法案》。由于参议院此前已表决通过同样版本，该法案将送交总统特朗普签署成为法律。
  <br/>
  <br/>
  美国众议院议长南希·佩洛西（Nancy Pelosi）在声明中指出，美国民主、共和两党当天以压倒性表决的立法坚定措施，反制北京当局对维吾尔人进行严重的侵犯人权行为。
  <br/>
  <br/>
  法案主要发起人、共和党籍参议院马克·鲁比奥（Marco Rubio）则表示，美国议会由此向中国政府发出明确信息，他们无法逍遥法外。

  4 赞比亚首都卢萨卡的华人仓库被抢劫焚烧，三名南通籍中国商人曹某、樊某、包某不幸遇害。其中曹某被烧焦的遗体已经找到，另两名受害者的遗体正在搜寻中。
  <br/>
  <br/>
  中共官媒则指责这次屠杀事件和排华有关连，并引用卢萨卡市长近期“新冠病毒源自中国”的言论以佐证。
  <br/>
  <br/>
  说到赞比亚，老一辈中国大陆人会直接想到坦赞铁路，这条几乎成为“中非友好关系”代名词，如今几乎荒废了的毛泽东时代“领导世界革命”的象征性中国援外产品。
  <br/>
  <br/>
  如今的赞中关系如江河日下，凶杀案只是一个表象。德国之声认为：中国在非洲大举投资、大兴土木，但喝彩寥寥。在赞比亚这个与中国拥有“传统友谊”的国家，近年来反华情绪明显上升。很多赞比亚人担心，国家资产会落入中国手中，自己的国家将完完全全地被中国控制。
  <br/>
  <br/>
  美国国务卿蓬佩奥在今年二月访问非洲三国的行程中曾数度不点名的批评中国在非洲大陆的殖民行为。
  <br/>
  <br/>
  5 以抗战史诗国画《浩气长流》闻名中外的中国独立学者、民间思想家王康周三在美国弗吉尼亚州的家中病逝，享年70岁。王康知识渊博、学贯中西，给后人留下了大量发人深省的政论纪录片和画作。
 </p>

 <p>
  八十年代末，王康以“布衣”的身份写下了“中国改革宪章”，名声鹊起。六四事件爆发后，王康因支持学运而一度流亡多年。
  <br/>
  <br/>
  九十年代初王康再度出山。1993年，他创作了中国九十年代唯一一部究诘国家发展道路的大型政论片《大道》。
  <br/>
  <br/>
  次年，王康成立了重庆陪都文化有限公司，并在此后几年间先后拍摄了《抗战陪都》、《中美西部开发启示录》、《重庆大轰炸》等纪录片。
  <br/>
  <br/>
  2004年，王康动员了几十名国内画家开始绘制名为《浩气长流》的巨幅画作。历时五年半，他们为200多名在抗日战争中牺牲的国军将领画像，声势浩大地展现了中国抗日战争史。2010年，这幅长八百余米、宽两米的画卷在台湾国父纪念馆展出，引发海内外各界的强烈反响。
  <br/>
  <br/>
  2015年，王康在美国短暂休息期间，与他的华人朋友开始筹划《浩气长流》的美国巡展事宜，却遭到中国政府强烈阻挠。在断然拒绝了当局要求之后，王康终于如愿以偿，先后在华盛顿、纽约、旧金山展出了这幅国画。但他却因此而无法回国，只能定居美国，直至去世。
  <br/>
  <br/>
  在最后的日子里，王康再度经历了精神洗礼。上个月，王康在家中受洗成为基督徒。就在半个月前，会说流利中文的美国共和党议员匡希恒(John Ream Curtis)还赠予了王康一面美国国旗，以满足他能在病床上看到星条旗的心愿。
  <br/>
  <br/>
  <br/>
