 <p>
  1  全国两会拍板5月底召开。
  <br/>
  <br/>
  受到武汉肺炎疫情延宕两个半月，新华社报导，政协全国委员会会议日前建议全国政协13届3次会议于5月21日召开。此外，人大会常委会第17次会议29日上午表决通过，13届全国人大3次会议将于5月22日在北京召开。
  <br/>
  <br/>
  2  随着全球武汉肺炎疫情渐趋缓和，很多国家，包括澳大利亚开始把焦点转向厘清病毒源头。中国驻澳大利亚大使日前表示，如果澳大利亚坚持呼吁国际就疫情源头进行独立调查，可能会引起中国消费者抵制。
  <br/>
  <br/>
  中国驻澳大利亚大使成竞业接受《澳大利亚金融评论报》采访时表示，中国社会对澳大利亚的反应感到沮丧和失望。如果澳大利亚坚持这样做，可能引起中国消费者杯葛：“（中国）消费者会思考为何要购买澳大利亚出产的红酒和农产品。”
  <br/>
  <br/>
  成竞业还抨击澳洲联邦政府对病毒起源进行独立调查是“迎合”美国。同时他反驳了病毒来自武汉海鲜市场的说法。根据成竞业的说法，中国民众可能出现的杯葛行动还包括，不再到澳大利亚旅游和留学。
  <br/>
  <br/>
  面对警告，澳大利亚卫生部长亨特强调，全球有近300万人确诊感染武汉肺炎、逾20万人染病死亡，进行独立调查符合澳大利亚及全球利益。
  <br/>
  <br/>
  澳大利亚外交部长佩恩也驳斥了成竞业的强硬言论。她表示，澳大利亚已原则性地呼吁对新冠状病毒疫情进行独立调查，这是一场前所未有的全球危机，对健康、经济和社会造成了严重影响。各国应该积极合作找出病毒的来源。
  <br/>
  <br/>
  澳洲工党领袖比尔.肖顿在接受本台采访时表示：“首先，现在全世界包括澳洲在内，就这个病毒做了很多研究，得出的结论基本一致的指向病毒来源于中国。“他接着表示，中国在全球的质疑中，一直闪烁其词，不敢正面回应。中国政府如果就如他们所说的是清白的，就应该允许全球开展此次病毒源头的调查，没有必要阻挡，用经济威胁。调查对他们也是有利的。
  <br/>
  <br/>
  3  中国维权律师王全璋星期一晚回到北京家中与妻儿团聚，这是王全璋被当局抓捕拘押4年半之后，首次与家人见面。
  <br/>
  <br/>
  709案著名律师王全璋遭当局非法拘押并判刑，但刑满释放后，被当局强制押送到山东济南居住，被拒绝回北京家中。但在一天前，妻子李文足突患急性阑尾炎在北京入院治疗，王全璋乘包车前往北京，欲探视照料生病的妻子和年幼的儿子，一度被警方拦截。后经协调，由济南圣井公安派出所警察当天开车送王全璋回北京家中。
  <br/>
  <br/>
  4   中国一消息人士日前披露，金正恩病情已无大碍，美国及韩国政府考虑到与朝鲜的关系，在金正恩健康问题上，相对谨慎。
  <br/>
  <br/>
  特朗普总统本周一在白宫草坪疫情简报会上回应记者就金正恩的健康情况提问时说，“我不能很准确地告诉你们什么”。他紧接着说：“是的，我很清楚他现在状况如何。但我现在不能谈。”
  <br/>
  <br/>
  他补充说：“我祝福他安好。我和金正恩的关系很好。如果我不是总统的话，你们现在（恐怕）就得上战场了，和朝鲜的战场。这就是我能告诉你的。那是他想要的。”
  <br/>
  <br/>
  5  据台湾媒体报道，4月25日，林荣基在抵台一周年之际重开“铜锣湾书店”，台湾总统蔡英文以个人名义送去花篮表达祝福。
  <br/>
  <br/>
  林荣基对在场媒体表示，“我开一个书店，也是一个反抗”。书店开业前夕，林荣基收到律师函指出“铜锣湾书店”在台已被注册，要求其停止使用该名称；他还在小巷内遭人泼红漆。其书店正式注册名为“中山铜锣湾书店”。
  <br/>
  <br/>
  原在香港的铜锣湾书店因出版揭露中共内幕的书籍而遭到中共打压，2015年10月起，书店内合共4名股东及店长、母公司业务经理，相继在外地及中国大陆失踪，包括东主桂民海。该书店的遭遇被认为是香港反送中运动的导火索之一。
  <br/>
  <br/>
  6  美国国际宗教自由委员会4月28日发布2020年度报告，缅甸、中国、厄立特里亚、伊朗、朝鲜、巴基斯坦、沙特阿拉伯、塔吉克斯坦、土库曼斯坦、印度、尼日利亚、俄罗斯、叙利亚和越南等14个国家，因“其政府参与或容忍系统性、持续的严重侵权行为”而被列入“特别关注国家”。
  <br/>
  <br/>
  报告有关中国的部分指出，中国宗教自由状况在2019年持续恶化，中国政府通过建立高科技监控体系，包括人脸识别、人工智能等对少数民族宗教团体进行监控。独立调查发现，90万至180万维吾尔、哈萨克、吉尔吉斯及其他穆斯林少数民族，在新疆多达1300个集中营遭到关押；近50万穆斯林儿童被迫与家庭分离。2019年期间，当局还从集中营进一步向各地棉花和建材工厂输送强制劳工；中国各地，多达数以千计的清真寺及阿拉伯语标志遭当局摧毁。
  <br/>
  <br/>
  在西藏，当局为控制达赖喇嘛转世继续实施强制措施，压制当地的佛教信仰。2019年，四川色达县喇荣寺五明佛学院遭强拆，多达600名僧侣、尼姑遭驱逐。而自2009年2月至今，至少有156名藏人因抗议中国政府压迫性治藏政策自焚。
  <br/>
  <br/>
  2019年，中国有多达数百家非官方家庭教会遭当局关闭查封，其中包括四川秋雨教会牧师王怡更被以颠覆国家政权罪名重判9年徒刑。多达数千名法轮功练习者因信仰或分发宣传品遭当局拘捕；人权活动人士和科学研究者继续大量披露，涉及法轮功学员的强制摘取人体器官问题的证据。
  <br/>
  <br/>
  报告还建议美国国会支持相关西藏、新疆政策法案，促进中国宗教自由，并且禁止涉及新疆强制劳工产品的贸易进口等。
 </p>
