 <p>
  1 国际人权组织“人权观察”9日发起致联合国秘书长古特雷斯、联合国人权事务高级专员巴切莱特及联合国各成员国的公开信，目前已获“国际特赦”、“自由之家”“香港观察”、“世界维吾尔大会”、“自由西藏”等六十多个国家321个人权团体的联署。
  <br/>
  <br/>
  公开信批评中国在全球范围实施人权侵犯，包括攻击海外人权人士，打压学术自由，从事互联网审查数字监控，同时对早前五十位联合国人权专员敦促国际社会采取果断行动，保护中国民众基本自由一事表达了支持。
  <br/>
  <br/>
  2 周二（8日）在北京举行的全国抗击新冠肺炎表彰大会上，多名医护人员获得习近平颁奖赞扬，国家卫健委高级别专家组组长钟南山获颁授“共和国勋章”，他与另外3名医学专家受到国宾式招待，在国宾护卫队护送下，由长安街进入人民大会堂。
  <br/>
  <br/>
  与此同时，世卫紧急项目执行主任瑞安在回应记者答问表示，中国仍持续有输入个案，没有自满的空间。唯有等到所有地方结束才叫真正结束。
  <br/>
  <br/>
  但引起广泛回响的“疫情吹哨人”李文亮却未见提及。近期央视推出的疫情纪录片对李文亮也只字不提，部分内容也引起民众强烈不满。
  <br/>
  <br/>
  表彰大会过后，大批网民涌入李文亮的微博留言。有人说，“大会欠下李文亮一个荣誉，一个公道，甚至把他当成不存在，虽然表彰大会没有李文亮的份儿，但是人民心里永远有他。”
  <br/>
  <br/>
  在这篇万言稿的照章宣读中习近平除了多次念白字之外，引用了一句"世上没有从天而降的英雄，只有挺身而出的凡人"，以此赞美抗疫有功的普通民众。但日后被有心人爆出，此句原创者是流亡海外的香港抗争者李珏熙在2015年为哥哥李政熙参选喊出的口号，反送中运动街头赫然可见。
  <br/>
  <br/>
  3 九月三日，习近平在北京人民大会堂举行的"纪念中国人民抗日战争暨世界反法西斯战争胜利七十五周年"座谈会上，语气严肃地一连说了五个"中国人民绝不答应"：
  <br/>
  <br/>
  ·任何人任何势力企图歪曲中国共产党的历史、丑化中国共产党的性质和宗旨，中国人民都绝不答应！
  <br/>
  <br/>
  ·任何人任何势力企图歪曲和改变中国特色社会主义道路、否定和丑化中国人民建设社会主义的伟大成就，中国人民都绝不答应！
  <br/>
  <br/>
  ·任何人任何势力企图把中国共产党和中国人民割裂开来、对立起来，中国人民都绝不答应！
  <br/>
  <br/>
  ·任何人任何势力企图通过霸凌手段把他们的意志强加给中国、改变中国的前进方向、阻挠中国人民创造自己美好生活的努力，中国人民都绝不答应！
  <br/>
  <br/>
  ·任何人任何势力企图破坏中国人民的和平生活和发展权利、破坏中国人民同其他国家人民的交流合作、破坏人类和平与发展的崇高事业，中国人民都绝不答应！
  <br/>
  <br/>
  4 中印两国军方各自发表声明，指责对方本周一（9月7日）在边境班公错湖鸣枪示警。这是四十五年来中印边境首次发生两国军人鸣枪事件。
  <br/>
  <br/>
  中印边界局势本周一再度紧张。中国军网周二凌晨报道，中国人民解放军西部战区新闻发言人张水利大校称，9月7日（星期一），印军非法越线进入中印边境西段班公湖南岸神炮山地域。印军在行动中悍然对前来交涉的中国边防部队巡逻人员鸣枪威胁，中国边防部队被迫采取应对措施稳控现地局势。不过，中方并未具体说明采取了哪些对应措施。
  <br/>
  <br/>
  5 中国周二（8日）发起“全球数据安全倡议”，并质疑有国家把数据安全政治化。
  <br/>
  <br/>
  中国国务委员兼外长王毅8日在北京的一个研讨会上宣读“全球数据安全倡议”时提出三项原则，包括秉持多边主义，反对个别国家大搞单边主义，以“清洁”为名向别国泼脏水；质疑把数据安全问题政治化，干扰和阻碍全球数字合作与发展；数字保护主义违背经济发展的客观规律。
  <br/>
  <br/>
  倡议内容包括：反对利用信息技术破坏其他国家关键基础设施或窃取重要数据；不得滥用信息技术对他国进行大规模监控；尊重他国主权、司法管辖权和对数据的管理权，不得直接向企业或个人调取位于他国的数据；科技企业不应在产品和服务中设置后门，非法获取用户数据。
  <br/>
  <br/>
  而在此前几天传出消息，美国正考虑将中国最大的芯片制造商中芯国际增列至出口管制清单。
  <br/>
  <br/>
  路透社9月4日引述未具名的美国国防部官员的话说，中芯国际和中国解放军关系密切，特朗普政府正考虑是否要采取对待华为的同样模式，将中芯国际列入美国的出口管制实体清单中。
  <br/>
  <br/>
  香港上市的中芯国际股价周一应声暴跌，盘中一度大跌两成，中芯国际发表声明指出，企业与中国军方“毫无关系”，也从未有任何涉及军事应用的经营行为。中芯国际还称，愿与美国各相关政府部门沟通，化解可能的歧见和误解。
  <br/>
  <br/>
  6 中国央视环球电视网（CGTN）主播、澳籍华裔成蕾上个月北京落网。中国外交部昨天星期二（8日）证实，有关部门以“涉嫌从事危害中国国家安全的犯罪活动”为由对她采取强制措施。
  <br/>
  <br/>
  另据星岛网引述外电报道，两名澳洲媒体驻华记者上周先后被中国国安人员约谈，两人一度藏身到澳洲驻华使馆及澳洲驻上海领馆。澳方和中方商议后，同意记者接受中方问话后离境。两人目前已离开中国并抵达悉尼。在这两名澳洲记者离开中国后，澳洲已无记者驻华，为50余年首次。
  <br/>
  <br/>
  本台记者专访上星期三被国安约谈后被驱离中国的澳洲记者之一比尔·博图斯，他告诉我们：“ 星期三晚上大概十二点，六个国保带一个翻译，一共是七个人，来我的房子敲门...（审问时）有三个警察、一个翻译，一共有四个人，然后有摄相机，都拍了...我说你们为什么选择我？我跟成蕾不熟悉。我觉得这可能是个大表演，这跟真正的调查没有什么关系。我感觉他们肯定不是想抓我、不是想逮捕我，就是想给我一些麻烦。”
  <br/>
